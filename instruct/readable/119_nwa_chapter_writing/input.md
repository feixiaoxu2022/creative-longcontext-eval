# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事



# 用户需求

女主是古董鉴定师，出身文物修复世家，外公是故宫退休的陶瓷修复师，她从小在修复室长大，本科读的考古、研究生读的文物鉴定，毕业后在拍卖行做了四年鉴定助理。嫁入豪门是因为婆婆看中她"识货"，想让她帮家族打理收藏，结果进门后发现只是要一个体面的儿媳妇撑场面，她的专业意见从来没被当回事。三年里她唯一能做的就是趁家宴无聊时偷偷翻看公婆库房里的藏品，心里默默做鉴定练手。离婚那天丈夫甩给她一箱"不值钱的破烂"当补偿——是公婆从老宅清出来准备扔掉的旧物件。她认出箱子最底层那个落满灰的青花罐是永乐年间的官窑孤品，不是靠灵光一闪，是因为她外公三十年前修复过同一窑口的残件，她从小看着那些碎片长大，对这个窑口的胎质、釉色、画工烂熟于心。保守估价八千万。她没声张，签了离婚协议拿着箱子走人。故事从这里开始：她要把这批"破烂"变成重新站起来的资本，但古董行的水比豪门深——造假的、设局的、抢货的、恶意压价的，她一个没有人脉没有资金的离婚女人，要在这个全是老狐狸的行当里站住脚。前夫家发现东西值钱后开始法律追讨，她要在鉴定专业、商业谈判和法律攻防三条线上同时作战。不要写成鉴宝爽文——她不是每次都能一眼看出真假，也会有看走眼的时候，也会被老手设局坑过。核心看点是一个有真本事但缺乏江湖经验的年轻女性，在一个吃人不吐骨头的行业里磕磕绊绊地站起来。感情线可以有但必须是势均力敌的平等关系，男性角色不能是她翻盘的关键。12万字。

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
第5章:林羽发现规则可以被操控,开始怀疑(幻灭)
第8章:林羽学会打破规则,变得像敌人一样冷酷(堕落边缘)
第10章:林羽选择坚守底线,用正义手段复仇(升华)
```

**成长弧关键点**:

- **信念动摇**: 原本坚信的东西被打破
- **试错挣扎**: 尝试新方法,但付出代价
- **顿悟时刻**: 找到真正的答案
- **行动证明**: 用行动体现新的价值观

#### 2.9.3 细节刻画(小动作/口头禅/习惯)

**原则**: 给每个重要角色设计**独特标签**,让读者一眼认出

**示例**:

```
林羽:紧张时会不自觉地推眼镜,即使没戴眼镜
幽灵:说话前总是先停顿三秒,像在搜索数据库
张伟:喝茶时小指会翘起,暴露精英阶层的傲慢
```

**标签类型**:

- 肢体习惯(咬指甲/转笔/挠头)
- 口头禅("嗯……怎么说呢""确实如此")
- 情绪反应(生气时耳根发红/尴尬时揉鼻子)

## 使用建议

**在创作chapters时**:
1. 应用**场景构建技巧**(五感描写+氛围营造)
2. 应用**对话写作技巧**(潜台词+节奏+人物语言差异)
3. 应用**情感渲染技巧**(Show Don't Tell+情感阶梯+对比反差)
4. 控制**节奏**(紧张-舒缓交替+信息释放节奏+场景长短搭配)
5. **开篇**必须有钩子(前300字制造冲突/悬念/反常)
6. **结局**完成情感承诺+回扣前文+余韵设计

**在创作全程**:
7. **人物刻画**保持矛盾性+成长弧+细节标签
8. **冲突设计**叠加外部冲突+内部冲突+层层升级
9. **超过8000字的长文**: 及时更新writing_log.md,下一章开始前先读log回顾

# 第一章 一箱破烂

民政局三楼的离婚登记窗口前，沈青瓷的手指轻轻按在协议书最后一页的签字栏上。

初秋的阳光透过窗帘缝隙落进来，在她手背上投下一道细长的光斑。窗口对面的工作人员是个中年女人，戴着金属框眼镜，目光在她和顾正霆之间游移了一下，例行公事地问："双方都确定吗？"

"确定。"

两个声音几乎同时响起，连语调都出奇地平淡。

沈青瓷没有抬头看旁边那个男人。三年婚姻，最后只浓缩成这一声"确定"，像结案陈词，像句号，像一扇门终于关上时发出的轻响。

笔尖在纸上划过，签下自己的名字。沈青瓷。三个字写得工整端正，和当初签结婚证书时一模一样。她垂着眼看了看那几个字，忽然觉得有些讽刺——三年前她签下这个名字的时候，以为自己是走进了一座金碧辉煌的宫殿；三年后她才明白，那不过是一座镀金的笼子。

顾正霆早她一步签完，已经站起身往外走了。他穿着一身深灰色的西装，袖口的袖扣在日光下闪了一下，是白金镶钻的款式，她认得那是他三十岁生日时顾母送的礼物。

沈青瓷刚要起身，就听见他的手机响了。

顾正霆脚步一顿，接起电话。他背对着她，但她清楚地听见了他说话的语气——温柔的、带着笑意的、像含着一颗糖。

"嗯，快了……马上就好……晚上？行，你定地方。"

沈青瓷的手指在椅子扶手上停了一秒。她面无表情地站起来，走向门口。经过顾正霆身边的时候，他正好挂断电话，侧头看了她一眼。

"东西在车上，你自己搬。"

他说这话的语气就像在吩咐佣人处理一件微不足道的杂务。说完也不等她回应，径直走了，皮鞋在大理石地面上敲出清脆的节奏，越来越远。

沈青瓷站在原地，看着他的背影消失在楼梯转角。

旁边经过的工作人员好奇地多看了她一眼，大概是觉得这个女人有点奇怪——既没有哭，也没有闹，就只是站在那儿，像一尊没有表情的雕塑。

她垂下眼，慢慢吐出一口气。

那点名为"被轻视"的微怒在胸口燃起又熄灭。三年了，她早该习惯了。

---

顾正霆的车停在民政局后面的停车场，黑色的宾利，车牌号她闭着眼睛都能背出来。沈青瓷走过去，后备箱是开着的，里面放着一个旧纸箱，用透明胶带缠了几道。

纸箱不大，目测也就能装下几本书的大小，灰扑扑的，边角有些磨损，看起来已经放了很久。她弯腰把箱子搬出来，意外地沉。

旁边的司机是顾家的老人了，姓周，跟了顾家十几年。周叔看着她搬箱子，嘴唇动了动，似乎想说什么，最后只是叹了口气，递过来一个环保袋："少奶奶……沈小姐，这里面是您的一些个人物品，太太让我交给您。"

沈青瓷接过来，低头看了一眼。是一些旧书、旧照片，和她出嫁前从家里带来的一些小物件。原来顾母的效率这么高，连她的东西都已经打包好了。

"谢谢周叔。"她淡淡道。

周叔张了张嘴，似乎还想说些什么，但沈青瓷已经拎起箱子和环保袋，转身走了。

九月的风里带着点凉意，吹动她的头发。沈青瓷走出停车场，站在路边打了辆出租车。司机从后视镜里打量她，一身素净的白衬衫和黑色直筒裤，长发披肩，脸色有些苍白，但五官很清秀，气质也好，不像是坐出租车的人。

"姑娘，去哪儿？"

沈青瓷报出一个老小区的地址。司机愣了一下，又看了看她手里抱着的旧纸箱，没再多问，启动车子。

车窗外的街景飞速后退。沈青瓷靠在座椅上，目光空洞地望着窗外。

三年。

三年前她嫁进顾家的时候，顾母是看中她"识货"。她出身文物修复世家，外公是故宫退休的陶瓷修复师，她从小在修复室长大，本科读的考古、研究生读的文物鉴定，毕业后在拍卖行做了四年鉴定助理。顾母说，顾家有不少收藏，正需要一个懂行的人打理。

可进门之后她才发现，所谓的"打理收藏"不过是个幌子。顾母需要的只是一个有文化背景的儿媳妇撑场面，在名流聚会上能说几句专业术语，让别人觉得顾家有品位。至于她的专业意见？从来没有人当回事。

三年里，她唯一能做的就是趁家宴无聊时偷偷溜进公婆的库房，翻看那些藏品，在心里默默做鉴定练手，就当给自己保持手感了。

那些藏品里有真有假，有精品也有平庸货。她看在眼里，却从来不说——因为顾母不想听，顾正霆不在乎，而她，不过是一个"识货的摆设"。

现在，她终于从那座笼子里走出来了。

代价是三年的青春，和一箱"破烂"。

出租车在一个老旧小区门口停下。沈青瓷付了钱，抱着箱子上楼。这是她昨天刚租下的房子，三楼，朝南的单间，月租两千三，押一付三。她用婚前攒下的私房钱付了房租，又添置了一些基本的生活用品，余额已经不多了。

门锁发出一声清脆的咔嗒，她推门进去。

屋子不大，三十多平米，一张床、一张桌、一把椅子、一个衣柜，简单得像个临时落脚的旅店。窗外是小区的天井，种着几棵老槐树，叶子已经开始泛黄。

沈青瓷把箱子放在地上，拉上窗帘，打开台灯。

黄色的灯光把屋子照得暖了一些，但也更显出这个房间的寒酸——和顾家那座别墅比起来，这里就像两个世界。

但沈青瓷没有太多感伤的时间。

她在床边坐下，看着面前那只旧纸箱。

顾正霆离开时说的那句话还在耳边回响——"这些老宅清出来的破烂，你识货，挑着玩吧"。

语气轻佻，像是施舍乞丐一把零钱。

沈青瓷盯着那箱子看了一会儿，慢慢撕开封口的透明胶带。

箱子打开，扑面而来的是一股陈旧的气息——灰尘、老木头、还有一种说不清的岁月沉淀的味道。她小心翼翼地把东西一件一件拿出来，摆在床上。

几个民国粉彩碟，边缘有磕碰，但画工还算细致；一对清代铜炉，包浆浑厚，应该是老东西；几件杂项，有些是民间普品，有些来历不明；还有一些边角料似的小物件，看起来像是从老宅仓库里随便清出来的。

沈青瓷一件件看过去，心里默默评估着价值。这些东西加起来，往高了说能值个几十万，但考虑到市场行情和出手难度，实际能卖出的价格要打对折。

她有些失望，但也在意料之中。顾家既然说是"破烂"，自然不会把真正值钱的东西给她。

她把东西都看完了，正准备把箱子收起来，手指却碰到了箱底的一个角落——那里还有一个东西，被牛皮纸层层包裹着，压在最下面，几乎和箱底融为一体。

沈青瓷的心跳漏了一拍。

她小心翼翼地把那团牛皮纸取出来，放在腿上，慢慢拆开包装。

牛皮纸一层层剥落，露出里面的东西——一只青花罐，落满灰尘，高约二十厘米，敞口、短颈、鼓腹，造型端庄秀雅。

沈青瓷的手指微微颤抖着，用衣袖轻轻拂去表面的灰尘。

釉面逐渐显露出来。

那一瞬间，她的呼吸停住了。

灯光下，青花的发色是一种深沉浓郁的钴蓝，在釉下晕散开来，带着若有若无的铁锈斑痕——那是苏麻离青的特征，是永乐年间特有的进口青料，烧制时会在釉面下留下不规则的深色晕染。

沈青瓷的脑海里轰然炸开无数画面。

她五岁那年，第一次被外公带进故宫文物修复室。那间屋子里弥漫着松节油和旧纸张的气味，阳光透过高窗落在工作台上，外公戴着老花镜，手里拿着一把小刷子，正在修复一堆碎瓷片。

"丫头，来看看这个。"外公把她抱到工作台前，"这是永乐年间的青花，看见没有？这个蓝色，叫苏麻离青，是从波斯进口的料，烧出来会有这种铁锈斑，像墨水洇开一样……"

她趴在台子边，眼睛瞪得溜圆，看着那些碎瓷片上深深浅浅的蓝色花纹，觉得好看极了。

"外公，这个能修好吗？"

"能。只要用心，再碎的东西都能修好。"外公笑着揉了揉她的头，"丫头，你把这个颜色记住了。以后要是见到类似的东西，就想想今天外公给你看的这些碎片……"

三十年前的记忆如潮水般涌来。

沈青瓷握着那只青花罐，手指止不住地颤抖。

她凑近台灯，仔细查看釉面的每一个细节。苏麻离青的晕染、釉层的肥厚、胎骨的细腻、画工的流畅……所有的特征都在指向一个答案——这不是仿品，不是高仿，不是"像"永乐的东西，而是真正的永乐官窑。

她翻过罐底，没有款识，但永乐青花本来就鲜少落款，这恰恰证明了它的真实性。

沈青瓷深吸一口气，放下青花罐，开始在屋里翻找自己的行李。

那个环保袋里有她从家里带来的一些旧物，其中有一本发黄的笔记本——那是外公当年的修复记录，她从小就喜欢翻看，后来外公把它送给了她，说"这是咱们沈家的传家宝，比金子还值钱"。

她找到那本笔记，快速翻到三十年前的记录。

"……修复永乐青花残件一批，计大盘两件、罐一件（残）、碗三件……胎骨洁白细腻，釉面肥厚莹润，青花发色深沉，苏麻离青特征明显，铁锈斑分布不规则，晕染自然……"

沈青瓷一边读一边比对手中的青花罐，字字句句都像是在描述同一件东西——同一个窑口，同一种青料，同一种画工风格。

她又翻了几页，找到了外公画的草图和详细的技术分析。三十年前那批永乐残件的特征，和她手里这只青花罐完全吻合。

沈青瓷缓缓合上笔记本，抬头看着那只青花罐。

灯光下，它静静地立在床头，落满尘埃的表面掩盖不住釉下青花的光华。三百年的时光在它身上留下沉淀，却没有磨灭那份属于帝王御用之物的风骨。

她闭上眼睛，深呼吸，努力让自己冷静下来。

——不能激动。

——要理性。

——要反复确认。

——万一是自己看错了呢？万一是太想翻身所以自己骗自己呢？

沈青瓷睁开眼，又把那只青花罐拿起来，从头到尾、从里到外、从釉面到胎骨，一点一点地重新检验。

时间一分一秒地流逝。窗外的天色从傍晚变成深夜，台灯的光圈把她和那只青花罐圈在一个小小的世界里。她就像一个虔诚的信徒在审视自己的信仰，不敢有丝毫松懈。

直到凌晨三点，她才终于放下手中的放大镜，长长地吐出一口气。

——是真的。

永乐官窑。孤品。

保守估价，八千万。

沈青瓷靠在床头，看着那只青花罐，忽然笑了。

笑容里带着点苦涩，带着点自嘲，也带着一种说不清的情绪——像是溺水的人终于抓住了一根救命稻草。

三年婚姻，她什么都没得到。被当成花瓶，被边缘化，被忽视，被轻慢。离婚的时候，顾正霆甩给她一箱"破烂"当补偿，语气像在打发乞丐。

可他不知道，他扔掉的"破烂"里，藏着一件价值八千万的孤品。

而她，只有她，能认出它来。

不是因为运气，不是因为巧合，是因为她从五岁开始就在修复室里看着外公工作，是因为她十几年如一日地学习钻研，是因为她对这个窑口的胎质、釉色、画工烂熟于心。

这是她的本事，是她的专业，是谁都拿不走的东西。

沈青瓷轻轻抚摸着青花罐的釉面，手指微凉。

---

第二天一早，沈青瓷抱着青花罐出了门。

城东有一片老式筒子楼，沈外公就住在那里。八十五岁的老人了，腿脚不太方便，但精神头还好，自己一个人住，每天种种花、喂喂鸟、翻翻旧书。沈青瓷的父母早年离异，她从小跟着外公长大，感情最深。

她敲开门的时候，外公正坐在书房里看报纸，听见声音转过头来。

"丫头？怎么这时候来？"

沈青瓷走进去，把怀里的锦盒放在桌上。昨晚她在出租屋里找到一个旧锦盒，刚好能装下那只青花罐，用棉布垫着，小心翼翼地包好。

"外公，帮我看个东西。"

沈外公眯起眼睛看了看她，没有多问，戴上老花镜，打开锦盒。

青花罐静静地躺在棉布上，被擦拭干净之后，釉面的光泽更加明显。沈外公愣了一下，然后慢慢地、郑重地把罐子取出来，放在窗边的光线下。

他拿起那把用了四十年的放大镜，凑近去看。

一看，就是二十分钟。

房间里安静得只能听见老式座钟的滴答声。沈青瓷站在一旁，大气都不敢出一口，眼睛紧紧盯着外公的表情。

二十分钟后，沈外公放下放大镜，缓缓抬起头。

"丫头，"他的声音有些沙哑，"这东西……你哪儿来的？"

沈青瓷把离婚的事简单说了一遍，包括那箱"破烂"的来历。

沈外公听完，沉默了很久。

然后他点了点头："胎骨、釉色、画工都对，是永乐甜白釉工艺的延续。三十年前我修过同窑的残件，这一只，品相完整。"

沈青瓷的心跳加速："外公，您确定吗？"

"确定。"沈外公把青花罐放回锦盒，又看了孙女一眼，"丫头，东西是好东西，但你要想好怎么出手。这行的水，比你想的深。"

沈青瓷点点头："我知道。"

"你有渠道吗？"

"还没有。我先找找。"

沈外公叹了口气，从抽屉里翻出一个旧本子，递给她："这是我这些年认识的一些老朋友，有些还在世，有些……"他顿了顿，"你可以先打听打听行情，但别急着出手。这东西一旦露了相，就藏不住了。"

沈青瓷接过本子，翻开看了看。上面记着一些人名和电话，字迹苍劲有力。她把本子收好，弯腰给外公倒了杯茶。

"外公，谢谢您。"

"谢什么，"沈外公摆摆手，眼里带着一丝复杂的神色，"丫头，你妈走得早，你爸指望不上，这些年就我带着你……"他停顿了一下，"你是个有本事的孩子，别被人小瞧了。"

沈青瓷鼻子有点酸，但还是笑了笑："外公，我知道。"

她弯腰把锦盒抱起来，准备离开。走到门口的时候，沈外公又叫住了她。

"丫头。"

"嗯？"

"记住，这行里，有眼力不等于吃得开。"沈外公的目光透过老花镜片，认真地看着她，"专业是专业，江湖是江湖，两回事。你小心点。"

沈青瓷点了点头，心里记下了这句话。

她抱着锦盒走出筒子楼，秋日的阳光明晃晃地照在身上，有一瞬间的暖意。

她低头看了看怀里的锦盒。

八千万。

这是她翻身的资本，是她重新站起来的筹码。

但外公说得对——东西是好东西，可要把它变成钱，变成她的资本，变成她在古董行里站稳脚跟的底气……这条路，不会简单。

沈青瓷深吸一口气，抬起头，走进人群中。

路还很长，但至少，她有了开始。

# 第二章 试水

离婚后的第五天，沈青瓷收到了一条微信消息。

发消息的人叫吴敏，是她大学时的同学，两人当年关系不错，毕业后各自忙碌，联系渐渐少了。吴敏现在在一家中型拍卖行做修复工作，算是还在这个圈子里混。

消息很简短："青瓷，听说你离婚了？方便出来聊聊吗？"

沈青瓷看着屏幕上的字，沉默了几秒。消息来得有些突然，但她确实需要一个切入口了解市场行情，而吴敏算是个现成的渠道。

她回复："好，你定时间地点。"

---

两天后，城西老街的一家旧茶馆。

沈青瓷推开木门走进去，空气里弥漫着茶叶和檀香混合的气息。茶馆装修是老式风格，竹编桌椅、青瓷茶具、墙上挂着几幅字画，看年头应该是近现代名家的作品。

吴敏已经到了，坐在角落的位置，看见她进来就站起身招手："青瓷，这边！"

沈青瓷走过去坐下。吴敏比大学时胖了一些，圆脸，笑起来眼睛眯成两条缝，看起来很好相处。

"好久不见啊，"吴敏给她倒了杯茶，"听说你从顾家那边出来了？"

"嗯，离了。"沈青瓷端起茶杯喝了一口，语气平淡。

"哎，那家人……"吴敏压低声音，"我当时就觉得，你嫁过去有点屈才了。你那专业水平，去哪儿不行，非得给人当少奶奶。"

沈青瓷淡淡一笑，没有接话。

吴敏看她不想多聊私事，便识趣地转了话题："对了，你现在有什么打算？是打算重新找工作，还是……"

"我想自己做点事。"沈青瓷放下茶杯，"手里有几件东西想出手，试试水。"

吴敏眼睛亮了一下："什么东西？你方便透露吗？"

"离婚时分到的一些旧物，民国的、清代的都有，不算特别名贵，但也还能看。"沈青瓷故意说得轻描淡写，没有提那只永乐青花罐，"主要是想找个靠谱的渠道出手，你在行里时间长，有什么建议吗？"

吴敏想了想："拍卖行的话，现在行情不太好，你东西不够分量的话，他们不一定愿意收。私下渠道倒是可以试试，我认识几个古玩圈的人，眼光挺毒的，出手也快。"

"哦？"沈青瓷来了兴趣。

"有个老板姓钱，人称钱永年，在南城古玩市场里算是'行头'了，做了四十年，人脉广得很。"吴敏说着压低声音，"他这人吧，有点……怎么说呢，老江湖，跟他打交道得小心点，但只要东西对路，他出价不会太离谱。"

沈青瓷点点头："能帮我牵个线吗？"

"没问题，"吴敏一口答应，"咱们同学，我还能坑你？回头我跟钱老板说一声，约个时间你们见面。"

沈青瓷看着吴敏，注意到她说"有几件东西想出手"的时候，吴敏的眼神闪了一下——那是一种很细微的变化，像是捕捉到了什么信息。但转瞬即逝，沈青瓷也没多想。

"行，那就拜托你了。"

"客气什么，"吴敏笑着给她添茶，"有什么需要帮忙的尽管说，咱们以前那关系，还用得着外道？"

---

三天后，南城古玩市场。

这片市场藏在老城区的巷子深处，门面不起眼，但在圈内名气不小。穿过一条狭窄的青石板路，两边是鳞次栉比的古董店铺，有的门脸敞亮，有的只挂一块旧匾，懂的人自然会进去，不懂的人看一眼就走了。

吴敏带着沈青瓷，七拐八拐地走到一家不起眼的店铺门口。

店铺门楣上挂着一块紫檀匾额，写着"永年斋"三个字，字体苍劲，应该是名家手笔。门口站着个穿灰色中山装的年轻人，看见吴敏就点了点头，侧身让她们进去。

店铺内部比外面看起来大得多，陈设考究，博古架上摆满了瓷器、玉器、铜器，灯光打得恰到好处，每一件东西都被照出了最好看的角度。空气里有淡淡的沉香味，和着窗外透进来的阳光，让人莫名地安心。

钱永年坐在靠里的太师椅上，正在喝茶。

他六十多岁，头发花白，面相富态，一双眼睛不大但特别有神，看人的时候带着笑，但那笑意只停留在皮肤表面，眼底深处是一种看穿世事的精明。

"钱老板，这就是我跟您说的沈青瓷，沈青瓷。"吴敏介绍道。

钱永年站起身，上下打量了沈青瓷一眼，笑呵呵地伸出手："沈小姐，久仰久仰。敏敏跟我说过，您是沈家修复一脉的传人？这可是名门正派啊。"

沈青瓷握了握他的手："钱老板客气了，我外公是做修复的，我只是跟着学了点皮毛。"

"哪里哪里，"钱永年请她坐下，亲自给她倒茶，"沈老先生的大名我也听过，故宫出来的国手，那是真正的手艺人。您能得他真传，在这行里走，有的是前途。"

客套话说完，钱永年的目光落在沈青瓷放在桌上的手提袋上。

"沈小姐今天带了什么宝贝来？"

沈青瓷打开手提袋，从里面取出两个锦盒，一一放在桌上。

她没有把永乐青花罐带来——那是她的底牌，现在还不是露相的时候。她带的是那箱"破烂"里其他的东西：一个民国粉彩碟，和一个青釉小瓶。

钱永年戴上老花镜，先拿起那个粉彩碟，对着光看了看。

"民国仿乾隆粉彩，画工细腻，品相也好，"他放下碟子，报出一个价格，"八万，您看怎么样？"

沈青瓷点点头。这个价格在她预期之内，算是合理的。

钱永年又拿起那个青釉小瓶，看了看，眉头微微皱起来。

"这个嘛……"他把瓶子翻过来看底，又看了看釉面，"普品，做工一般，两万吧。"

沈青瓷看着那个青釉小瓶。老实说，她也没太仔细看过这件东西——那天晚上她的注意力全在青花罐上，其他的东西只是粗略过了一遍。这个小瓶她记得是放在箱子中层的，釉色泛青，造型普通，当时她也觉得是件普品。

"行。"她点头同意。

钱永年笑起来，示意旁边的年轻人去办手续。

交易很顺利，十万块钱现金，沈青瓷当场清点完毕。走出永年斋的时候，吴敏挽着她的胳膊，语气里带着兴奋："怎么样，钱老板这人还不错吧？出价够意思的。"

沈青瓷点点头："确实，谢谢你帮忙牵线。"

"这算什么，"吴敏摆摆手，"以后有什么需要尽管找我，咱们同学嘛。"

---

一周后的晚上。

沈青瓷窝在出租屋的床上，漫无目的地刷着手机。这几天她一直在研究古董市场的行情，看各大拍卖行的成交记录，也关注了一些业内的公众号和论坛。

十万块到手后，她先还了一部分外债，又留出三个月的生活费，剩下的打算作为启动资金，看看有没有合适的机会。

正刷着，一条新闻推送吸引了她的注意。

是一家拍卖行的成交快报，标题写着：【古韵秋拍】青釉小瓶以十八万元落槌。

沈青瓷愣了一下，点进去看详情。

新闻里附了图片——一只青釉小瓶，高约十五厘米，直颈、圆腹、圈足，釉面匀净，呈现一种淡雅的青绿色。

她的心跳漏了一拍。

这只瓶子……她认识。

就是她上周卖给钱永年的那只，两万块。

沈青瓷放大图片，仔细看瓶身的每一个细节，又把图片下载下来和手机里之前随手拍的照片对比。造型、釉色、尺寸，都对得上。新闻里提到，这只青釉小瓶是清中期官窑作品，底款"大清乾隆年制"，流传有序，估价十五到二十万，最终以十八万落槌。

沈青瓷握着手机的手开始发抖。

底款？

她当时……没有翻过来看底款？

不对，她看了的。她记得当时翻过来看了一眼，圈足里面灰蒙蒙的，好像什么都没有……

不，不是什么都没有。是灰尘太厚，把款识盖住了。她当时急着看那只青花罐，根本没仔细清理其他东西。

沈青瓷把手机扔在床上，用双手捂住脸。

八万块钱。

她亏了整整八万块钱。

这是她半年的生活费，是她离婚后仅有的积蓄的一大块，就这么被她自己送出去了。

愤怒、懊恼、自我怀疑，各种情绪一起涌上来。

她是学这个的，从小学到大，读了那么多年书，做了那么多年鉴定，怎么会犯这种低级错误？

可是她确实犯了。

沈青瓷坐在床上，盯着天花板看了很久。

冷静下来之后，她开始复盘那天的场景。

钱永年先看那个民国粉彩碟，给了八万的合理价。然后"顺便"看了那个青釉小瓶，漫不经心地说"普品，两万"。他的语气很自然，表情也没有任何异常，就好像那真的只是一件不值钱的东西。

——他是故意的。

他用一个合理的大价来麻痹她，让她觉得这次交易很公平，然后在小件上设局。两万对八万，比例不大，不容易引起警觉。而她……她太大意了，太急着证明自己可以独立操作，结果被人卖了还帮人数钱。

沈青瓷闭上眼睛，外公的话在耳边响起——

"有眼力不等于吃得开。专业是专业，江湖是江湖，两回事。"

她终于明白这句话的含义了。

她的专业能力没有问题，但她的江湖经验是零。她不懂这行的规矩，不懂这些老江湖的手段，以为只要东西对、价格合理就可以成交。可古董行不是学术界，这里是丛林，每个人都在算计，稍有不慎就会被吃干抹净。

那个钱永年，表面上和和气气，背地里却把她当成了韭菜。

还有吴敏……

沈青瓷想起吴敏那天的表现，想起她听到"有几件东西想出手"时眼神的闪动，想起她那句"我还能坑你"——

如果吴敏和钱永年之间有什么默契呢？如果吴敏从一开始就是在给钱永年牵线搭桥，而不是单纯地帮老同学介绍渠道呢？

沈青瓷不敢深想下去。

但她记住了这一课。

---

那天晚上，沈青瓷没有睡觉。

她把那只青釉小瓶的所有资料都查了一遍——器型、年代、窑口、市场价格、历年成交记录。越查越后悔，因为所有的特征都指向清中期官窑，如果她当时仔细看过底款，仔细查过资料，根本不会犯这种错误。

但后悔没有用。

她犯了错，就要承担后果。

八万块钱的学费，让她明白了一个道理：这个行当里，信任是奢侈品，谨慎才是必需品。

凌晨四点，沈青瓷合上笔记本电脑，靠在床头，看着窗外渐渐泛白的天空。

她没有被打倒的打算。

八万块钱，就当是交学费了。她还有那只永乐青花罐，那才是她真正的底牌。

现在她需要做的，是更谨慎地研究市场，更小心地选择渠道，不再给任何人可乘之机。

而钱永年……

沈青瓷的眼神在黎明的微光中变得锐利。

她会记住这个人的。

总有一天，她要让他知道，小瞧她是要付出代价的。

# 第三章 传票

被钱永年坑了之后，沈青瓷消沉了三天。

第四天早上，她从床上爬起来，洗了个冷水脸，对着镜子里憔悴的自己说："够了。"

够了，不能再这样下去了。

八万块钱已经打了水漂，再怎么后悔也回不来。她现在需要做的不是沉溺在自我怀疑里，而是总结教训、重新出发。

她开始系统地研究古玩市场的规矩。看论坛、看案例、看别人是怎么被坑的、又是怎么爬起来的。她发现这个行当里的水比她想象的还要深——造假的、设局的、恶意压价的、洗钱的、倒手骗贷的……每一条路子都有无数坑等着新人往下跳。

而她，作为一个没有人脉、没有资金、刚刚被坑过一次的新人，想要在这个行当里立足，难度不亚于赤手空拳闯虎穴。

但她没有别的选择。

她手里只有那只永乐青花罐。那是她唯一的筹码，她必须把它变成钱，变成她重新站起来的资本。

---

这天上午，沈青瓷正在电脑前研究拍卖行的秋拍预告，手机响了。

是外卖？她没点外卖。是快递？她没买东西。

"您好，请问是沈青瓷女士吗？"

"是我。"

"这里是顺达速递，您有一封法院专递，请签收。"

法院专递？

沈青瓷愣了一下，披上外套下楼。快递员站在楼下，拿着一个牛皮纸大信封，让她签字确认。她签完字，快递员把信封递给她，头也不回地走了。

沈青瓷站在单元门口，拆开信封。

里面是一份法院传票，和一份民事诉状副本。

原告：顾正霆、顾徐氏（顾母）
被告：沈青瓷
案由：离婚后财产纠纷
诉讼请求：一、撤销双方离婚协议中关于财产分割的条款；二、判令被告返还因欺诈取得的原告家族财产。

沈青瓷一行行往下看，手指不自觉地攥紧了纸张边缘。

诉状里写得很详细：原告顾正霆与被告沈青瓷于某年某月某日协议离婚，离婚时原告将一箱家族旧物作为补偿给予被告。后原告发现该箱旧物中可能存在具有重大价值的文物，被告作为文物鉴定专业人士，在签订离婚协议时对此明知却故意隐瞒，构成欺诈，导致原告重大利益受损……

沈青瓷看到最后，忽然笑了一下。

笑容里带着冷意。

顾家动作真快。

她才刚刚开始试水，他们就已经布好了局。传票来得这么迅速，说明顾家早就在关注她的动向。她在古玩市场上的活动，她跟钱永年的接触，他们都一清二楚。

而钱永年……

沈青瓷想起那天离开永年斋时钱永年送她出门的表情，那种不动声色的精明。他是不是从一开始就知道她的身份？是不是顾家的人让他盯着她？

太多疑问，但现在不是追究的时候。

她需要一个律师。

---

下午，法律援助中心。

这是市里设立的公共法律服务机构，专门为经济困难的当事人提供免费或低价的法律援助。沈青瓷不是付不起律师费，但她现在的处境需要精打细算每一分钱，能省则省。

接待她的律师姓冯，三十多岁，女性，短发利落，说话干脆，看起来是个务实的人。

冯律师仔细看完了诉状和离婚协议，抬起头看着沈青瓷。

"沈女士，我问你一个问题，你要如实回答。"

"你说。"

"签离婚协议的时候，你知不知道那箱东西里有值钱的东西？"

沈青瓷直视她的眼睛："不知道。离开顾家之后才发现的。"

冯律师点点头，在本子上记了几笔。

"这是关键点。他们的诉求是撤销离婚协议中的财产分割条款，理由是你'欺诈'。但要构成欺诈，必须满足几个要件：一，你当时明知那批东西有重大价值；二，你故意隐瞒这个信息；三，对方因此做出了错误的决定。"

她顿了顿，继续说："你是学文物鉴定的，这点他们会抓住不放。但'学过'不等于'看过就认识'，你在顾家三年，从来没有正式参与过他们的藏品鉴定工作，对不对？"

"对。"沈青瓷点头，"顾母不让我碰那些东西。我只是偶尔趁家宴无聊的时候偷偷看看，从来没有做过正式的评估或记录。"

"好。"冯律师的语气稍微放松了一点，"这对我们有利。他们现在的证据链是不完整的，只能证明你有鉴定能力，不能证明你在签协议的时候就已经知道那批东西的价值。"

沈青瓷松了口气，但很快冯律师的下一句话又让她的心悬了起来。

"不过，有一点你要注意。"冯律师看着她，"诉讼期间，你手里的那批东西最好别动。一旦你私下交易，对方会说你在转移财产，那时候就说不清了。"

沈青瓷的心沉了下去。

诉讼期间不能交易，意味着她的翻身计划被迫暂停。那只永乐青花罐躺在保险柜里，价值八千万，却动不了。

"官司大概要打多久？"她问。

冯律师想了想："快的话半年，拖的话……一两年也有可能。取决于对方的策略和举证情况。"

一两年？

沈青瓷闭上眼睛，深吸一口气。

"冯律师，"她睁开眼睛，声音很平静，"这个案子您接吗？"

冯律师看了她一眼，点点头："接。但我先把丑话说在前面——我不打包票，只能说尽力。法律援助中心的资源有限，我能做的就是帮你把证据理顺，把程序走完。最后的结果，看法官怎么判。"

"我明白。"沈青瓷从包里拿出一个信封，"这是委托费，您看看够不够。"

冯律师接过去数了数，是三千块钱。

"够了。"她把信封收起来，站起身跟沈青瓷握手，"沈女士，这官司不好打，但也不是没有机会。你有什么问题随时联系我。"

---

从法律援助中心出来，天色已经暗了下来。

沈青瓷站在路边，等着红绿灯。手机在口袋里震动了一下，她掏出来一看——是一个陌生号码。

她犹豫了一下，还是接了。

"喂，是沈小姐吗？"

电话那头传来一个熟悉的声音，带着点笑意，像是故意压低了嗓子。

是钱永年。

"钱老板。"沈青瓷的语气淡淡的。

"哎，沈小姐，别这么生分嘛，"钱永年在电话那头笑呵呵地说，"我听说你最近摊上点麻烦事？官司打起来了？"

沈青瓷握着手机的手紧了紧。

他怎么知道？传票今天上午才收到，她下午才去找律师，钱永年的消息这么灵通？

"承蒙关心。"她的语气更冷了。

"小沈啊，我给你一个建议，"钱永年的语气变得"语重心长"起来，"这种时候可别乱动，古玩这行消息灵得很，你要是私下卖什么东西，对方律师肯定第一时间知道。到时候吃不了兜着走的是你自己。"

沈青瓷没有说话。

"当然了，我这是好心提醒你，"钱永年继续说，"你一个年轻姑娘，初来乍到的，不懂这行的规矩，吃点亏也正常。以后要是有什么需要帮忙的，尽管来找我，咱们认识一场，我不会亏待你的。"

"谢谢钱老板。"沈青瓷的声音平板得像一张纸，"没别的事的话，我先挂了。"

"哎，等等——"

她按下挂断键，把手机塞回口袋。

站在街边，夜风吹过来，带着深秋的寒意。沈青瓷感觉自己像站在一个巨大的包围圈中央，四面八方都是敌人的眼睛。

钱永年怎么知道她打官司的事？是顾家告诉他的？还是他自己的消息网？他那通电话看似是"好心提醒"，实际上是在警告——警告她不要轻举妄动，警告她在这个圈子里没有人脉、没有靠山，任人拿捏。

法律上被顾家追讨，商业上被钱永年封堵。

她孤立无援。

沈青瓷回到出租屋，第一件事就是把那只永乐青花罐从柜子里拿出来，锁进保险柜。

她跪在保险柜前，看着那只罐子静静地躺在铁盒子里，灯光照在釉面上，折射出幽深的蓝色光芒。

八千万。

这是她翻身的唯一筹码。

可现在，它被卡住了。顾家的官司像一道锁链，把它牢牢锁死，动弹不得。

沈青瓷关上保险柜的门，调好密码，站起身。

窗外，夜色浓得像墨。远处的城市灯火依旧璀璨，但那些光亮照不到她这间小小的出租屋里。

她坐到书桌前，打开笔记本电脑，开始搜索这个城市里有哪些古董拍卖会、有哪些值得关注的行业活动。

官司是官司，生活还要继续。

她不能坐以待毙。在等待法律程序走完的这段时间里，她得想办法在这个行当里建立存在感，积累人脉，学习规矩。等官司结束、东西解冻的那一天，她要有能力把它顺利变现。

而不是像现在这样，被人当成待宰的羔羊。

沈青瓷点开一个拍卖行的官网，看到一条消息：下周末有一场中小型艺术品秋拍，门票五百元，持票者可入场观摩。

五百块，对现在的她来说不算小钱。但她决定去。

因为她需要让这个圈子看到她，记住她。

沈青瓷，不是谁的附属品，不是谁的花瓶。

她是一个鉴定师，一个有真本事的人。

她要证明给所有人看。

# 第四章 拍卖会

离婚后的第十二天，沈青瓷走进了城南艺术酒店的宴会厅。

入场门票五百块，是她从生活费里抠出来的。这场秋拍规模不大，主要是中小型艺术品和一些流传有序的老物件，不是那种动辄上亿的顶级拍卖会，但对于现在的她来说，已经是难得的学习机会。

她没打算买东西。钱永年那八万块的教训还在隐隐作痛，加上官司悬着、青花罐冻住，她现在没有任何资本去参与竞拍。她来这里只有一个目的——观察，学习，在这个圈子里建立存在感。

宴会厅布置得颇为讲究。暖色调的射灯打在展示台上，周围摆着几排皮质座椅，中央留出通道。空气里飘着淡淡的咖啡香和某种昂贵香水的气息，穿着考究的男男女女三三两两地交谈，低声议论着即将上拍的藏品。

沈青瓷找了个靠后的位置坐下。

她穿得很素净，白色衬衫、黑色西裤、头发简单扎成马尾，整个人干净利落，但放在这群珠光宝气的买家中间，显得有些格格不入。旁边一个戴着翡翠耳坠的中年贵妇瞥了她一眼，目光里带着一丝打量，然后若无其事地移开了。

沈青瓷没有在意。她的注意力全在预展图录上。

入场时领到的图录有三十多页，彩色印刷，每一件拍品都有详细的描述、估价和来源说明。她从第一页开始翻，一件件看过去，在心里默默做着鉴定练习。

第8号拍品，明万历青花五彩龙纹盘。图录上标注"来源清晰，传承有序"，估价六十万至八十万。沈青瓷看了看图片，又看了看描述，微微皱眉——五彩的发色有点问题，不太像万历年间的特征，倒更像是清代仿明的风格。

她在图录空白处用铅笔轻轻画了个问号。

第15号拍品，清康熙豇豆红太白尊。估价三百万至四百万。沈青瓷仔细看了看图片，这件的釉色和造型都没问题，应该是真品，但价格偏高，市场上同等品相的成交价一般在两百五到三百之间。

她又画了个向下的箭头。

就这样一路翻下去，等翻到第23号拍品时，她的手指停住了。

"清乾隆青花碗（存疑）"。

图录上用红色小字标注了"存疑"二字，估价也因此被压低，只有十万至十五万。沈青瓷盯着那张图片看了很久，然后翻到后面的详细描述。

描述里写道："此碗青花发色纯正，绘工精细，足墙略有修补痕迹。因胎底款识模糊，真伪存疑，买家自辨。"

沈青瓷把图录凑近眼前，仔细观察那只碗的足墙部分。

灯光下，图片的分辨率有限，但她还是注意到了一些细节——足墙和圈足交接处的角度不太对，过于平滑，缺少乾隆官窑特有的那种细腻而略带棱角的处理方式。这个细节很小，小到一般人根本不会注意，但沈青瓷见过太多真品了，那种微妙的差异一眼就能看出来。

她在心里做出判断：这不是"存疑"，是确假。民国仿品。

---

拍卖会正式开始。

主持人是个穿西装的中年男人，声音洪亮，节奏把握得很好。前面几件拍品顺利成交，价格都在估价范围内浮动，没有太大的惊喜也没有冷场。

沈青瓷坐在后排，默默观察着现场的情况。

她注意到，真正的大买家很少自己举牌。他们要么派代理人出面，要么通过电话委托竞拍，自己只是坐在角落里喝咖啡，偶尔抬抬眼皮看一眼场上的情况。而那些举着号牌频繁竞价的，反倒大多是新入行的藏家或者纯粹来凑热闹的有钱人。

"第23号拍品，清乾隆青花碗，起拍价十万。"

主持人的声音把沈青瓷的注意力拉回来。

展示台上，那只青花碗被工作人员小心翼翼地摆放在转盘上，射灯打下来，釉面泛着温润的光泽。单从外观来看，确实很漂亮，造型规整、发色纯正，不仔细看很难发现问题。

"十万起拍，有没有感兴趣的藏家？"

现场举起了两个号牌，一个出价十万，另一个加到十二万。

沈青瓷看着那只碗，不由自主地皱起眉头。

"这不是存疑，是民国仿品。"

她的声音很轻，几乎只是自言自语，但在周围安静的环境里还是显得有些突兀。

旁边座位上的人转过头来。

是一个男人，三十岁出头的样子，穿着深灰色的休闲西装，没打领带，衬衫领口敞开一颗扣子，整个人看起来很放松。他的五官端正但不算特别出众，真正让人印象深刻的是那双眼睛——很沉静，像是深潭一样，看人的时候带着一种审视的意味。

"怎么看出来的？"他问。

声音不高不低，语气平和，却带着一种认真询问的态度，不像是在质疑，更像是真的想知道答案。

沈青瓷愣了一下，意识到自己多嘴了。

她不认识这个人，不知道他的来历，也不清楚他在圈子里是什么身份。万一是哪家拍卖行的人呢？万一跟钱永年有关系呢？她现在的处境本来就够麻烦了，不应该随便得罪人。

但那人一直看着她，目光里没有敌意，只是单纯的好奇。

沈青瓷犹豫了两秒，还是开口了。

"足墙和圈足交接处的处理方式不对。"她指着图录上的细节说，"乾隆官窑的圈足处理很讲究，交接处会有一个微妙的棱角感，不是纯圆的。但这件碗的交接处太平滑了，像是用砂纸打磨过——这是民国仿官窑的通病，他们模仿造型和发色很到位，但细节处理上总差那么一点意思。"

她说完才意识到自己说得太专业了，可能暴露了太多东西。

那男人没有立刻回应，而是低头认真看了看图录，沉默了几秒。

然后他点了点头："你说得对。"

语气很平静，像是在陈述一个事实。

台上的竞价还在继续，那只"存疑"的乾隆青花碗最终以十八万落槌，买家是一个穿貂皮大衣的中年妇女，举着号牌时脸上带着志得意满的笑容。

沈青瓷收回目光，不再多说什么。

中场休息时间。

宴会厅外的休息区摆着几张高脚桌，提供免费的咖啡和点心。沈青瓷没有去凑热闹，而是找了个角落的沙发坐下，继续翻看图录后半部分的拍品。

"我请你喝杯咖啡？"

一个声音在头顶响起。

她抬起头，是刚才坐在她旁边的那个男人。他手里端着两杯咖啡，一杯递给她，自己拿着另一杯在对面的沙发上坐下。

沈青瓷迟疑了一下，还是接过了咖啡。"谢谢。"

男人从口袋里掏出一张名片递过来。

"程砚洲，做拍卖顾问的。"

沈青瓷接过名片看了一眼。简洁的白色卡纸，上面只印着名字和一个电话号码，没有公司名称，也没有花哨的头衔。

"刚才那件碗，你说得对。"程砚洲喝了口咖啡，"我之前查过它的著录，来源就有问题。图录上写的'传承有序'是编的，三年前这件东西在另一场小拍卖会上出现过，当时就被人质疑过真伪。"

沈青瓷挑了挑眉："既然来源有问题，为什么还能上拍？"

"因为拍卖行标注了'存疑'，风险转嫁给买家。"程砚洲淡淡道，"这行就是这样，明着告诉你可能有问题，但还是有人愿意赌。刚才那位女士，大概是觉得十八万买个'可能是真的'很划算。"

沈青瓷沉默了一下，然后从包里翻出自己的名片。

是旧的，离婚前印的，上面还写着"顾家藏品管理"几个字。她递出去的时候有一瞬间的窘迫，犹豫要不要解释一下，但话到嘴边又咽了回去。

"名片有点旧了，"她最终只是简单说道，"我现在……自己做。"

程砚洲接过名片，目光在"顾家"两个字上停了一下。

那视线只有短短一瞬，但沈青瓷还是捕捉到了。他的眼神变了——不是惊讶，也不是轻蔑，而是某种更复杂的东西，像是突然确认了什么猜测。

但他什么都没说，只是把名片收进口袋里。

"顾家的藏品，我听说过一些。"他的语气很平淡，"是个讲究的人家。"

沈青瓷不知道该怎么接这话，干脆低头喝咖啡。

两人沉默了一会儿。

"你是沈家修复一脉的人？"程砚洲忽然问道。

沈青瓷的手指微微一紧："你怎么知道？"

"刚才你分析那只碗的时候，用的是修复师的角度，"程砚洲的目光很平静，"普通鉴定师看东西，先看釉面、发色、画工；修复师看东西，先看胎骨、接缝、做旧痕迹。你刚才说的'足墙和圈足交接处'，这是修复时最先检查的位置。"

沈青瓷没有说话。

程砚洲继续说道："我查过一些资料。沈家修复一脉在故宫系统里很有名，老沈先生——也就是你外公——三十年前修复的那批永乐青花残件，在圈子里被传为经典。"

沈青瓷抬起头看着他。

程砚洲的表情很坦然，没有遮掩的意思："我调查过你的背景，所以今天才坐在你旁边。"

沈青瓷的手指在咖啡杯上收紧了一些。

"你想干什么？"

"请你帮个忙。"程砚洲直接说道，"我手里有一批瓷器需要鉴定，大部分我自己能看，但有几件吃不准。你如果愿意帮忙，报酬按市场价。"

沈青瓷盯着他看了几秒。

"就因为我是沈家的人？"

"因为你有真本事。"程砚洲的语气很平静，"刚才那只碗，现场两百多号人，只有你看出问题。而且你不是猜的，你是真的懂。"

沈青瓷没有立刻回答。

她在心里快速权衡着利弊。这个程砚洲来路不明，突然出现、主动搭话、调查过她的背景——这些行为放在任何一个刚被坑过一次的人身上，都应该引起警觉。

但她同时也意识到，她现在需要机会。

官司悬着，青花罐动不了，钱永年在背后虎视眈眈，顾家的势力随时可能出手。她没有人脉、没有资金、没有靠山，唯一的筹码就是自己的专业能力。如果有正规渠道愿意给她工作机会，她没有理由拒绝。

"我可以考虑。"她最终说道，"但我需要先知道这批东西的来源，还有你找我的真正原因。"

程砚洲点了点头，似乎早就预料到她会这么问。

"那批东西的委托人叫白行舟，是个海外回流藏家。来源……不太干净。"

他顿了顿，直视沈青瓷的眼睛："我找你，是因为那批东西里可能有几件，是你外公当年经手过的。我需要有人能帮我确认它们的真实流转历史。"

沈青瓷的心跳漏了一拍。

外公经手过的东西？什么意思？那些东西是怎么到了白行舟手里的？

她张了张嘴想问，但话到嘴边又停住了。这里不是深谈的地方，而且她还需要更多信息才能判断程砚洲的话有几分可信。

"我需要时间考虑。"她站起身，"有结果了会联系你。"

程砚洲也站了起来，把自己的咖啡杯放在桌上。

"随时都可以。"他说，"有机会聊聊。"

两人交换了一个点头，然后程砚洲转身走向另一边的人群，很快被几个穿西装的男人围住了，看起来像是在谈什么事情。

沈青瓷独自站在休息区，看着他的背影消失在人群中。

---

拍卖会下半场她没再进去，而是直接离开了酒店。

傍晚的天色灰蒙蒙的，街道上人来人往。沈青瓷走到最近的公交站台，等车的时候掏出手机，在搜索引擎里输入了"程砚洲"三个字。

搜索结果跳出来一堆。

第一条是某财经杂志的专访，标题是《新生代艺术品顾问程砚洲：用专业说话》。

她点进去看了看。

程砚洲，三十二岁，毕业于海外某知名大学艺术品鉴定专业硕士，曾在苏富比实习两年，回国后为多位私人藏家提供顾问服务。履历干净漂亮，在圈子里口碑不错，既能跟老一辈的传统藏家打交道，也能和新兴的投资型买家沟通。

照片里的他穿着正装出席某个拍卖会，表情沉稳，看起来确实像个靠谱的人。

沈青瓷又翻了几条相关报道，内容大同小异，基本上都是在夸他专业、稳重、有底线。没有负面新闻，没有争议事件，简历干净得像一张白纸。

太干净了。

沈青瓷微微眯起眼睛。

在这个鱼龙混杂的古董圈里，一个三十出头的年轻人能做到这种程度的"干净"，要么是真的本事够硬，要么就是背后有人在护着。

公交车来了。沈青瓷收起手机，上了车，在最后一排的位置坐下。

车窗外的街景飞速后退，霓虹灯开始在暮色中亮起来。

她靠着座椅，脑子里转着各种念头。

程砚洲找她的原因，他说的"白行舟"，那批"可能是外公经手过"的瓷器……这些信息串在一起，让她隐隐觉得不安。

如果那批东西真的有问题，如果它们的来源跟某些不干净的渠道有关联，那她贸然卷进去，可能会惹上更大的麻烦。

但如果她不去接触，就永远不知道那些东西是什么、从哪里来、为什么会出现在白行舟手里。

而且——

沈青瓷想起程砚洲看到她名片上"顾家"两个字时的眼神。

那不是单纯的好奇。

他认识顾家？还是说，他早就知道她的身份？他今天坐在她旁边，真的只是巧合吗？

车厢里有些闷热，空调吹出来的风带着一股陈旧的味道。沈青瓷把车窗推开一条缝，让外面的凉风灌进来。

不管怎样，今天这趟拍卖会没有白来。

她至少证明了一件事：自己的眼力没有退步。在两百多号人里面，她是唯一一个看出那只碗有问题的人——不是因为运气，不是因为猜测，是因为真本事。

钱永年可以坑她一次，顾家可以打压她，圈子里的人可以看不起她，但她的专业能力是实打实的，谁都拿不走。

公交车经过一条商业街，车窗外的玻璃橱窗里反射出她模糊的影子。沈青瓷看着那个影子，忽然想起自己包里那张旧名片——上面还印着"顾家藏品管理"几个字，像个撕不掉的标签。

她从包里翻出那张名片，看了几秒，然后收了起来。

该重新印一张了。

新名片上不要什么"顾家"，不要什么"藏品管理"，就写六个字——

沈青瓷，文物鉴定。

够了。

---

公交车在老小区门口停下。沈青瓷下了车，走进暮色中的街道。

路灯亮了，橙黄色的光晕落在老旧的路面上。她走得不快，脑子里还在想着今天的事情。

程砚洲的邀请，那批来源不明的瓷器，外公可能经手过的东西……

这些线索隐隐约约地指向某个方向，但她现在还看不清全貌。

不过没关系。

她有的是时间，有的是耐心。

走到单元楼门口时，她停下脚步，抬头看了看三楼的窗户。灯没开，窗帘拉着，黑漆漆的一片。

那间小小的出租屋里，保险柜里躺着那只永乐青花罐。八千万的筹码，被官司锁住，动弹不得。

但她不能就这样干等着。

官司要打，生活要继续，她得在等待的时间里积累人脉、学习规矩、建立存在感。等到官司结束、东西解冻的那一天，她要有能力把它顺利变现——而不是像现在这样，被人当成待宰的羔羊。

沈青瓷推开单元门，走进楼道。

脚步声在狭窄的空间里回响，一级一级往上。

她想起外公说的那句话——"专业是专业，江湖是江湖，两回事"。

现在她开始明白这句话的意思了。

有眼力不等于吃得开，会看东西不等于会做生意，懂鉴定不等于懂人心。这个行当里的水比她想象的深得多，钱永年那样的老狐狸遍地都是，稍不留神就会被人算计。

但她不怕。

她是沈家的人，是从故宫修复室里泡大的孩子，是能在两百多号人中一眼看出问题的专业人士。

这是她的底气，是她的资本，是她在这个江湖里安身立命的根本。

沈青瓷打开出租屋的门，在黑暗中摸索着开了灯。

暖黄色的光亮起来，照亮了那间简陋的小屋。

她把包放在桌上，走到保险柜前，蹲下身，用手指轻轻抚过冰冷的金属表面。

保险柜里的青花罐她看不见，但她知道它就在那里。静静地等待着，等待着她一步步走出困境，等待着她有一天能够光明正大地把它变现。

"等着吧，"她轻声说，"不会太久的。"

然后她站起身，走到书桌前，拿出手机，存下了程砚洲的电话号码。

窗外，夜色渐浓，远处的城市灯火依旧璀璨。

沈青瓷坐在桌前，开始整理今天拍卖会上的笔记。

新的一天明天就会到来，而她要为此做好准备。

# 第五章 第一次开庭

开庭的日子定在一个阴天。

沈青瓷早上六点就醒了，躺在床上盯着天花板发呆，脑子里反复过着冯律师叮嘱的那些话——"不要激动"、"不要主动攻击"、"问什么答什么"、"没把握的就说不记得了"。

她把这些话像背书一样在心里默念了十几遍，然后爬起来洗漱。

镜子里的自己脸色不太好，眼下有淡淡的青黑。她用冷水拍了拍脸，换上昨晚准备好的衣服——深灰色的西装套裙，头发盘起来，整个人看起来干练利落，但又不会太过张扬。

冯律师说得对，法庭上第一印象很重要。

九点钟，沈青瓷坐地铁到了区人民法院。

法院门口的安检排着长队，她跟着人群慢慢往前挪。周围的人神色各异，有的面无表情，有的愁眉苦脸，有的在低声打电话——每个人都背着自己的故事，等着法律给一个说法。

过了安检，冯律师已经在大厅等着了。

冯律师五十来岁，头发花白，戴一副金丝边眼镜，整个人透着一股老派知识分子的儒雅气质。他是法律援助中心指派的，虽然不收费，但办事认真得很，每次见面都带着厚厚一叠材料，上面密密麻麻做满了批注。

"来得挺早。"冯律师点点头，"准备好了？"

"准备好了。"沈青瓷的声音很平静，但握着包带的手指微微收紧。

"走吧，三号法庭。"

---

三号审判庭比沈青瓷想象的要小。

木质的审判台、原告席、被告席、旁听席整齐排列，头顶的日光灯散发着惨白的光芒，空气里飘着一股淡淡的消毒水味道，混合着某种说不清的压抑感。

顾家那边已经到了。

沈青瓷走进法庭的一瞬间，感觉到好几道目光同时射过来。她没有回避，而是不卑不亢地扫了一眼原告席。

顾正霆坐在最边上，穿着一身剪裁考究的深蓝色西装，表情淡漠，看她的眼神像在看一个陌生人。他旁边是顾母，五十多岁的女人，保养得很好，头发一丝不乱，脸上带着一种居高临下的冷淡。

顾家的律师团队有两个人。为首的是个四十来岁的男人，鹰钩鼻，薄嘴唇，目光锐利得像刀子。沈青瓷认得他——周海涛，顾家长期合作的律师，在商界口碑不错，打官司很有一套。

两边对视了一眼，都没有说话。

冯律师示意沈青瓷坐下，自己在旁边落座，低声说："对方来了两个律师，阵仗不小。别紧张，按我们之前说的来。"

沈青瓷点了点头。

她其实没有那么紧张。这些天她把可能出现的情况在脑子里过了无数遍，已经做好了心理准备。反倒是真正坐在这里的时候，心里出奇地平静，像是一场迟早要来的考试终于开始了。

"全体起立——"

书记员的声音打断了她的思绪。

法官走进来，在审判台后面落座。是个五十多岁的女法官，戴着眼镜，表情严肃但不苛刻，给人一种公正稳重的感觉。

"坐下。现在开庭，审理原告顾正霆、顾美珍诉被告沈青瓷离婚财产分割纠纷一案……"

法官念完开场白，翻开卷宗："请原告陈述诉讼请求。"

周海涛站起来，声音洪亮："审判长，我方当事人请求：一、依法撤销原、被告离婚协议中关于财产分割的条款；二、判令被告返还因欺诈取得的财物。事实与理由如下——"

他停顿了一下，目光扫过沈青瓷，嘴角微微上扬。

"被告沈青瓷在与原告顾正霆离婚前，系顾家藏品管理人员，具备专业的文物鉴定背景。离婚时，原告将一批旧物作为补偿交付被告。该批旧物中包含若干具有较高价值的文物艺术品，而被告凭借其专业知识，在签署离婚协议时已明知或应知这些物品的价值，却故意隐瞒，使原告在不知情的情况下作出错误的财产处分决定。"

他翻开一份材料："我方现提交以下证据——"

"第一，被告的学历及从业背景资料，证明被告具备专业的文物鉴定能力。"

"第二，顾家库房的监控录像截图若干，显示被告在婚姻存续期间多次在库房内长时间停留、观察藏品，时间从十分钟到四十分钟不等。"

"第三，被告离婚后的通讯记录，显示被告在离婚后第三天就开始联系古玩市场的相关人员，试图出售物品。"

周海涛把材料递给书记员，继续说道："综上，被告作为专业人士，在签署离婚协议时明知或应知涉案物品具有较高价值，却未如实告知原告，构成欺诈。据此，我方请求依法撤销离婚协议中关于财产分割的条款。"

他说完坐下，脸上带着胸有成竹的表情。

沈青瓷听着这些话，指甲悄悄掐进掌心。

她知道会是这样。顾家有的是资源去调查她，那些监控录像、通讯记录，普通人根本拿不到，但对顾家来说不过是几个电话的事情。

但她没有慌。

冯律师早就分析过：这些证据看起来很多，其实都是间接证据，不能直接证明她"签协议时就已经知道价值"。

法官示意被告方发言。

冯律师站起来，语气不紧不慢："审判长，被告对原告陈述的事实和理由均不予认可。"

"首先，关于被告的专业背景。原告提交的学历及从业资料确实表明被告具有文物鉴定相关知识，但'具备鉴定能力'不等于'见过就能认识'。被告的专业领域是陶瓷，尤其是明代官窑，而涉案物品种类繁多，包括瓷器、铜器、杂项等，被告不可能对所有类别都同样精通。"

他翻开自己的材料："我们提交被告在大学期间的课程记录和专业方向说明，证明其研究领域的具体范围。"

"其次，关于监控录像。原告所称的'长时间停留、观察藏品'，实际上是被告在婚姻存续期间的正常行为。被告作为家庭成员，在自己家中观看藏品完全合法合理，不能据此推断其具有非法目的。而且——"

冯律师加重语气："被告从未正式参与过顾家的藏品鉴定工作。她只是偶尔在家宴等场合观看过部分藏品，从未进行过系统评估或记录。原告无法证明被告在签署离婚协议时对涉案物品进行过专业鉴定。"

"最后，关于离婚后的通讯记录。被告离婚后联系古玩市场人员，是为了处理自己合法获得的财产，这是正当行为，不能作为其'签协议时已知价值'的证据。原告的这一推理逻辑是：因为被告后来想卖东西，所以被告当时就知道东西值钱。这是典型的事后推断，不具有证明力。"

冯律师最后说道："综上，原告无法证明被告在签署离婚协议时具有'明知故意隐瞒'的主观故意，其关于'欺诈'的指控缺乏事实和法律依据，请求法庭依法驳回原告的诉讼请求。"

他说完坐下。

法庭里安静了几秒。

法官翻看材料，问原告方："原告是否有补充意见？"

周海涛站起来："有。审判长，我方还有一份证人证词需要提交。"

沈青瓷的心跳漏了一拍。

证人？什么证人？

周海涛把一份材料递给书记员，同时说道："证人吴敏，系被告大学同学，目前在拍卖行从事修复工作。据吴敏陈述，被告离婚后第一次与她见面时，表现得'非常懂行，像是早有准备'，并且主动询问古玩市场的渠道——"

沈青瓷的脑袋嗡了一下。

吴敏。

她的眼前闪过那张熟悉的脸。城西老茶馆里，吴敏热心地说"我认识古玩圈的人，帮你牵线"；钱永年的店铺里，吴敏笑盈盈地坐在旁边，眼神闪烁……

从一开始，吴敏就是钱永年的眼线。

而现在，她连法庭上都没放过。

"——证人还表示，"周海涛继续念道，"'沈青瓷对古玩鉴定很精通，她要是看中什么东西，肯定一眼就能认出来。'"

他收起材料，看向法官："审判长，这份证词说明被告在离婚时已经具备识别涉案物品价值的能力，且其离婚后的行为表明她早有打算。结合前述证据，足以证明被告构成欺诈。"

沈青瓷死死攥着拳头，指甲掐进肉里。

她想反驳，想站起来说吴敏在撒谎，说那些话根本不是她的本意——但她知道不能。冯律师开庭前反复叮嘱过，法庭上不能情绪化，更不能随意打断对方陈述，否则只会让自己显得心虚。

冯律师在旁边轻轻碰了一下她的手臂，示意她冷静。

然后他站起来。

"审判长，对于原告提交的证人证词，我方有以下意见——"

冯律师的语气依然平稳，但眼神变得锐利起来。

"第一，证人吴敏与被告是大学同学关系，但据我方了解，证人目前所在的拍卖行与原告方存在业务往来。证人的证词是否客观中立，需要进一步核实。"

周海涛皱了皱眉，想说什么，但法官示意他稍等。

"第二，证人所说的'非常懂行''像是早有准备'，这是主观评价，不是客观事实。被告作为文物鉴定专业出身，表现得'懂行'是正常的，不能据此推断她在签署离婚协议时就已经识别出涉案物品的价值。"

"第三，"冯律师加重语气，"即便证人的陈述属实，也只能证明被告在离婚后表现得'像是有准备'，而不能证明她在离婚前就已经知道价值。原告需要证明的是'签协议时明知'，而不是'签协议后懂行'。这是两个完全不同的概念。"

他看向法官："原告方始终无法证明被告在签署离婚协议的那一刻，对涉案物品进行过专业鉴定并明知其价值。所有的证据都指向'事后'，而不是'事前'。这不符合欺诈的构成要件。"

法庭里再次安静下来。

法官低头看了看材料，然后抬起头："双方还有补充吗？"

周海涛站起来："审判长，我方认为现有证据已经形成完整证据链——"

法官摆了摆手："本案的争议焦点在于被告是否存在欺诈的主观故意。从目前的证据来看，原告提供的证据能够证明被告具备专业能力，但不能直接证明被告在签署协议时明知涉案物品的具体价值。"

她合上卷宗："法庭需要更多证据来判断被告的主观状态。本案择日再审，双方继续举证。休庭。"

"全体起立——"

---

走出法庭的时候，沈青瓷的腿有点软。

不是因为害怕，而是因为憋了太久的情绪终于可以释放。她一直绷着，从进法庭到出来，脸上始终保持着平静，但内心早已翻江倒海。

吴敏。

她闭上眼睛，深吸了一口气。

那个一起吃过饭、一起聊过天、一起说"咱们同学我还能坑你"的人，从一开始就在背后捅她刀子。第一次见钱永年是她牵的线，第一次被坑的信息是她传出去的，现在连法庭上的证词都安排好了。

沈青瓷不知道该愤怒还是该心寒。

也许两者都有。

"小沈。"

一个声音从身后传来。

沈青瓷转过头，看到顾母站在法庭门口的走廊里，旁边站着周海涛和一个助理，但她们都保持着距离，只有顾母一个人朝她走过来。

"顾太太。"沈青瓷站住脚步，语气冷淡。

顾母走到她面前，目光上下打量了她一番。

"瘦了，"顾母说，语气里听不出是关心还是讽刺，"日子过得不太好吧。"

沈青瓷没有接话。

顾母也不在意，继续说道："青瓷，你是聪明人，应该知道今天这只是开始。我们手里的证据还有很多，没有一次性拿出来，是给你留体面。"

她的声音不高不低，就像在谈一桩普通的生意："那批东西，你吃不下的。早点吐出来，我当你是不懂事，不追究。继续拖下去……"

她顿了顿，嘴角微微上扬："你在这个圈子里的名声，我一句话就能毁掉。到时候就算你打赢了官司，东西也出不了手——我在这个行当里的面子，够让你一件都卖不出去。"

沈青瓷看着她，沉默了几秒。

这个女人说的不是虚话。顾家在古董圈经营了三十多年，人脉遍布全国，真要打个招呼封杀她，没几个人敢不给面子。

但她更清楚，如果现在服软，就什么都没有了。

"顾太太，"沈青瓷开口，声音平静得出奇，"您说得对，我是吃不下那批东西。"

顾母眼睛一亮，以为她要松口。

"但那是我的东西。"沈青瓷一字一顿，"协议上写得清清楚楚，是您儿子亲手签的字、亲手给我的。您要是觉得自己儿子眼瞎了、脑子糊涂了、签协议的时候被人骗了——那您应该去找他算账，不是来找我。"

顾母的脸色变了。

沈青瓷没有给她说话的机会，转身就走。

走出两步，她听到身后传来顾母压低的声音："沈青瓷，你会后悔的。"

沈青瓷没有回头。

她的背影很直，脚步很稳，但只有她自己知道，心脏正在狂跳，握着包带的手指几乎发白。

---

法院门口的台阶上，冯律师等着她。

他看了看沈青瓷的脸色，没有多问，只是递过来一瓶矿泉水："喝点水。"

沈青瓷接过去，拧开盖子喝了一口。凉意顺着喉咙滑下去，让她稍微冷静了一些。

"今天打成平手。"冯律师说，"法官的态度你也看到了，她认为他们的证据链不完整。只要我们能在下次开庭前找到更有力的反证，胜算还是有的。"

沈青瓷点了点头。

"但是，"冯律师话锋一转，"吴敏那份证词是个隐患。她是你的熟人，她的话法官可能会作为参考。如果对方下次还传更多证人，或者找到新的证据……你要有心理准备。"

"我知道。"沈青瓷说。

冯律师看了她一眼，问："官司还要打多久？"

这本来应该是她问律师的问题。沈青瓷愣了一下，然后苦笑："您是想告诉我，可能会很久？"

# 第六章 老同学

屏幕上显示的是一个陌生号码，但归属地是本市。她犹豫了一下，还是接了。

"喂？"

"沈青瓷小姐？"电话那头传来一个低沉的男声，"我是程砚洲，上次拍卖会上见过。"

沈青瓷愣了一下。

程砚洲。

那个在拍卖会上主动搭话的男人，海外艺术品鉴定硕士，苏富比实习经历，私人藏家顾问。

"程先生，"沈青瓷的声音恢复了冷静，"有什么事吗？"

"有个事情想请你帮忙。"程砚洲的语气很直接，没有多余的客套，"我手里有一批瓷器需要鉴定，大部分我能判断，但有几件吃不准，想请你看看。报酬按市场价。"

沈青瓷没有立刻答应。

"为什么找我？"

电话那头沉默了两秒，然后程砚洲说："拍卖会上你说那件碗是民国仿品，我后来查过它的著录，确实有问题。你的眼力很准。"

"只是因为眼力？"

"还有一个原因。"程砚洲的声音微微压低，"我查过你的背景——你是沈家修复一脉的人，你外公是故宫退休的陶瓷修复师。那批瓷器里，可能有几件是你外公当年经手过的。"

沈青瓷的心跳漏了一拍。

"你查过我？"

"是。"程砚洲没有否认，"我需要找一个可靠的人合作，所以做了一些调查。如果这让你不舒服，我道歉。"

沈青瓷沉默了一会儿。

说实话，被人调查背景的感觉并不好，有一种被人扒光了看的窥探感。但程砚洲的坦诚让她稍微放下了一点戒备——至少他没有藏着掖着。

"见面聊吧。"她说。

"好。明天下午两点，我把地址发给你。"

---

程砚洲的工作室在城东的一个艺术园区，是一栋独立的三层小楼。

沈青瓷下了出租车，站在门口打量了一下。外墙是简洁的灰白色，入口处种着几丛翠竹，整体风格低调但不失品位。

她按了门铃。

几秒钟后，门开了。

程砚洲站在门口，穿着一件深蓝色的亚麻衬衫，袖子挽到手肘，看起来比拍卖会上那天随意了一些。

"沈小姐，请进。"

沈青瓷点了点头，跟着他走进去。

一楼是接待区和会客厅，装修风格偏中式，但细节处有现代感。墙上挂着几幅书画，沈青瓷扫了一眼，看出其中两幅是真迹。

"茶还是咖啡？"程砚洲问。

"茶吧。"

程砚洲泡了一壶龙井，给她倒了一杯："那批东西在二楼，你先喝口茶，我跟你说说情况。"

沈青瓷接过茶杯，没有喝，而是直接问："是什么来路？"

程砚洲在她对面坐下，双手交叉放在桌上："委托人叫白行舟，是个海外回流藏家。他手里有一批瓷器想出手，找我帮忙评估和对接买家。我看了大部分，判断没问题，但有几件……"

他顿了顿："有几件的著录有问题。"

"著录？"

"对。"程砚洲从旁边拿过一份文件夹，递给她，"这是那批东西的清单和相关资料。你看第七页。"

沈青瓷翻到第七页，上面是一件青花大盘的照片和说明。

照片上的青花大盘直径约四十厘米，盘心绘着缠枝莲纹，釉色润泽，看起来是件不错的东西。说明文字写着"清雍正青花缠枝莲纹大盘，来源：民国陈氏旧藏"。

"这件有什么问题？"

"来源有问题。"程砚洲说，"所谓的'民国陈氏旧藏'出自一本叫《陈氏藏瓷录》的手记。但我请人查过，这本手记的最早流通记录是十五年前，而且只在一个小圈子里出现过。换句话说——"

"这本手记可能是伪造的。"沈青瓷接上他的话。

"对。"程砚洲点头，"伪造著录是造假产业链的常见手法。先做几件高仿，再伪造一本'民国藏家手记'，声称东西是某某名家旧藏，来源可靠。买家一看著录俱全，往往就信了。"

沈青瓷低头继续看资料。

往后翻了几页，她看到了另一件东西的照片——一只青花缠枝牡丹纹大罐。

她的动作突然停住了。

这件东西……她见过。

不是在拍卖会上，不是在古玩市场，而是在外公的修复记录里。

"这件……"她指着那张照片，声音有些发紧。

程砚洲看了她一眼，说："这件青花大罐，著录说是'故宫旧藏，1960年代外贸口调拨出境'。我让人查了故宫的档案，确实有这样一件东西被调拨过，但档案显示它在1983年就被追回了。"

"1983年追回？"沈青瓷皱眉，"那现在怎么会在白行舟手里？"

"这就是问题所在。"程砚洲的语气变得严肃起来，"要么是档案有误，要么是这件东西被人动了手脚，真品被掉包，白行舟手里的是仿品。要么……"

"要么是有人把追回的东西又倒卖出去了。"沈青瓷接上他的话。

两人对视了一眼，都从对方眼中看到了同样的猜测。

"你找我，就是因为这个？"沈青瓷问。

程砚洲点头："这件青花大罐的修复记录里，有你外公的名字。1978年，故宫组织了一次文物修复，这件罐子的底部有一处裂纹，是沈老先生修补的。"

沈青瓷沉默了。

她想起外公的修复室，想起那些泛黄的笔记本，想起外公说过的话——"这行的水，比你想的深"。

原来外公说的不只是商业上的尔虞我诈，还有这些不见天日的黑幕。

"你想让我做什么？"她抬起头，直视程砚洲的眼睛。

"帮我鉴定这批瓷器。"程砚洲说，"尤其是那几件有疑问的。我需要知道它们的真伪，也需要知道它们的真实流转历史。"

"你怀疑这批东西和什么人有关系？"

程砚洲没有直接回答。他站起身，走到窗边，背对着沈青瓷说："伪造《陈氏藏瓷录》的人，叫徐鸣。景德镇的，做仿古瓷的，在圈子里有点名气。"

沈青瓷的瞳孔微微收缩。

徐鸣。

她记得这个名字——钱永年的电话里提过，吴敏也说过，钱永年最近在找徐鸣"做一件永乐青花"。

"徐鸣和钱永年什么关系？"她问。

程砚洲转过身，看着她："你知道钱永年？"

"知道。"沈青瓷没有隐瞒，"他坑过我。"

程砚洲点了点头，似乎并不意外："钱永年是徐鸣的主要销售渠道。徐鸣在景德镇做货，钱永年在这边负责洗白和出货。他们合作很多年了。"

"那白行舟呢？他在这条链里是什么角色？"

"不确定。"程砚洲说，"可能是被蒙骗的买家，也可能是这条链的一部分。我需要更多信息才能判断。"

沈青瓷低头想了一会儿。

她想起自己这段时间的遭遇——被钱永年设局坑钱、被顾家法律追讨、被吴敏背叛出卖。这些事情看似独立，但现在想来，似乎有一条隐秘的线把它们串在了一起。

钱永年不只是一个"坑新人"的老狐狸，他背后还有一整条造假产业链。

而她，无意中踩到了这条链的边缘。

"你说的合作，具体是什么条件？"她问。

程砚洲走回来，在她对面坐下："你帮我鉴定这批瓷器，报酬按市场价结算。如果在鉴定过程中发现了什么，我们共享信息。另外——"

他顿了顿，目光变得认真起来："如果查到什么和你外公有关的事情，我会第一时间告诉你。"

沈青瓷沉默了几秒。

她知道程砚洲找她不只是因为她的专业能力，还因为她和那批东西有某种"渊源"。程砚洲想通过她查清来龙去脉，而她也想知道——那些流转了几十年的东西背后，究竟藏着什么秘密。

"好。"她说，"我答应你。"

程砚洲点了点头，脸上的表情微微松了松。

"但我有一个条件。"沈青瓷补充道。

"说。"

"如果查到什么，我有知情权。"她的目光直视他的眼睛，"不管是好消息还是坏消息，不管和谁有关，你不能瞒我。"

程砚洲看了她几秒，然后伸出手："成交。"

沈青瓷握了握他的手。

这是一次合作，也是一次博弈。她不完全信任程砚洲，正如程砚洲也不可能完全信任她。但在这个充满尔虞我诈的行当里，有限度的合作有时候比单打独斗更有效。

"那批东西在二楼，"程砚洲站起身，"跟我来吧。"

---

二楼是一个专业的鉴定工作区。

房间四面都是窗户，自然光充足。中央摆着一张大型工作台，台面铺着深灰色的绒布，旁边的架子上整齐地摆放着放大镜、紫光灯、卡尺等各种鉴定工具。

靠墙的柜子里，一排锦盒整整齐齐地排列着。

"一共十二件。"程砚洲说，"你慢慢看，有问题随时问我。"

沈青瓷点了点头，走到工作台前，戴上手套。

程砚洲从柜子里取出第一件——就是那只青花缠枝牡丹纹大罐。

沈青瓷深吸一口气，小心翼翼地接过来。

罐子入手的瞬间，一种熟悉的感觉涌上心头。她轻轻转动罐身，让目光从口沿一路扫到底部，观察釉面的光泽、青花的发色、胎体的厚度。

然后她把罐子翻过来，看底部。

那里有一道极细的修复痕迹，如果不是专业人士，根本看不出来。但沈青瓷太熟悉这种手法了——这是外公的风格，用特殊的材料填补裂缝，再用与原釉相近的颜料覆盖，几乎天衣无缝。

"是外公修的。"她轻声说。

程砚洲站在旁边，没有说话。

沈青瓷把罐子放回工作台，目光复杂地盯着它看了很久。

这件东西，外公四十多年前修复过。然后它被追回、被档案记录、又不知何时流出、辗转到了白行舟手里。

这中间究竟发生了什么？

"我需要时间仔细看。"她说，"所有十二件都看完，可能要两三天。"

"没问题。"程砚洲说，"这期间你可以随时来，我把备用钥匙给你。"

沈青瓷接过钥匙，握在手心。

她看了一眼那只青花大罐，心里有一种说不清的预感——这次鉴定，也许会揭开很多她从前不知道的事情。

也许是好事，也许不是。

但无论如何，她都要知道真相。

# 第八章 看走眼

沈青瓷是第二天上午去程砚洲工作室的。

她起得很早，比预定的时间提前了半小时到达。程砚洲还没来，她用备用钥匙开了门，独自上了二楼。

清晨的阳光从东面的大窗户斜斜照进来，落在工作台的灰色绒布上，像铺了一层淡金色的薄纱。沈青瓷站在窗边看了一会儿，心里莫名地安定下来。

这是她熟悉的环境——安静、专业、只有她和器物。

她戴上手套，从柜子里取出第一件东西。

是一只青花小盘，直径约十五厘米，盘心绘着折枝花卉，釉面匀净，青花发色沉稳。沈青瓷把它放在工作台上，打开专业照明灯，用放大镜仔细观察。

胎质细腻，釉面莹润，青花呈色偏蓝灰，有明显的"苏麻离青"特征——那种深入胎骨的铁锈斑，是永乐、宣德时期青花瓷的标志性特点。她翻过来看底部，款识规整，但风格偏晚。

"康熙仿宣德。"她在笔记本上写下判断，"青花发色到位，但胎体厚度比真宣德略薄，画工也更趋工整，少了宣德的自然随性。"

她放下小盘，又取出第二件。

是一只粉彩茶碗，外壁绘着缠枝莲纹，彩料明艳，画工细腻。沈青瓷看了很久，眉头微微皱起。

粉彩是清代创烧的品种，她对这个领域的熟悉程度不如青花。她拿起紫光灯照了照釉面，又用高倍放大镜观察彩料的状态，最终在笔记本上写下：

"乾隆官窑。彩料叠压有序，画工精细规整，符合乾隆时期的审美特征。但粉彩非本人强项，建议复核。"

---

程砚洲是九点半到的。

他推开二楼的门，看到沈青瓷正埋头工作，脚步不由放轻了些。

"来多久了？"

"一个多小时。"沈青瓷头也没抬，手里正拿着一只青花瓶转来转去。

程砚洲走到她旁边，看了眼工作台上已经摆开的几件器物，又看了眼她的笔记本。

"进度不错。"

"这批东西质量参差不齐。"沈青瓷放下青花瓶，"目前看了四件，一件康熙仿宣德，一件乾隆官窑，一件民国仿品，还有一件……"她指了指角落里那只青花瓶，"这件有问题，著录说是'清早期青花'，但我看更像是近三十年的高仿。"

程砚洲挑了挑眉："怎么看出来的？"

"釉面太亮了。"沈青瓷说，"真正的老瓷器，釉面会有一种'熟旧感'，是时间沉淀出来的包浆。这件的釉面光泽过于均匀，缺乏那种自然的润泽。而且你看这里——"她指着瓶底的一处，"做旧的痕迹。用酸蚀刻出来的'土沁'，和真正的窖藏痕迹质感不同。"

程砚洲凑近看了看，点了点头："我也怀疑这件有问题，但没你说得这么具体。"

"我从小跟着外公泡在修复室里，见过的高仿品不比真品少。"沈青瓷淡淡地说，"造假的手段再高明，只要仔细看，总能找到破绽。"

程砚洲看了她一眼，没再说什么，转身去泡茶。

---

上午的工作进行得很顺利。

沈青瓷一件一件地看过去，大部分都能给出明确判断。程砚洲在旁边记录她的意见，偶尔问一两个专业问题，两人配合得越来越默契。

十一点多，沈青瓷看到了那件青花缠枝牡丹纹大罐——外公修复过的那件。

她的动作慢了下来。

这是第二次近距离接触这件东西。她戴着手套，小心翼翼地把它从锦盒里取出来，放在工作台正中央。

阳光照在釉面上，青花发色浓艳而沉稳，那种永乐时期特有的"苏麻离青"蓝，带着历史的厚重感扑面而来。她转动罐身，目光从口沿一路扫到底部，每一处细节都看得极为仔细。

"确认是真品。"她说，"永乐官窑，品相完好，釉面包浆自然，画工流畅生动。底部的修复痕迹是……"

她停顿了一下，声音微微发涩："是我外公的手法。"

程砚洲站在旁边，安静地等着。

沈青瓷把罐子翻过来，指着底部一处极细微的痕迹："看到了吗？这里。这是外公独创的填补技法，用特殊配比的材料填充裂缝，再用与原釉成分相近的覆盖料打磨，几乎天衣无缝。他修过的东西，我都能认出来。"

"所以这件确实是故宫档案里记载的那件。"程砚洲说。

"是。"沈青瓷把罐子放回工作台，目光复杂，"问题是，它怎么会到了白行舟手里？档案显示1983年就追回了。"

"这就是我要查的。"程砚洲说，"你能确认它是真品，这很重要。接下来我会从流转渠道入手，看看中间有没有人动过手脚。"

沈青瓷点了点头，没有再说什么。

她把那只青花大罐小心地放回锦盒，却迟迟没有盖上盖子——眼睛一直盯着那釉面上的流光，像是透过它在看很久以前的什么东西。

---

午饭是程砚洲叫的外卖。

两人在一楼的会客区吃饭，沈青瓷要了一份简单的鸡丝面，程砚洲则是一份牛肉饭。

"上午看了八件，还剩四件。"程砚洲边吃边说，"下午能看完吗？"

"应该可以。"沈青瓷挑着面条，"剩下那四件，有一件我昨天翻图录的时候就觉得有点问题，想亲眼看看。"

"哪一件？"

"一只粉彩笔筒。"沈青瓷说，"图录上标注的是'清代粉彩山水笔筒'，但照片拍得不太清楚，看不出具体年份。"

程砚洲点了点头，没有多说。

吃完饭，两人休息了十几分钟，又上楼继续工作。

---

下午的第一件就是那只粉彩笔筒。

笔筒高约十二厘米，直径八厘米左右，外壁绘着一幅山水人物图，远山近水，几个高士在松下对弈，画面清雅秀丽。

沈青瓷把它放在工作台上，打开照明灯，开始仔细观察。

她先看整体造型——笔筒呈直筒形，上下口沿平齐，胎体厚度适中。再看釉面——白釉底子，粉彩绘画，彩料鲜艳但不刺眼。

她用放大镜观察画工。

山石的皴法有些生硬，人物的衣纹线条偏软，松针的画法也有点……不太对劲。

她皱起眉头，又拿起紫光灯照了照釉面。

在紫光灯下，釉面呈现出一种均匀的淡紫色荧光——这是现代釉料的特征。老瓷器的釉面在紫光灯下，荧光应该是不规则的、有深浅变化的，不会这么均匀。

她翻过来看底部。

底款是"大清乾隆年制"六字篆书款，字迹规整，但笔力不足，有些地方甚至显得犹豫迟疑。

沈青瓷放下放大镜，在笔记本上写下："现代仿品。理由：1.釉面在紫光灯下呈均匀荧光，疑为现代釉料；2.画工生硬，缺乏古人笔意；3.底款书写软弱，与乾隆官窑款识风格不符。"

她合上笔记本，把笔筒放到一边，拿起下一件。

---

一个多小时后，十二件东西全部看完。

沈青瓷站起身，活动了一下僵硬的脖子和肩膀。坐了一整天，后背有些酸痛。

"结果出来了。"她把笔记本递给程砚洲，"十二件东西，真品七件，高仿三件，还有两件需要进一步确认——那只乾隆粉彩碗我不太确定是官窑还是民窑精品，那只青花梅瓶的断代也有两种可能。"

程砚洲接过笔记本，仔细看了一遍。

看到粉彩笔筒那一栏时，他的目光停了一下。

"这件，你确定是现代仿品？"

"确定。"沈青瓷点头，"紫光灯测试的结果很明显，加上画工和款识的问题，综合判断是近二三十年的高仿。"

程砚洲没有说话，而是拿起手机打了一个电话。

"周老，是我。今天下午有空吗？……好，麻烦您过来一趟，有件东西想请您掌掌眼……对，就是那批……好，三点半见。"

他挂断电话，对沈青瓷说："我请了一位老先生过来复核一下。"

"复核？"沈青瓷微微一愣，"复核哪一件？"

"你刚才说的那只粉彩笔筒。"程砚洲说，语气平淡。

沈青瓷的眉头皱了起来。

她已经给出了明确判断，程砚洲却要请人复核——这是什么意思？不相信她的眼力？

"你觉得我的判断有问题？"她直接问。

程砚洲看了她一眼："不是不相信你，是这批东西来路特殊，我需要多一个人的意见做参照。而且……"

他顿了顿："周老是清代官窑瓷器的权威，尤其是中晚清这一块，国内顶尖。有些东西，他比我们看得准。"

沈青瓷沉默了几秒，点了点头。

她听出了程砚洲没有明说的意思——他对她在明代官窑方面的眼力是认可的，但清代中晚期这一块，她的经验可能不够。

这让她有点不舒服，但她知道程砚洲说的是事实。

"行。"她说，"那就等周老来看看。"

---

三点半，门铃响了。

程砚洲下楼去开门，沈青瓷站在二楼的窗边，看到一个身形瘦削的老人走进来。

老人看起来七十多岁，头发花白，穿着一件灰色的中式对襟外套，步伐稳健。他手里拎着一个旧皮包，看起来用了很多年，包面的皮质已经磨得发亮。

"周老，辛苦您跑一趟。"程砚洲的声音从楼下传来。

"不辛苦。"老人的声音沙哑但中气十足，"那批东西还在？让我看看。"

两人上了楼。

沈青瓷站在工作台旁边，看着周老走进来。老人目光锐利，进门先扫了一眼屋子里的布置，然后落在工作台上那排锦盒上。

"周老，这位是沈青瓷，做鉴定的。"程砚洲介绍道，"沈小姐，这位是周建德周老，业内公认的清代官窑研究权威。"

"周老好。"沈青瓷点头致意。

周建德看了她一眼，点了点头，没有多寒暄，直接走到工作台前坐下。

"东西呢？"

程砚洲把那只粉彩笔筒从锦盒里取出来，放在老人面前。

周建德从旧皮包里取出自己的放大镜——一只用了不知道多少年的老式折叠放大镜，镜片边缘的铜框已经被摩挲得发亮。

他戴上老花镜，把笔筒拿起来，凑近灯光仔细看。

沈青瓷站在旁边，目光跟着老人的动作移动。

周建德先看了整体造型，又看釉面，然后把笔筒翻过来看底款。他看得很慢，每一处都要停留很长时间，还时不时调整笔筒的角度，让光线从不同方向照射。

大约十分钟后，他放下放大镜，问程砚洲："小沈给的结论是什么？"

程砚洲看了沈青瓷一眼，把笔记本递过去。

周建德看了一眼，嘴角动了动，没有说话。

他又拿起笔筒，这次重点看的是外壁的画工。他用放大镜一寸一寸地看过去，有时候还用手指轻轻触摸釉面，感受那细微的凹凸质感。

又过了大约五分钟，他放下笔筒，把放大镜收回皮包里。

"小姑娘，"他转头看向沈青瓷，"这件东西，你怎么断的现代仿品？"

沈青瓷上前一步，指着笔筒说："三点。第一，紫光灯测试，釉面呈均匀荧光，这是现代釉料的特征。第二，画工问题，山石皴法生硬，人物线条软弱，缺乏古人笔意。第三，底款书写犹豫迟疑，与乾隆官窑款识风格不符。"

周建德听完，沉默了一会儿。

然后他开口了，声音不紧不慢，却带着一种不容置疑的笃定：

"小姑娘，你说的三点，有两点是对的，但结论错了。"

沈青瓷的脸色微微变了。

"哪里错了？"

周建德拿起笔筒，指着外壁的画工："你说画工生硬，这个判断没问题。光绪年间的官窑瓷器，画工确实比不上乾隆、嘉庆时期，有一种'力不从心'的感觉——不是画工不努力，而是整体水平在下滑，这是晚清瓷器的通病。"

他又指着底款："你说底款软弱，这个也对。光绪官窑的款识书写，确实不如早期那么刚劲有力，有些显得畏畏缩缩。但这恰恰说明——"

他抬起头，目光锐利地看着沈青瓷：

"这件东西不是乾隆的，是光绪的。你用乾隆的标准去套，当然觉得哪里都不对。"

沈青瓷的心猛地沉了下去。

"光绪？"

"对。"周建德把笔筒翻过来，指着底款，"你看这个款识的写法——'大清乾隆年制'六字，是光绪年间仿乾隆款的典型风格。光绪朝做了很多仿乾隆的东西，款识也照着乾隆的写，但写得没那么自信，一看就是临摹。"

他又把笔筒翻回来，指着釉面："至于紫光灯测试，你说釉面荧光均匀，这个判断也有问题。光绪官窑的釉料配方和乾隆时期略有不同，在紫光灯下的反应也不一样。不能用乾隆的标准来套所有清代瓷器。"

沈青瓷的脸彻底白了。

她站在原地，手指攥紧了笔记本的边缘，指节泛白。

"所以……这件是真品？"她的声音有些发涩。

"光绪官窑，仿乾隆粉彩笔筒。"周建德点头，"有一定收藏价值，但和真正的乾隆官窑没法比。市场上这种东西不少，内行人一眼就能看出来——但你要是对晚清瓷器不熟悉，容易误判。"

他把笔筒放回工作台，看着沈青瓷，语气变得和缓了一些：

"小姑娘，你明代的东西看得不错，我刚才翻了你的笔记，那几件永乐、宣德的判断都很准。但清代中晚期这一块，你经验还不够。这不是丢人的事，术业有专攻，没人能样样精通。"

沈青瓷咬着嘴唇，点了点头。

"谢谢周老指教。"她说。

她的声音平静，但攥紧笔记本的手在微微发抖。

---

周建德走的时候，程砚洲送他下楼。

沈青瓷一个人留在二楼，站在窗边，看着外面渐渐暗下来的天色。

她的脑子里反复回放着刚才的场景——周建德拿起笔筒、指出她的错误、最后那句"经验还不够"。

每回想一次，她的脸就烫一分。

她是沈家修复一脉的传人，从小跟着外公学手艺，自认为在瓷器鉴定方面有真本事。来给程砚洲帮忙，她本以为能展示自己的专业能力，没想到……

一件光绪仿品都没看出来。

她闭上眼睛，深吸一口气。

脑海里突然浮现出婆婆——不，应该叫"前婆婆"了——顾母的那张脸。顾母曾经当着一屋子客人的面说："青瓷是个会看东西的小姑娘，小打小闹还行，真正的大场面上不了台。"

那时候她气得浑身发抖，觉得顾母是故意贬低她。

可是现在……

她真的有那么强吗？一件光绪官窑都能看走眼，她凭什么觉得自己能在古董行站稳脚跟？

"沈小姐。"程砚洲的声音从身后传来。

沈青瓷睁开眼睛，转过身。

程砚洲不知道什么时候上来了，正站在工作台旁边，手里端着两杯茶。

"喝点茶。"他把其中一杯递给她。

沈青瓷接过来，没有喝，只是握在手里。

"我看走眼了。"她说。

"是。"程砚洲点头，没有安慰她。

"我应该……"沈青瓷的声音有些涩，"我应该在给结论之前说一句'粉彩不是我的强项'。我太自信了。"

"你确实太自信了。"程砚洲说，"但看走眼是正常的。这行里没人敢说自己什么都懂，连周老都有打眼的时候。"

沈青瓷抬起头看他。

程砚洲的表情很平静，既没有嘲讽，也没有刻意的安慰。他只是在陈述一个事实。

"重要的是知道自己不懂什么。"他说，"你明代官窑看得很准，这是你的长处。清代中晚期经验不足，这是你的短板。知道短板在哪里，才能避免下次再犯同样的错误。"

沈青瓷沉默了一会儿，点了点头。

"程先生，"她说，"谢谢你今天请周老来。如果不是他指出来，我可能到现在还以为那件是假的。"

程砚洲看了她一眼，嘴角微微扬起。

"你比很多老手强。"他说。

"什么意思？"

"他们看走眼了不会承认，你会。"程砚洲说，"这行里，敢认错的人比会看东西的人更稀缺。"

沈青瓷愣了一下，然后低下头，轻轻笑了。

笑容里带着一点苦涩，但更多的是释然。

"谢谢。"她说。

---

离开工作室的时候，天已经黑了。

程砚洲送她到门口，说："那批东西的鉴定报告，你整理好发给我就行，那件粉彩笔筒的结论改一下。"

"好。"沈青瓷点头。

她走了两步，又停下来，转头看着程砚洲。

"程先生，"她说，"我想找我外公借一些晚清官窑的资料，系统补一补这方面的知识。以后如果有这类东西需要鉴定，我希望能派上用场，而不是……"

她没有说完，但程砚洲明白她的意思。

"好。"他说，"等你准备好了告诉我。"

沈青瓷点了点头，转身走向地铁站的方向。

夜风吹在脸上，带着初秋的凉意。她边走边想今天的事情——那只粉彩笔筒、周老的指教、程砚洲的话。

"知道自己不懂什么。"

这句话在她脑海里反复回响。

她一直以为自己够专业，够有眼力。但今天才意识到，她的专业有边界，她的眼力有盲区。明代官窑是她的主场，清代中晚期却是她的软肋。

想要在古董行站稳脚跟，光有一技之长是不够的。她得知道自己的短板，然后一点一点补上去。

她想起外公说过的话："这行的水，比你想的深。"

原来这句话，说的不只是人心险恶、尔虞我诈，还有……专业本身的浩瀚无边。

沈青瓷在地铁口站了一会儿，然后掏出手机，给外公发了一条消息：

"外公，明天我去看您，想借几本关于清代官窑的资料。"

消息发出去没多久，外公就回了：

"好。"

沈青瓷看着那个简短的回复，嘴角微微弯起。

她走进地铁站，消失在人流里。

---

地铁摇摇晃晃地向城西开去。

沈青瓷靠在车厢的扶手杆上，闭着眼睛，回想今天在工作室的经历。

那只青花缠枝牡丹纹大罐的釉面在脑海里浮现——那是外公四十年前修复过的东西，如今却辗转到了白行舟手里。中间究竟发生了什么？有多少人经过了那件东西？有多少秘密藏在那些流转的痕迹里？

还有那只粉彩笔筒……

她睁开眼睛，苦笑了一下。

光绪官窑，被她看成了现代仿品。

如果今天不是程砚洲请了周老来，如果她把这个错误的判断写进正式报告里，那会是多大的笑话？一个号称"专业鉴定师"的人，连光绪仿乾隆都认不出来？

她想起婆婆的嘴脸，想起那些"上不了台面"的讥讽，忍不住攥紧了拳头。

但随即，她又松开了。

看走眼就是看走眼，没什么好狡辩的。她唯一能做的，就是承认错误、弥补短板，让自己下次不要再犯。

"知耻而后勇"——外公教过她这四个字。

地铁报站的声音响起，下一站就是她要下的地方。

沈青瓷站直身子，理了理衣服，走向车门。

明天去外公那里借资料，然后回来好好补课。清代中晚期的东西，她要从头学起，直到再也不会看走眼为止。

她走出地铁站，夜风迎面吹来，带着几分寒意。

但她的步子很稳，目光很坚定。

这条路还很长，但她已经知道该往哪个方向走了。

# 第九章 圈套

三天后，沈青瓷正在外公家翻看晚清官窑的资料，吴敏的微信消息跳了出来。

"有事，见面聊。"

沈青瓷看了眼时间，下午两点。她把手里那本《清代瓷器鉴赏》合上，跟外公打了声招呼，出门坐地铁去了城西。

吴敏租住的地方在一栋老旧的筒子楼里，楼道的灯管坏了一半，忽明忽暗的。沈青瓷摸黑爬到三楼，敲了敲门。

门开了条缝，吴敏的脸从门缝里探出来，左右张望了一下，才把门拉开。

"进来。"

沈青瓷走进去，发现吴敏的屋子比上次来还要乱。茶几上堆着外卖盒子，沙发上扔着几件没叠的衣服，空气里弥漫着一股闷热的气息。

"什么事？"沈青瓷直接问。

吴敏关上门，反锁，然后走到窗边往外看了一眼，才转过身来。

"钱永年那边放出消息了。"她压低声音说，"有人要私下出手一件永乐青花大罐，开价八百万。"

沈青瓷的眉头微微皱起。

八百万——永乐官窑青花大罐的市场价少说也要八千万，这个价格低得离谱。

"什么来路？"她问。

"说是卖家急着用钱，不走拍卖渠道，要求现金一次性付清。"吴敏说，"钱老板让我打听有没有人感兴趣，我第一个想到的就是你。"

沈青瓷没有说话，脑子里快速转动着。

永乐青花大罐。八百万。私下交易。不走拍卖渠道。

这几个关键词组合在一起，怎么看都像是一个圈套。

她想起自己那只青花罐——钱永年应该已经从各种渠道听到了风声，知道她手里有件好东西。现在突然放出这么一个消息，是巧合还是有意为之？

"卖家是谁？"沈青瓷问。

"不知道。"吴敏摇头，"钱老板只说是'圈里的朋友'，没透露具体是谁。"

"东西放在哪里？"

"古玩城那边，有个店铺专门看货。"吴敏说完，犹豫了一下，又补充道，"青瓷，我觉得这事儿有点怪。你手里那件……"

她没有说完，但意思沈青瓷听懂了。

"你觉得是冲着我来的？"

吴敏点了点头，声音更低了："我在钱老板身边待了这么久，从没见过他主动放这种消息。永乐官窑的东西，真要卖，不愁找不到买家，何必私下放风？"

沈青瓷沉默了一会儿。

吴敏说得对。钱永年是老江湖，不会做亏本买卖。他放这个消息，要么是想引蛇出洞，要么是想设局——不管是哪种，目标很可能就是她。

"那件东西，你见过吗？"沈青瓷问。

"没有。"吴敏说，"只听说了，没亲眼看到。"

沈青瓷点了点头，站起身来。

"我去看看。"

吴敏愣住了："你要去？那不是往陷阱里跳吗？"

"我去看看是什么货色。"沈青瓷说，"放心，我不会买。"

她走到门口，又停下来，回头看着吴敏。

"你继续打听，有什么消息随时告诉我。"

吴敏点了点头，眼神里带着一丝担忧。

---

古玩城在城东，一栋五层的仿古建筑，里面密密麻麻地挤着上百家店铺。

沈青瓷下午四点到的时候，正是一天里生意最冷清的时段。大部分店铺都半掩着门，店主要么在喝茶，要么在打瞌睡。

她按照吴敏给的地址，穿过主通道，拐进一条偏僻的侧廊。

这边的店铺明显冷清许多，招牌陈旧，门面狭小，看起来都不是什么大买卖。沈青瓷走到廊道尽头，看到了那块写着"雅韵斋"的招牌。

店门紧闭，但门缝里透出微弱的灯光。

她深吸一口气，推门走了进去。

店里空间不大，四面墙上挂满了字画，博古架上摆着些杂七杂八的瓷器和玉件，看起来都是些中低档的货色。柜台后面坐着一个四十多岁的男人，穿着灰色的唐装，正低头看手机。

听到门响，男人抬起头来。

"找什么？"

"听说你们这儿有件永乐青花大罐要出手？"沈青瓷开门见山。

男人的眼神微微变了，上下打量了她一番。

"你是……"

"姓沈。"沈青瓷说，"朋友介绍来的。"

男人放下手机，站起身来。他的目光在沈青瓷脸上停留了几秒，似乎在判断什么，然后点了点头。

"跟我来。"

他绕过柜台，走向店铺深处。沈青瓷跟在后面，注意到店里还有一个小门，门后是一条窄窄的楼梯。

男人上了楼梯，推开二楼的门。

二楼是个小房间，灯光昏暗，只有一张方桌和几把椅子。桌上放着一个锦盒，看起来有些年头了。

"东西在这儿。"男人指了指锦盒，"你自己看。"

沈青瓷走到桌前，打开锦盒。

一只青花大罐静静地躺在红色的绸缎里。

她的呼吸微微一滞。

从外观看，这只罐子确实像是永乐时期的东西——造型端庄，比例协调，通体绘着缠枝莲纹，青花发色浓艳而沉稳。

沈青瓷从包里取出放大镜和手套，戴好手套，把罐子小心翼翼地从锦盒里取出来。

她先看整体。

罐口微撇，短颈丰肩，肩下渐收，圈足外撇。造型确实是永乐时期的风格，没有明显的破绽。

再看釉面。

釉色白中泛青，釉质肥厚莹润，有一种温润如玉的质感。她用放大镜仔细观察，釉面下隐约可见细密的气泡，排列不太规则——这是老瓷器的特征之一。

然后是青花。

青花发色深沉凝重，略带蓝紫色调，有明显的"苏麻离青"特征。她把放大镜凑近，寻找那些标志性的铁锈斑——苏麻离青因为含铁量高，烧制后会在青花表面形成深黑色的结晶斑点，俗称"铁锈斑"，是鉴定永乐、宣德青花瓷的重要依据。

铁锈斑确实有。

沈青瓷的眉头皱了起来。

她转动罐身，让光线从不同角度照射在青花上。铁锈斑的分布……

太均匀了。

她心里一动，把罐子放回桌上，又从包里取出一支紫光灯笔，对着釉面照了照。

在紫光灯下，釉面呈现出淡淡的紫色荧光——颜色分布均匀，没有明显的深浅变化。

这不对。

真正的永乐官窑，釉面在紫光灯下的荧光应该是不规则的，因为六百年的岁月会让釉面发生微妙的变化，每一处的老化程度都不同。而这只罐子的荧光太过均匀，像是……新釉。

沈青瓷把紫光灯收起来，又拿起放大镜，重新看那些铁锈斑。

她这次看得更仔细了。

铁锈斑的分布确实太均匀。真正的苏麻离青铁锈斑，是在烧制过程中自然形成的，分布应该是随机的、不规则的，有的地方密集，有的地方稀疏，有的大有的小。但这只罐子上的铁锈斑，虽然形态逼真，但分布过于"完美"——每一处斑点之间的距离几乎相等，大小也差不多。

这是人工做旧的痕迹。

沈青瓷心里有了判断，但面上不动声色。

她把罐子翻过来，看底部。

底部无款，圈足处理得很干净，露胎处呈现出火石红色。她用手指轻轻触摸圈足的边缘，感受那细微的质感——

手感不对。

永乐官窑的圈足，因为年代久远，边缘应该有一种"滑熟"的感觉，是岁月磨砺出来的温润。但这只罐子的圈足边缘，虽然做了打磨处理，但还是有一丝"涩"的感觉，像是……时间不够。

她把罐子放回锦盒，摘下手套。

男人一直站在旁边看着她，这时候开口了：

"怎么样？"

沈青瓷没有直接回答，而是问："这件东西什么来路？"

"卖家是圈里的朋友，急着用钱，不方便透露身份。"男人说，"价格你也知道了，八百万，一口价。"

"能少吗？"

"不能。"男人摇头，"这价格已经是朋友价了，真要送拍卖会，翻十倍都不止。"

沈青瓷沉默了一会儿，做出一副犹豫的样子。

"我再考虑考虑。"她说。

男人的眉头皱了皱："这价格，过了这村就没这店了。你要是诚心想买，我可以帮你跟卖家说说。"

"东西是好东西。"沈青瓷说，"但八百万不是小数目，我得回去筹钱。"

男人看了她一眼，似乎在判断她是不是真心想买。

"行。"他最终说，"我给你三天时间。三天之后，我就找别人了。"

"好。"沈青瓷点头，"三天后我给你答复。"

她转身下楼，穿过店铺，推门走出去。

夏末的阳光刺得她眯起眼睛。

她深吸一口气，快步走向古玩城的出口。

---

出了古玩城，沈青瓷没有直接回家，而是在附近的一家咖啡店坐了下来。

她要了一杯美式，坐在靠窗的位置，盯着窗外发呆。

那只青花罐的影像在脑海里反复浮现——造型、釉面、青花、铁锈斑……

是假的。

她有百分之九十的把握。

那些铁锈斑的分布太均匀了，不像是自然形成的，更像是人工点缀上去的。真正的苏麻离青铁锈斑，因为是釉料中的铁元素在高温下自然析出结晶而成，分布必然是随机的、无序的。但那只罐子上的铁锈斑，却呈现出一种隐约的"秩序感"——这是高仿的通病。

做高仿的人，往往太追求"完美"，反而暴露了破绽。

真品的美，恰恰在于它的"不完美"——那些随机的铁锈斑、不规则的气泡、深浅不一的釉色，都是时间留下的印记，是任何人工手段都无法复制的。

沈青瓷端起咖啡喝了一口，苦涩的味道让她清醒了一些。

现在的问题是：钱永年设这个局，目的是什么？

她在脑子里推演了几种可能。

第一种，让她上当买假货。

如果她真的花八百万买了那只假罐子，不但亏了钱，还会落下"看走眼"的笑柄。以钱永年的行事风格，这个消息肯定会被散播出去，她在圈子里的名声就彻底毁了。

但这种可能性不大。

钱永年应该知道她没有八百万的现金——她的全部家当也就九万多块，买一只青花罐？做梦呢。

第二种，诱她私下交易，给顾家把柄。

顾家的官司还没完全了结，冯律师说过，诉讼期间最好不要有大额交易，否则会被对方抓住"转移财产"的把柄。如果她真的去买那只罐子，不管买不买成，只要有交易的意向和行为，都可能被顾家律师利用。

这种可能性比较大。

钱永年和顾家有没有联系？她不知道。但顾母既然能在古玩圈子里封锁她，就说明她在这个圈子里有人脉。钱永年会不会是顾母的棋子？

第三种，试探她的鉴定水平。

如果她去看了货却没买，钱永年就能知道她看出了这是假货。这样一来，她对永乐瓷器的鉴定能力就暴露了——以后钱永年再设局，就会更加小心，更加难以防范。

这种可能性也有。

钱永年是老江湖，每一步都在算计。他放出这个消息，未必只有一个目的，很可能是一石多鸟。

沈青瓷放下咖啡杯，揉了揉太阳穴。

不管钱永年的真实目的是什么，她现在最好的策略是：不动声色。

让钱永年以为她没有看穿那只罐子是假的。

如果她表现出"很想买但买不起"的样子，钱永年就会以为她被骗住了，对她的鉴定水平产生误判。这样一来，她在暗处，钱永年在明处，以后对付他会更容易。

但如果她直接揭穿那是假货，钱永年就会知道她对永乐瓷器很在行，下次设局会更加谨慎。

沈青瓷想了想，掏出手机，给吴敏发了条消息：

"我去看过了，东西确实好，但我没钱买。你跟钱老板说，我很心动，但拿不出八百万。"

消息发出去没多久，吴敏回了：

"收到。我帮你转达。"

沈青瓷看着那条回复，嘴角微微扬起。

演戏要演全套。

她又翻了翻相册，找到几天前拍的外公那些晚清官窑资料的照片，发到一个古董爱好者群里，说："最近在补课，有没有大佬指点一下光绪仿乾隆的鉴定要点？"

发完之后，她把手机收起来，叫服务员结账。

钱永年肯定会通过各种渠道监视她的动向。让他看到她在"补课"，看到她对那只罐子"很心动但买不起"——这些信息加在一起，会让他觉得她只是一个有点眼力但经验不足的新手，还没有真正看穿他的局。

这正是她想要的效果。

---

从咖啡店出来，天已经擦黑了。

沈青瓷走向地铁站，边走边想。

钱永年的这个局，她暂时避开了。但这只是第一回合，后面肯定还有更多的招数等着她。

她想起上次在程砚洲工作室看到的那件青花大盘——外公修复过的东西，辗转到了白行舟手里。那批瓷器的来源有问题，和钱永年、徐鸣的造假链有关联。

程砚洲在查白行舟，她在对付钱永年，两个人查的其实是同一伙人。

或许……可以联手？

她掏出手机，犹豫了一下，还是给程砚洲发了条消息：

"钱永年最近在卖一件'永乐青花大罐'，开价八百万。我去看过了，是高仿。"

消息发出去后，她把手机揣回口袋，走进地铁站。

---

晚上九点多，沈青瓷刚洗完澡，手机响了。

是程砚洲的电话。

"你说的那件'永乐青花'，在哪里看的？"他开门见山。

"古玩城那边，一个叫'雅韵斋'的店铺。"沈青瓷擦着头发说，"二楼有个小房间，专门看货的。"

"你怎么断定是高仿？"

"铁锈斑的分布太均匀了。"沈青瓷说，"真正的苏麻离青铁锈斑是自然形成的，分布应该是随机的。那件东西上的铁锈斑虽然形态逼真，但排列过于整齐，是人工点缀的。"

电话那头沉默了几秒。

"你能确定？"

"有百分之九十的把握。"沈青瓷说，"剩下百分之十……我需要更多时间和工具才能完全确认。但以钱永年放出这个消息的方式来看，大概率是假的。他不会好心卖真东西给别人。"

程砚洲又沉默了一会儿，然后说："我让人去查查这件东西的来路。如果真是徐鸣做的，应该能找到痕迹。"

"你怀疑是徐鸣的作品？"

"你之前跟吴敏打听的消息——钱永年让徐鸣做一件永乐青花。时间上对得上。"

沈青瓷想起来了。那天吴敏确实透露过这个信息：钱永年在找徐鸣"做一件永乐青花"。当时她还没放在心上，现在看来，那就是今天这只"永乐青花大罐"的来历。

"如果能查到徐鸣的进货记录或者制作记录，就能坐实这件是假货。"她说。

"我有渠道。"程砚洲说，"你那边继续盯着钱永年，有什么消息随时告诉我。"

"好。"

电话挂断了。

沈青瓷放下手机，坐在床边发呆。

她和程砚洲，本来是两条平行线，因为白行舟和钱永年的关系，渐渐交织到了一起。

他查白行舟，是为了追回祖父的旧藏。

她对付钱永年，是为了在古董行站稳脚跟。

目标不同，但敌人是同一拨人。

这大概就是所谓的"敌人的敌人是朋友"吧。

沈青瓷苦笑了一下，躺倒在床上。

天花板上的灯光刺得她眯起眼睛。她伸手关掉灯，黑暗中只剩下窗外透进来的微弱光芒。

今天在雅韵斋，她演了一场戏。

让钱永年以为她被骗住了，以为她只是个"有点眼力但经验不足"的新手。

但她知道，这只是暂时的。

钱永年迟早会发现她的真实水平，到时候，才是真正的战斗。

她闭上眼睛，脑海里浮现出那只青花罐的影像——那些排列过于整齐的铁锈斑，那层均匀得不自然的釉面荧光。

高仿。

这个词在她嘴边滚了滚，带着一丝讽刺的意味。

钱永年以为她会上当，以为她看不出真假。

他错了。

她是沈家修复一脉的传人，从小跟着外公在修复室里长大。真品、高仿、粗仿——她见过的东西，比大多数人一辈子见过的都多。

想用一件高仿来试探她？

太小看人了。

沈青瓷翻了个身，把脸埋进枕头里。

明天还要去外公那里继续补课。晚清官窑的知识，她确实还有欠缺——上次在程砚洲工作室的那次"看走眼"，给她敲响了警钟。

但明代官窑，尤其是永乐、宣德这一块，是她的主场。

钱永年想在这里设局？

让他来。

她等着。

# 第十章 造假链

三天后，程砚洲发来消息，让沈青瓷去他的工作室。

"查到东西了。"消息很简短，但沈青瓷能感觉到其中的分量。

她收起手机，跟外公打了声招呼，出门打车。

程砚洲的工作室在城西一栋老式写字楼里，和古玩城的喧嚣不同，这里安静得几乎能听到空调的嗡鸣声。沈青瓷敲门进去，发现程砚洲正坐在一张长桌前，桌上铺满了文件和照片。

"来，看这个。"他招呼她坐下，把一叠资料推到她面前。

沈青瓷低头看去。

最上面是一本旧书的照片——封面泛黄，书脊破损，看起来有些年头了。书名是《雅庐藏瓷手记》，落款是"民国十七年 吴县钱氏"。

"这本书是三年前被'发现'的。"程砚洲说，"据说是从苏州一户老宅里流出来的，里面记录了一位民国收藏家的藏品清单，包括瓷器、字画、玉器，有照片、有尺寸、有来源。"

他翻到下一页，是书中的内页照片。沈青瓷看到一件青花缠枝牡丹纹大罐的图片，旁边用毛笔小楷写着"永乐官窑青花缠枝牡丹纹大罐，高三十二公分，径四十公分，口沿小磕，购自粤商陈某，价银三百两……"

"看起来很专业。"沈青瓷说。

"问题就在这里。"程砚洲指了指图片下方的一行小字，"你看这个印章。"

沈青瓷凑近去看。印章很小，有些模糊，但依稀能辨认出"雅庐珍藏"四个字。

"这个印章有问题？"

"印泥。"程砚洲说，"这本书号称是民国十七年的，也就是1928年。但根据我找专业人士做的分析，印泥的成分里含有一种1960年代以后才开始使用的添加剂。"

沈青瓷瞳孔微缩。

"这本书是伪造的。"

"对。"程砚洲点头，"整本书都是近几年做的，专门用来给一批瓷器'洗白'来源。"

他翻出另一叠资料，是几张表格。

"这是我这三个月查到的东西。这本《雅庐藏瓷手记》里一共记录了四十七件瓷器，其中有二十三件已经流入市场，被不同的藏家买走。而这些瓷器有一个共同点——"

他停顿了一下，目光直视沈青瓷。

"它们都经过一个人的手：钱永年。"

沈青瓷的呼吸微微一滞。

钱永年。又是钱永年。

"这本书的'发现者'是谁？"她问。

"名义上是一个叫徐鸣的古董商，景德镇人，家族世代做仿古瓷生意。"程砚洲说，"但实际上，这本书从一开始就是钱永年策划的。徐鸣只是执行者。"

他把另一张照片推过来。

照片上是一个五十多岁的男人，瘦削精干，戴着一副金丝眼镜，正站在一个瓷窑前。

"徐鸣。"程砚洲说，"他父亲徐守正是景德镇有名的仿古瓷大师，手艺精湛，但一辈子只做'仿古'，不做'造假'——凡是他手里出去的东西，都会在底部刻一个隐蔽的记号，表明这是仿品。但徐鸣不一样。"

他的语气变得冷厉。

"徐鸣把父亲的手艺用来造假牟利。他负责做高仿，钱永年负责伪造来源和著录，然后找人出货。三个人分工明确，形成了一条完整的造假产业链。"

"三个人？"沈青瓷注意到这个词，"还有谁？"

程砚洲沉默了一会儿。

"白行舟。"

沈青瓷的眼睛微微睁大。

白行舟——就是那个拥有程砚洲祖父旧藏的"收藏家"。

"白行舟在这条链里是什么角色？"她问。

"洗白出货。"程砚洲的声音很冷，"他有干净的身份——企业家、收藏家、慈善人士。他买下这些有'著录'的瓷器，养几年，然后再以自己的名义送拍卖会，或者私下转让给其他藏家。因为经过了'白行舟收藏'这一环，东西的来源就被洗白了，买家不会怀疑。"

沈青瓷听得心惊。

这是一条多么精密的产业链——制假、造源、洗白、出货，每一个环节都有专人负责，每一步都设计得滴水不漏。

"所以白行舟手里那批瓷器……"

"有一部分是真品，但也有相当一部分是徐鸣做的高仿。"程砚洲说，"包括那件青花缠枝牡丹纹大罐——你外公修复过的那件。"

沈青瓷的心猛地一沉。

那件青花大盘，是她在程砚洲工作室看到的。当时她就觉得不对劲——造型、画工、釉色都没问题，但有一种说不出来的"新"。现在想来，那应该就是徐鸣的仿品。

"等等。"她突然想到什么，"那件大盘上有我外公的修复痕迹，是真的修复痕迹，不是伪造的。这说明什么？"

程砚洲看了她一眼，点头道："你想到点子上了。"

他从桌上拿起一份档案。

"这是我托人查到的。那件大盘确实被你外公修复过——1970年代，故宫库房里有一批受损瓷器需要修复，其中就包括这件明代青花缠枝牡丹纹大盘。你外公当时是修复组的组长，亲手修复了这件东西。"

"但后来呢？"

"1978年，这件大盘和另外十几件瓷器一起，被调拨给外贸部门，用来换外汇。"程砚洲说，"官方记录到此为止。但根据我的调查，这批瓷器后来被一个香港商人买走，辗转几手之后，下落不明。"

"直到出现在白行舟手里。"沈青瓷接过话。

"对。"程砚洲点头，"但问题是，出现在白行舟手里的这件大盘，和1978年调拨出去的那件，不是同一件东西。"

沈青瓷愣住了。

"你的意思是……"

"真品不知去向，白行舟手里的是仿品。"程砚洲说，"但徐鸣为了以假乱真，想办法把你外公当年修复时留下的痕迹复制到了仿品上。他甚至找到了你外公用过的修复材料配方——1970年代故宫修复组用的材料是特制的，和现在市面上的不一样。徐鸣花了很大功夫才仿制出来。"

沈青瓷只觉得背脊发凉。

这已经不是普通的造假了。这是有组织、有预谋、处心积虑的犯罪。

"我外公知道这件事吗？"她问。

"不清楚。"程砚洲说，"但我觉得你应该问问他。他修复过的东西，他最清楚。"

---

从程砚洲工作室出来，沈青瓷直接打车去了外公家。

外公住在城南的一个老小区里，房子不大，但收拾得干净整洁。客厅里摆着一张老式的红木方桌，桌上放着几本旧书和一副放大镜——这是外公平时看书用的。

沈青瓷推门进去，看到外公正坐在窗边的藤椅上，手里拿着一本《中国陶瓷史》，戴着老花镜慢慢翻看。

"外公。"她喊了一声。

沈外公抬起头，看到是她，脸上露出慈祥的笑容。

"青瓷来了？吃饭了没有？"

"吃过了。"沈青瓷在外公对面坐下，从包里掏出手机，翻到那张青花大盘的照片，"外公，我想问您一件事。"

她把手机递过去。

"您看看这件东西，认识吗？"

沈外公接过手机，眯着眼睛看了一会儿。

他的表情渐渐变了。

"这件东西……你从哪里看到的？"他的声音有些发紧。

"一个朋友那里。"沈青瓷说，"外公，这件东西您修复过，对吗？"

沈外公沉默了很久。

他把手机放下，摘下老花镜，揉了揉眼睛。

"修复过。"他终于开口，"1973年，故宫库房清点时发现的。当时这件大盘碎成了三块，我花了两个月才修好。"

他的目光变得有些遥远，像是在回忆几十年前的事情。

"那件大盘是明代中期的东西，景德镇官窑出品。缠枝牡丹纹画得极好，一看就是当时最好的画师画的。我修复的时候特别小心，生怕破坏了原来的画工。"

"后来呢？"

"后来……"沈外公叹了口气，"1978年，上面说要换外汇，从库房里调了一批东西出去。这件大盘也在里面。我当时还心疼了好久——那么好的东西，就这么流出去了。"

他看向沈青瓷，眼神里带着疑惑。

"你怎么会看到这件东西？它不是早就出国了吗？"

沈青瓷犹豫了一下，还是决定把实话告诉外公。

"外公，我看到的这件，可能不是真品。"

沈外公愣住了。

"什么意思？"

"有人仿了一件，把您当年修复的痕迹也仿上去了。"沈青瓷说，"现在这件仿品在一个叫白行舟的收藏家手里，他们还给它伪造了一本著录，说是民国某个藏家的旧藏。"

沈外公的脸色变得很难看。

"仿我的修复痕迹？"他的声音带着一丝颤抖，"谁干的？"

"一个叫徐鸣的人，景德镇的。"沈青瓷说，"您认识吗？"

沈外公听到这个名字，眉头皱得更紧了。

"徐鸣……徐守正的儿子？"

"您认识他父亲？"

"认识。"沈外公点头，"徐守正是景德镇的老手艺人，做仿古瓷的一把好手。我年轻的时候去景德镇学习，和他打过交道。他这个人手艺好，人品也正，做出来的仿古瓷都会做记号，从不骗人。"

他叹了口气。

"但他儿子……我听说不太成器，走了歪路。没想到走得这么歪。"

沈外公站起身，走到书柜前，从里面翻出一个旧本子。

本子的封面已经发黄，边角卷起，看起来有几十年的历史了。

"这是我当年的修复记录。"他翻开本子，找到其中一页，"你看，这是那件大盘的修复记录。我写了修复的日期、用的材料、修复的部位，还画了图。"

沈青瓷接过本子，仔细看着。

上面的字迹工整清晰，画的图也很详细。她看到外公标注的修复部位——盘口沿有一道裂缝，盘底有两处碎片接合处，还有一处釉面剥落需要补釉的地方。

"如果有人要仿我的修复痕迹，"沈外公指着图说，"他必须知道我修复的具体位置和方法。这些信息，除了我自己，只有当时的修复组同事知道。"

他的表情变得严肃起来。

"青瓷，你说的那个徐鸣，他是怎么知道这些的？"

沈青瓷摇了摇头。

"我不知道。但我会去查。"

她把本子还给外公，又问道："外公，您对徐家的仿古瓷了解多少？他们有什么特点？"

沈外公想了想，说："徐家的仿古瓷，最大的特点是'胎'。他们家用的瓷土是景德镇高岭山上的老矿，和现在市面上的不一样。而且徐守正有一套独特的配方，能让瓷胎的质感非常接近明清时期的老瓷。"

"那有没有什么破绽？"

"有。"沈外公说，"徐家的东西，再好也有一个问题——火气。"

"火气？"

"新瓷器烧出来都有火气，就是那种'新'的感觉。老瓷器经过几百年的岁月，火气早就退尽了，摸起来温润如玉。徐家的仿古瓷虽然做得好，但火气退不干净，仔细上手还是能感觉到。"

沈外公看着沈青瓷，眼神里带着担忧。

"青瓷，你是不是在查什么事？"

"嗯。"沈青瓷点头，"跟钱永年有关。"

听到这个名字，沈外公的脸色更差了。

"钱永年那个人，我听说过。"他压低声音说，"古董圈里的老油条，专门吃两头——从小藏家手里低价收东西，再高价卖给大藏家。手段不干净，但从来没有被抓住过把柄。"

他握住沈青瓷的手。

"你要小心。这种人，不好对付。"

"我知道。"沈青瓷说，"外公，您放心，我有分寸。"

---

从外公家出来，天已经黑了。

沈青瓷走在街上，脑子里转着无数念头。

钱永年、徐鸣、白行舟——三个人，一条产业链。制假、造源、洗白，每一步都精心设计，环环相扣。

而她和程砚洲，一个在对付钱永年，一个在追查白行舟，从两个方向逼近同一个核心。

她想起外公说的话——徐家的仿古瓷有火气，退不干净。

这是一个破绽。

如果能证明白行舟手里那批瓷器有问题，证明它们是徐鸣做的仿品而不是真品，那整条产业链就会曝光。

但问题是，怎么证明？

白行舟不会让她接近那些瓷器。钱永年更不会给她机会。

她需要找到一个突破口。

沈青瓷停下脚步，抬头看着夜空。

城市的灯光太亮，看不到星星，只有一轮模糊的月亮挂在天边。

她深吸一口气，掏出手机，打开笔记软件。

她新建了一个页面，在上面画了一张图：

```
钱永年（销售渠道）
    |
    ↓
徐鸣（制作高仿）
    |
    ↓
白行舟（洗白出货）
    |
    ↓
    ？
```

她在最下面又加了一行字：

```
顾家——是否也在链条上？
```

顾母的藏品里，会不会也有这条产业链上的东西？

如果有，那就意味着顾家也是受害者。而如果能证明这一点，官司的走向就会完全不同。

但这只是猜测。她需要证据。

沈青瓷收起手机，继续往前走。

她想起上次在法庭上的对峙，想起顾母在庭外的威胁，想起吴敏那双闪烁不定的眼睛。

这场战争，才刚刚开始。

---

晚上回到公寓，沈青瓷洗完澡，坐在床边整理思路。

她拿出一张白纸，把今天知道的信息一条一条写下来：

1. 《雅庐藏瓷手记》是伪造的，印泥成分暴露了年代问题
2. 这本假著录是徐鸣"发现"的，背后策划者是钱永年
3. 书中记录的瓷器有23件已经流入市场，都经过钱永年的手
4. 白行舟负责洗白，用他的"收藏家"身份给这些瓷器背书
5. 外公修复过的那件青花大盘是仿品，真品下落不明
6. 徐家仿古瓷的特点：瓷土好、火气退不尽

她看着这些信息，眉头紧锁。

链条很清晰，但证据链还不完整。要扳倒这几个人，需要的不是推测，而是铁证。

她想起钱永年设局让她看的那只"永乐青花大罐"。

如果能证明那件东西是徐鸣做的，就能把钱永年和徐鸣的关系坐实。

但她只是去看了一眼，没有拿到任何实物证据。而且她现在装作"没看出是假货"，如果突然翻脸揭穿，反而会打草惊蛇。

沈青瓷揉了揉太阳穴，感到一阵头疼。

手机响了，是程砚洲的消息：

"我有个想法。你明天有空吗？"

她回复："有。什么想法？"

"见面说。"

沈青瓷放下手机，躺倒在床上。

天花板上的灯光刺得她眯起眼睛。她伸手关掉灯，黑暗中只剩下窗外透进来的微弱光芒。

明天又是新的一天。

她不知道程砚洲有什么想法，但她有种预感——事情正在朝一个她无法预料的方向发展。

钱永年、徐鸣、白行舟、顾家……这些人、这些事，像一张巨大的网，把她罩在中间。

但她不会束手就擒。

她是沈家修复一脉的传人，她有外公传授的本事，她有识别真假的眼力。

这些人想用假货蒙骗世人，她偏要把他们的真面目揭开。

沈青瓷闭上眼睛，在黑暗中慢慢睡去。

梦里，她看到了外公年轻时的样子——穿着白大褂，戴着手套，正在修复一件破碎的青花大盘。碎片在他手中一块块拼合，裂缝被精心填补，最后，一件完整的瓷器出现在灯光下，釉面温润，画工精美。

外公转过头来，对她笑了笑。

"青瓷，记住，真的假不了，假的真不了。"

她醒来时，窗外已经泛起了鱼肚白。

新的一天开始了。

# 第十一章 证人

第二次开庭定在一周后的周三。

沈青瓷提前一天收到了冯律师的电话。

"顾家那边申请了新的证人，是你在拍卖行的前同事，叫赵婷婷。你认识吗？"

沈青瓷握着手机，脑海里浮现出一张脸——圆脸，戴眼镜，说话时喜欢眨眼睛。

"认识。"她说，"鉴定部的，跟我同一届入职的。"

"她会出庭作证。"冯律师的声音很沉，"我不知道顾家是怎么找到她的，但这不是好消息。你们之间有没有什么过节？"

沈青瓷想了想，摇头："没有。我们关系一般，不好也不坏。"

"那她为什么愿意给顾家作证？"

"不知道。"沈青瓷说，"但我猜，顾家应该给了她一些好处。赵婷婷这个人……比较现实。"

冯律师沉默了一会儿。

"明天开庭，你做好心理准备。不管她说什么，不要在庭上失态。"

"我知道。"

---

第二天，区人民法院第三审判庭。

沈青瓷坐在被告席上，看着对面的原告席。顾家派来的还是那个西装革履的律师，旁边坐着一个戴金丝眼镜的中年男人——据冯律师说，这是顾家的法务总监，专门来"督战"的。

旁听席上，顾母没有出现。但沈青瓷知道，那双眼睛无处不在。

书记员宣读完开庭词，法官敲响法槌。

"传证人赵婷婷出庭作证。"

沈青瓷转过头，看到一个熟悉的身影从门口走进来。

赵婷婷比几年前瘦了一些，但还是那副圆脸眨眼的样子。她穿着一身职业装，看起来紧张又兴奋，像是第一次参加这种场合。

她走到证人席，在书记员的指引下宣读证言承诺，然后坐下。

顾家律师站起身，走到她面前。

"赵女士，请您描述一下您与被告沈青瓷的关系。"

赵婷婷眨了眨眼睛，声音有些发紧："我们是同事。2019年一起入职的，都在鉴定部工作。"

"你们平时关系如何？"

"还可以。"赵婷婷说，"我们经常一起吃午饭，有时候也会讨论专业上的事情。"

"好。"顾家律师点头，"请问在你们的日常交流中，被告有没有提到过她对瓷器的鉴定能力？"

赵婷婷犹豫了一下，然后点头。

"有。她……她对永乐官窑特别有研究。"

沈青瓷的眉头微微皱起。

"请您具体说说。"律师追问。

"有一次我们一起看拍卖图录，里面有一件永乐青花。"赵婷婷说，"她只看了一眼就说，这件东西开门，是真品。我问她怎么看出来的，她说永乐官窑的东西她从小就接触，一眼就能认出来。"

她停顿了一下，又补充道："她还说过，她外公是故宫的修复师，专门修复明代官窑瓷器的。她从小在修复室里长大，对永乐、宣德那一时期的东西烂熟于心。"

顾家律师露出满意的神情。

"也就是说，被告具备鉴定永乐官窑瓷器的专业能力，对吗？"

"应该……是的。"赵婷婷说。

律师转向法官："法官大人，根据证人证词，可以证明被告在离开顾家之前，已经具备了识别永乐官窑瓷器的专业能力。结合此前提交的证据——被告在顾家生活三年期间，多次接触原告家族的藏品库房——我方认为，被告在签署离婚协议时，完全有能力识别出涉案青花罐的价值，却故意隐瞒，涉嫌欺诈。"

冯律师站起身，语气平静但有力。

"法官大人，我方对证人证词有异议。"

他走到赵婷婷面前。

"赵女士，请问您能否证明被告在顾家生活期间，曾经正式参与过藏品鉴定工作？"

赵婷婷愣了一下："这个……我不清楚。"

"您能否证明被告在签署离婚协议当天，亲眼见过涉案青花罐并进行了鉴定？"

"我也不清楚。"

"那您凭什么认为被告'隐瞒'了涉案物品的价值？"冯律师的声音锐利起来，"您刚才的证词只能证明被告有一定的专业知识，但'有知识'不等于'使用了知识'，更不等于'故意隐瞒'。我当庭提交的证据显示，被告在顾家三年期间，从未被正式授权接触藏品库房，也从未参与任何鉴定工作。她的专业能力，与本案涉案物品的发现过程，没有任何直接关联。"

他转向法官。

"法官大人，原告方提交的证人证词，只能证明被告'可能具备'识别能力，但无法证明被告'实际使用'了这种能力。根据'谁主张谁举证'的原则，原告方需要提供更直接的证据，才能支持其欺诈指控。"

法官点了点头，在记录本上写了几笔。

"原告方有补充吗？"

顾家律师脸色有些不好看，但还是保持着镇定。

"法官大人，我方会继续收集证据。但我们坚持认为，被告的专业背景足以说明她具备识别涉案物品的能力，这一点不容忽视。"

法官敲了敲法槌。

"本案将择日再审。双方继续准备证据。退庭。"

从法院出来，沈青瓷跟着冯律师走到外面的小公园。

九月的阳光还有些燥热，公园里的梧桐树叶子开始泛黄，风一吹，沙沙作响。

两人在一张石凳上坐下。

"今天的情况比我预想的好一些。"冯律师说，"但也只是暂时挡住了。"

"他们的证据链还是不完整。"沈青瓷说。

"对。"冯律师点头，"但他们会继续找。顾家不缺钱，不缺人脉，只要他们想，总能找到更多的证人、更多的'证据'。"

他转头看着沈青瓷。

"我之前说过，拖下去对你不利。现在我还是这个观点。你需要主动出击，找到能反转局面的东西。"

沈青瓷沉默了一会儿。

"我有个想法。"她说。

"什么想法？"

沈青瓷看着远处的梧桐树，声音很平静。

"如果顾家自己也买过假货呢？"

冯律师愣住了。

"什么意思？"

"顾母是收藏家，她的藏品里有很多瓷器。"沈青瓷说，"但古董这行水深，就算是行家也有看走眼的时候。如果能证明顾家的藏品里有假货，就能说明两件事：第一，鉴定瓷器真假本来就很难，就算是专业人士也会犯错；第二，顾家在古董鉴定上并不比我更专业，他们没有资格指责我'故意隐瞒'。"

冯律师听完，眼睛亮了起来。

"这是个好思路。但你有证据吗？"

"还没有。"沈青瓷说，"我在查。"

她站起身，拍了拍裙子上的灰。

"冯律师，给我一些时间。我会找到证据的。"

下午，某茶楼的包间里。

沈青瓷和吴敏相对而坐，中间隔着一壶龙井。

"你让我查的事，有点眉目了。"吴敏压低声音说。

沈青瓷的眼睛微微眯起。

"康熙五彩？什么器型？"

"花觚。"吴敏说，"我见过照片。"

她从手机里翻出一张图片，递给沈青瓷看。

图片上是一只花觚，敞口，长颈，鼓腹，撇足。通体绘着牡丹蝴蝶纹，彩料鲜艳，看起来确实像是康熙时期的东西。

沈青瓷把图片放大，仔细看那些红色的部分。

吴敏说得对。康熙五彩的红彩是用矾红釉料在低温下烧制的，因为工艺限制，颜色偏暗，有一种沉稳的质感。但图片上这件花觚的红彩，亮得有些刺眼，确实不太对劲。

"应该还在顾家。"吴敏说，"顾母买回去之后很得意，还专门开了个赏鉴会，请了一帮收藏圈的朋友来看。"

沈青瓷把图片保存下来，又问："当时有没有人提出质疑？"

"没有。"吴敏摇头，"来的人都是给顾母面子的，谁敢说她买了假货？"

沈青瓷点了点头，心里有了计较。

如果这件康熙五彩花觚确实是赝品，那就是她需要的证据。但问题是，她现在接触不到这件东西。顾家跟她撕破脸了，不可能让她进库房鉴定。

除非……有别的办法。

"吴敏，"她看着对方，"你能帮我弄到这件花觚更多的照片吗？最好是高清的，有细节的。"

吴敏犹豫了一下。

"我试试。钱老板那边可能留了档案。"

"还有，"沈青瓷又说，"这件东西的来源，你能查到吗？钱永年是从哪里弄来的？"

"这个……"吴敏皱起眉头，"有点难。钱老板对货源看得很紧，不会轻易透露。"

"想想办法。"沈青瓷说，"如果能查到这件花觚是徐鸣做的，那就更有说服力了。"

吴敏听到"徐鸣"这个名字，表情变了变。

"你怎么知道徐鸣？"

"我有渠道。"沈青瓷没有解释太多，"你只管帮我查，其他的我来处理。"

吴敏看了她一会儿，最终点了点头。

"行。我帮你查。"

沈青瓷站在路边，看着来来往往的车流，脑子里想着刚才的对话。

康熙五彩花觚。三百万。彩料发色有问题。

如果这件东西真的是假货，那顾母就是被钱永年坑了。而如果能证明钱永年跟徐鸣有关联，那整条造假产业链就串起来了：

徐鸣做高仿 → 钱永年伪造来源、销售 → 买家上当受骗

顾母也是受害者之一。

而她沈青瓷，不过是被甩了一箱"破烂"的离婚女人，凭什么要为顾家的眼力不济背锅？

沈青瓷勾了勾嘴角，掏出手机，给程砚洲发了条消息：

"钱永年三年前卖过一件康熙五彩花觚给顾家，开价三百万。我怀疑是徐鸣做的仿品。你那边能查到相关信息吗？"

消息发出去没多久，程砚洲就回了：

"我查查。"

沈青瓷收起手机，打车回家。

今天在法庭上，赵婷婷的证词给她敲响了警钟——顾家在一步步收紧包围圈，她不能坐以待毙。

但她也有自己的武器。

她有外公传授的本事，有程砚洲的情报支持，有吴敏这个"双面间谍"在钱永年身边。

三线作战，压力很大，但也不是没有胜算。

回到公寓，沈青瓷打开电脑，把今天拿到的那张康熙五彩花觚的照片导入进去，放大到最大倍数，一寸一寸地看。

红彩的发色确实不对。不是那种沉稳的枣皮红，而是亮得有些"飘"的矾红。

还有那些蓝彩和绿彩，颜色倒是没什么问题，但线条……

她盯着那些线条看了很久，突然发现一个细节——

花瓣的勾边，太流畅了。

康熙五彩的画工，虽然精湛，但因为是手工绘制，线条总会有一些细微的顿挫和起伏。但这件花觚上的线条，流畅得有些"机械"，像是用现代工具辅助过的。

沈青瓷把这个发现记在笔记本上，继续往下看。

她需要更多的证据。

但至少，方向是对的。

# 第十二章 旧地

沈青瓷用了两天时间做准备。

她翻出自己当年结婚时寄存在顾家的几本书——《中国陶瓷史》《故宫藏瓷图录》《明清瓷器鉴赏》——这些都是她读研时用的教材，当时放在顾家书房里，一直没有拿回来。

现在，这些书成了她的入场券。

她拨通了顾家管家的电话。

"老周，我是青瓷。"她的声音平静，听不出任何波澜，"我有几本书之前放在书房里，想过来拿一下。"

电话那头沉默了几秒。

"沈小姐，这个……我得请示一下太太。"

"麻烦您了。"

电话挂断。沈青瓷攥着手机，指节发白。

她知道这个电话有多冒险。顾家现在把她当敌人，她主动上门，就是把自己送到对方眼皮底下。但她没有别的办法——那件康熙五彩花觚就摆在顾家客厅的博古架上，她需要近距离看一看，拍到更多细节。

五分钟后，电话回来了。

"沈小姐，太太说可以来拿。"老周的声音有些僵硬，"但您只能在客厅等着，由我去书房帮您找。"

"好，谢谢老周。"

"明天上午十点方便吗？"

"方便。"

---

第二天，沈青瓷站在顾家大宅门口。

这栋别墅她太熟悉了。三年的时间，她在这里吃饭、睡觉、应酬、假笑，像一个精致的人偶被摆放在合适的位置。门口那两棵龙爪槐还是老样子，但经过一个夏天，枝叶茂密了许多，像两把撑开的绿伞。

她深吸一口气，按响门铃。

开门的是老周。六十多岁的老人，穿着灰色的中式褂子，脸上的表情比上次见面时更拘谨了。

"沈小姐，请进。"

沈青瓷跟着他走进去。

玄关还是那个样子，一幅张大千的山水画挂在正中央。客厅里的布置也没有变——欧式的沙发，中式的博古架，巨大的水晶吊灯，每一样东西都在彰显着这个家族的财力和品位。

她的目光不动声色地扫过博古架。

在那里——第二层，靠右的位置。

那件康熙五彩花觚静静地立在灯光下，敞口长颈，牡丹蝴蝶纹在彩釉下熠熠生辉。

"沈小姐，请坐。"老周指了指沙发，"我去书房帮您找书。您说是《中国陶瓷史》、《故宫藏瓷图录》和《明清瓷器鉴赏》，对吧？"

"对，麻烦您了。"

老周点了点头，转身往楼上走去。

沈青瓷坐在沙发上，眼睛一直盯着老周的背影，直到他消失在楼梯转角处。

然后她站起来。

她的心跳得很快，但手很稳。她走向博古架，掏出手机，调到静音模式，打开相机。

康熙五彩花觚就在眼前，比她预想的更大——高约四十公分，通体绘满缠枝牡丹和飞舞的蝴蝶。彩料在灯光下闪着柔和的光泽，看起来确实像是有些年头的东西。

但沈青瓷知道，"看起来像"和"真的是"，是两回事。

她举起手机，快速拍了几张整体照。然后凑近去看细节。

红彩。她首先看的是红彩。

康熙五彩的红彩是枣皮红，偏暗沉，有一种内敛的质感。但眼前这件花觚的红彩……确实亮了一些，就像吴敏说的那样，有点像现代的矾红。

她把镜头对准一朵牡丹花的红色花瓣，按下快门。

然后是蛤蜊光。

五彩瓷因为是低温彩釉，时间久了会产生一种虹彩效果，俗称"蛤蜊光"。她调整角度，让光线从侧面照射在釉面上——

有蛤蜊光。但分布得太均匀了。

真品的蛤蜊光是自然形成的，分布应该不规则，有的地方浓，有的地方淡，有的地方干脆没有。但这件花觚的蛤蜊光，像是被人工"安排"过一样，每一处都恰到好处。

沈青瓷又拍了几张蛤蜊光的细节照。

最后是底款。

她小心翼翼地把花觚转了一下，看到底部的款识——"大清康熙年制"，青花双圈六字楷书款。

她对准底款拍了一张。

然后她听到了脚步声。

老周从楼上下来了。

沈青瓷飞快地把花觚转回原来的位置，收起手机，走回沙发坐下。当老周出现在楼梯口时，她正若无其事地看着茶几上的茶杯。

"找到了。"老周捧着三本书走过来，"沈小姐，您看是这几本吗？"

沈青瓷接过书，翻了翻。

"是的，就是这几本。谢谢老周。"

她站起身，准备离开。

老周的目光落在她手里的手机上，停留了一秒。

"沈小姐，您刚才……"

沈青瓷的心跳漏了一拍，但她的脸上没有任何表情变化。

"怎么了？"

"没什么。"老周收回目光，"我送您出去。"

走出大门的那一刻，沈青瓷才悄悄松了口气。

她没有回头，但能感觉到老周的目光一直跟着她，直到她走出了门口的龙爪槐，拐上了大路。

---

回到公寓，沈青瓷迫不及待地把照片导入电脑。

她把每一张照片都放大到最大倍数，一寸一寸地看。

首先是整体照。器型没有问题，康熙五彩花觚的标准形制——敞口、长颈、鼓腹、撇足。比例协调，线条流畅。

然后是红彩。

她把红彩的照片和网上搜到的真品对比照片放在一起看。差别很明显——真品的红彩暗沉内敛，像成熟的枣皮；顾家这件的红彩明亮外放，像刚出窑的新彩。

再是蛤蜊光。

她打开另一组照片，仔细观察蛤蜊光的分布。就像她在现场看到的那样，分布得太均匀了。真品的蛤蜊光是岁月在彩釉上留下的痕迹，三百多年的日晒雨淋、温度变化，会让每一处彩釉的老化程度都不同。但这件花觚的蛤蜊光，像是用化学试剂处理过的，每一处都有，每一处都差不多。

最后是底款。

沈青瓷盯着底款的照片看了很久。

"大清康熙年制"，青花双圈六字楷书款。笔法还算规整，但……

等等。

她把照片放大，对准"年"字。

康熙官窑款识的"年"字，有一个特点——下面的"田"部，右下角的竖笔会稍微往外撇一点，形成一个微小的"勾"。这是康熙时期御窑厂书写款识的惯例，熟悉的人一眼就能认出来。

但顾家这件花觚的"年"字，右下角的竖笔是直的，没有那个"勾"。

沈青瓷的眼睛亮了起来。

这是个破绽。

她立刻翻出自己收藏的康熙官窑款识资料，一张一张地对比。没错——十几件真品的"年"字，右下角都有那个微小的"勾"。而顾家这件花觚的款识，虽然模仿得很像，但在这个细节上露了马脚。

她有八成把握了。

这件康熙五彩花觚，是高仿。

但八成把握还不够。她需要更确凿的证据——最好是能找到这件东西的出处，证明它是徐鸣做的。

沈青瓷靠在椅背上，揉了揉太阳穴。

她想起程砚洲说过的话——徐鸣做高仿，钱永年伪造来源、销售出货。如果能查到这件花觚是通过钱永年卖给顾家的，又能证明它和徐鸣的仿品特征吻合，那就能把整条链串起来。

但她现在只有照片，没有实物。光凭照片，很难说服法官。

她需要找专家鉴定——但又不能暴露目的。顾家在古董圈子里人脉很广，如果她贸然找人鉴定，消息很可能传到顾母耳朵里。

沈青瓷想了想，掏出手机，给程砚洲发了几张照片过去。

"我拍到了那件康熙五彩，你帮我看看。"

几分钟后，程砚洲回了消息：

"红彩发色偏亮，蛤蜊光太均匀，款识的'年'字写法有问题。我看有八九成是仿品。"

沈青瓷追问："能确定是徐鸣做的吗？"

"不能。徐家的仿古瓷特征主要在胎质上，光凭照片看不出来。"

"那怎么办？"

"我有个办法。"程砚洲发来一条语音，"景德镇有个人，专门研究现代仿古瓷的工艺特征，和徐家有些渊源。如果能把这件花觚的照片给他看，他应该能判断出是不是徐鸣的手法。"

"那个人靠得住吗？"

"靠得住。"程砚洲说，"他是徐守正的徒弟，后来和徐家闹掰了。他对徐鸣的手法比任何人都清楚，而且他恨徐鸣——恨一个人，就会想办法扳倒他。"

沈青瓷想了想，说："我想见见这个人。"

"行。我帮你约。"

挂断电话，沈青瓷看着电脑屏幕上的照片，眼神变得锐利起来。

康熙五彩花觚。三百万。钱永年卖给顾母的。

如果这件东西真的是徐鸣做的仿品，那顾母就是被骗了三百万。而卖给她假货的钱永年，还在外面活得好好的，继续坑别人。

顾家告她"欺诈"，说她故意隐瞒那只青花罐的价值。

可顾家自己呢？买了三百万的假货，连真假都没看出来。

这不是讽刺是什么？

沈青瓷勾了勾嘴角，关掉电脑，走到窗边。

夜色已经降临，城市的灯火在窗外闪烁。远处的高楼大厦，近处的车水马龙，每一盏灯下都有人在为自己的生活奔波。

而她，也是其中一个。

三年的豪门生活，像一场漫长的梦。现在梦醒了，她要为自己而活。

那只永乐青花罐，是她重新站起来的资本。

而这件康熙五彩花觚，是她反击的武器。

顾家想用官司逼她就范？

那她就让他们知道，谁才是被"欺诈"的那一个。

# 第十三章 源头

三天后，程砚洲的消息来了。

"有进展。来一趟。"

沈青瓷放下手里正在看的晚清瓷器资料，打车去了城西。

程砚洲的工作室还是老样子——长桌上铺满了文件和照片，空气里弥漫着淡淡的檀香味。但今天桌上的东西明显比上次多了，摞成了好几叠。

"你看这个。"程砚洲指了指桌上的一份文件。

沈青瓷走过去，拿起文件看了看。

是一份手写的账目记录，纸张已经泛黄，字迹有些潦草。

"这是什么？"

"钱永年的进货账本。"程砚洲说，"我托朋友从他的合作伙伴那里拿到的影印件。"

沈青瓷仔细看那些字迹。账本记录的是各种瓷器的进货信息——时间、器型、来源、价格。她翻了几页，目光突然停住了。

"2020年8月15日，康熙五彩花觚，徐氏，六十万。"

她的呼吸微微一滞。

"徐氏就是徐鸣？"

"对。"程砚洲点头，"钱永年从徐鸣那里六十万拿货，然后翻了五倍卖给顾家。"

他又推过来另一份文件。

"这是顾家的拍品记录。三年前，顾母参加了一场私人交易会，花三百万买了一件康熙五彩花觚。卖家是钱永年。"

沈青瓷把两份文件放在一起对比。时间对得上，器型对得上，卖家对得上。

"证据链完整了。"她说。

"对。"程砚洲的表情却没有轻松下来，"但有一个问题。"

"什么问题？"

"这些证据，"程砚洲指了指桌上的文件，"都是我托朋友从灰色渠道拿到的。账本是偷拍的，交易记录是内部人透露的。这些东西在法庭上不会被采信。"

沈青瓷愣了一下。

"你的意思是……"

"如果你直接把这些证据交给法官，对方律师会质疑来源的合法性。"程砚洲说，"而且就算法官采信了，顾家也可以反咬一口，说你非法获取他人商业信息。"

沈青瓷沉默了。

她看着桌上那些文件——钱永年的账本、顾家的交易记录、徐鸣的进货单据……每一份都指向同一个事实，但每一份都不能直接使用。

"那这些东西有什么用？"她问。

"有用。"程砚洲说，"只是不能直接用。"

他看着沈青瓷，眼神里带着一丝探究。

"你对顾母这个人了解多少？"

沈青瓷想了想。

"她很要面子。在她的圈子里，名声比什么都重要。她可以输掉官司，但不能输掉名声。"

"那就对了。"程砚洲说，"你掌握了她买假货的证据，她最怕的不是在法庭上输，而是这件事被传出去。一个号称'收藏家'的人，花三百万买了件假货——这个笑话够她丢一辈子的脸。"

沈青瓷明白了。

"你是说，让我用这些证据去威胁她？"

"不是威胁。"程砚洲纠正道，"是谈判。你让她知道你掌握了这些东西，然后给她一个台阶下——撤诉，各退一步。"

沈青瓷垂下眼睛，盯着桌上的文件。

这个办法确实可行。顾母是聪明人，她会权衡利弊。如果撤诉能保住面子，她应该会选择撤诉。

但如果她不吃这套呢？

"你要小心。"程砚洲像是看出了她的顾虑，"这等于是在刀尖上跳舞。如果顾母不吃这套，反咬一口，说你敲诈勒索，你会更被动。"

沈青瓷抬起头，看着程砚洲。

"你觉得我应该怎么做？"

程砚洲沉默了一会儿。

"我不能替你做决定。"他说，"但如果你问我的意见——这个办法有风险，但也是目前最有可能成功的办法。关键是你怎么谈，用什么方式让她知道这件事，又不至于让她觉得你在威胁她。"

沈青瓷点了点头。

"我知道了。"

她收起桌上的文件，放进自己的包里。

"程砚洲，谢谢你。"

"不用谢。"程砚洲说，"我们查的是同一伙人。你扳倒钱永年，对我查白行舟也有帮助。"

---

两人一起走出工作室。

外面的阳光很好，但已经有了秋天的凉意。梧桐树的叶子开始变黄，风一吹，沙沙作响。

"白行舟那边有进展吗？"沈青瓷问。

"有一些。"程砚洲边走边说，"他的货源和徐鸣有关联，有几件东西的来历和那本《雅庐藏瓷手记》对得上。我正在收集证据。"

"你打算怎么做？"

"找到足够的证据之后，把他告上法庭。"程砚洲说，"他手里有我祖父的旧藏，那些东西本来就是被骗走的。我要拿回来。"

沈青瓷看了他一眼。

程砚洲的侧脸在阳光下显得很硬朗，眉眼间有一种冷峻的气质。他说话的语气很平静，但沈青瓷能感觉到那平静下面压着的东西——愤怒、执念、还有某种她说不清楚的情绪。

"那批瓷器对你很重要？"她问。

程砚洲没有直接回答。他停下脚步，看着远处的天际线。

"我祖父是个书画家，也是收藏家。1949年之前，他在上海有一些名气，藏品也不少。后来……"他顿了顿，"后来时局变了，他的很多东西都散失了。有些是被没收的，有些是被迫卖掉的，还有些是被骗走的。"

他转过头，看着沈青瓷。

"我从小就听他讲那些瓷器的故事。他说，那些东西不只是瓷器，是他一辈子的心血，也是我们家族的记忆。他去世的时候，握着我的手说，如果有机会，把它们找回来。"

沈青瓷听着，心里有些触动。

她想起自己的外公——那个在修复室里度过一辈子的老人，手里修过无数珍贵的瓷器，却从来没有拥有过一件属于自己的藏品。

"你会找到的。"她说。

程砚洲没有说话，只是点了点头。

两人继续往前走。走到路口的时候，程砚洲突然开口：

"你一个人扛这些，累吗？"

沈青瓷愣了一下。

她没想到他会问这个问题。

"累也得扛。"她说，"没有别的选择。"

程砚洲看了她一会儿，没有再说什么。

"那我送你到地铁站。"

"不用了，我自己走就行。"

沈青瓷转身，朝地铁站的方向走去。走出几步之后，她又停下来，回头看着程砚洲。

"程砚洲，"她说，"我们查的是同一伙人。我会把我这边的进展告诉你，你也一样。"

程砚洲点了点头。

"好。"

沈青瓷转身离开。

她没有回头，但能感觉到程砚洲的目光一直跟着她，直到她走进了地铁站。

---

晚上，沈青瓷一个人坐在公寓里，看着那些文件发呆。

钱永年的账本、顾家的交易记录、徐鸣的进货单据——这些东西像一把把刀，每一把都能刺中顾家的要害。

但怎么用，是个问题。

直接找顾母摊牌？太冒险。顾母不是软柿子，如果她觉得被威胁了，很可能会鱼死网破，把事情闹大。

通过中间人传话？也不好。中间人可能会扭曲她的意思，让事情更复杂。

沈青瓷揉了揉太阳穴，感到一阵疲惫。

她想起程砚洲问她的话——"你一个人扛这些，累吗？"

累。当然累。

三年前她嫁入顾家，以为找到了归宿，结果只是从一个笼子换到了另一个笼子。三年后她离婚出来，以为可以重新开始，结果又被官司缠上，被人围追堵截。

她有时候会想，如果当初没有认出那只青花罐，会不会轻松一些？

但她知道，那不是她想要的生活。

她是沈家修复一脉的传人，她有真本事。那只青花罐是她应得的——不是因为她欺骗了谁，而是因为她识货，因为她有眼力。

顾家看不上她的专业能力，把她当花瓶摆了三年。现在他们发现那个"花瓶"比他们更懂瓷器，就急了，就要告她欺诈。

可笑。

沈青瓷站起身，走到窗边，看着窗外的夜景。

她想起外公说过的话——"真的假不了，假的真不了。"

顾家买了三百万的假货，还觉得自己是行家。

她被甩了一箱"破烂"，却从里面认出了八千万的真品。

这就是差距。

这就是她沈青瓷的本事。

沈青瓷深吸一口气，转身回到桌前。

她拿出一张白纸，开始写计划。

她要找一个合适的方式，让顾母知道这件事。

不是威胁，是提醒。

提醒她，在这场官司里，她不是唯一有把柄的人。
