# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事



# 用户需求

民国上海租界连环密室命案。法医（主角）被召去调查法租界三起密室杀人案：死者全是通灵会的常客，死因均为"溺毙"，但现场无一滴水。法医发现三间密室的墙纸上都印着同一套"通灵守则"——七条规则，遵守即安全。然而当他逐条验证，发现第三条和第五条在逻辑上互斥：同时遵守两条等于违反第七条。他开始怀疑凶手就是写规则的人，但调查越深越发现：这套规则并非人为编写，最早的版本出现在1842年《南京条约》签订当日的一份英文档案中。更诡异的是，法医自己正在不自觉地执行第六条。15万字，要求至少三次认知反转，结尾必须填完所有坑——规则的真正来源、密室手法的物理解释、法医为何执行第六条，全部要有合理答案。

## CONSISTENCY_MANAGEMENT_GUIDE.md

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。

也许是新的城市,也许是……另一个故事的开始。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
第5章:林羽发现规则可以被操控,开始怀疑(幻灭)
第8章:林羽学会打破规则,变得像敌人一样冷酷(堕落边缘)
第10章:林羽选择坚守底线,用正义手段复仇(升华)
```

**成长弧关键点**:

- **信念动摇**: 原本坚信的东西被打破
- **试错挣扎**: 尝试新方法,但付出代价
- **顿悟时刻**: 找到真正的答案
- **行动证明**: 用行动体现新的价值观

#### 2.9.3 细节刻画(小动作/口头禅/习惯)

**原则**: 给每个重要角色设计**独特标签**,让读者一眼认出

**示例**:

```
林羽:紧张时会不自觉地推眼镜,即使没戴眼镜
幽灵:说话前总是先停顿三秒,像在搜索数据库
张伟:喝茶时小指会翘起,暴露精英阶层的傲慢
```

**标签类型**:

- 肢体习惯(咬指甲/转笔/挠头)
- 口头禅("嗯……怎么说呢""确实如此")
- 情绪反应(生气时耳根发红/尴尬时揉鼻子)

**在创作chapters时**:
1. 应用**场景构建技巧**(五感描写+氛围营造)

# 第一章 霞飞路的溺亡者

雨是从傍晚开始下的。

沈兆清坐在书桌前，台灯投下一圈昏黄的光。他面前摊着一本德文版《法医病理学》，翻到肺组织切片的彩色图版那一页，铅笔在页边空白处写了半行批注。窗外的秋雨敲打着老虎窗的玻璃，发出细密而持续的声响，像有人用指甲不断地叩击。

他的公寓在法租界金神父路的一栋老石库门房子的三楼，两间屋子加一个亭子间，家具少得近乎寒酸——一张书桌、一把藤椅、一个装满书的旧书柜、一张单人铁架床。书柜里的书比衣柜里的衣服多出四倍。厨房里只有一只搪瓷锅和半袋大米。

这是一个三十一岁的独身法医应有的住所。干净，冷清，像验尸间的延伸。

时钟走过十一点。沈兆清准备合上书——他有个习惯，入睡前必须把当天阅读的重点内容在心中默念三遍。这个习惯从幼年起就有了，是母亲教的："读了就要记住，念三遍，刻在脑子里。"他从不觉得这有什么奇怪。嘴唇无声地翕动了三次，然后合上书，拧灭台灯。

黑暗中，敲门声响起。

不是雨声。是实实在在的拳头捶在木门上的闷响，急促，带着不容忽视的力度。

沈兆清重新拧亮台灯，披上一件旧呢大衣走到门口。门外站着裴孝廉，法租界巡捕房的华人探长，他的直属上级。裴孝廉穿着黑色雨衣，帽檐上滴着水，整个人像从黄浦江里捞出来的一样。

"兆清，"裴孝廉的声音因为淋了雨而有些发哑，"出事了。"

"现在？"沈兆清看了一眼墙上的钟，"十一点了。"

裴孝廉没有回答，而是从雨衣内袋里掏出一张照片，直接拍在了沈兆清身后的书桌上。

一块深红色的波斯地毯，书架，一张写字台，一盏落地灯——这是一间书房。地毯上没有水渍，写字台上没有水杯倒翻的痕迹，整个画面中唯一的液体就是死者口鼻间那一层泡沫。

沈兆清的目光从照片上抬起来，盯着裴孝廉。

"溺毙？"

"法籍验尸官初步判断是溺毙。"裴孝廉的声音压得很低，像是怕隔墙有耳，"但房间里没有一滴水。门窗从里头反锁，佣人砸不开，我们到了以后巡捕破的门。"

"死者叫吕鹤年，霞飞路上有头有脸的人物，做过洋行买办。"裴孝廉补了一句，"他是法租界通灵会的常客。"

"通灵会？"沈兆清的眉头皱了一下。

"你先来看看，路上说。"

沈兆清没有再问。他回到书桌旁，从抽屉里取出他的法医箱——一只棕色牛皮手提箱，边角磨损发白，铜扣因为反复开合已经磨出了弧形凹痕。这只箱子跟了他七年，从震旦大学医学院到巡捕房验尸间，比任何人都忠诚。

他拎起箱子走向门口。经过衣帽架时，他停了一下，从架子顶层摸下一顶呢帽扣在头上。

"走吧。"

巡捕房的黑色福特轿车在雨中驶过霞飞路。

车窗外的梧桐树被雨水洗得发亮，路灯的光在湿漉漉的柏油路面上拖出长长的倒影。霞飞路是法租界最繁华的马路，白天这里洋行、百货公司、咖啡馆鳞次栉比，入夜后却安静得像另一个世界。

"知道名字。法租界的洋人太太们搞的降灵聚会，请灵媒在黑暗中敲桌子。"沈兆清的语气里带着不加掩饰的轻蔑，"骗子集会。"

"吕鹤年也是？"

"入会六年。每周三晚上雷打不动参加仪式。"

沈兆清转过头看着车窗外掠过的雨幕，没有说话。他不信任何超自然的东西。所有的鬼魂、灵媒、降神术，在他眼里都是对活人心理弱点的精准利用。他相信的只有刀下的组织和显微镜里的细胞——人死了，就是一具需要被解剖的标本，没有灵魂出窍，没有冤魂索命。

这个信念不是凭空而来的。他的母亲沈柳芸，二十年前死于一次所谓的"通灵仪式"中的"意外"。他那年十一岁，被人从学堂叫回家，看见母亲躺在卧室的地板上，面色青紫，口鼻间有白色泡沫。

和照片里的吕鹤年一模一样。

那之后，幼年的沈兆清立了一个志：他要搞清楚所有"不可解释"的死亡。搞清楚了，它们就不可怕了。

汽车在一栋三层石库门宅邸前停下。门口已经拉起了白色警戒绳，两名巡捕撑着伞站在雨中，见裴孝廉下车，立刻敬了礼。

"尸体在二楼书房，"其中一名巡捕用法语报告，"我们到的时候门从里头用铁闩锁着，窗子也是从里面扣上的。破门之后没人动过现场。"

沈兆清提着法医箱走进吕宅。

一楼前厅的灯亮着，吕家的几个佣人缩在走廊角落里，有人在低声抽泣。沈兆清没有停留，跟着裴孝廉径直上了楼。

二楼走廊尽头是书房的门。门板已经被撬开，铁闩从门框上被硬生生拗断，断口处的铁锈像干涸的血迹。沈兆清侧身走进去。

他闻到的第一样东西是气味。

一种浓郁的、焦甜的、带着烟熏质感的香气，像是把蜂蜜放在炭火上烤焦后的味道，但更深沉、更粘稠，仿佛这股气味有实体一样附着在空气里，黏在鼻腔内壁上，赖着不走。安息香——他在东南亚传教士的药房里闻过这种树脂。

第二样是温度。虽然已入深秋，外面秋雨阵阵，但书房里异常闷热，像是房间在很长时间里处于完全密封的状态，空气没有任何流通。他下意识地检查了窗户——两扇木框玻璃窗，从内侧用铁插销扣死，窗框与窗台之间还塞了布条。这不是普通的关窗，这是刻意的密封。

第三样才是尸体。

吕鹤年的尸体保持着照片中的姿势，仰面倒在那块深红色的波斯地毯上。沈兆清放下法医箱，蹲到尸体旁边，戴上白色棉纱手套。

他先看脸。面部呈暗紫色，这是缺氧的典型表现。嘴唇微张，唇缘有干涸的白色泡沫。他用拇指掀开死者的右眼上眼睑——瞳孔完全散大，巩膜上有点状出血。再掀左眼，同样。

他俯下身，将耳朵凑近死者的口鼻，停了三秒。没有气息，但他闻到了一丝极其微弱的、不同于安息香的气味——酸涩的，像没酿好的醋。

他从法医箱中取出手术刀和一只注射器。手术刀沿着死者的颈前正中线划开一道浅口，暴露出气管。他将刀尖探入气管切口，挑起一小块黏膜组织，举到台灯光下端详。黏膜充血水肿，表面有一层微薄的灰白色薄膜，用刀尖一刮就脱落。

"化学灼伤。"他自言自语。

然后他用注射器的针头刺入死者右侧胸壁第四肋间，缓缓抽吸。针管中迅速涌入透明的、微微泛黄的液体。他抽了整整十毫升，才将针管拔出。举起针管对着光看——液体通透，有极微弱的乳光，不是水。

他用手指按压死者的胸骨。力一下去，胸腔内部传来一声沉闷的、咕噜咕噜的液体晃动声，像是有人在死者身体里藏了一只装满水的皮囊。

"这不是水，"沈兆清对身后的裴孝廉说，"是肺泡自己渗出来的浆液。他的肺已经变成了两只装满液体的水袋。"

裴孝廉站在门口，没有走近。他看起来不太适应尸体的气味，但脸上的表情不是厌恶——更像是一种沈兆清当时没有读懂的复杂神情。

"溺死在自己的肺液里？"裴孝廉问。

"可以这么说。"沈兆清将注射器里的液体转移到法医箱中的玻璃试管里，塞上软木塞。"问题是，什么东西让他的肺在短时间内渗出了这么多液体。"

他站起身，开始环视房间。

书房大约十来个平方米。三面墙是书架，第四面是窗户。但让他注目的不是书架上的藏书，而是墙壁——所有书架与墙壁之间的缝隙、窗框周围的空白区域、甚至天花板的边角，都被一种米黄色的印花墙纸覆盖。墙纸的花纹不是普通的装饰图案，而是文字。

沈兆清走近一面墙壁，凑到离墙纸不到半尺的距离。

文字是中英双语印刷的，字体端正，排列整齐，像某种仪式用的经文。英文在上，中文在下，总共七条：

**THE SEVEN RULES OF COMMUNION**

*通灵守则*

一、通灵时必须保持室内密封（不可有任何通风口开启）

二、通灵前必须在墙壁四面张贴守则全文

三、通灵时必须点燃安息香以净化空间

四、通灵结束前不得触碰房间内任何水源

五、通灵全程必须保持呼吸均匀且深长

六、每次阅读守则后必须默诵三遍方可离开

七、以上六条须同时遵守，违反任意一条即为对亡灵的亵渎

沈兆清逐条读完，眉头越皱越紧。

他从法医箱里取出一本随身笔记本和一支钢笔，将七条守则一字不差地抄录了下来。写到第六条时，他的笔尖停顿了一下。

*每次阅读守则后必须默诵三遍方可离开。*

一种说不清的不适感从胃部升起，像是吞了一口冰水。他几乎是条件反射地想到了自己的习惯——读完重要文字默念三遍。母亲教的。但这不过是巧合。他快速写完最后一行，合上笔记本。

房间中央的地面上，在尸体头部上方约一尺的位置，放着一只铜制小香炉。炉内残留着黑色的灰烬和几块没有完全烧尽的树脂碎块。沈兆清用镊子夹起一块碎块，凑近鼻子——安息香，确认无误。

他正要将碎块装入取证袋时，身后传来一个女人的声音。

"请把那个放下。"

声音不大，但冷得像秋雨，每一个字都带着不容置疑的硬度。

沈兆清没有回头。他不喜欢在工作时被打断，尤其是被一个语气命令式的陌生女人。

"这是犯罪现场，"他说，镊子上的树脂碎块稳稳当当没有动，"我是巡捕房法医。"

"这不是犯罪现场。这是通灵会的圣域。"

沈兆清这才转过身。

门口站着一个年轻女人，二十七八岁的样子，穿一件深灰色旗袍，外面罩着一件黑色天鹅绒披肩，头发挽成一个低髻，没有任何首饰。她的脸在走廊灯光的逆照下有些看不真切，但沈兆清注意到她的站姿——背脊挺直，双手交叠在身前，下巴微微抬起。不是佣人，不是家眷，不是看热闹的邻居。

"方鸿雪，"她自我介绍，声音平稳，"通灵会现任祭司。吕先生是我们的会员。他是在神圣的通灵仪式中离世的，这间房间里的一切——墙纸、香炉、灰烬——都是仪式的组成部分，不可以被带走。"

裴孝廉在一旁咳嗽了一声。沈兆清注意到他没有出面干涉，也没有帮自己说话——这不像他的作风。

"方小姐，"沈兆清将镊子上的树脂碎块放回香炉里，但动作缓慢，带着一种刻意的敷衍，"一个人死在了密封的房间里。不管你管这叫仪式还是降灵还是跳大神，对我来说都一样——这是一起死因不明的案件，我需要取证。"

"守则有规定。"方鸿雪说。

"什么守则？"

"墙上贴的那七条。第二条，通灵前必须在墙壁四面张贴守则全文。守则是圣域的一部分，移除即为亵渎。"

沈兆清几乎笑出来了。"一个人死了，你跟我说亵渎？"

方鸿雪的目光与他对上。那双眼睛让沈兆清的笑意凝固了——不是因为美或者威严，而是因为冷静。那双眼睛里没有悲伤、没有恐惧、没有愤怒，也没有宗教狂热者那种燃烧般的执念。那种冷静是计算性的，像一个外科医生在审视手术台上的器官。

这个女人不像是信徒。她在观察他。

但这个念头只闪了一瞬，沈兆清就把它压了下去。他不是来分析活人心理的。

"裴探长，"他对裴孝廉说，"我需要带走墙纸样本和香炉残渣做化验。"

裴孝廉张了张嘴，目光在沈兆清和方鸿雪之间移动了一下，最后说："墙纸……先不动。方小姐那边的关系不好处理。你把该记的都记下来，样本以后再说。"

沈兆清盯了裴孝廉两秒钟。裴孝廉避开了他的目光。

他没有再争辩。转过身，默默地将笔记本翻到抄录七条守则的那一页，又从头到尾看了一遍，确认一字不差。然后他拎起法医箱——里面装着那管珍贵的肺液样本——走出了书房。

她在看他的嘴。

他没有多想。

一楼前厅。

巡捕已经把吕家的佣人赶到了后院，前厅里只剩下沈兆清和裴孝廉。一张红木太师椅，一盏煤油灯，满墙的对联和福字在昏黄的光线里显得阴沉沉的。

从头再来——第二遍。

再来——第三遍。

他合上笔记本，长长地呼出一口气。

裴孝廉端着一杯热茶走过来，放在他手边的茶几上。"有什么初步判断？"

"死因确实是溺毙，"他说，"但液体来自死者自己的肺组织。某种原因导致了急性肺水肿，肺泡壁破裂，大量浆液涌入肺腔。他是溺死在自己的肺液里的。"

"什么原因？"

"目前不知道。气管黏膜有化学灼伤的痕迹，说明死者在死前吸入过某种刺激性气体。但具体是什么物质，需要实验室分析。"

裴孝廉点了点头，像是在消化这些信息。然后他看似随意地问了一句：

"你刚才嘴里在念什么？"

沈兆清愣了一下。"什么？"

"刚才你看笔记本的时候，嘴在动。"

沈兆清低下头看着手里的笔记本，一时竟觉得有些窘迫——他确实没有意识到自己在念。

"习惯。"他说，"读完东西要念三遍才记得住。我妈从小教的。"

话出口的瞬间，一个微小的变化发生了——但沈兆清当时没有捕捉到。裴孝廉端茶的手顿了一下，极短暂的、不超过半秒的停顿。然后他若无其事地喝了一口茶，点了点头。

"好习惯。"裴孝廉说。

他的语气平淡得像在说天气。但如果沈兆清此刻抬头仔细看一眼裴孝廉的眼睛，他会在那双老练的、什么风浪都见过的眼睛里，看到一种极力克制的东西。

不是惊讶，不是好奇。

是确认。

沈兆清没有抬头。他将试管从法医箱里取出来，对着煤油灯的光再看了一眼那管浅黄色的透明液体。液体在灯光下泛着微弱的乳光，像某种不属于人体的东西。

"我先回去做尸检，"他站起来，"有结果再报告你。"

"好。"裴孝廉也站了起来，"我让车送你。"

"不用，走回去就行。雨小了。"

沈兆清提着法医箱走出吕宅大门。雨确实小了一些，从瓢泼变成了绵密的细雨。他站在台阶上戴好帽子，呼吸了一口潮湿的夜风。空气里还残留着安息香的甜腻气味，从他的衣服上、头发上渗出来，像是那间密室的幽灵追了出来，附着在他身上不肯离去。

他沿着霞飞路向东走。法医箱在左手里沉甸甸的，试管里的肺液随着他的步伐轻微晃动。

他没有回头。

在他身后，吕宅二楼亮灯的窗口，裴孝廉站在那里，目送他走远。探长的脸隐没在雨雾之中，看不清表情。他一直站到沈兆清的背影消失在霞飞路尽头的黑暗里，才转过身，走回书房。

他走到墙纸前，看着第六条下面的那行中文——

*每次阅读守则后必须默诵三遍方可离开。*

裴孝廉闭上了眼。

二十年前的那个夜晚，也是这样的秋雨。沈柳芸倒在辣斐德路19号的地板上时，她十一岁的儿子还在学堂里背课文。那个孩子不知道的是，他一辈子都在执行的那条"读三遍"的习惯，不是什么好记性的诀窍。

那是一条守则。

裴孝廉睁开眼，将煤油灯吹灭，在黑暗中独自站了很久。
