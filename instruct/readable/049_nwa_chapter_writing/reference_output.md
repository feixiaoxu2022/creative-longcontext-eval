# 第四章循环

第四个副本是"镜像迷宫"。

沈执渊站在迷宫的入口，看着眼前无尽的镜面走廊，眉头紧锁。

这是一个由镜子构成的迷宫。每一面墙都是镜子，每一条路都是镜子，每一个转角都是镜子。他能看到无数个自己的倒影，从各个角度、各个方向注视着他。

"沈哥，"苏念的声音从身后传来，"这个副本好漂亮。"

沈执渊回头看了她一眼。苏念正站在一面镜子前，好奇地看着自己的倒影。

"漂亮？"沈执渊皱眉，"这是C级副本，不是观光景点。"

"可是真的很漂亮啊。"苏念转过身，笑着说，"你看，这些镜子的排列好像有规律。"

沈执渊顺着她指的方向看去。果然，镜子的排列不是随机的。它们形成了一条隐约的路径，通向迷宫深处。

"你怎么发现的？"他问。

"感觉。"苏念笑着说，"我说了，我的感觉很准的。"

沈执渊沉默了。

他不相信感觉，但他不得不承认，苏念的"感觉"确实帮他们避开了很多危险。

"走吧。"他说，"跟紧我。"

两人沿着镜子指引的路径前进。迷宫里很安静，只有他们的脚步声在回荡。偶尔，沈执渊会看到镜子里的倒影做出和他不同的动作，但当他仔细看时，一切又恢复了正常。

"沈哥，"苏念的声音响起，"你有没有觉得……这些镜子里的倒影，好像在看着我们？"

沈执渊停下脚步，回头看了她一眼。

"你也感觉到了？"

苏念点点头："嗯。它们的眼神……很奇怪。"

沈执渊皱了皱眉。他也有同样的感觉。那些倒影的眼神，不像是在反射，更像是在……观察。

"不要看它们的眼睛。"他说，"专注于路径。"

"好。"苏念点头。

两人继续前进。迷宫的路径越来越复杂，有时候需要转弯，有时候需要原路返回。沈执渊发现，这个迷宫的设计很巧妙，如果不按照正确的路径走，很容易迷失方向。

但苏念的"感觉"再次帮了他们。每当他们走到岔路口时，她总能指出正确的方向。

"沈哥，走左边。"

"沈哥，这里要原路返回。"

"沈哥，前面有陷阱，绕一下。"

沈执渊开始习惯听从她的指引。他不知道为什么，但他就是相信她。

终于，他们来到了迷宫的中心。

那里有一面巨大的镜子，比其他镜子都要大。镜子前站着一个人影。

那个人影穿着黑色的风衣，背对着他们。

"是之前那个怪物？"苏念问。

"不知道。"沈执渊说，"小心点。"

他握紧了手中的武器，慢慢靠近。

那个人影听到脚步声，缓缓转过身来。

沈执渊愣住了。

那不是怪物。

那是他自己。

镜子里的倒影走了出来，站在他面前。它的脸和他一模一样，但眼神却完全不同——那是一种冰冷的、没有任何感情的眼神。

"你是谁？"沈执渊问。

倒影没有回答。它只是看着他，嘴角慢慢扬起一个诡异的笑容。

然后，它动了。

它的速度快得不可思议，眨眼间就来到了沈执渊面前。它的拳头砸向沈执渊的脸，沈执渊侧身躲开，但还是被拳风扫中，整个人飞了出去。

"沈哥！"苏念的声音响起。

她冲了过来，挡在沈执渊面前。

"你是什么东西？"她问倒影，"为什么要攻击我们？"

倒影歪着头，像是在思考什么。

然后，它笑了。

"因为……"它的声音和沈执渊一模一样，"你们不该来这里。"

沈执渊挣扎着站起来，看着面前的倒影，心里涌起一股寒意。

这个东西，是他的镜像。

它拥有他所有的能力，所有的记忆，所有的……弱点。

"苏念，退后。"他说，"这是我的战斗。"

"可是——"

"退后！"

苏念咬了咬唇，退到了一边。

沈执渊握紧武器，冲向倒影。

两人在镜子迷宫中展开了激烈的战斗。他们的动作一模一样，速度一样快，力量一样强。每一次攻击都被对方挡下，每一次闪避都被对方预判。

这是一场势均力敌的战斗。

但沈执渊知道，这样下去他赢不了。因为倒影不会累，不会痛，不会犯错。

他必须找到它的弱点。

"苏念！"他喊道，"帮我！"

苏念冲了过来，但她不是攻击倒影，而是——闭上了眼睛。

"沈哥，"她说，"我看不见了。你来引导我。"

沈执渊愣了一下，然后明白了她的意思。

倒影是他的镜像，它的所有能力都来自于他。如果他闭上眼睛，倒影也会闭上眼睛。如果他停下来，倒影也会停下来。

但苏念不是他的镜像。她不受这个规则的限制。

"好。"沈执渊说，"我数到三，你攻击它的后颈。"

"好。"

"一……"

沈执渊闭上眼睛。

倒影也闭上了眼睛。

"二……"

沈执渊停下脚步。

倒影也停下了脚步。

"三！"

苏念动了。她冲向倒影，双手抓住它的肩膀，用力一扭。

倒影发出一声惨叫，轰然倒地。

【副本通关！】

白光闪过，两人回到了等待区。

沈执渊睁开眼睛，看着面前的苏念，心里涌起一股复杂的情绪。

"你……"他想说什么，但话到嘴边又咽了回去。

苏念笑着说："沈哥，我们又通关了！"

沈执渊看着她的笑脸，心里那种异样的感觉越来越强烈。

这个女孩，到底是什么人？

她的直觉、她的反应、她的配合……都不像是一个普通的新人。

但他没有证据。

"休息一下吧。"他说，"明天还有副本。"

"好的沈哥！"苏念开心地说。

她转身向休息区走去，脚步轻快。

沈执渊看着她的背影，心里那种异样的感觉越来越强烈。

他不知道那是什么。

但他知道，从今天开始，他不会再把苏念当作一个普通的新人搭档了。

她是……不一样的。

沈执渊看着这条提示，皱了皱眉。

绑定关系稳固度又增加了。

这到底是什么东西？

他想查看详细说明，但系统依然没有给出任何解释。

沈执渊摇了摇头，不再多想。

不管怎样，明天还有新的副本。

他需要保持清醒，保持警惕。

至于苏念……

他会继续观察她。

但这一次，他不会再让她死。

绝对不会。
