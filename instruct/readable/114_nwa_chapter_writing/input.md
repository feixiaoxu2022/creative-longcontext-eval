# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事

## CONSISTENCY_MANAGEMENT_GUIDE.md

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。

也许是新的城市,也许是……另一个故事的开始。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
```

# 第一章：透明的开始

##场景一

**场景编号**:1
**场景位置**:旧教学楼三楼储物间，狭小黑暗，只有门缝透进一丝光
**人物**: 沈默
**场景目的**: 建立沈默的困境，引入透明人规则
**核心动作**: 沈默靠在墙角，看着手机上的时间，想起那个都市传说，开始在心里默数秒
**冲突点**:门外周薇的嘲笑声 vs 沈默内心的绝望与决绝
**情感变化**: 从"又是这样"的麻木 → "也许这次不一样"的微弱希望
**场景结果**: 三分钟到，沈默的手指开始变得透明，她惊恐地捂住嘴，不敢发出声音

---

储物间的门被锁上的那一刻，沈默就知道今天不会善了。

她背靠着冰冷的墙壁，听着门外周薇和她那群跟班的笑声。那些笑声像针一样扎进她的耳朵里，又顺着脊椎一路往下，在胃里搅成一团。

"沈默，你就在里面好好反省吧。"周薇的声音从门缝里挤进来，带着那种她特有的、甜腻的恶意，"想想你做错了什么。"

沈默没有回答。她早就学会了不回答。

手机屏幕亮着，显示着下午3点47分。她被锁在这里已经快一个小时了。储物间里没有窗户，只有门缝透进来的一线光，照在她苍白的脸上。

空气里有一股陈旧的霉味，混着灰尘和不知道什么东西腐烂的气息。沈默能感觉到自己的后背被墙壁的潮气浸透，冷意顺着皮肤往骨头里钻。

她想起了那个传说。

那是学校里流传已久的都市怪谈——如果你在监控死角停留超过三分钟，你就会变成透明人，持续24小时。

没有人知道这是不是真的。但沈默已经不在乎了。

她低头看着自己的手。那双手很瘦，指节分明，因为长期握笔而在中指上磨出了茧。此刻，这双手正微微发抖。

"三分钟。"她在心里默念，"只要三分钟。"

她闭上眼睛，开始数秒。

一。

门外的笑声还在继续，但她已经听不清她们在说什么了。她的注意力全部集中在那个数字上，像溺水的人抓住最后一根稻草。

二。

手机屏幕的光在她眼皮上投下一片橙红色的暖光。她能感觉到自己的心跳，咚、咚、咚，在寂静的储物间里显得格外清晰。

三。

什么都没有发生。

沈默睁开眼睛，苦笑了一下。果然是骗人的。这种都市传说怎么可能是真的？她真是疯了才会相信。

但就在她准备放弃的时候，她看见了自己的手。

她的手指——正在变得透明。

那种透明不是渐渐的、模糊的，而是像有人在她的皮肤上涂了一层看不见的油，让她的血肉一点点从视线中消失。她能看见手指后面的墙壁，那面斑驳的、布满灰尘的墙壁，此刻正透过她的指骨显露出来。

沈默猛地捂住嘴，把即将冲出喉咙的尖叫咽了回去。

她的手还在变。手腕、前臂、手肘……透明像潮水一样蔓延，所过之处，她的身体就从这个世界上一点点抹去。

不是消失。是变得看不见。

她还在这里。她能感觉到自己的心跳，能感觉到后背贴着的冰冷墙壁，能感觉到恐惧像一只手紧紧攥住她的胃。

但她的身体，正在变成透明。

门外，周薇的笑声还在继续。她们完全没有注意到，储物间里的沈默，已经不再是原来的沈默了。

**人物**: 沈默

沈默低头看着自己的身体。

透明化已经蔓延到了她的肩膀。她能看见自己的锁骨，那两根细细的骨头像两道月牙，悬浮在空气中。锁骨下面是她的胸腔，她能看见自己的心脏在跳动——那颗小小的、红色的器官，此刻正透过她透明的皮肤一下一下地收缩、舒张。

恐慌像冰水一样浇遍全身。她拼命地想要抓住什么，手指在空气中胡乱挥舞。她的手穿过了储物间的门把手，穿过了那扇紧锁的铁门，像穿过一层薄雾。

然后，她的手穿过了墙壁。

那种感觉很奇怪。不是疼痛，也不是冰冷，而是一种……虚无。就像她的手伸进了一团棉花里，没有阻力，没有触感，只有一片空洞的、什么都没有的空间。

沈默愣住了。

她低头看着自己穿过墙壁的手，又抬头看着那面斑驳的墙壁。墙上有一道裂缝，裂缝里长着一丛细小的青苔，此刻正透过她透明的手臂清晰可见。

她试着把手收回来。

手很顺利地穿了回来，就像穿过一层水幕。她低头看着自己的手——已经完全透明了，只剩下一个模糊的轮廓，像一团淡淡的烟雾。

透明化还在继续。

她的躯干、她的腿、她的脚……一点点从这个世界上消失。不，不是消失，是变得看不见。她还在这里，她能感觉到自己的重量压在地面上，能感觉到空气从她透明的皮肤上流过。

但她的身体，已经不再属于这个世界了。

沈默站起来。她的脚穿过了地面，又穿了回来。她低头看着自己透明的双脚，突然觉得这一切荒谬得可笑。

她变成了透明人。

那个她以为是骗人的都市传说，竟然是真的。

门外，周薇的声音还在继续："……她以为她是谁啊？不就是个没人要的孤儿吗？她妈一年到头都不在家，她爸早就不知道死哪儿去了……"

沈默的身体僵住了。

那些话像刀子一样扎进她的心里。她早就习惯了被这样说，习惯了被忽视，习惯了在这个学校里像一团空气一样存在。

但此刻，她真的变成了空气。

一种奇异的平静笼罩了她。既然已经这样了，还有什么好怕的呢？

她走向储物间的门。这一次，她没有去碰门把手，而是直接穿过了那扇门。

沈默站在那里，看着周薇的背影，心里涌起一种奇怪的感觉。不是愤怒，不是委屈，而是一种……解脱？

不用再听那些话，不用再忍受那些目光，不用再假装自己不存在。

但沈默没有听见。

她还不知道，在这个透明的世界里，有一个人，一直在等她。

透明层的图书馆很安静。

沈默跟着顾深走进那扇灰色的大门，里面的一切都是灰色的。书架是灰色的，书本是灰色的，连空气都是灰色的。唯一有颜色的，是顾深的眼睛——那双眼睛里有一种沈默从未见过的光。

"这里的书都是灰色的。"顾深说，"但内容和现实世界一样。我这三年，把这里所有的书都读完了。"

沈默走到书架前，抽出一本书。书的封面是灰色的，但她翻开之后，里面的文字是清晰的。那是一本小说，讲的是一个人在荒岛上独自生存的故事。

"你一个人……读了三年的书？"她问，声音有些干涩。

顾深点了点头。他走到窗边，看着外面灰色的天空。

"刚开始的时候，我很害怕。"他说，"这里什么都没有，没有声音，没有颜色，没有任何人。我以为我会疯掉。"

沈默放下书，走到他身边。

"后来呢？"

"后来我发现了这个图书馆。"顾深说，"这里有书，有知识，有另一个世界。我开始读书，读很多很多的书。读着读着，我就忘记了时间。"

他转过头，看着沈默。他的眼神很平静，但沈默能感觉到那平静下面藏着的东西。

"三年，一千多个日夜。"他说，"我读完了这里所有的书。哲学、历史、文学、科学……我什么都读。"

沈默的心揪了一下。

她想象着顾深一个人坐在这个灰色的图书馆里，一页一页地翻着那些灰色的书。没有人说话，没有人陪伴，只有他自己的呼吸声和翻书的声音。

"你不孤独吗？"她问。

顾深沉默了一会儿。

"孤独。"他说，"但我有你。"

沈默愣住了。

"我？"

"对。"顾深说，"虽然你听不见，但我知道你在。我对着你说话，对着你读书，对着你思考。你是我这三年来唯一的……陪伴。"

沈默的眼眶有些发酸。

她想起了那些时刻。做作业的时候，她会不自觉地点头。发呆的时候，她会叹气。被周薇欺负的时候，她会靠在墙上。

她一直以为那是自己的习惯。

原来，那是在回应他。

"对不起。"她说，声音很轻，"我不知道。"

"没关系。"顾深说，"现在你知道了。"

他伸出手，想要触碰她的脸。但他的手穿过了她透明的皮肤，什么都没有碰到。

沈默看着他的手，心里涌起一种奇异的感觉。她想要抓住那只手，但她知道，她抓不住。

他们都是透明的。

他们都是不存在的。

但此刻，他们在一起。

"不能了。"他说，"他们已经在这里待了很久，比我还久。他们的身体已经快要完全消失了。"

沈默的心沉了下去。

她看着那两个模糊的身影，心里涌起一种说不出的感觉。恐惧？悲伤？还是别的什么？

"他们……会怎么样？"她问。

"消失。"顾深说，"彻底消失。就像从来没有存在过一样。"

沈默沉默了。

她不想变成那样。她不想彻底消失。她想要活着，想要被看见，想要被记住。

"有办法离开吗？"她问，"有办法不变成那样吗？"

顾深看着她，眼神里有一种沈默看不懂的东西。

"有。"他说，"但我不确定你愿不愿意。"

"什么办法？"

顾深沉默了一会儿，然后说：

"放弃。"

沈默愣住了。

"放弃什么？"

"放弃你最珍贵的记忆。"顾深说，"透明层的入口分布在城市的各个角落。但只有特定的地点才能触发。那些地点都有一个共同点——它们都是'被遗忘的地方'。"

"如果我们能找到其他入口，也许就能找到出去的方法。但问题是，入口被封印了。只有愿意放弃最珍贵记忆的人，才能打开封印。"

沈默的心揪了一下。

放弃最珍贵的记忆？那意味着什么？意味着忘记自己最爱的人，忘记自己最珍贵的经历，忘记自己之所以是自己的一切。

"如果我放弃了……"她的声音有些颤抖，"我会忘记什么？"

"你最珍贵的东西。"顾深说，"可能是一个人，可能是一段记忆，可能是一种感觉。每个人都不一样。"

沈默沉默了。

她最珍贵的东西是什么？是父亲？是母亲？还是……

如果她放弃了这段记忆，她会忘记他吗？

"我不想忘记你。"她说，声音很轻，但很坚定。

顾深看着她，眼睛里闪过一丝光芒。

"我也不想让你忘记。"他说，"但如果这是唯一的办法……"

"一定还有别的办法。"沈默打断他，"我们再想想，一定还有别的办法。"

顾深沉默了一会儿，然后点了点头。

"好。"他说，"我们再想想。"

沈默不知道自己在透明层里待了多久。她只知道，当她再次睁开眼睛的时候，她已经回到了储物间。

储物间的门还是锁着的，但她的身体已经恢复了正常。她能看见自己的手，能感觉到后背贴着的冰冷墙壁，能闻到空气中的霉味。

她不再是透明的了。

但她知道，这只是暂时的。

24小时的倒计时已经开始了。她能感觉到，有什么东西正在从她的记忆里慢慢消失。像沙子从指缝间流走，像水从杯子里蒸发。

沈默点了点头。

"我知道。"她说，"但我会回来的。"

"你怎么回来？"

沈默沉默了一会儿。

"我不知道。"她说，"但我会找到办法的。"

顾深看着她，眼睛里闪过一丝光芒。

"好。"他说，"我等你。"

"等我。"她说，"我一定会回来的。"

她的皮肤已经不再是皮肤了。它变成了一层透明的薄膜，让风直接穿过，却带不走任何温度。

沈默站在教学楼前的空地上，抬头看着天空。

天很灰，像一块洗旧了的抹布。远处的操场上有人在上体育课，篮球撞击地面的声音断断续续地传过来。

她不知道自己该去哪里。

她不知道这个状态会持续多久。

她不知道24小时之后，会发生什么。

但此刻，她只想走。走得越远越好，离开这个让她窒息的地方，离开那些嘲笑她的人，离开那个永远把她当成空气的世界。

她迈开脚步，朝校门口走去。

她不知道的是，在她身后的旧教学楼里，有一双眼睛，正看着她的背影。

那双眼睛的主人，嘴角浮起一丝苦涩的微笑。

"没关系。"他轻声说，"我会一直在这里等你。"

沈默没有听见。

她已经忘记了。

但她的心里，还是空了一块。

##场景一

沈默躺在床上，看着天花板。

房间里很暗，只有窗外的路灯透进来一点光。她能听见隔壁房间母亲打电话的声音，断断续续的，像是在和什么人吵架。

她已经习惯了。

母亲很少在家，每次回来都是为了吵架。吵架的对象永远是同一个人——她的父亲。那个在她很小的时候就离开的男人。

沈默不记得父亲的样子了。她只记得一些模糊的片段：一双温暖的手，一个宽厚的背影，还有一种淡淡的烟草味。

但那些记忆已经很淡了，淡得像一张褪色的照片。

她翻了个身，把脸埋进枕头里。

她不知道自己为什么会想起父亲。也许是因为最近太累了，也许是因为心里空了一块。

是的，空了一块。

自从那天从储物间出来之后，她就一直觉得心里空了一块。像是有什么重要的东西被挖走了，留下一个黑洞，怎么填都填不满。

她试着回忆那天发生了什么。她记得自己被周薇锁在储物间里，记得自己在里面待了很久，记得林小鹿来救她。

但除此之外，什么都没有了。

她不记得自己是怎么出来的，不记得自己在储物间里做了什么，不记得……

"有人在叫我。"

这个念头突然冒出来，沈默猛地坐起来。

有人在叫她。她能感觉到。那声音很轻，很远，像是从很深的地方传来。

但她听不清那声音在说什么。

她闭上眼睛，仔细听。那声音还在，断断续续的，像是在呼唤她的名字。

"沈默……沈默……"

她睁开眼睛，房间里一片寂静。只有母亲打电话的声音还在继续。

是幻听吗？

沈默不知道。她只知道，那个声音让她心里涌起一种奇异的感觉。像是想念，又像是悲伤。

她想要抓住那个声音，但她抓不住。

"我不知道。"她说，"但我总觉得，这里有什么东西在等着我。"

林小鹿皱起眉头，走到她身边。

"你确定吗？"她问，"也许只是你想多了。"

林小鹿沉默了一会儿，然后点了点头。

"好。"她说，"我陪你。"

沈默看着她，心里涌起一种温暖的感觉。

"谢谢。"她说。

她不记得这些。她只记得自己被锁在里面，然后就什么都不知道了。

"没有了。"她说，"就是这些。"

沈默低下头，看着那扇门。

她知道，林小鹿没有说谎。但她也知道，那天一定发生了什么。只是她忘记了。

"对。"沈默说，"我想查一些东西。"

# 第十二章：离开

房间里很安静，只有他们的呼吸声。灰色的墙壁，灰色的天花板，灰色的地板，一切都是灰色的，像是一个没有颜色的世界。

照片在他手中化为灰烬，飘散在空气中。

她忘记了父亲。忘记了他的笑容，忘记了他的声音，忘记了他给她的温暖。

他们撕下照片后，房间开始变化。

灰色的墙壁开始变得透明，像是有人在上面涂了一层水。灰色的天花板开始变得明亮，像是有人在上面点了一盏灯。灰色的地板开始变得坚实，像是有人在上面铺了一层砖。

整个房间像是从灰色的梦境中醒来，变得真实起来。

"看。"顾深指着前方。

沈默转过头，看见了一条路。

那条路出现在房间的尽头，像是凭空出现的一样。路是灰色的，但路的尽头有一片光。那片光很亮，很温暖，像是太阳的光。

"那是出口。"顾深说，"我们要走那条路。"

沈默点了点头。

他们一起朝那条路走去。

路很长，很远，但他们没有停下。他们知道，只要一直走，总会到达终点。

走着走着，沈默感觉到有什么东西在变化。

她的身体开始变得沉重，像是有什么东西在拉扯她。她的视线开始变得模糊，像是有什么东西在遮挡她。

她知道，那是记忆在消失。

她忘记了父亲。忘记了他的笑容，忘记了他的声音，忘记了他给她的温暖。

但她没有停下。她知道，只要一直走，总会到达终点。

路的尽头是一片白光。

那片白光很亮，很温暖，像是太阳的光。沈默和顾深走进那片白光，感觉到有什么东西在包裹他们，像是一双温柔的手。

然后，白光消失了。

沈默睁开眼睛，发现自己站在旧教学楼的天台上。

天已经黑了。星星在天空中闪烁，月亮挂在天边，洒下一片银色的光。风很大，吹得她的头发乱飞。

"我们……出来了？"她的声音有些颤抖。

顾深站在她身边，看着天空。他的身体还是透明的，但在月光下，他的轮廓反而更加清晰了。

"出来了。"他说，声音很轻，"我们出来了。"

沈默看着他，心里涌起一种奇异的感觉。她想起了刚才发生的事情——撕下照片，放弃记忆，走出那条路。

她忘记了什么？

她想不起来。她只知道，她的心里空了一块，像是有什么重要的东西被挖走了。

"现在几点了？"她问。

顾深沉默了一会儿。

"我不知道。"他说，"但我知道，我们在里面待了很久。"

沈默的心跳加速了。

很久？多久？

她拿出手机，想要看时间。但手机的屏幕是黑的，没有任何反应。

"没电了。"她说，声音有些干涩。

顾深点了点头。

"我们下去吧。"他说，"去看看外面的世界。"

沈默点了点头。

他们一起走下天台，走下楼梯，走出旧教学楼。

外面的世界很安静。街道上没有人，路灯发出昏黄的光。远处的高楼大厦亮着灯，像是一片星海。

沈默看着这一切，心里涌起一种奇异的感觉。

她不知道自己离开了多久。一天？一周？一个月？

她不知道外面的世界发生了什么。她不知道林小鹿怎么样了，不知道陈老师怎么样了，不知道……

她不知道。

她只知道，她回来了。

她和顾深一起回来了。
