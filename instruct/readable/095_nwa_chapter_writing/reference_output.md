# 第十八章 余波
天元329年春，苍梧互市迎来了开年的第一个大集，人特别多，仙魔两界的商人都来摆摊，热闹得很。
陈安和阿糯也过来摆摊卖他们自制的草药膏，刚摆好摊，就看到几个鬼鬼祟祟的人穿着黑袍，在卖干粮的摊子边晃悠，趁老板不注意，往粮食里撒了什么东西。
阿糯碰了碰陈安的胳膊，低声说：“你看那几个人，不对劲，好像在往粮食里下毒。”
陈安点点头，不动声色地走过去，故意撞了其中一个人一下，那人手里的纸包掉在了地上，白色的粉末撒了出来。
“你干什么？”那人脸色一变，伸手就要去抢。
陈安往后退了一步，拿出剑指着他：“你往粮食里撒的是什么？是不是毒药？”
周围的人听到了，都围了过来，卖干粮的老板赶紧拿起粮食闻了闻，脸色大变：“是软筋散！吃了会浑身无力，任人摆布！”
那几个人见事情败露，转身就想跑，刚好红缨带着巡逻的士兵过来，直接把他们抓了起来，押回了苍梧宫。
审问之后才知道，这几个人是夜枭的旧部，不甘心仙魔和平，就想在互市的粮食里下毒，嫁祸给魔族，挑起仙魔两界的矛盾，让战争重新爆发。
谢无妄知道之后，脸色特别沉，下令严查所有夜枭的余党，一个都不放过。他怕沈砚担心，一开始没告诉他，想等处理完了再说，结果沈砚还是从苏清鸢那里知道了。
“这么大的事你居然不告诉我？”沈砚有点生气，皱着眉看着他。
“我这不是怕你担心嘛，”谢无妄走过去，拉住他的手，低声哄他，“几个小喽啰而已，已经都抓住了，没闹出什么事，你别生气了好不好？”
“我不是生气，我是担心你，”沈砚叹了口气，“夜枭的余党肯定还有不少，你以后处理这种事，一定要告诉我，我们一起面对。”
“好，我知道了，下次一定告诉你。”谢无妄点点头，在他嘴角亲了一下。
后来经过彻查，一共抓了三十多个夜枭的余党，都依法处置了，还在互市加了巡逻的士兵，以后再也没出现过这种事。
经过这件事，仙魔两界的人反而更团结了，大家都知道，和平来之不易，不会轻易被别有用心的人挑拨。有次沈砚去互市巡查，听到两个人族的商人聊天，说“要不是谢陛下和沈少宗主，我们现在还在打仗呢，谁会来搞破坏我们都不答应”，沈砚听了，心里特别暖。
回去之后，他把这件事告诉了谢无妄，谢无妄笑了笑，抱住他：“你看，大家都很珍惜现在的和平，我们的努力没有白费。”
“嗯，”沈砚靠在他怀里，点点头，“以后会越来越好的。”
后来谢无妄还专门出台了政策，鼓励仙魔两界的人互相举报反对和平的人，举报属实的还有奖励，从那之后，夜枭的余党再也掀不起什么风浪了。
有年春节，谢念安和樱樱在院子里放烟花，沈砚和谢无妄坐在廊下看，谢无妄忽然说：“再过两年，我们把魔界君主的位置传给长老，然后找个地方隐居好不好？我们去江南住，那里气候好，适合养老。”
沈砚愣了一下，转头看他：“你舍得丢下魔界的事务吗？”
“有什么舍不得的，”谢无妄笑了，捏了捏他的脸，“我当君主就是为了给你一个安稳的未来，现在和平已经稳定了，长老们都能处理好事务，我想多陪陪你，我们还有很多地方没去玩呢。”
“好啊，”沈砚笑了，眼睛亮得像星星，“等安安长大了，能独当一面了，我们就去隐居，游山玩水，过我们自己的日子。”
“嗯，都听你的。”谢无妄低头，在他额头上亲了一下，烟花在天上炸开，照亮了两个人温柔的侧脸。
后来谢念安十八岁的时候，确实成了仙魔两界的使者，能力特别强，不管是仙门的事还是魔界的事，他都能处理得井井有条，谢无妄就放心地把魔界的事务交给了长老们，和沈砚一起去江南隐居了。
他们在江南买了一个小院子，种了满院的桃花，还有桂花树，每年秋天，桂花开了，整个院子都是香的。沈砚每天在院子里看书画画，谢无妄就给他做桂花糕，炖鸡汤，日子过得特别悠闲。
有时候谢念安会带着樱樱来看他们，给他们带最新的消息，说仙魔两界最近又建了几个新的互市，又有很多仙魔情侣成婚了，每次听到这些，沈砚和谢无妄都特别开心。
有次沈砚躺在院子的摇椅上，看着旁边给他剥橘子的谢无妄，笑着说：“我们现在的日子，比我以前想象的还要好。”
谢无妄把剥好的橘子递给他，笑了笑：“我说过，我们会一直幸福下去的。”
阳光透过桃花的缝隙洒下来，落在两个人的身上，暖融融的。他们找了一百年，等了一百年，终于等到了属于他们的，最安稳的幸福。
而仙魔两界的和平，也在所有人的努力下，一直延续了下去，再也没有发生过战争，百姓们安居乐业，有情人都能终成眷属，这大概就是最好的结局了。
