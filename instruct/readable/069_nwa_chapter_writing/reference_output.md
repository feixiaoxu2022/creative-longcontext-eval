# 第四章 共犯

霍深离开后，沈时宜在黑暗中坐了很久。

她不知道他去见霍父会面对什么。

也许是被发现了什么，也许是家族的其他安排，也许是和她完全无关的事务。但她无法不去想。

**他吻了她。**

**他在那个被监视的房间里，在李副官就站在门外的时候，吻了她。**

**他疯了吗？**

**还是她疯了？**

她低头看着自己的双手，它们在微微颤抖。

月光从窗帘缝隙里漏进来，照在床头柜上，照在那本旧书上。

《战地日记》。

十年前她送给他的书，扉页上有她写的"活着回来"四个字。

她伸手，把那本书拿了起来。

书页已经泛黄，边角磨损，但保存得很好——被主人翻阅过无数次的痕迹，却又被小心翼翼地呵护着。

她翻开扉页。

熟悉的字迹映入眼帘：

**"活着回来。——时宜"**

她的名字。

十年前她写下这四个字的时候，炮火就在不远处轰鸣。他坐在她对面，接过书，看了很久。

"活着回来。"他念出声，然后抬起头看着她，"好。"

那是他给她的承诺。

三天后，他被上报阵亡。

她以为他食言了。

她花了十年恨他、想他、悼念他、在酒精里溺死自己，又从酒精里爬出来继续活着。

**而他把这本书带在身边十年，带着它完成了"死亡"，带着它度过了她不知道的岁月，现在又带着它出现在她的新房里。**

沈时宜的眼泪落在书页上，洇开一个浅浅的印记。

她把书合上，放在枕边。

**这是她今夜唯一能拥有的他的东西。**

窗外的天色开始变化了。

最深的黑暗过去了，东边的天际有了一丝模糊的灰白色。

沈时宜从床上起身，走到窗边。

她轻轻拉开一条窗帘缝隙，看着窗外的世界。

军区大院还是那个军区大院。整齐的宿舍楼，笔直的道路，路灯在黎明的光线里渐渐失去存在感。哨兵正在换岗，两个身影在路口交接，然后一个离开，一个接替。

规律、沉闷、永不停歇。

**十五分钟一次的巡逻。**

**二十四小时不间断的监视。**

**这就是她从今往后的生活。**

但她此刻看着那些哨兵，却没有像今夜刚开始时那样觉得窒息。

**因为她知道了。**

**霍深还活着。**

**那个她以为死了十年的男人，还活着。**

**他就住在这个大院里，睡在她的新房里，和她用只有彼此能懂的暗语交流。**

**他记得她。**

**他带着她送的书活了十年。**

**他吻了她。**

她的嘴角浮现出一个微笑——十年来第一个真实的、发自内心的微笑。

**哪怕明天是毁灭，今夜也值了。**

就在这时，她听见了脚步声。

不是哨兵的脚步——那个节奏她已经熟悉了。

是另一种脚步，沉稳、有力，从走廊尽头一步步向这扇门逼近。

**是他。**

**他回来了。**

沈时宜的心跳加速了。

她站在窗边，没有动，只是看着门的方向。

门把手转动了。

门开了。

晨光从他身后涌进来，给他的轮廓镀上了一层淡金色的边。他穿着军装，笔挺得像一把出鞘的剑，但他脸上的表情——

她看不清。

光线太亮了，她只能看见他的轮廓，看不清他的脸。

他走进来，带上门。

房间重新陷入相对的昏暗，只有窗帘缝隙透进来的一线晨光。

他朝她走来。

一步、两步、三步。

他在她面前停下，距离她不到一米。

现在她能看清他的脸了。

他的表情很复杂，有疲惫，有释然，还有一些她说不清楚的东西。他的眼睛里有红血丝，像是一夜没睡。

"霍父……"她开口。

"家里的事。"他打断她，声音低沉，"和你无关。"

她点点头，没有追问。

沉默了几秒。

然后他的手动了。

他在她面前比了一个手势——

**战场暗语。**

意思是："我们需要谈谈。"

沈时宜的心跳漏了一拍。

**需要谈谈。**

**他在用暗语告诉她——我们需要谈谈。**

**不是用嘴说的话，是用只有他们两个人能懂的手势。**

**因为这个房间里可能有窃听器，有监视，有无数双看不见的眼睛。**

**但手势是安全的。**

**手势是只属于他们两个人的语言。**

她的手也动了，比出回应：

**"我等你很久了。"**

他看见她的手势，瞳孔微微收缩。

**我等你很久了。**

**不只是今夜。**

**是十年。**

**十年来她一直在等他。**

**等一个她以为再也不会回来的人。**

他垂下眼帘，睫毛在脸上投下一小片阴影。然后他抬起头，对上她的目光。

他的眼神变了。

不再是那个冰冷的、疏离的少将的眼神，而是另一种眼神——她曾经无比熟悉，却以为再也不会见到的眼神。

**十年前，他最后一次看她的时候，也是这样的眼神。**

**那时候他满身是血，把她塞上直升机，用沙哑的声音说："活着，等我回来。"**

**她等了十年。**

**他终于回来了。**

晨光从窗帘缝隙里涌进来，在他们之间的地板上画出一道明亮的线。

他们隔着那道线对视，谁都没有说话。

因为有些话不需要说出口。

因为他们已经说过了——用手势，用眼神，用一本旧书，用一道疤痕，用那个额头上的吻。

**他们是共犯了。**

**是那种把彼此拖进深渊、却谁都不后悔的共犯。**

窗外，哨兵的脚步声再次响起。

规律、沉闷，十五分钟一次的巡逻。

但这一次，沈时宜听着那些脚步声，却笑了。

**让他们巡逻去吧。**

**让那些眼睛看着去吧。**

**他们什么都看不见。**

**因为真正的对话，发生在只有两个人能懂的沉默里。**

霍深看见她的笑容，愣了一下。

然后他的嘴角也微微弯起，勾出一个若有若无的弧度。

那是她十年来第一次看见他笑。

不是档案照片上的那种标准微笑，不是婚礼上对宾客的那种礼貌微笑。

是真的笑。

是她记忆中的那个人才会有的笑。

沈时宜忽然觉得眼眶有些发热。

**他还是他。**

**十年的时光、阵亡的假象、少将的军衔，都没有改变他。**

**他还是那个在战场上为她挡枪的人。**

**还是那个把她从雷区背出来的人。**

**还是那个她爱了十年、恨了十年、以为再也见不到的人。**

窗外的天彻底亮了。

晨光涌进房间，把红色的喜字照得发亮，把那些压抑的装饰变成了某种温暖的东西。

沈时宜看着窗外的天空，看着那一抹正在蔓延的金色。

新的一天开始了。

她不知道等待他们的是什么——是更大的风暴，是霍父的追问，是组织的审视，还是这场政治婚姻背后更深的漩涡。

**但她知道一件事。**

**她不再是一个人了。**

**不管前面是什么，他们会一起面对。**

**因为他们已经是共犯了。**

**是那种即使毁灭，也要一起毁灭的共犯。**

她转过身，面对霍深。

他也在看着她。

晨光照在他们身上，把两个人的影子重叠在一起，投射在地板上。

"早安。"她说。

这是她在这个新房里对他说的第一句不是表演的话。

"早安。"他说。

他的声音里有一丝笑意。

窗外，军区大院开始苏醒。早操的号声响起，士兵们列队跑步的脚步声整齐如鼓点。

这是他们的新生活。

这是他们的牢笼，也是他们的战场。

但至少——

**他们在一起。**

**活着，在一起。**
