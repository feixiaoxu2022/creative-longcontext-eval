# 第十一章 真相与和解

下午的训练馆里，阳光透过窗户洒在平衡木上，映出金色的光芒。苏清禾独自站在更衣室的长凳前，手里握着几张泛黄的照片。她的眼神里充满了复杂的情绪，有困惑，有怀疑，还有一丝期待。

她想起中午苏晚在器械区对她说的话——"前世的事故...是意外。"她知道，苏晚的眼神里没有欺骗，但她还是需要时间来接受这个真相。

"清禾。"

她猛地转身，看到陈小雅站在更衣室门口。阳光透过窗户洒在她的脸上，映出她担忧的神色。

"小雅，你怎么这么早？"苏清禾的声音有些沙哑，"今天不是休息日吗？"

陈小雅走到她身边，在长凳上坐下。"清禾，我担心你。你下午...一直心不在焉吧？"

苏清禾低下头，不敢看她的眼睛。"小雅，我...我需要时间。"

"时间？"陈小雅歪着头，"你需要时间做什么？"

苏清禾深吸一口气。"小雅，如果...如果一个人做了什么，但那不是故意的，你会原谅她吗？"

陈小雅看着她，眼神里充满了担忧。"清禾，你是不是想起了什么？"

苏清禾摇了摇头，但眼泪却不争气地流了下来。"小雅，我...我可能误会了什么。"

陈小雅轻轻抱住她。"清禾，不管发生了什么，我都会站在你这边。"

苏清禾靠在陈小雅的肩膀上，感受着朋友的温暖。她想起前世在病床上度过的那些日子，想起苏晚站在病床前冷漠的眼神。

但现在，一切都变得不一样了。苏晚的眼神里没有冷漠，只有愧疚和小心翼翼。

"小雅，"苏清禾的声音很轻，"如果...如果前世的事故真的是意外，你会原谅那个人吗？"

陈小雅愣了一下。"清禾，你在说什么？"

苏清禾抬起头，看着她。"小雅，前世的事故...可能是意外。"

陈小雅的脸色变得严肃起来。"清禾，你发现了什么？"

苏清禾从包里拿出几张泛黄的照片，递给陈小雅。"小雅，你看这些。"

陈小雅接过照片，仔细地看着。"道具检查正常...2023年3月10日？还有这个...道具故障记录...2023年3月12日？"

苏清禾点了点头。"小雅，前世的事故发生在3月15日，而道具检查是在3月10日，道具故障记录是在3月12日。如果道具真的检查正常，那么事故的原因就只有一个——道具在检查后出现了故障。"

陈小雅的脸色变得苍白。"清禾，你是说...苏晚教练推你的时候，是想保护你？"

苏清禾的眼泪再次流了下来。"小雅，我...我不知道该相信什么。"

陈小雅轻轻抱住她。"清禾，不管真相是什么，我都会支持你。"

苏清禾靠在陈小雅的肩膀上，感受着朋友的温暖。她知道，陈小雅会支持她，不管真相是什么。

与此同时，在训练馆的器械区，苏晚正站在平衡木旁边发呆。她的手指轻轻抚摸着平衡木的表面，感受着那熟悉的触感。

"苏晚。"

她猛地转身，看到苏清禾站在器械区入口处。阳光透过窗户洒在她的脸上，映出她苍白的脸色。

"清禾，你怎么这么早？"苏晚的声音有些紧张，"今天不是休息日吗？"

苏清禾慢慢走向她，每一步都带着犹豫。"苏晚，我能问你一个问题吗？"

苏晚的心跳加速了。"什么问题？"

"道具..."苏清禾的声音很轻，"道具检查真的正常吗？"

苏晚的脸色瞬间变得苍白。她知道，苏清禾已经发现了什么。但她不知道该怎么回答，因为真相比她想象的要复杂得多。

"清禾，"苏晚深吸一口气，"有些事情...不是你想的那样。"

"那是什么样？"苏清禾的声音里带着一丝颤抖，"苏晚，你到底在隐瞒什么？"

苏晚低下头，不敢看她的眼睛。"清禾，我...我需要时间解释。"

"时间？"苏清禾冷笑一声，"你已经让我等了这么久，还要我继续等吗？"

苏晚抬起头，看着她。"清禾，我知道你恨我。但有些事情...不是你想的那样。"

苏清禾的眼泪再次流了下来。"不是我想的那样？那是什么样？苏晚，你告诉我，前世的事故到底是怎么回事？"

苏晚的身体僵住了。她没想到苏清禾会直接问出这个问题。她知道，苏清禾已经想起了前世的事情。

"清禾，"苏晚的声音很轻，"前世的事故...是意外。"

"意外？"苏清禾的声音里充满了怀疑，"你说是意外？"

苏晚点了点头。"清禾，道具在检查后出现了故障。我...我推你的时候，是想保护你。"

苏清禾愣住了。她没想到会听到这样的解释。前世她一直以为苏晚是故意推她的，但现在苏晚却说那是意外。

"清禾，"苏晚走到她面前，"我知道你很难相信，但那是真的。我重生回来，就是想补偿你。"

苏清禾看着她，眼神里充满了困惑。"重生？你也重生了？"

苏晚点了点头。"是的，我和你同一天重生。"

苏清禾后退了一步，不敢相信自己的耳朵。"所以...前世的事故真的是意外？"

苏晚点了点头。"是的，清禾。道具在检查后出现了故障，我推你的时候是想保护你。"

苏清禾的眼泪再次流了下来，但这次不是因为恨，而是因为困惑。她想起前世在病床上度过的那些日子，想起苏晚站在病床前冷漠的眼神。

但现在，一切都变得不一样了。苏晚的眼神里没有冷漠，只有愧疚和小心翼翼。

"苏晚，"苏清禾的声音很轻，"如果真的是意外...你为什么要隐瞒？"

苏晚低下头，不敢看她的眼睛。"清禾，我...我害怕。我害怕你不会原谅我。"

苏清禾的心软了下来。她知道，苏晚的愧疚是真实的。她想起前世在病床上度过的那些日子，想起苏晚站在病床前冷漠的眼神。

但现在，一切都变得不一样了。苏晚的眼神里没有冷漠，只有愧疚和小心翼翼。

"苏晚，"苏清禾轻轻握住她的手，"我...我需要时间。"

苏晚点了点头。"清禾，我会给你时间。但请你相信，我从来没有想过要伤害你。"

苏清禾看着她，眼神里充满了复杂的情绪。她不知道该相信什么，但她知道，苏晚的眼神里没有欺骗。

与此同时，在训练馆的休息区，林教练正坐在长凳上发呆。她的目光落在窗外的阳光中，思绪飘得很远。

"林教练。"

她猛地转身，看到苏晚站在休息区入口处。阳光透过窗户洒在她的脸上，映出她苍白的脸色。

"苏晚，你怎么这么早？"林教练的声音有些关切，"今天不是休息日吗？"

苏晚走到她身边，在长凳上坐下。"林教练，我...我需要你的建议。"

林教练看着她，眼神里充满了担忧。"苏晚，你怎么了？"

苏晚深吸一口气。"林教练，清禾...她发现了什么。"

林教练的眉头立刻皱了起来。"发现了什么？"

"道具故障的真相。"苏晚的声音很轻，"她问我前世的事故到底是怎么回事。"

林教练的脸色变得严肃起来。"苏晚，你告诉她了？"

苏晚点了点头。"是的，我告诉她了。"

林教练叹了口气。"苏晚，你做得对。清禾需要知道真相。"

苏晚看着她，眼神里充满了困惑。"林教练，我...我不知道该怎么做。"

林教练轻轻拍了拍她的肩膀。"苏晚，你只需要做你自己。清禾会理解的。"

苏晚点了点头，但眼神里仍然充满了担忧。她知道，苏清禾需要时间来接受这个真相。

"林教练，"苏晚的声音很轻，"如果清禾不原谅我...怎么办？"

林教练看着她，眼神里充满了理解。"苏晚，清禾会原谅你的。她只是需要时间。"

苏晚点了点头，但眼神里仍然充满了担忧。她知道，苏清禾需要时间来接受这个真相。

阳光透过窗户洒在训练馆的每一个角落，映出金色的光芒。苏清禾独自站在更衣室的长凳前，手里握着几张泛黄的照片。

她的手指轻轻抚过照片的背面，那里有一行小字："道具检查正常，2023年3月10日。"还有一行小字："道具故障记录，2023年3月12日。"

她想起苏晚今天在训练中的表现——那个总是小心翼翼地关注她，却又在她靠近时退缩的身影。

"苏晚..."她低声念出这个名字，声音里带着一丝颤抖。

她知道，真相可能比她想象的要复杂得多。但她也知道，苏晚的眼神里没有欺骗。

她需要时间来接受这个真相，但她也知道，苏晚会给她时间。

阳光如金，训练馆的灯光在白昼中显得格外明亮。苏清禾站起身，走向更衣室的门口。

她知道，今天会是一个新的开始。
