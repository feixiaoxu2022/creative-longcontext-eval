# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。

也许是新的城市,也许是……另一个故事的开始。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
```

## 正文

### 场景一

废弃的城郊仓库里，血迹已经干涸，在昏暗的光线下呈现出触目惊心的暗褐色。

陆沉蹲在受害者身边，眉头紧锁。死者是本地著名的企业家，三天前失踪，今天被发现死在这里。致命伤是胸口的两刀，但从伤口的痕迹来看，凶手似乎并不熟练——刀口的方向凌乱，显示凶手在刺出第一刀后犹豫了一瞬，然后才补上第二刀。

“陆队，”身后传来同事小刘的声音，“技术组那边有新发现。”

陆沉站起身，走向技术组的方向。他的右眼皮突然跳了一下——这是今天第二次了。从早上开始，他就总觉得有什么事情要发生。

作为刑侦支队副队长，他处理过上百起凶案，见过太多死亡。但这一次，他说不清为什么，心里有一种莫名的不安。

他弯下腰，检查地上的血迹。死者倒在血泊中，右手向前伸着，像是在临死前想要抓住什么。他的手指微微弯曲，指甲缝里有皮屑组织。

这是搏斗的痕迹。

陆沉直起身，准备让人把证物带走。他的目光落在死者的脸上——那双眼睛还睁着，瞳孔已经涣散，但仿佛在最后时刻凝固了某种情绪。

恐惧？

不，不是恐惧。是一种……疑惑？

陆沉皱起眉头。一个人在临死前露出疑惑的表情，这意味着什么？

他伸出手，想要合上死者的眼睛。就在他的手指即将触及眼皮的瞬间——

他的额头突然一阵剧痛。

像是有烧红的铁丝从眉心刺入大脑，陆沉闷哼一声，弯下腰去。周围的同事纷纷转头看他。

“陆队？你怎么了？”

陆沉摆摆手，示意自己没事。那阵剧痛来得快去得也快，但当他抬起头时，他看到了一些不该看到的东西。

死者的头顶，浮现出一行血红色的字。

【距离死亡：3分钟】

陆沉眨了眨眼，以为自己出现了幻觉。那行字还悬在那里，颜色鲜艳得像刚流出的血。

他环顾四周。仓库里除了他之外还有四名同事——小刘、技术组的老张、还有两名痕检人员。每个人的头顶都浮现出了倒计时。

小刘：【距离死亡：47年2个月13天】

老张：【距离死亡：12年8个月】

两名痕检人员头顶的数字也在正常范围内。

只有这个死者，头顶显示的是【距离死亡：3分钟】。

3分钟。

陆沉看了看手表。现在是下午5点27分。按照字面意思，这具尸体应该在5点30分死亡。

但他已经死了至少三天。

“这不可能……”陆沉喃喃自语。

“陆队？你在说什么？”

陆沉没有回答。他的视线死死盯着那行字，心里涌上一个荒谬的念头——也许这不是死者的死亡时间，而是……

他的手机突然震动。是一条陌生号码发来的短信。

【系统绑定成功。犯罪预警系统已激活。】

【检测到宿主试图规避预言。】

【警告：规避行为将加速死亡。】

【当前预计死亡时间：22:30】

陆沉的手指开始颤抖。他豁然抬头，看向仓库大门的方向。

5点29分。

距离那个“死亡时间”，还有一分钟。

“快跑！”他突然大吼一声，冲向门口，“所有人离开这里！”

“陆队？怎么了？”

“ 别废话！跑！”

他冲出仓库的瞬间，身后传来一声巨响。

仓库的承重柱突然断裂，整面墙壁向内坍塌。尘土飞扬中，陆沉只来得及扑倒在地。碎砖块砸在他背上，剧痛让他眼前发黑。

但他还记得那声巨响之前的最后一幕——

一根断裂的钢筋，从天而降，刺穿了刚才他站立的位置。

如果他还在那里，现在被刺穿的就是他的身体。

废墟归于平静。陆沉从地上爬起来，满身灰尘和血迹。他看着那具已经被埋入废墟的尸体，终于明白了那行字的意义。

那不是死者的死亡时间。

那是他的。

### 场景二

傍晚6点15分，陆沉站在自己家的门前，迟迟没有打开门。

他的后背还在隐隐作痛——被砖块砸中的地方已经淤血，但他没有去医院。这种伤对他来说不算什么，他以前受过更重的伤。

让他犹豫的是进门之后的事。

距离那个“预言”给出的死亡时间，还有不到五个小时。他的未婚妻苏清禾现在就在门里面，按照平时的时间，她应该正在厨房准备晚餐。

陆沉靠在门框上，点了根烟，深深吸了一口。

系统的声音在他脑海中响起——是的，系统有声音，像是一个没有任何感情的电子合成音。

【警告：检测到宿主处于危险目标附近。】

【目标：苏清禾】

【预言内容：将在今晚23:00杀害你】

陆沉又吸了一口烟，然后慢慢吐出来。烟雾在走廊的灯光下缓缓散开，像是某种消散不去的执念。

他不知道这是不是某种恶意的玩笑。系统绑定后的这四十分钟里，他试图研究这套所谓的“预言系统”到底是怎么回事，但除了那行不断变化的倒计时之外，他什么都得不到。

他尝试过查询系统——用意识在脑海中提问，比如“系统是什么”、“预言能否改变”、“苏清禾为什么要杀我”。系统只会回复一些冰冷的提示音，像是某种设定好的自动回复。

唯一明确的信息是：规避会加速死亡。

而他已经规避过一次了——在仓库里喊了那一声“快跑”之后，他的预计死亡时间从23:00变成了22:30。

提前了半小时。

陆沉掐灭烟头，扔进门口的垃圾桶。然后他深吸一口气，露出一个疲惫的笑容，推开门。

“回来了？”

苏清禾的声音从厨房传来，带着一贯的温柔。她的身影在油烟机嗡嗡的声音中若隐若现，哼着一首陆沉没听过的歌。

“回来了。”陆沉关上大门，换好拖鞋。他的目光不由自主地飘向苏清禾的头顶。

没有倒计时。

取而代之的，是那行血红的字：

【将在今晚23:00杀害你】

陆沉盯着那行字，感到一阵眩晕。他强迫自己移开视线，装作若无其事地走进客厅，一屁股坐在沙发上。

“今天怎么这么早？”苏清禾从厨房探出头，“我还在做饭呢，你先去坐会儿，饭马上好。”

“嗯。”陆沉应了一声，声音干涩。

他的视线一直盯着地板，不敢再看她头顶的那行字。但那行字像是印在他视网膜上一样，无论如何都挥之不去。

他在心里默念：也许这是个错误。也许系统出错了。也许……

“陆沉？”

他抬起头。苏清禾站在他面前，手里端着一盘菜，表情有些担忧。

“你脸色很不好。怎么了？”

“没事，”陆沉勉强笑了笑，“可能是最近太累了。”

“那就早点休息。”苏清禾把菜放在茶几上，俯下身摸了摸他的额头，“没发烧。可能是最近降温，着凉了。”

她的手很凉，触碰在陆沉的额头上，让他一个激灵。

就是这双手，会在今晚23:00杀死他吗？

陆沉抓住了她的手腕。

苏清禾愣了一下：“怎么了？”

“没什么，”陆沉松开手，“就是……想看看你。”

苏清禾笑了。她的笑容很温柔，带着一丝调侃：“怎么了？突然这么肉麻？”

“可能就是突然觉得……”陆沉顿了顿，喉咙有些发紧，“你挺好看。”

“贫嘴。”苏清禾点了点他的额头，直起身，“去洗手，准备吃饭了。”

陆沉看着她转身走回厨房的背影，眼神复杂。

那行字还挂在她头顶，像是一个永不消失的诅咒。

### 场景三

深夜11点47分。

陆沉站在阳台边缘，指尖夹着一根已经燃到一半的烟。城市的夜景在脚下蔓延，路灯像一条流动的星河，延伸到视线无法触及的远方。

他在这里已经站了两个多小时。

中间苏清禾出来过一次，问他为什么还不睡。他说在看星星——这个理由蠢得他自己都不相信，但苏清禾没有追问，只是给他披了一件外套，然后回了卧室。

她睡着了吗？陆沉不知道。

他现在脑子里一片混乱。系统的存在、预言的内容、苏清禾头顶的那行字……所有信息交织在一起，像是一团无法解开的乱麻。

他试着和系统对话。

“系统，你到底是什么？”

【犯罪预警系统已绑定。功能：预测犯罪行为及后果。】

“预测犯罪行为？那为什么显示的是……死亡时间？”

【死亡是犯罪行为的最严重后果。】

“那苏清禾呢？她要杀我？这算是犯罪行为？”

【预言已锁定。目标将在今晚23:00杀害宿主。】

陆沉烦躁地抓了抓头发：“那我该怎么办？等死？”

【系统无法提供建议。】

“屁话。”

他深吸一口气，掐灭烟头。城市夜晚的风带着深秋的寒意，吹在他脸上，让他稍微清醒了一些。

还有不到15分钟就是23:00。

在这最后的时刻，他反而平静了下来。也许是因为已经接受了命运的安排，也许是因为……

“也许她不会呢？”

这个念头刚冒出来，就被系统无情的提示音打断。

【检测到宿主试图规避预言。】

【警告：规避行为将加速死亡。】

【当前预计死亡时间：22:30】

陆沉苦笑。22:30，也就是说他还有不到四十分钟的时间。

系统说得对，他的每一次侥幸心理都在加速死亡的到来。如果他真的想反抗，也许从一开始就不该抱有任何幻想。

他转身准备回卧室。无论如何，他想在最后的时刻和她待在一起。也许……也许还有转机呢？

就在他转身的瞬间，他看到了从卧室走出来的苏清禾。

她穿着白色的睡袍，长发披散，看起来刚刚睡醒。月光照在她脸上，让她看起来有些不真实。

“这么晚了还不睡？”她的声音很轻，像是怕吵到邻居。

“就睡。”陆沉走过去，想要拥抱她。

但苏清禾侧身躲开了。

她的这个动作让陆沉的动作僵在了原地。

“怎么了？”他问，声音干涩。

苏清禾没有回答。她的视线越过陆沉的肩膀，看向阳台外面的夜空。

“你看今天的月亮，”她说，“是不是特别圆？”

陆沉愣了一下，不明白她为什么突然说这个。但他还是点了点头：“嗯。”

“我记得我们第一次见面的时候，也是这样的满月。”苏清禾的声音很轻，像是在自言自语，“那天晚上也是这样的月亮，也是这么晚……你还记得吗？”

“记得。”陆沉说，心里涌上一股不安。

苏清禾转过头，直视着他的眼睛。

“那天晚上，你跟我说，”她的声音突然变得很低，“你说……”

她顿了顿，嘴角浮现出一丝苦涩的笑意。

“你说，如果有一天我不在了，你会好好活着。”

陆沉的心脏猛地收缩。

“你……说什么？”

“没什么。”苏清禾笑了笑，笑容里带着陆沉看不懂的情绪，“进去吧，外面冷。”

她转身往卧室走。

陆沉站在原地，看着她的背影，突然开口：“清禾。”

“嗯？”

“如果……”他顿了顿，喉咙像是被什么东西堵住了，“如果我出了什么事，你会难过吗？”

苏清禾的脚步顿住了。

她没有回头。

“会。”她说，声音很轻但是很坚定，“会非常难过。”

“那就不要……”

“陆沉。”她打断他，转过身来，脸上带着一种复杂的表情——像是心疼，又像是决绝，“有些事情，不是我们能决定的。”

她的视线落在他的额头上，眼神闪烁了一下。

那一瞬间，陆沉几乎以为她看到了什么。

但她只是叹了口气，说：“进去吧。明天还要上班。”

然后她走进了卧室，轻轻关上了门。

陆沉在原地站了很久。

他低头看了看手表。22:58。

还有两分钟。

### 场景一

清晨8点，市公安局刑侦支队办公室。

陆沉准时打卡上班，看起来和平时没有任何区别。身上的警服熨烫得平整干净，领带打得一丝不苟——如果忽略他眼底淡淡的青黑色，这副模样完全就是个标准的警队精英。

他熬过了昨晚。

22:58的时候，他以为一切都要结束了。但苏清禾只是问了几句就回了卧室，之后再也没有出来。23:00，系统显示的“预言时间”悄然过去，没有任何事情发生。

陆沉不知道这是好事还是坏事。系统只是冰冷地提示：

【预言内容已更新。】

【预计应验时间：明晚23:00】

应验时间从“今晚”变成了“明晚”，但这并没有让陆沉感到轻松多少。该来的还是会来——也许苏清禾只是把计划推迟了一天。

“陆队！早啊！”

周扬的大嗓门从身后传来，打断了陆沉的思绪。他转过身，看到禁毒支队副队长周扬正朝他走过来，脸上挂着热情的笑容。

“早。”陆沉淡淡地点头。

“昨天那个案子怎么样？有什么进展吗？”周扬自来熟地凑过来，“我听说你们在现场发现了重要证据？”

陆沉看了他一眼。周扬是禁毒支队的，跟他们的案件本来没有直接交集，但最近因为赵翎的案子，两队有些业务交叉。

“还在调查中。”陆沉说，“有什么问题？”

“没什么没什么。”周扬摆摆手，笑容里带着点别的意味，“就是关心一下。对了陆队，明天营救赵翎的行动，你一定要来啊。”

“营救？”陆沉皱眉，“赵翎的案子不是你们禁毒在办吗？”

“话是这么说，但赵翎是重要证人，上头指定要刑侦配合。”周扬拍了拍陆沉的肩膀，“你是我们队最优秀的，有你在我放心。”

陆沉没有说话。周扬的的热情让他觉得有些不对劲。但具体哪里不对劲，他又说不上来。

“行，我先过去了。”周扬转身要走，又突然停下，“哎陆队，你未婚妻是法医部的吧？听说她最近经常请假？”

陆沉的眼神冷了一瞬：“你关心这个干什么？”

“没什么，随口问问。”周扬笑了笑，“那先忙。”

看着周扬离开的背影，陆沉的眉头皱得更紧了。

周扬刚才的话听起来像是随意聊天，但陆沉知道，赵翎的案子涉及机密，周扬不应该随便向刑侦的人透露行动细节。除非……

除非他想故意让他知道。

陆沉走进自己的办公室，关上门。他需要一个人静静。

桌上摊着昨天晚上的案件报告，但他一个字都看不进去。脑子里全是苏清禾头顶那行字，还有周扬刚才意味深长的表情。

他拿起手机，拨通了师傅陈国栋的电话。

“师傅，是我。”

“臭小子，终于想起给我打电话了？”陈国栋的声音从听筒里传出来，带着一贯的爽朗，“昨天一天都没见你人，跑哪去了？”

“有点事。”陆沉顿了顿，“师傅，你现在有空吗？我想见你。”

电话那头沉默了几秒。

“来我办公室。”

---

陈国栋的办公室在走廊尽头，地方不大但收拾得很整洁。陆沉进去的时候，陈国栋正站在窗边抽烟，看到他进来，把烟掐灭在烟灰缸里。

“坐。”陈国栋指指对面的椅子，“脸色这么差，昨晚没睡好？”

陆沉坐下去，犹豫了一下该怎么开口。

“师傅，”他低声说，“我身上的系统……又出现了。”

陈国栋的表情瞬间凝固。

“你说什么？”

“昨天傍晚，我绑定了一个系统。”陆沉详细地把昨天发生的事情说了一遍，包括系统显示的预言、苏清禾头顶的字，还有规避后的时间加速。

陈国栋听完之后，久久没有说话。

他起身走到门口，打开门左右看了看，确认走廊里没有人。然后他关上门，回到办公桌前坐下，点了根烟。

“七年前。”陈国栋缓缓开口，“你导师死之前，也出现过类似的情况。”

陆沉心里咯噔一下。

“导师？”

陈国栋深吸一口烟，烟雾缭绕中，他的表情变得有些模糊。

“你导师是七年前意外死亡的——官方说法是心脏病发。但我知道真相。他死前一个月，开始变得神神叨叨的，经常说一些我听不懂的话。他说有人在他脑子里放了一个'东西'，能看到人的死亡时间。”

陆沉的心脏狂跳起来。

“后来呢？”

“后来他就死了。”陈国栋掐灭烟头，“官方结论是过度劳累导致的心脏病发。但我始终觉得不对。小沉……”

他抬起头，直视着陆沉的眼睛。

“你导师死之前，说过一句话。他说——'有些东西，不该被人类知道。'”

办公室里的空气仿佛凝固了。

陆沉沉默了很长时间，才开口：“师傅，你觉得这个系统……是哪里来的？”

“我不知道。”陈国栋摇头，“但我可以告诉你一件事。”

“什么？”

“苏清禾那孩子，”陈国栋的声音压得更低了，“她父亲是我的老战友，三年前因公殉职。她是个孤儿，由我和你阿姨看着长大的。”

陆沉愣住了。

他从来不知道这件事。苏清禾从来没有跟她提过她的家庭。

“所以，”陈国栋继续说，“我可以保证，清禾这孩子不是坏人。但她最近……确实有些不对劲。”

“师傅知道她在做什么？”

“我不知道。”陈国栋叹息，“她只是偶尔会来找我，打听一些赵翎案子的细节。小沉，有些事……知道得太多未必是好事。你明白吗？”

陆沉点头，但心里已经下定了决心。

无论苏清禾在做什么，无论她背后有什么秘密，他都要查清楚。

不是为了打破那个预言，而是为了……

为了证明，系统看见的未来，未必就是真正的未来。

---

### 场景二

中午12点30分，市公安局停车场。

陆沉借口买咖啡，从单位溜了出来。他没有开车，而是步行离开了公安局，然后在两条街外打了一辆出租车。

“师傅，去锦绣家园。”

那是昨天苏清禾去过的那个老旧小区。

出租车开了十五分钟，停在了锦绣家园门口。这是一个建于上世纪九十年代的老小区，外墙斑驳，绿化带里杂草丛生，住的大多是老年人和租户。

陆沉付了钱，下车环顾四周。

小区门口没有门卫，只有一道锈迹斑斑的铁门。他走进去，按照记忆中的路线，找到了3栋。

3栋是一栋六层高的居民楼，外墙是暗淡的水泥灰色，楼道口的铁门同样锈迹斑斑。

501室。

陆沉站在单元门口，抬头往上望去。501的窗帘拉着，从外面什么都看不到。阳台上有几盆花，看起来像是有人住的样子。

他走进楼道。

楼梯间的灯是坏的，陆沉只能摸着黑往上走。501室门口装着一个摄像头，正对着楼梯口。

陆沉在摄像头下方停住脚步，抬起头。

这种老式小区的监控通常都是摆设，但他不想冒险。他从口袋里掏出一张名片——这是技术组小王的名片，专门搞监控破解的。

名片背面有一个微型摄像头屏蔽器的贴片，撕下来贴在衣领上，可以干扰三米内的监控信号。

陆沉把贴片贴好，然后按下501的门铃。

没有回应。

他又按了几次，还是没人开门。就在他准备放弃的时候，身后突然传来脚步声。

“小伙子，你找谁？”

陆沉转过身，看到一个六十多岁的老太太正端着菜篮子站在楼梯口，警惕地看着他。

“大妈，”陆沉出示了一下警官证，“我是警察。请问这家住的是什么人？”

“警察？”老太太狐疑地看了他一眼，“501？那户人家搬走半年多了。”

“搬走了？”陆沉皱眉，“那现在是谁住？”

“不知道，反正没见人回来过。”老太太摇头，“哎小伙子，那户人家可邪门了，半夜经常有人进进出出的，吵死人了。”

“半夜？”

“就是最近几个月的事。”老太太压低声音，“我晚上起来上厕所，经常听到楼道里有脚步声，还有说话声……但是打开门看，又什么都没看到。你说邪门不邪门？”

陆沉心里一动。

“大妈，这户人家之前住的是什么人？”

“说是叫什么……赵教授？”老太太想了想，“对，赵教授，听说是农大的教授。后来不知道犯了什么事被抓起来了，房子就空着了。”

赵教授。

赵翎。

陆沉的心沉了下去。

“大妈，谢谢你。”

他转身往外走，心里已经大概猜到了是怎么回事。苏清禾来这里的次数不少——也许她是在暗中调查赵翎，或者……

或者是在秘密接触什么人。

陆沉走出3栋单元门，突然看到一辆熟悉的车从小区门口驶过。

苏清禾的车。

陆沉迅速闪身躲到一棵大树后面，看着那辆车缓缓停在3栋前面的停车位上。

苏清禾从车上走下来。

她看起来和平时一样，穿着白大褂——那是法医部的工作服。她左右看了看，确定没人注意，然后走向3栋的单元门。

她的手里，提着一个黑色的袋子。

陆沉躲在大树后面，默默地看着她的背影消失在单元门里。

他在等。

大约过了十五分钟，苏清禾从单元门里走出来。她的脸色看起来有些苍白，额头上还有细细的汗珠。

她左右看了看，然后快速走向自己的车。

就在她拉开车门的瞬间，她的动作突然顿住了。

她抬起头，看向了陆沉藏身的方向。

两人的视线在空中交汇。

陆沉知道自己暴露了。他从树后面走出来，表情复杂地看着苏清禾。

苏清禾站在原地，没有任何惊慌的表情。

她只是静静地看着他，嘴唇微微动了动。

两人对视了很久。

最后，苏清禾打开车门，坐了进去。

她降下车窗探出头来，看着陆沉。

“上车。”她说，声音平静，“我们回家说。”

---

### 场景三

下午3点，锦绣家园3栋501室。

苏清禾打开门走进去，室内的光线很暗，窗帘拉着，几乎不透光。她打开灯，环顾四周。

这是一个两室一厅的老房子，家具很简单，只有一张桌子和几把椅子。墙上挂着几幅已经被灰尘覆盖的壁画，看起来很长时间没有人住过了。

她走到其中一个房间，推开门。

房间里除了一张床，什么都没有。床上的被褥叠得整整齐齐，像是有人定期打扫。

她把手中的黑色袋子放在桌上，从里面拿出来一部老式手机、一叠现金，还有一个小型的USB存储设备。

然后她打开手机，拨通了一个号码。

“是我。”她说，声音压得很低，“计划有变。他发现了。”

电话那头不知道说了什么，苏清禾的表情变得有些难看。

“我知道轻重。”她说，“但他已经开始怀疑了。再这样下去会出问题。”

沉默了几秒。

“好，我尽量。”她挂断电话，把手机关机，然后拿起那叠现金和存储设备。

这些东西不能带回家。

她把东西塞进墙角的缝隙里藏好，然后走出房间。

就在她准备离开的时候，门外突然传来了脚步声。

苏清禾的表情瞬间变得警惕。她从口袋里掏出一把小型折叠刀，握在手里。

门被推开了。

一个陌生的中年男人站在门口，穿着快递公司的制服，戴着帽子和口罩，遮住了大半张脸。

“你是谁？”苏清禾冷声问。

男人没有说话，只是缓缓摘下口罩。

看清他的脸，苏清禾的表情瞬间变得煞白。

“是你……”

男人笑了笑，笑容里带着说不出的阴冷。

“好久不见，苏法医。”他说，“看来你还记得我。”

苏清禾握紧了手中的刀。

“你来干什么？”

男人向前迈了一步。

“当然是来提醒你，”他说，“别忘了你的任务。”

“我的任务完成了没有，跟你没关系。”

“呵。”男人冷笑，“组织很关注你的进度。你已经拖了太长时间了。”

“要你管。”

男人又往前走了一步，距离苏清禾只有两步之遥。

“别忘了你哥哥是怎么死的。”他的声音突然变得阴森，“如果不想再发生那种事，就按计划行动。”

苏清禾的身体明显颤抖了一下。

她的眼眶瞬间红了，但声音依然强硬。

“滚。”

男人看了她一眼，似乎对她的反应很满意。

“明晚之前，”他说，“你应该知道该怎么做。”

然后他重新戴上口罩，打开门，头也不回地走了。

苏清禾站在原地，握刀的手在微微发抖。

她的眼泪终于忍不住掉了下来。

“哥哥……”她喃喃自语，声音哽咽，“我一定会为你报仇的……”

不知道过了多久，她才收拾好情绪，擦干眼泪，走出了501室。

下楼的时候，她的手机响了。

是陆沉打来的。

苏清禾看着屏幕上跳动的名字，犹豫了很久，最后还是接通了。

“在哪？”陆沉的声音听起来很疲惫。

“单位。”她顿了顿，“怎么了？”

“没什么，”他说，声音很轻，“就是想问问你今晚几点回来。”

苏清禾闭上眼睛。

“八点。”她说，“我尽量早一点。”

“好。”

电话挂断了。

苏清禾站在楼道里，握着手机的手指微微发白。

她不知道陆沉发现了多少。但她知道，计划必须加快进行了。

明晚。

明晚就是最后的期限。

## 正文

### 场景一

下午2点，陆沉和苏清禾的家中。

苏清禾说八点才回来，但现在才两点。距离她回家还有六个小时，足够陆沉做很多事情了。

陆沉站在书房门口，犹豫着要不要进去。

这里是苏清禾的书房——虽然两人同居三年，但这间房间她一直锁着，理由是里面有很多法医专业的资料，不方便让他看到。

陆沉当时没有在意。现在想来，一切都有迹可循。

他从口袋里掏出一根铁丝。这是当警察这行必备的技能，开这种普通的门锁不在话下。

门锁发出轻微的咔哒声，开了。

陆沉推开门，走了进去。

书房不大，只有十几个平方。一面墙是书架，摆满了各种法医学的书籍。另一面墙上挂着一幅画，是梵高的《星空》，苏清禾最喜欢的作品。

书桌在窗边，上面摆着一台笔记本电脑和几个文件夹。

陆沉没有动电脑——苏清禾是法医，电脑里可能有很多敏感资料，他不想留下指纹或者其他痕迹。

他把注意力放在那几个文件夹上。

文件夹是普通的文件夹，没有任何标记。陆沉打开第一个，里面是一些纸张文件。他翻看起来，呼吸渐渐变得急促。

这些是赵翎的调查记录。

详细到令人发指。

赵翎的个人资料、社会关系、行动轨迹、医疗记录……甚至连他每天几点起床、几点吃饭、几点出门买报纸都记得清清楚楚。

陆沉继续往下翻，然后看到了让他血液凝固的内容。

那是一张手绘的地图，标注着几条路线。旁边写着“押运路线A/B/C”，还有各种备注——哪里有监控、哪里是盲点、哪里可以设置路障。

这完全是针对赵翎押运行动的策划书。

陆沉的手开始颤抖。

他翻开第二个文件夹。

里面是另一些资料，但这次是关于他本人的。

陆沉的出勤记录、他最近在处理的案件清单、他每天的行动轨迹……甚至还有他下周的工作安排。

文件夹最下面，压着一张纸。

那是一张手写的计划表，上面密密麻麻写满了字。

陆沉把纸拿出来，凑到眼前仔细看。

【行动人员：陆沉】

【时间：10月2日 09:00】

【地点：云山路】

【方式：】

后面的内容被涂掉了，但前面的信息已经足够让陆沉明白——这是针对他的行动计划。

明天的行动。

陆沉把文件放回原处，心里涌上一种复杂的情绪。

愤怒？恐惧？还是……

悲伤？

他不知道。但有一点很清楚——苏清禾确实在策划什么，而且和她说的“加班”完全无关。

他的视线落在书桌的抽屉上。

那个抽屉上着锁。

陆沉撬开锁，打开抽屉。

里面是一个黑色的笔记本。

他拿出来，翻开第一页。

笔记本里是密密麻麻的文字，记录着一些……陆沉看不太懂的东西。

“收割者”、“证人保护计划”、“转移时间”、“组织目标”……

这些词陆沉都认识，但组合在一起，他就完全不知道是什么意思了。

唯一清楚的是，这些内容和一个叫“收割者”的组织有关。

还有一页，引起了陆沉的注意。

那上面写着：

【警告：目标已发现异常。计划需提前。】

【最终方案：23:00前完成。】

23:00。

又是23:00。

陆沉合上笔记本，放回原处，然后把抽屉锁好。

他需要时间消化这些信息。

但首先，他要把这些东西放回原位——不能让苏清禾发现他进来过。

陆沉把书房恢复成原来的样子，轻手轻脚地走出去，关上门。

然后他坐在客厅的沙发上，等待苏清禾回家。

### 场景二

傍晚6点30分，门锁转动的声音。

苏清禾打开门，踩着高跟鞋走进玄关。她看起来有些疲惫，但整体状态还好。

“回来了？”陆沉坐在沙发上，头也不抬地说。

“嗯。”苏清禾换上拖鞋，抬起头，“你今天怎么这么早？”

“想你了。”陆沉终于抬起头，看着她。

苏清禾的动作顿了一下。

“怎么了？突然这么肉麻。”她笑了笑，走过来，“吃了吗？”

“还没。等你一起吃。”

苏清禾看了他一眼，眼神里带着一丝审视。

“你今天很奇怪。”

“有吗？”陆沉站起身，走向她。

“有。”苏清禾没有动，“你平时可不会说'想你了'这种话。”

陆沉走到她面前停下。

“那我该说什么？”

“比如'回来了'、'辛苦了'之类的。”苏清禾嘴角带着笑，“你今天吃错药了？”

“可能吧。”陆沉伸出手，搭在她的肩膀上，“让我抱一下。”

苏清禾没有拒绝。

她的身体很凉，可能是在外面吹了太久的风。陆沉抱着她，闻到她身上有淡淡的消毒水味——那是法医部特有的味道。

但除此之外，还有另一种味道。

血腥味。

很淡，但他确定自己没有闻错。

“你受伤了？”陆沉松开她，皱眉问。

“没有啊。”苏清禾愣了一下，“怎么了？”

“你身上有血腥味。”

苏清禾的表情僵了一瞬，但很快就恢复正常。

“可能是在单位沾到的吧。”她说，“今天有一具尸体送来，腹部被刺伤，血流了很多。”

“是吗。”陆沉没有追问。

他知道她在说谎。

那个血腥味不是尸体上的——尸体的血腥味不会那么淡，也不会带着一种……温热的感觉。

那更像是刚沾上不久的血。

“去换衣服吧，”陆沉说，“我去做饭。”

苏清禾看了他一眼，点点头，走进了卧室。

陆沉看着她的背影消失在门后，眼神变得复杂起来。

### 场景三

晚上8点，市区某条僻静的小巷。

陆沉把车停在路边，点燃一根烟。烟雾在车厢里缓缓升起，让他的表情变得有些模糊。

副驾驶座上坐着陈国栋。

两人沉默了很久，最后还是陈国栋先开口。

“查到了什么？”

陆沉把下午在书房看到的一切都告诉了师傅，包括那个笔记本、那些文件、还有赵翎的押运计划。

陈国栋听完之后，沉默了很久。

“你确定你看清楚了？”

“确定。”陆沉说，“那些文件不是假的。”

陈国栋叹了口气。

“小沉，有些事……我本来不想让你卷进来的。”

“师傅知道什么？”

“我知道'收割者'这个组织。”陈国栋的声音很低，“这是一个跨国犯罪组织，专门渗透各国的司法系统，暗杀关键证人。他们已经暗杀了至少七个重要证人了。”

陆沉心里咯噔一下。

“赵翎……”

“赵翎是唯一活下来的。”陈国栋说，“他掌握着这个组织的核心证据。所以他们一定要除掉他。”

“那清禾她……”

陈国栋摇头。

“清禾这孩子……我也不知道她在里面扮演什么角色。”他说，“我只知道她的身份不简单。具体情况，只有她自己知道。”

“师傅……”

“小沉，”陈国栋打断他，“有些事，不是你想的那样。清禾她……唉。”

他顿了顿，似乎在犹豫要不要说。

“清禾她，可能有苦衷。”

“苦衷？”

陈国栋没有回答，而是转移了话题。

“明天赵翎的转移行动，我怀疑有内鬼。”他说，“但我没想到这个人会是清禾。”

“你怀疑她？”

“我不知道。”陈国栋摇头，“但如果那些文件是真的，那她确实在策划什么。”

陆沉沉默了很久。

“师傅，”他突然问，“如果她真的在策划什么……你觉得她的目的是什么？”

陈国栋看了他一眼。

“你现在还爱她吗？”

陆沉愣住了。

“爱。”他说，声音很坚定，“我爱她。”

“那就对了。”陈国栋说，“也许她的目的，根本就不是伤害你。”

“你是说……”

“小沉，”陈国栋打断他，“有时候，眼睛看到的未必是真相。”

他推开车门，准备下车。

“师傅，”陆沉叫住他，“你知道清禾的哥哥吗？”

陈国栋的身体僵了一下。

“你怎么突然问这个？”

“今天在她的书房里，看到了一些东西。”陆沉说，“她有个哥哥？”

陈国栋沉默了几秒。

“死了。”他说，“三年前，意外死亡。”

“三年前？”

“具体情况我不清楚，”陈国栋说，“但清禾一直认为她哥哥的死另有隐情。这可能就是她……卷进这些事情的原因。”

他说完，下车走了。

陆沉坐在车里，看着师傅的背影消失在夜色中，心里涌上一种说不清的情绪。

也许陈国栋说得对。

也许苏清禾做的事情，并不是他想的那样。

但不管她的目的是什么，有一个事实无法改变——

系统预言，她会在明晚11点，杀害他。

陆沉发动车子，汇入了城市的车流中。

苏清禾背对着陆沉，侧躺在床上。月光从窗外照进来，在她的侧脸上投下一片阴影。

“陆沉。”她突然开口。

“嗯？”

“你……爱我吗？”

陆沉沉默了几秒。

“爱。”

“那就够了。”苏清禾的声音很轻，“别恨我。真的。”

陆沉想问为什么，但话到嘴边又咽了回去。

有些问题，也许永远没有答案才是最好的。

窗外的月亮很圆，像极了他们第一次见面的那个晚上。

### 场景一

晚上9点，客厅。

墙上的时钟指向9点整。

陆沉坐在沙发上，看着秒针一下一下地走着。每一下都像敲在他的心上，让他感到一种说不出的压迫感。

距离苏清禾说要“杀害”他的时间，还有不到14个小时。

14个小时，说长不长，说短不短。足够做很多事情——包括杀人。

陆沉打开手机，搜索一些关键词。

“杀人犯的心理特征”、“如何识别要杀你的人”、“谋杀未遂的征兆”……

一条一条地看下去，越看越觉得自己可笑。

他可是刑侦副队长，侦破过上百起凶案。现在却在这里像普通市民一样搜索“如何识别杀手”？

但他不得不承认，这种未知的恐惧，比任何已知的情况都让人煎熬。

至少面对罪犯的时候，他能知道对方想要什么。

但苏清禾……

他完全不知道她想要什么。

“咔哒。”

门口传来门锁转动的声音。

陆沉迅速关掉手机屏幕，抬起头。

苏清禾推开门，提着一个纸袋走进来。

“还没睡？”她换着拖鞋，随口问。

“等你。”陆沉说。

苏清禾的动作顿了一下。

“等我？”她抬起头，嘴角带着一丝笑意，“干嘛等我？”

“想你了。”

“又来。”苏清禾笑着摇头，把纸袋放在玄关，“我给你带了点夜宵，楼下那家烧烤店的烤串，你最喜欢吃的那家。”

陆沉看着那个纸袋，心里涌上一股复杂的情绪。

她还记得他喜欢吃什么。

一个要杀你的人，会记得你喜欢吃什么吗？

“谢谢。”他说，声音有些干涩。

苏清禾走过来，在他身边坐下。

“今天怎么样？工作顺利吗？”

“还行。”陆沉点头，“你呢？”

“我也还行。”苏清禾靠在他肩膀上，“就是有点累。”

她闭上眼睛，看起来确实很疲惫。

陆沉闻到她身上有酒味。

“你喝酒了？”

“陪客户喝了一点。”苏清禾说，“法医部最近接了个大案子，应酬比较多。”

她的解释听起来天衣无缝。

但陆沉知道，她在说谎。

那个酒味不是陪客户喝的那种应酬酒——更像是烈酒，而且不是市面上的品牌。

他没说话，只是伸出手臂揽住她的肩膀。

“清禾。”

“嗯？”

“如果……”他顿了顿，“我是说如果，有一天你发现我做了什么对不起你的事，你会怎么样？”

苏清禾睁开眼睛。

“为什么要问这个？”

“随便问问。”

她沉默了几秒，然后笑了。

“那要看是什么事了。”

“如果是很严重的事呢？”

苏清禾直起身，转过来面对他。

“有多严重？”

“比如……”陆沉看着她的眼睛，“杀人。”

房间里的空气仿佛凝固了。

苏清禾的表情没有任何变化。

“你在说什麼？”她笑着问，“杀人？你吗？”

“假设。”

“没有意义的假设。”苏清禾重新靠回他肩膀，“你不会杀人，我也不會。”

“你怎么知道？”

“因为我了解你。”她说，声音很轻但很坚定，“你是个好人。”

陆沉没有再说什么。

好人。

也许吧。

但他现在已经不是一个“好人”该有的样子了——他撬开了自己未婚妻的书房，偷看了她的秘密，还准备了一套应对方案。

如果这都不算背叛，那什么才是？

“早点休息吧。”苏清禾站起来，“明天你不是还有行动吗？”

“嗯。”

“记得小心点。”

苏清禾转身走进卧室。

陆沉看着她的背影消失在门后，眼神变得复杂起来。

### 场景二

晚上10点30分，市公安局门口。

陆沉借口要回单位拿东西，从家里溜了出来。他没有告诉苏清禾自己去哪——反正她也不关心，或者说她根本不在意。

实际上，苏清禾吃完夜宵就上床睡觉了，看起来真的很累。

陆沉开车来到单位，但没有进大门。他把车停在附近的停车场，然后步行走到市局侧面的一个小门。

这里是技术组的专用出入口。

他需要找林小满。

林小满是痕检技术员，也是苏清禾在单位里最好的朋友。如果苏清禾最近有什么异常行为，她应该会知道。

小门旁边有一个监控盲区，陆沉从那里翻墙进去——对他来说这不算什么。

技术组在二楼东侧，24小时有人值班。

陆沉走到门口，看到里面亮着灯。一个年轻的技术员坐在电脑前，正在处理什么文件。

“小王。”陆沉打招呼。

“陆队？”小王抬起头，“这么晚了你来干嘛？”

“查点东西。”陆沉走过去，“林小满呢？”

“你找林姐？她刚走。”

“走了？”陆沉皱眉，“去哪了？”

“她说身体不舒服，请假先回去了。”小王说，“大概走了十几分钟吧。”

陆沉转身就往外走。

“陆队？你干嘛去？”

“没事。”

他冲出技术组大楼，快步往停车场走。

林小满身体不舒服？请假？

不对。

陆沉突然想起来，今天下午他在单位门口遇到林小满的时候，她还好好的，没有任何不舒服的样子。

而且她看他的眼神……

当时他没注意，现在回想起来，那眼神里带着一丝……

愧疚？

还是别的什么？

陆沉打开手机，拨通林小满的电话。

响了几声之后，电话通了。

“陆队？”林小满的声音听起来有些紧张，“这么晚了有什么事吗？”

“你在哪？”

“我……我在家啊。”

“哪个家？”

电话那头沉默了几秒。

“陆队，你到底想问什么？”

陆沉停下脚步。

“清禾最近……有没有什么异常？”

“异常？”林小满的声音提高了半度，“没有啊，她挺好的。”

“小满。”陆沉的声音沉了下来，“你骗不了我。说吧，她到底在干什么？”

又是沉默。

然后，林小满的声音低了下来。

“陆队，”她说，“有些事……我不能说。”

“为什么？”

“因为……”她顿了顿，“清禾会杀了我的。”

“哈？”

“总之，你別再问了。”林小满的声音带着哭腔，“清禾她……她最近真的很奇怪。她经常一个人发呆，有时候还会突然消失几个小时……我还以为她在……”

“在什么？”

“没什么！”林小满突然提高了音量，“陆队，我真的不能说。你别问了。”

然后电话就被挂断了。

陆沉站在夜风中，看着手机屏幕，眉头紧锁。

林小满最后的反应……

“他还以为她在……”

在什么？

陆沉突然有了一个不好的猜测。

他重新拨通林小满的电话。

这次响了很多声她才接。

“还有什么事？”

“如果，”陆沉压低声音，“清禾要对我做什么……你知道吗？”

电话那头彻底的沉默。

然后，林小满的声音响起：

“陆队，”她几乎是咬着牙说，“明天晚上10点之前，离开这座城市。”

“为什么？”

“别问那么多。”她的声音在发抖，“相信我，这是为了你好。”

“清禾她……”

“嘟——”

林小满挂断了电话。

陆沉站在原地，夜风吹在他脸上，让他感到一阵刺骨的寒意。

明天晚上10点之前，离开这座城市。

也就是说，明天晚上10点，就是苏清禾计划动手的时间？

不对，系统显示的是11点……

陆沉突然意识到一个问题。

系统显示的时间，可能不是准确的。

它只是给出一个“预言”，但实际发生的时间，可能会因为各种因素而改变——包括他的规避行为。

也就是说，他每一次试图逃避，都在加速死亡的倒计时。

现在，预计死亡时间已经从23:00变成了……

他不知道。

但肯定比23:00更早。

### 场景三

晚上11点，卧室。

陆沉轻手轻脚地推开门，走进卧室。

苏清禾侧躺在床上，看起来已经睡着了。月光从窗外照进来，在她脸上投下一片柔和的光影。

陆沉站在床边，静静地看着她。

她的睫毛很长，呼吸很均匀，看起来很平静。

但他知道，这只是表面。

他需要找到决定性的证据。

说服自己的证据。

说服自己，她真的要杀他。

陆沉转身走出卧室，来到玄关。

苏清禾的包挂在衣架上——这是她每天用的那个包，黑色的皮质，单肩带，款式很简单。

陆沉把包拿下来，拉开拉链。

里面是一些日常用品：手机、钱包、钥匙、口红、粉底……

很正常。

他把东西一件一件拿出来，放在地上。

还是没有。

难道是他想错了？

不，不可能。

系统的预言不可能出错——它已经证明了这一点。

那么证据在哪？

陆沉把手伸进包的夹层。

那里有一个暗袋。

他把手伸进去，指尖触到了冰凉的金属。

陆沉的心脏猛地收缩。

他把东西掏出来。

一把手术刀。

刀刃大约十五厘米长，锋利得在月光下闪着寒光。刀柄上有防滑设计——这是专业的外科手术刀，不是普通的水果刀。

陆沉放下手术刀，继续往包里掏。

然后是一卷绳索。

很细，但很结实——那种可以轻易勒死人的绳索。

是一个小最后玻璃瓶，里面装着透明的液体。

陆沉把瓶子举到眼前，借着月光看。

标签已经被撕掉了，但他闻了一下——是某种化学药品，具体是什么他不清楚。

三样东西。

齐全的杀人工具。

陆沉把东西一件一件塞回包里，放回原处。

他的手指在发抖。

不是因为害怕，而是因为……

愤怒？

悲伤？

他自己也说不清。

原来一切都是真的。

系统没有骗他。

苏清禾真的要杀他。

陆沉站在玄关，看着那扇虚掩的卧室门。

现在只要推门进去，把她叫醒，质问她就可以了。

然后呢？

没有倒计时。

没有预言。

只有苏清禾的哭声和他的心跳。

那种关切……

那种紧张……

确实不太正常。

### 场景一：真相

晚上11点30分，陆沉家中。

客厅里只开了一盏台灯，昏黄的光线在墙上投下暧昧的影子。

陆沉和苏清禾面对面坐着，中间隔着一张茶几。

气氛凝重得几乎要结冰。

"开始吧。"陆沉说，"把所有事都告诉我。"

苏清禾低头看着自己的手，沉默了很长时间。

"我该怎么开始……"

"从最开始的地方。"陆沉说，"你到底是谁？"

苏清禾深吸一口气。

"我是一名警察。"她说，"但不是法医。"

陆沉点头，这个答案在他意料之中。

"我的真实身份是安全部门特别行动组的成员。"苏清禾继续说，"三年前，我奉命接近你。"

"监视我？"

"观察你。"苏清禾说，"组织需要一个能够接近你的人，而我正好符合条件。"

"什么条件？"

“法医的身份，和……"苏清禾犹豫了一下，"你的类型。"

陆沉冷笑。

"所以你故意变成我喜欢的样子？"

"不。"苏清禾摇头，"我爱你是真的。"

陆沉的心脏猛地收缩。

"你……"

"我从一开始就知道这是任务。"苏清禾说，"接近你，接近系统，观察预言的发展。但是我没想到……"

"没想到什么？"

"没想到我会真的爱上你。"苏清禾的声音很低，"这不在计划里。"

陆沉默然。

他不知道该怎么回应。

"你还记得我们第一次见面吗？"苏清禾问。

"记得。"陆沉说，"市局的技术鉴定科。"

"对。"苏清禾点头，"那天是我刻意安排的。我想看看你到底是什么样的人。"

"然后呢？"

"然后我发现，你和我想象的不一样。"苏清禾说，"你很温柔，很正直，也很……傻。"

"傻？"

"你还记得那次追捕逃犯的事吗？"苏清禾问，"你一个人追了三条街，差点把腿摔断。"

陆沉当然记得。

"我只是履行职责。"

"但在我看来，那就是傻。"苏清禾说，"世界上怎么会有这么不要命的人。"

陆沉没有说话。

"后来我们在一起了。"苏清禾继续说，"我越来越发现自己离不开你。每次看到你笑，我就会忍不住想陪你笑。看到你皱眉，我就会想帮你分担。"

"所以你就假装法医，假装爱我？"

"一开始是假装。"苏清禾说，"但后来不是了。"

她抬起头，看着陆沉的眼睛。

"陆沉，我是真的爱你。"

陆沉感到自己的心脏被什么东西攥紧了。

"如果你真的爱我，"他说，"为什么要杀我？"

苏清禾闭上眼睛。

"因为预言。"

"预言说我会死。"

"对。"苏清禾说，"系统绑定在你身上的那一刻，我就看到了预言。"

"你看到了？"

"嗯。"苏清禾点头，"我的系统和你的不一样。我只能看到片段，而不是完整的预言。"

"什么片段？"

"我看到你倒在血泊里。"苏清禾的声音在颤抖，"看到救援队抬着担架，看到……看到你闭上眼睛。"

陆沉的心沉了下去。

"然后呢？"

"然后我开始调查。"苏清禾说，"我利用组织的资源，查到了'收割者'的真正目的。他们在研究一种可以预知未来的系统，而你，是他们的实验品。"

"实验品……"

"对。"苏清禾说，"你导师是第一个实验品，我是第二个，你是第三个。但是你的系统，比我们的都要强大。"

"什么意思？"

"你的系统可以看到完整的预言。"苏清禾说，"包括死亡时间、死亡方式。而我只能看到片段。"

陆沉想起他看到的那些倒计时。

那些在他眼前归零的生命。

"所以组织想得到我的系统。"

"对。"苏清禾说，"他们想把你带回去，研究你的大脑，然后批量生产这种系统。"

"但是你阻止了他们。"

"我试过。"苏清禾说，"但我发现，预言是无法打破的。"

她睁开眼，眼神里带着绝望。

"我试过让你远离这个案子，试过让你辞职，甚至试过让你离开这座城市。但每次这样做，预言就会加速应验。"

陆沉想起他之前试图规避的时候，系统显示死亡时间提前了。

原来如此。

"所以你只能用最后一个办法。"

"对。"苏清禾说，"让预言换个方式应验。"

"假死。"

"对。"苏清禾的声音很低，"在预言应验之前，我先'杀死'你。不是真的杀死，而是让你'死'一次。然后我们改名换姓，远走高飞。"

"你觉得这样可行？"

"我不知道。"苏清禾说，"但这是唯一的办法。"

陆沉沉默了很长时间。

"那然后呢？"他问，"你打算怎么处理你自己？"

苏清禾没有回答。

"你打算怎么办？"陆沉追问，"你杀了我之后，你自己去自首？还是逃一辈子？"

"我不知道。"苏清禾说，"我只想让你活下来。"

"那我爸妈呢？我妹妹呢？"陆沉的声音突然提高了，"他们会怎么看我？他们会以为我死了！你有没有想过他们！"

苏清禾低下了头。

"对不起……"

"对不起有什么用！"陆沉站起来，激动地在客厅里走来走去，"你知不知道你在做什么！你这是在毁掉我的生活！"

"我知道……"

"你不知道！"陆沉转身指着她，"如果你真的爱我，你不会背着我做这种决定！"

"那我应该怎么办！"苏清禾也站了起来，眼泪在眼眶里打转，"眼睁睁看着你死吗？"

"至少……"

"至少什么！"苏清禾哭了出来，"你知道我看到预言的时候是什么感觉吗！我看到你倒在地上，血流了一地！我看到你闭上眼睛，再也醒不过来！我每天都做噩梦，每天都在害怕失去你！"

她蹲下来，哭得像个孩子。

"我真的很害怕……我怕你出事……我怕组织杀你……我怕预言成真……我该怎么办……我该怎么办……"

陆沉看着眼前这个哭泣的女人，心里涌上无数复杂的情绪。

愤怒、心痛、怜惜、绝望……

所有情绪交织在一起，让他几乎无法呼吸。

"清禾……"

他走过去，蹲下来，把她抱进怀里。

苏清禾哭得更大声了。

"我真的很爱你……"她抽泣着说，"所以我才会这样做……我不奢求你原谅我……我只希望你活着……"

陆沉抱着她，什么都没说。

他能感觉到她的泪水浸湿了他的肩膀，能感觉到她的身体在颤抖。

这个说要杀他的女人，其实比任何人都爱他。

爱到愿意背负罪名，爱到愿意毁掉自己。

"清禾，"他终于开口，"我们一起面对。"

"什么？"

"我说，我们一起面对。"陆沉捧起她的脸，"不管预言说什么，不管组织要做什么。我们一起面对。"

"但是……"

"没有但是。"陆沉说，"如果你真的爱我，就不要为我做任何傻事。"

苏清禾泪眼婆娑地看着他。

"你真的愿意和我一起面对？"

"愿意。"陆沉说，"因为我也爱你。"

苏清禾再次哭了出来。

但这次是感动的泪水。

### 场景二：书房夜谈

凌晨1点，书房。

两人坐在书桌两侧，面对面。

苏清禾的情绪已经平静下来，但眼睛还是有些红肿。

"接下来我们怎么办？"她问。

陆沉思考了一会儿。

"首先，我需要知道组织的真正目的。"

"我刚才说了，他们想得到你的系统。"

"我知道。"陆沉说，"但他们为什么要研究这个系统？只是为了预知未来？"

苏清禾犹豫了一下。

"其实……我也不是很清楚全部。"

"把你知道的告诉我。"

苏清禾深吸一口气。

"'收割者'是一个跨国犯罪组织，表面上是一个洗钱集团，但实际上，他们渗透了多个国家的司法系统。"

"然后呢？"

"他们暗杀证人，操控审判，操纵选举。"苏清禾说，"只要是有利于他们的判决，他们就会不择手段地实现。"

"证人保护计划……"

"对。"苏清禾点头，"他们已经暗杀了七个关键证人了。每个都是在证人保护计划内被杀。"

"赵翎是第八个？"

"对。"苏清禾说，"赵翎手里有组织的核心名单和资金流向证据。只要他出庭作证，组织就会完蛋。"

"所以组织要杀他。"

"对。"苏清禾说，"但是上面决定提前转移他，打组织一个措手不及。"

陆沉想起周扬说的话。

这次行动提前了。

而且周扬怀疑有内鬼。

"清禾，"陆沉问，"你觉得周扬是内鬼吗？"

苏清禾犹豫了一下。

"我不确定。但是他最近行为很奇怪。"

"怎么奇怪？"

"他总是在问关于你的事。"苏清禾说，"你的行踪，你的系统能力，还有预言的内容。"

"他问这些干什么？"

"我不知道。"苏清禾说，"但我觉得他在向组织汇报。"

陆沉沉默了一会儿。

"明天我去试探他。"

"不要！"苏清禾突然抓住他的手，"太危险了！"

"我会小心的。"

"不。"苏清禾摇头，"如果他真的是内鬼，你去找他就是自投罗网。"

"那你说怎么办？"

苏清禾思考了一会儿。

"这样吧，我让我师父去查。"

"你师父？"

"对。"苏清禾说，"我的上线。他可以暗中调查。"

"好。"陆沉点头，"还有一件事。"

"什么？"

"许知行。"陆沉说，"他说他是组织的叛徒，想帮我们。是真的吗？"

苏清禾的表情变得复杂。

"我不确定。"

"为什么？"

"他以前确实是组织的人。"苏清禾说，"但是七年前他'死'之后，我就没有他的消息了。"

"你觉得他在骗我们？"

"我不知道。"苏清禾说，"但是他说的预言闭环……"

"怎么了？"

"我觉得他说的是真的。"苏清禾的声音很低，"预言确实是一个陷阱。"

陆沉想起许知行说的话。

除非有第三人愿意牺牲，否则预言会不断应验。

"那我们该怎么办？"

"我不知道。"苏清禾说，"但至少现在，预言没有显示。"

她指的是系统。

陆沉看向自己的前方。

没有倒计时。

没有预言。

"也许是因为我们和解了。"他说。

"也许。"苏清禾说，"但我不认为预言会这么容易失效。"

"什么意思？"

"它只是在等待机会。"苏清禾说，"等我们放松警惕的时候，给我们致命一击。"

陆沉沉默了。

他不知道该说什么。

"总之，"苏清禾站起来，"我们先按兵不动。等我师父调查清楚之后，再做决定。"

"好。"

苏清禾转身往外走。

"清禾。"

"嗯？"

"你今天……没事吧？"

苏清禾回过头，勉强笑了笑。

"我没事。"她说，"只是有点累。"

"那你去休息吧。"

"好。"

苏清禾走出书房。

陆沉坐在原地，看着她的背影消失。

不知道为什么，他突然有一种不安的感觉。

### 场景三：崩溃

凌晨2点30分，卧室。

陆沉躺在床上，辗转反侧，无法入睡。

苏清禾背对着他躺在一侧，呼吸均匀，像是睡着了。

但陆沉知道，她没睡。

她在装睡。

"清禾。"他轻声唤道。

没有回应。

"清禾？"

还是没有回应。

陆沉翻了个身，面向她。

月光下，他可以看到她紧锁的眉头，还有睫毛上未干的泪痕。

她在哭。

即使睡着了，她也在哭。

陆沉心里涌上一股说不出的难受。

他伸出手，想触碰她的肩膀。

但就在这时，苏清禾的身体突然抽搐了一下。

"清禾！"

陆沉坐起来。

苏清禾紧闭着眼睛，脸色苍白，额头布满冷汗。

"清禾！醒醒！"

他摇晃着她的身体。

苏清禾缓缓睁开眼睛，眼神涣散。

"陆沉……"她的声音很虚弱，"我……"

"你怎么了？"

"我……"她试图说什么，但声音越来越弱，"好冷……"

她的身体在颤抖。

陆沉摸了一下她的额头。

滚烫。

"你发烧了！"他说着就要下床，"我去拿退烧药！"

"不……"苏清禾抓住他的手腕，"不要走……"

"我不走。"陆沉重新坐回来，"我一直在这。"

苏清禾的意识开始模糊。

她感觉自己在一个黑暗的隧道里行走，前方看不到光亮。

"陆沉……"她喃喃地说，"对不起……"

"我……我真的……"

她的声音越来越小，最后完全消失了。

陆沉看着怀里的女人，心如刀割。

她的呼吸很微弱，身体很烫。

她到底怎么了？

陆沉突然想起一件事——

今天晚上，苏清禾在码头见了他之后，就和他一起回家了。

在这之前，她在做什么？

她在准备那个"计划"。

那个要杀死他的计划。

她是不是……

是不是在那之前，服用了什么药物？

陆沉的心猛地沉了下去。

如果苏清禾为了那个计划，已经提前服用了什么药物……

那现在药物发作……

他不敢往下想。

"清禾！"他抱起她，"坚持住！我送你去医院！"

苏清禾没有任何反应。

陆沉跳下床，迅速穿好衣服，然后抱起苏清禾冲出家门。

"清禾！清禾！"

他一边跑一边喊，但她始终闭着眼睛。

救护车。

需要叫救护车。

陆沉掏出手机，颤抖着拨打了120。

"喂！120吗！我未婚妻晕倒了！地址是……"

他报出了地址。

挂断电话，他抱着苏清禾坐在地上，让她的头靠在他的肩膀上。

"清禾，"他哽咽着说，"你不会有事的……你不会有事的……"

苏清禾的呼吸越来越微弱。

她的嘴唇开始发紫。

"清禾！"

陆沉疯了一样地喊着她的名字。

但她再也没有回应。

## 正文

### 场景一：医院

凌晨3点30分，市中心医院，急诊室。

苏清禾躺在病床上，脸色苍白如纸。护士刚刚给她打完点滴，正在整理医疗设备。

陆沉坐在床边，握着她的手。

她的体温还是很低，但呼吸已经平稳下来。

"她怎么样？"陆沉问护士。

"暂时脱离危险了。"护士说，"但需要住院观察。"

"谢谢。"

护士离开后，病房里只剩下他们两个人。

陆沉看着苏清禾苍白的睡颜，心里五味杂陈。

她到底怎么了？

为什么会突然晕倒？

是因为那个"计划"吗？

"陆沉……"

苏清禾缓缓睁开眼睛，声音虚弱。

"我在。"陆沉握住她的手，"你感觉怎么样？"

"我……"苏清禾试图起身，但失败了，"我没事……"

"别说话。"陆沉说，"好好休息。"

"不……"苏清禾摇头，"我有话要告诉你……"

"什么话？"

苏清禾深吸一口气，像是做了什么重大决定。

"我……"她的声音很轻，"我中毒了。"

"中毒？"

"嗯。"苏清禾闭上眼睛，"组织给我下的毒。"

陆沉的心猛地沉了下去。

"什么意思？"

"他们在我身上装了监控。"苏清禾说，"每次我和你在一起的时候，他们都知道。"

"所以……"

"所以他们给我下了毒。"苏清禾的声音在颤抖，"如果我完不成任务，就会毒发身亡。"

陆沉感到一阵眩晕。

"他们给你下了毒？什么时候？"

"就在我来见你之前。"苏清禾说，"许知行给我的。"

"许知行？"

"对。"苏清禾说，"他骗我，说只要我按计划杀了你，就会给我解药。"

"所以你……"

"我别无选择。"苏清禾的眼泪掉了下来，"我不想死……但我更不想你死……"

陆沉的心如刀割。

原来如此。

苏清禾不是真的要杀他，而是被组织胁迫。

她想用"假死"的方式打破预言，同时也是为了拿到解药。

但最后她选择了放弃。

她宁可自己死，也不愿意伤害他。

"清禾……"陆沉的声音沙哑，"你怎么这么傻……"

"我不傻。"苏清禾虚弱地笑了笑，"我只是爱你。"

陆沉说不出话来。

他只能紧紧握着她的手，仿佛一松开就会失去她。

"陆沉。"苏清禾突然说。

"嗯？"

"组织不会放过你的。"她的眼神变得坚定，"他们要抓你回去研究。"

"我知道。"

"所以你要小心。"苏清禾说，"特别是……"

她突然顿住了。

"特别是什么？"

"特别是你身边的人。"苏清禾说，"周扬……他真的是内鬼。"

陆沉的心沉了下去。

"他……"

"我亲耳听到的。"苏清禾说，"他在向组织汇报你的行踪。"

陆沉沉默了很久。

"我知道了。"

"还有……"苏清禾的声音越来越弱，"许知行……他不是好人……"

"我知道。"

"不，你不知道。"苏清禾摇头，"他……他想要你的系统……"

"什么意思？"

"他说……他说只要得到你的系统，他就可以……"苏清禾突然剧烈地咳嗽起来。

"清禾！"

陆沉帮她拍背。

"好了，别说了。"他说，"好好休息。"

"不……"苏清禾抓住他的手腕，"你一定要小心……他们要对你妹妹下手……"

"什么？"

陆沉的心脏猛地收缩。

"你说什么？"

"他们要绑架你妹妹……"苏清禾的声音越来越弱，"作为人质……逼迫你就范……"

陆沉松开苏清禾的手，站起来。

手机。

他的手机。

他快速掏出手机，拨打陆晚棠的电话。

"对不起，您拨打的电话已关机……"

再打。

还是关机。

陆沉的心沉到了谷底。

"清禾，"他尽量保持冷静，"你知道他们会把晚棠带到哪里吗？"

"城北……码头……"苏清禾的声音几乎微不可闻，"他们约你……今晚10点……"

又是城北码头。

"我知道了。"陆沉说，"你好好休息。我去找她。"

"陆沉……"

"嗯？"

"不要去……"苏清禾的眼泪掉下来，"那是陷阱……"

"我知道是陷阱。"陆沉说，"但她是我妹妹。"

他转身往外走。

"陆沉！"

苏清禾的声音从身后传来。

"小心周扬……"

陆沉停下脚步，但没有回头。

"我知道。"

他走出病房。

### 场景二：走廊

凌晨4点，医院走廊。

陆沉站在窗边，看着外面的夜色。

他的手机刚刚收到一条彩信。

照片上，陆晚棠被绑在一张椅子上，嘴巴被胶带封住，眼睛也被蒙住。周围是废弃仓库的背景。

照片下面有一行字：

"想让她活命，今晚10点之前，一个人来城北码头。不要报警，否则撕票。"

陆沉的手指在发抖。

他强迫自己冷静下来。

深呼吸。

再深呼吸。

现在不是慌张的时候。

他需要思考。

周扬是内鬼。

苏清禾说的。

许知行也不可信。

他需要帮助。

陆沉拨通陈国栋的电话。

"师傅。"

"小沉？"陈国栋的声音听起来很疲惫，"这么晚了什么事？"

"晚棠被绑架了。"

电话那头沉默了几秒。

"什么？"

"我妹妹被绑架了。"陆沉说，"绑匪要我今晚10点之前一个人去城北码头。"

"他妈的！"陈国栋爆了一句粗口，"你在哪？"

"中心医院。"

"清禾呢？"

"她在住院。"陆沉说，"中毒了。"

"中毒？"

"回去再跟您解释。"陆沉说，"师傅，我需要您的帮助。"

"你说。"

"我怀疑周扬是内鬼。"陆沉说，"您能不能帮我查一下他的背景？"

"周扬？"陈国栋的语气变得严肃，"你确定？"

"清禾亲口告诉我的。"陆沉说，"她听到周扬向组织汇报我的行踪。"

"我知道了。"陈国栋说，"我这就去查。你先不要轻举妄动。"

"我知道。"

挂断电话，陆沉靠在墙上，闭上眼睛。

城北码头。

又是那里。

苏清禾约他见面的是那里。

现在绑匪约他去的也是那里。

这是巧合，还是组织故意设的局？

陆沉突然意识到一个问题。

如果周扬是内鬼，那他今天的行动……

他今天没有参加营救行动。

这意味着什么？

营救行动是下午2点开始的。

他在那之前请假了，去了码头。

然后苏清禾晕倒，他把她送到医院。

也就是说，他完美地错过了营救行动。

这会是巧合吗？

还是说，从一开始，这就是组织的计划？

陆沉的心沉了下去。

如果营救行动出了什么问题……

如果赵翎死了……

那他就是最大的嫌疑人。

临阵脱逃，涉嫌渎职。

而现在他妹妹又被绑架。

这是一个完美的陷阱。

让他百口莫辩的陷阱。

"陆沉。"

一个声音从身后传来。

陆沉转过身。

陈国栋站在走廊尽头，正向他走来。

"师傅。"

陈国栋走到他面前，表情凝重。

"我查过了。"

"周扬？"

"嗯。"陈国栋点头，"他的账户最近有一笔海外汇款，金额很大。"

陆沉的心沉了下去。

"果然是他。"

"不只是这样。"陈国栋说，"我让人调取了周扬的通话记录，发现他最近经常联系一个神秘号码。"

"能查到是谁吗？"

"查不到。"陈国栋摇头，"对方用了加密技术。"

陆沉沉默了一会儿。

"师傅，您能不能帮我一个忙？"

"什么？"

"我想请您暗中保护清禾。"陆沉说，"我怀疑组织不会放过她。"

"我会的。"陈国栋说，"但是你……"

"我去找晚棠。"陆沉说，"今晚10点，城北码头。"

"那是陷阱！"

"我知道。"陆沉说，"但我必须去。"

陈国栋沉默了很久。

"好吧。"他说，"我陪你。"

"不。"陆沉摇头，"您帮我照顾清禾和晚棠。如果我回不来……"

"说什么屁话！"陈国栋打断他，眼眶红了，"你他妈的给我活着回来！"

陆沉笑了。

"我尽量。"

他转身往外走。

"小沉！"

陈国栋的声音从身后传来。

"嗯？"

"小心许知行。"陈国栋说，"他不是个省油的灯。"

"我知道。"

陆沉走进电梯。

### 场景三：天台

凌晨5点30分，医院天台。

陆沉站在天台边缘，看着脚下的城市。

天还没亮，街道上只有零星的灯光。

他的手机突然响了。

是一条短信。

"如果你想救你妹妹，今晚10点，一个人来城北码头。不要耍花招。"

没有署名。

但陆沉知道是谁发的。

组织。

或者说是许知行。

陆沉拨通那个号码。

"喂？"

"陆沉。"许知行的声音听起来很平静，"你收到我的礼物了？"

"你什么意思？"

"字面意思。"许知行说，"你妹妹在我们手里。"

"你们想怎么样？"

"很简单。"许知行说，"今晚10点，一个人来城北码头。"

"如果我不去呢？"

"那你就等着给她收尸吧。"许知行说，"我说到做到。"

陆沉握紧手机。

"许知行，"他压抑着怒气，"你到底想干什么？"

"我想干什么？"许知行笑了，"我想帮你。"

"帮我？"

"对。"许知行说，"只要你把系统让给我，我就放了你妹妹。"

"系统？"

"把你的系统转移到我身上。"许知行说，"这样预言就会应验在我身上，而不是你。"

陆沉的心猛地沉了下去。

又是系统。

每个人都想得到他的系统。

"你觉得我会相信你？"

"你可以选择不相信。"许知行说，"但你妹妹的命在你手里。"

陆沉沉默了很长时间。

"好。"他说，"我答应你。"

"很好。"许知行说，"今晚10点，城北码头见。"

"嘟——"

电话挂断了。

陆沉站在天台边缘，看着远处的天空。

天边开始泛起鱼肚白。

新的一天即将到来。

但对他来说，这可能是最后一天。

"陆沉。"

一个声音从身后传来。

陆沉转过身。

陈国栋站在天台门口，手里提着一个塑料袋。

"师傅。"

"给你带了早餐。"陈国栋走过去，把塑料袋递给他，"吃点东西。"

陆沉打开袋子，里面是两个包子和一杯豆浆。

"谢谢师傅。"

他拿起包子，咬了一口。

，味道很好。

但他吃不出任何味道。

"小沉。"陈国栋开口。

"嗯？"

"你真的决定要去？"

"嗯。"陆沉点头，"我必须去。"

陈国栋沉默了很久。

"我跟你一起去。"

"不行。"陆沉说，"如果被发现，他们会撕票的。"

"那怎么办？"

"我自有办法。"陆沉说，"您帮我照顾好清禾和晚棠就够了。"

陈国栋眼眶红了。

"臭小子……"

"师傅。"陆沉突然跪下来，给陈国栋磕了个头。

"你干什么！快起来！"

陈国栋赶紧扶起他。

"师傅，"陆沉说，"如果我回不来，请您帮我照顾我妹妹和我爸妈。"

"说什么屁话！"陈国栋的眼泪掉了下来，"你他妈的给我活着回来！"

陆沉笑了。

"我尽量。"

他把最后一口包子塞进嘴里，然后转身走向天台门口。

"师傅，"他头也不回地说，"谢谢您这么多年的照顾。"

"臭小子……"

陈国栋的声音在身后哽咽。

陆沉走进电梯。

电梯门缓缓关上。

他最后看了一眼师傅苍老的脸。

然后电梯开始下行。

### 场景一：码头

晚上9点，城北码头。

海风呼啸，月光在海面上投下粼粼波光。

陆沉站在废弃的仓库门口，深吸了一口气。

他只有一个人。

没有带武器——绑匪说一个人来，他就真的一个人来。

但他在衣服里面藏了一把刀。

一把很小的折叠刀，只有十厘米长。

这是他最后的底牌。

"来了。"

一个声音从黑暗中传来。

陆沉抬起头。

许知行从仓库里走出来，身后跟着四五个穿着黑色西装的男人。

"你很准时。"许知行笑着说。

"我妹妹呢？"陆沉问。

"在里面。"许知行指了指身后的仓库，"她很好。"

"我要见她。"

"可以。"许知行做了个请的手势，"跟我来。"

陆沉跟着许知行走进仓库。

仓库里很暗，只有几盏昏暗的灯泡挂在天花板上。

陆晚棠被绑在一张椅子上，嘴巴贴着胶带，眼睛蒙着黑布。

"晚棠！"

陆沉想冲过去，但被两个保镖拦住了。

"别急。"许知行说，"先把事情说清楚。"

"还有什么好说的？"陆沉压抑着怒气，"放了她！"

"可以。"许知行说，"但是你要配合。"

"怎么配合？"

许知行从口袋里掏出一个小型遥控器，按了一下。

仓库的灯突然全部亮了。

陆沉这才看清周围的环境。

仓库里有很多设备，看起来像是一间实验室。

最引人注目的是一台巨大的机器，像是一个金属圆环，中间是空的。

"那是什么？"陆沉问。

"预言转换机。"许知行说，"可以把系统从一个宿主转移到另一个宿主身上。"

陆沉的心沉了下去。

又是系统。

每个人都想得到他的系统。

"你想干什么？"

"很简单。"许知行说，"把系统转移到我身上。"

"为什么？"

"因为我想活下去。"许知行的声音变得低沉，"你不知道被预言折磨是什么感觉。"

"什么意思？"

"我身上也有系统。"许知行说，"但是它坏了。"

"坏了？"

"对。"许知行说，"系统的预言功能出现了bug，我看不到完整的预言。只能看到一些碎片。"

"所以你想抢我的系统？"

"对。"许知行说，"你的系统是最完美的。只要得到它，我就能看到完整的预言。"

"然后呢？"

"然后我就可以改变自己的命运。"许知行的眼睛里闪着光，"我就不用死了。"

陆沉沉默了。

他不知道该说什么。

"你觉得我会相信你？"陆沉说，"如果你得到系统之后杀了我妹妹怎么办？"

"我不会杀她。"许知行说，"我只要你。"

"你觉得我会相信你？"

"你可以选择不相信。"许知行说，"但你妹妹的命在你手里。"

陆沉看向陆晚棠。

小姑娘被绑在椅子上，身体一直在发抖。

她很害怕。

"晚棠。"陆沉轻声唤道，"别怕，哥哥来救你了。"

陆晚棠抬起头，虽然眼睛被蒙着，但她还是点了点头。

"哥……"她的声音从胶带下面传出来，含糊不清。

"听话。"陆沉说，"哥哥一定会救你出去的。"

他转向许知行。

"好，我答应你。"

"明智的选择。"许知行笑着说，"跟我来。"

他带着陆沉走向那台巨大的机器。

"站进去。"许知行指着金属圆环说。

陆沉犹豫了一下，然后走进圆环。

"然后呢？"

"等一下。"许知行走到控制台前，开始操作，"系统转移需要时间。"

"需要多长时间？"

"大概五分钟。"许知行说，"在这期间，你不能动。"

陆沉站在圆环里，感觉到身体开始发热。

好像有什么东西在往他脑子里钻。

不，不是往脑子里。

而是往外抽。

"好了。"许知行说，"系统正在转移。"

陆沉感觉到大脑里有什么东西正在离开。

像是灵魂被抽走的感觉。

很痛苦。

但他不能动。

一旦他动了，转移就会失败。

"许知行。"陆沉咬着牙说，"你答应过我的，放了我妹妹。"

"我会的。"许知行说，"只要系统转移完成，我就放了她。"

"你最好说话算话。"

"我从来都说话算话。"许知行笑着说，"除非……"

"除非什么？"

"除非我反悔了。"

陆沉的心沉了下去。

"你……"

"开玩笑的。"许知行说，"别紧张。"

转移还在继续。

陆沉感觉意识开始模糊。

不行。

他不能在这里倒下。

他还有妹妹要救。

还有清禾要照顾。

他必须活着出去。

"嗯……"

突然，陆沉闷哼了一声。

疼痛达到了顶峰。

但随后，疼痛开始消退。

意识也开始恢复。

"系统转移完成了。"许知行的声音听起来很兴奋，"太好了！"

陆沉缓缓睁开眼睛。

他感觉到大脑里空荡荡的。

系统……不在了。

"许知行。"陆沉的声音很虚弱，"你现在满意了吗？"

"满意，非常满意。"许知行笑着说，"现在，你可以去死了。"

他挥了挥手。

两个保镖走向陆沉，每人手里拿着一把刀。

"等等。"陆沉说，"你答应过我，放了我妹妹。"

"我反悔了。"许知行说，"不行吗？"

陆沉的心沉到了谷底。

他早该想到的。

许知行根本不可信。

"动手。"许知行下令。

两个保镖走向陆沉。

就在这时——

"砰！"

仓库的大门突然被踢开了。

所有人都是一惊。

陆沉抬起头。

苏清禾站在门口，手里拿着一把枪。

她穿着病号服，脸色苍白，但眼神异常坚定。

"清禾……"陆沉的声音里带着不可置信。

"我来救你了。"苏清禾说。

---

### 场景二：兄妹

"是你？"许知行的脸色变得很难看，"你不是中毒住院了吗？"

"我命大。"苏清禾说，"没死成。"

她举起枪，对准许知行。

"放了他们。"

"你觉得我会听你的？"许知行冷笑着说。

"你可以选择不听。"苏清禾说，"但我会杀了你。"

许知行犹豫了一下。

他旗下虽然有四个人，但苏清禾有枪。

而且她看起来很冷静，不像是在虚张声势。

"好。"许知行说，"我放人。"

他挥了挥手。

两个保镖放开了陆沉，去解陆晚棠身上的绳子。

"清禾……"陆沉走向苏清禾，"你怎么来了？"

"我来救你。"苏清禾说，"我们一起走。"

"但是……"

"没有但是。"苏清禾打断他，"快走！"

陆沉跑向陆晚棠，帮她解开绳子。

"晚棠，快走！"

"哥……"陆晚棠泣不成声，"你怎么办？"

"我随后就来。"陆沉说，"听话，快跑！"

他把妹妹推向门口。

"想跑？"许知行突然冷笑，"没那么容易！"

他按下手里的遥控器。

"轰！"

仓库突然发生了爆炸。

确切地说，不是爆炸，而是震动。

天花板上的灯开始摇晃，灰尘簌簌落下。

"不好！"苏清禾大喊，"这个仓库要塌了！"

陆沉一把抱起陆晚棠往外跑。

"清禾！"

他回头看苏清禾。

她还在门口，举枪对准许知行。

"走啊！"她大喊，"别管我！"

"不！"陆沉摇头，"一起走！"

"来不及了！"

一块巨大的水泥板从天花板上掉落下来。

苏清禾迅速闪开，但还是被余波震得后退了几步。

"清禾！"

陆沉把妹妹推出门外，然后冲回去找苏清禾。

"你疯了吗！"他大喊，"快走！"

"我走不了！"苏清禾指着许知行，"他还在里面！"

"别管他了！"

"不行！"苏清禾说，"他是组织的核心成员！必须抓到他！"

"命都不要了？"

"我的命不重要。"苏清禾说，"重要的是你活着。"

陆沉看着她，眼泪突然掉了下来。

"你这个傻瓜……"

"别说了。"苏清禾拉着他往外跑。

"轰！"

又一块水泥板掉落下来。

这一次，正好砸在苏清禾的腿上。

"啊！"

她惨叫一声，倒在地上。

"清禾！"

陆沉赶紧扶起她。

"走……"苏清禾推开他，"快走……"

"不！"陆沉摇头，"我背你！"

"来不及了！"

"来得及！"

陆沉把苏清禾背起来往外跑。

身后，仓库在继续坍塌。

砖块不断掉落，扬起漫天的灰尘。

"陆沉……"苏清禾的声音在耳边响起，"对不起……"

"别说话！"

"听我说……"苏清禾的声音越来越弱，"我爱你……"

"我知道！"陆沉的眼泪掉下来，"我也爱你！所以别说话！"

"如果我死了……"

"你不会死的！"

"如果我死了……"苏清禾的声音很轻，"你要好好活着……"

"清禾！"

陆沉冲出仓库。

身后，仓库轰然坍塌。

尘土飞扬，遮天蔽月。

### 场景三：废墟

凌晨0点30分，码头废墟外。

陆沉跪在地上，苏清禾躺在他身边。

她的腿被水泥板砸中，鲜血淋漓。

但她还活着。

"清禾……"陆沉抱着她，"你怎么样？"

"我……"苏清禾虚弱地睁开眼睛，"我没事……"

"别说话。"陆沉说，"我送你去医院。"

他抱起苏清禾，往码头外面跑。

身后，废墟里传来许知行的声音。

"陆沉！"

陆沉停下脚步。

"你给我等着！"许知行大喊，"我一定会得到你的系统！"

陆沉没有回头。

他继续往前跑。

身后，废墟里传来咳嗽声。

许知行还活着。

但陆沉已经顾不了那么多了。

现在最重要的是苏清禾的伤。

"清禾，"他一边跑一边说，"坚持住！"

苏清禾靠在他肩膀上，闭上眼睛。

她的呼吸很微弱。

"清禾！"

"我没事……"苏清禾的声音像蚊子一样轻，"只是……有点累……"

"别睡！"陆沉大喊，"清禾！别睡！"

"让我……休息一下……"

"不行！"陆沉的眼泪掉下来，"我们马上到医院了！"

他冲出码头，拦下一辆出租车。

"去医院！最快的医院！"

司机看到他满身是血，愣了一下，然后立刻发动车子。

"清禾，"陆沉抱着她，"坚持住……"

苏清禾没有回应。

她已经昏迷了。

出租车在夜色中飞驰。

陆沉的心跳得很快。

清禾，你一定要没事。

你一定要活着。

我们说好了要一起面对的。

你不能食言。

出租车飞快地驶向医院。
