# Judge Criteria (1 条)

## 剧本核心设定吸引力 [basic]

### 评分标准（matched: true/false）
### true = 设定抓人，有看下去的欲望
### false = 设定平淡，无辨识度

**【本检查项只管一件事】**：从大纲和角色设定来看，这部剧的核心设定是否抓人。
- ✅ 检查：设定层面是否有"一句话就能引起兴趣"的核心概念
- ✅ 检查：是否有身份反差/阶层对撞/认知颠覆/禁忌关系等抓人元素
- ❌ 不检查：剧本执行质量（那是其他检查项的范围）
- ❌ 不检查：题材是否匹配（那是 genre_fit 的范围）

**什么是"抓人的设定"**：
短剧的设定需要在一句话内让人产生"这个有意思，想看"的反应。
好的设定自带冲突势能，不需要复杂铺垫就能让观众理解"这里面有戏"。

**抓人设定的常见模式**：
- **身份反差**：白天外卖员，晚上隐藏富二代；表面乖乖女，暗地黑客高手
- **阶层对撞**：豪门千金嫁入农村/底层逆袭打脸上流社会
- **禁忌关系**：仇人之子的恋情/师生禁忌/敌对家族
- **极端处境**：重生复仇/穿越求生/绝症倒计时/身份互换
- **高概念**：一句话就能说清的强概念（"如果你能看到别人头上的死亡倒计时"）

**不抓人的设定**：
- 平淡的日常故事，没有核心冲突势能（"一个女生在城市打拼的故事"——然后呢？）
- 设定太常规，没有任何差异化（"霸道总裁爱上灰姑娘"且无任何新鲜元素）
- 角色设定缺乏反差或记忆点（所有角色都很"正常"，没有让人好奇的点）

**判断标准**：
- 核心测试：**用一句话概括这部剧的设定，听到的人会不会追问"然后呢"？**
- 有明确的冲突势能 + 至少一个抓人元素 → matched=true
- 设定平淡无奇，听完不会产生好奇心 → matched=false

**注意：不要跟题材过不去**
- 有些题材天然设定感强（穿越、重生），有些题材天然偏日常（家庭、甜宠）
- 关键是在其题材范围内是否做了有辨识度的设定，而非所有剧都得有高概念
- 甜宠剧也能有好设定（"大理古城银饰匠人×独立书店老板"比"总裁×秘书"强）
