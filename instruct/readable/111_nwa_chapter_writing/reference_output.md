# 第二十二章 并肩弑神

“轰——！”

诛魔剑带着毁天灭地的气势，狠狠斩在沈傲天的护体罡气上。

沈傲天闷哼一声，连退数步，嘴角溢出一丝鲜血。

“好！很好！”他擦了擦嘴角的血迹，眼中满是疯狂，“没想到你们两个小辈，竟然能把我逼到这个地步！既然如此，那就别怪我不客气了！”

说完，他猛地咬破舌尖，一口精血喷在定魂珠上。

“以血为引，魔神降临！”

随着他一声怒吼，定魂珠瞬间爆裂开来，化作无数道血红色的光芒钻入他体内。

沈傲天的身体开始剧烈膨胀，皮肤表面浮现出诡异的魔纹，双眼变得赤红如血，周身散发着令人窒息的恐怖气息。

“不好！他要化身血魔！”萧妄脸色大变，“快阻止他！”

两人对视一眼，无需多言，默契已生。

“双剑合璧！”

沈清澜手中的诛魔剑与萧妄手中的墨鳞软剑同时挥出。一白一黑两道剑气在空中交织缠绕，化作一条巨大的阴阳鱼，向着沈傲天绞杀而去。

“雕虫小技！”

沈傲天冷笑一声，随手一挥，一道血色巨掌凭空出现，竟硬生生抓住了那条阴阳鱼。

“给我破！”

他用力一捏，阴阳鱼瞬间崩碎。

巨大的反噬力传来，沈清澜和萧妄同时喷出一口鲜血，倒飞而出。

“清澜！”萧妄顾不上自己的伤势，连忙扶住沈清澜，“你怎么样？”

“没事……”沈清澜脸色苍白，气息紊乱，“这家伙太强了……”

“去死吧！”

沈傲天根本不给他们喘息的机会，身形一闪，瞬间出现在两人面前，一掌拍向沈清澜的天灵盖。

这一掌蕴含着血魔的全部力量，若是拍实了，沈清澜必死无疑。

“不！”

萧妄瞳孔骤缩，想也不想地扑了过去，用后背挡住了这一掌。

砰——！

一声闷响，伴随着骨骼碎裂的声音。

萧妄连哼都没哼一声，身体软软倒下，鲜血染红了沈清澜的衣衫。

“萧妄——！”

沈清澜目眦欲裂，发出一声撕心裂肺的嘶吼。

“为什么……为什么又要救我……”

他抱着萧妄渐渐冰冷的身体，泪水模糊了视线。

“别哭……”萧妄艰难地睁开眼，嘴角带着一丝满足的笑意，“我说过……要保护你的……”

“啊——！”

沈清澜仰天长啸，体内的灵力在这一刻彻底暴走。

悲伤、愤怒、绝望……所有的情绪汇聚在一起，化作一股前所未有的力量。

诛魔剑仿佛感应到了主人的心意，发出一声清越的剑鸣，剑身上的红光瞬间暴涨，化作一道通天彻地的光柱。

“沈傲天！我要你偿命！”

沈清澜缓缓站起身，双眼赤红，宛如从地狱归来的修罗。

他一步步走向沈傲天，每走一步，身上的气势就强上一分。

“你……”沈傲天看着他那副模样，心中竟然涌起一股莫名的恐惧，“你别过来！”

“死！”

沈清澜一剑挥出。

这一剑，没有花哨的招式，只有纯粹的力量和杀意。

噗——！

剑光闪过，沈傲天的身体僵住了。

他低下头，难以置信地看着插在自己胸口的诛魔剑，眼中满是不甘和恐惧。

“我……不甘心……”

他张了张嘴，想要说什么，却最终无力地倒了下去。

轰——！

随着沈傲天的陨落，周围的血色结界瞬间崩塌。无数道被囚禁的生魂从定魂珠的碎片中飞出，化作点点星光，消散在天地间。

“赢了……”

“我们赢了！”

广场上爆发出震耳欲聋的欢呼声。

然而，沈清澜却仿佛听不到这些。他扔下手中的剑，踉跄着跑到萧妄身边，将他紧紧抱在怀里。

“萧妄，醒醒……求求你，醒醒……”

他一遍遍地呼唤着那个名字，泪水滴落在萧妄苍白的脸上。

可是，怀里的人却再也没有回应。
