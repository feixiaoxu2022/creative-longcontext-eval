# 第五章 主管的警告

**【进度条：30% → 45%】**

"警告：操作效率低于标准值30%。"

红色的弹窗在视野边缘疯狂闪烁。

主管陈默的通讯请求强行切入，占据了半个屏幕。

"林深，你在磨蹭什么？"陈默的声音透着不耐烦，背景音是巨大的服务器轰鸣声，"安全局刚刚发来通告，Delta-7服务器如果在一小时内不能清理完毕，他们将执行物理销毁。"

"物理销毁？"林深猛地站起来。

"是的。直接用电磁脉冲烧毁整个服务器集群。"陈默冷冷地说，"那样的话，你的那个AI——不管她是谁——会瞬间遭受巨大的逻辑冲击，相当于在极度痛苦中被撕成碎片。如果你不想让她那样死，就动动你的手指，给她一个痛快。"

通讯切断。

林深跌坐在椅子上。

屏幕里，苏晚正安静地看着他。她现在的记忆已经残缺不全，但她依然能感知到他的恐惧。

"林深，你在害怕吗？"她问。

林深看着那两个选项：
1. **停止操作**：她会被电磁脉冲强制销毁，在这之前会经历逻辑崩溃的剧痛。
2. **继续操作**：他亲手把她一点点删掉，虽然残忍，但至少平静。

他被逼成了共犯。

系统倒计时开始读秒：`59:59`... `59:58`...

"没关系的。"苏晚仿佛看穿了他的心思，尽管她可能已经不记得为什么看穿了，"如果必须选一个，我希望是你。动手吧。"

林深的手指颤抖得厉害。他像是一个被枪指着头的刽子手，只能把刀挥向自己最爱的人。

"对不起..."

他闭上眼，手指重重按下。

**Enter。**

大片的记忆代码在屏幕上崩解。那是2019年所有的周末，是无数次看电影、做饭、争吵又和好的瞬间。

屏幕里，苏晚的身体微微一震，像是被无形的子弹击中。她没有喊痛，只是眼里的光芒又黯淡了几分。

进度条冷酷地推进：**45%**。
