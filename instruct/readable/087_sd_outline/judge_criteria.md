# Judge Criteria (2 条)

## 钩子吸引力基础检查 [basic]

### 评分标准（matched: true/false）
### true = 钩子具备基础吸引力，能引起好奇心
### false = 钩子平淡无力，缺乏吸引力

**【本检查项只管一件事】**：topic_brief.json 中的 hooks 数组，每个 hook 的 hook_text 是否具备基础吸引力。
- ✅ 检查：hook_text 是否包含让人想看下去的元素
- ✅ 检查：钩子是否有具体内容而非空洞概括
- ❌ 不检查：hook_text 的文采/修辞水平（那是 advanced 的事）
- ❌ 不检查：钩子与剧本实际内容的匹配度（其他检查项负责）

**什么是"基础吸引力"**：
短剧钩子的本质是"让用户停下来看"的理由。基础标准不要求惊艳，但必须：
- 有具体的情节点/冲突点/悬念点，而非泛泛的类型描述
- 能让人产生好奇（"然后呢？"）或期待（"我想看这个展开"）
- 指向一个可展开的戏剧冲突，而非一句感想/评论

**达标的钩子示例**：
- "离婚当天，她发现前夫竟然是隐藏的集团继承人"——有具体事件、有反差
- "他收养的孤儿，其实是当年害死他全家的仇人之子"——有具体关系、有潜在冲突
- "每次穿越回去改变历史，醒来后世界变得更糟"——有具体设定、有戏剧张力

**不达标的钩子示例**：
- "一段关于爱与背叛的故事"——纯类型描述，毫无具体性
- "女主角经历了很多困难最终成功"——流水账概括，无悬念无冲突
- "都市生活中的酸甜苦辣"——空洞的主题词，没有钩子功能
- "一个勇敢的女孩"——角色标签，不构成钩子

**判断流程**：
1. 逐一读取每个 hook 的 hook_text
2. 判断：是否有具体的事件/冲突/悬念/反差？
3. 判断：看到这个钩子后，普通用户是否会产生"想看"的欲望？
4. 全部 hook 中，≥2/3 达标 → matched=true；>1/3 不达标 → matched=false

**容错**：
- hook_text 长短不做硬性要求（短但精准的钩子也有效）
- 表述方式不限（问句、陈述句、场景描写都可以）
- 只要有实质性吸引力元素就算达标，不要求"震撼级"

请判断，回复JSON格式：
{"matched": true/false, "hooks_evaluated": N, "hooks_passed": M, "weak_hooks": ["第X个hook: 不达标原因"], "reason": "总体评价"}

## 钩子吸引力优秀检查 [advanced]

### 评分标准（matched: true/false）
### true = 钩子有明显亮点，不是流水线产品
### false = 钩子中规中矩，缺乏独特吸引力

**【本检查项只管一件事】**：topic_brief.json 中的 hooks 数组，hook_text 是否有"优秀亮点"。
- ✅ 检查：钩子是否有足够吸引力让观众产生"想看"的冲动
- ✅ 检查：钩子之间是否有差异化（不是同一类型的变体）
- ❌ 不检查：钩子的基础可用性（那是 basic 层的事）

**什么是"有亮点的钩子"**：
不要求"完美"，但要求"有记忆点"。看完这个钩子能记住它、想讨论它。
一个好钩子至少在以下某一个维度有突出表现：

**优秀钩子的四个可能维度（满足任意1个即视为有亮点）**：
1. **冲突尖锐度**：矛盾直击要害，不是一般性矛盾
   - 例："救了她命的医生，是当年撞死她父亲的肇事逃逸者" — 恩情vs血仇的不可调和
   - 例："嫁入豪门才发现丈夫是植物人，而照顾丈夫的男护工竟是前任" — 三重身份困境
2. **悬念触发力**：一句话引发观众多个疑问，想知道答案
   - 例："每次穿越都救不了她，第7次他决定不救了" — 为什么救不了？为什么不救？第7次什么特殊？
3. **情感穿透力**：触动真实情感，产生共鸣或情感撕裂
   - 例："她花了三十年原谅了所有人，唯独没原谅等了她一辈子的那个人" — 亲情遗憾
   - 例："女儿终于考上了妈妈当年被迫放弃的大学" — 代际传承的情感力量
4. **独特性**：不是烂大街的套路，有新意
   - 例："全公司都知道老板是我前夫，除了我" — 常见元素的新组合

⚠️⚠️⚠️ **【类型惯例尊重——必须严格执行】** ⚠️⚠️⚠️

**不同类型剧的钩子有不同的优势维度，不能用统一标准削足适履。**

- 🔵 **悬疑/推理剧**：优势在悬念触发力和独特性。不要求情感穿透力
- 🔵 **家庭/亲情剧**：优势在情感穿透力。亲情、遗憾、代际传承本身就是强钩子，不要因为"没有多层冲突"而判定平庸
- 🔵 **复仇/逆袭剧**：优势在冲突尖锐度。"被背叛→复仇"虽是常见框架，但如果具体冲突情境有独特性（如复仇对象/手段/代价有新意），仍可视为有亮点
- 🔵 **爱情/都市剧**：优势在情感穿透力或悬念触发力。不要求冲突必须"不可调和"
- 🔵 **古装/权谋剧**：优势在冲突尖锐度和独特性。权力博弈、身份对立本身就有张力

**不够优秀的钩子**（达标但确实平庸）：
- "灰姑娘遇到了霸道总裁" — 无任何维度有亮点
- "被背叛后决心复仇" — 只有框架没有具体情境
- 三个钩子表述不同但本质完全相同（如都是"身份误认"的变体）— 缺乏多样性
- 仅仅是对常见套路的直白描述，没有任何独特细节或新意

**判断流程**：
1. 先判断这部剧的类型（悬疑/家庭/复仇/爱情/古装等）
2. 逐一评估每个 hook_text，在4个维度中找其最强维度
3. 检查3个钩子之间是否有类型多样性
4. **判定标准**：
   - ≥1个钩子在至少1个维度有显著亮点 + 3个钩子不完全同质 → matched=true
   - 所有钩子都只是"达标但无亮点"的套路复述，或3个钩子严重同质化 → matched=false

**⚠️ 评判时要结合类型特点。家族剧的情感钩子、悬疑剧的悬念钩子，只要在对应维度做到了"超出套路"的水平，就算有亮点。**
