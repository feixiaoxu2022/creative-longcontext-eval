# 第七章 剑仙与弟子

仙侠副本的加载界面是一片云海。

苏晚御剑站在云端，裙摆被风吹得猎猎作响。她低头看着脚下翻涌的云海，又抬头看向不远处的仙山楼阁，感觉自己像是在做梦。

"御剑飞行副本，任务是在限定时间内抵达天机峰。"陆溯的声音从前方传来，"注意控制平衡，不要掉下去。"

"我、我在努力！"苏晚紧紧抓着剑柄，身体摇摇晃晃。

她的脚下是一把系统分配的飞剑，银白色的剑身上刻着复杂的符文。作为一个Lv.10的玩家，她理论上已经掌握了御剑飞行的基础技巧，但实际操作起来完全是另一回事。

"放松。"陆溯飞回来，停在她身边，"剑随心动，不要和身体对抗。"

"可是它不听我的话……"苏晚委屈地说。

话音未落，飞剑突然一个急转弯，她整个人失去平衡，向前栽去——

陆溯眼疾手快，一把揽住她的腰，将她拉进自己怀里。

苏晚撞进他怀中，脸贴在他的胸膛上，能清楚地听到他有力的心跳。他的手臂紧紧箍着她的腰，温度透过薄薄的衣料传递过来，让她整个人都僵住了。

"师、师尊……"她结结巴巴地说。

陆溯愣了一下，才意识到自己在副本中的身份——这个仙侠副本的设定是师徒任务，他是"师尊"，她是"弟子"。

"……没事吧？"他问，声音有些沙哑。

"没、没事……"苏晚的脸红得像熟透的苹果，连忙从他怀里挣脱出来，"对不起，我太笨了……"

"不笨。"陆溯说，"只是需要练习。"

他伸出手："来，我带你。"

苏晚犹豫了一下，然后小心翼翼地把手放进他的掌心。陆溯握紧她的手，两人共乘一剑，向天机峰飞去。

云海中，他们的身影如同一对神仙眷侣。

"师尊，"苏晚突然开口，"你以前也带过别的弟子吗？"

"没有。"陆溯说，"你是第一个。"

"真的？"苏晚的眼睛亮了起来，"那我是不是最特别的？"

陆溯低头看着她期待的眼神，心里涌起一股温柔。

"是。"他说，"你是最特别的。"

这是实话。在他三年的无限流生涯里，从来没有一个人能让他如此在意。其他玩家也好，NPC也好，都只是过客。但她不同。

她是唯一一个，让他愿意一次次读档重来的人。

天机峰是一座悬浮在云海中的仙山，山顶有一座古老的道观。他们降落在道观前的广场上，系统提示音响起：【师徒任务开启：弟子需在师尊指导下完成三项试炼。】

"第一项试炼是什么？"苏晚问。

陆溯查看任务面板："剑术对决。你需要在师尊手下支撑十招。"

"十招？"苏晚瞪大眼睛，"我怎么可能……"

"我会放水。"陆溯说，嘴角微微上扬，"而且，我会教你。"

他们在广场上开始了教学。陆溯握着苏晚的手，一招一式地演示。他的呼吸就在她耳畔，他的体温就在她身后，这种亲密的距离让苏晚的心跳始终无法平静。

"专心。"陆溯低声说，"感受剑的流动。"

"我、我在努力……"苏晚的声音有些发抖。

"不要紧张。"陆溯说，"相信我。"

苏晚深吸一口气，努力让自己冷静下来。她感受着他握着她的手，感受着他的动作，感受着他的呼吸。

渐渐地，她进入了一种奇妙的状态。仿佛整个世界都消失了，只剩下他们两个人，和手中这把剑。

"很好。"陆溯的声音里带着赞许，"你学得很快。"

十招对决，苏晚勉强支撑了下来。系统判定通过，开启了第二项试炼。

第二项试炼是心性考验，他们需要进入一个幻境，面对内心最深的恐惧。

苏晚进入幻境后，发现自己站在一片黑暗中。四周什么都没有，只有无尽的虚无。

"师尊？"她惊慌地叫道，"师尊你在哪里？"

没有回应。

她开始奔跑，但不管跑多久，周围都是同样的黑暗。恐惧慢慢爬上心头，她感到呼吸困难，心跳加速。

"不要慌。"陆溯的声音突然响起，"这是幻境，不是真实的。"

苏晚转过头，看到他就在自己身边，伸出手等着她。

"师尊……"她扑进他怀里，眼泪不受控制地流了下来，"我以为你不见了……"

"我不会不见。"陆溯抱住她，轻轻拍着她的背，"我说过，我会一直在。"

幻境中的恐惧慢慢消散，取而代之的是温暖的安全感。苏晚靠在他怀里，听着他的心跳，感到无比安心。

【第二项试炼通过。】

第三项试炼是疗伤。系统模拟了苏晚受伤的场景，需要陆溯为她治疗。

他们在道观后的桃花林里坐下。粉色的花瓣纷纷扬扬地落下，在地上铺了一层花毯。

"把衣服脱了。"陆溯说。

苏晚的脸瞬间红了："诶？"

"疗伤需要接触伤口。"陆溯的声音很平静，"放心，我不会看不该看的地方。"

苏晚犹豫了很久，终于红着脸解开了外袍。她的肩膀上有一道虚拟的伤口，是系统模拟的剑伤。

陆溯的手指轻轻触碰她的皮肤，带来一阵酥麻。他的动作很轻柔，像是在对待什么珍贵的宝物。

"疼吗？"他问。

"不疼……"苏晚小声说，"就是有点痒……"

陆溯的嘴角微微上扬。他继续为她疗伤，指尖在她的皮肤上划过，留下一道道温暖的光芒。

桃花落在他们身上，像是为他们披上了一层粉色的纱衣。

"师尊……"苏晚突然开口，声音很轻，"你会一直陪着我吗？"

陆溯的手顿了一下。

"会。"他说，声音低沉而坚定，"永远。"

苏晚转过头，看着他的眼睛。那双眼睛里有着她看不懂的深邃，但更多的是温柔和专注。

"真的吗？"她问，"即使我笨，即使我总是拖后腿，即使我……"

"即使你什么？"

苏晚低下头，声音细若蚊蝇："即使我不是……不是一个好弟子……"

陆溯看着她，感到心脏被什么东西狠狠击中。

她不是弟子。她不是玩家。她甚至可能不是真实存在的。

但他不在乎。

"你不是弟子。"他说。

苏晚惊讶地抬起头。

"对我来说，你不是弟子。"陆溯说，他的声音很轻，但每一个字都清晰无比，"你是……"

他顿了顿，似乎在斟酌用词。

"你是我想一直保护的人。"他说，"不管你是谁，不管你是什么。"

苏晚愣住了。她看着他的眼睛，看着里面燃烧的情感，感到心脏漏跳了一拍。

"师尊……"她喃喃道，"你是说……"

陆溯没有回答。他低下头，在她的额头上落下一个轻柔的吻。

"试炼结束了。"他说，"我们回去。"

【第三项试炼通过。】

【师徒任务完成，奖励：师徒羁绊+10，亲密度+50。】

他们御剑离开天机峰，云海在脚下翻涌。苏晚靠在陆溯怀里，回想着刚才那个吻，脸上泛着幸福的红晕。

"师尊。"她轻声说。

"嗯？"

"我会努力变强的。"她说，"强到可以保护你。"

陆溯低头看着她，眼神温柔得能滴出水来。

"好。"他说，"我等着。"

【副本通关。】

【绑定对象存活率：58%。】

回到系统大厅后，苏晚的心情格外好。她蹦蹦跳跳地跟在陆溯身后，嘴里还哼着不成调的歌。

"这么开心？"陆溯问。

"嗯！"苏晚用力点头，"因为师尊说会一直陪着我！"

陆溯看着她灿烂的笑容，感到心里某个地方被填满了。

即使这一切都是虚假的，即使她只是一个幻觉，即使这个"永远"可能随时会结束——

他也选择相信。

相信此刻的幸福，相信她的笑容，相信那句"永远"。

因为这就是他想要的全部。
