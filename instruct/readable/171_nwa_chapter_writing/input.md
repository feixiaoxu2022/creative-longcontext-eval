# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事

## CONSISTENCY_MANAGEMENT_GUIDE.md

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。

也许是新的城市,也许是……另一个故事的开始。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
```

# 第一章 隐形人

沈听晚知道方芷萱又要开始了，因为第三节课的铃声还没响，方芷萱就从前排转过身来，手里捏着一沓试卷，指尖涂着浅粉色的甲油，笑得很好看。

"来，传一下。"

试卷从第一排开始往后递。沈听晚坐在最后一排靠窗的位置，她能看到白色纸张像接力棒一样在每张课桌上短暂停留，然后被抽走一份、继续向后。一份、两份、三份——纸张越来越薄，速度越来越快。

递到倒数第二排的赵玉婷手上时，赵玉婷抬头看了一眼方芷萱的方向，然后直接把剩余的试卷全部递给了右边的同学，跳过了沈听晚。

沈听晚的手甚至没来得及伸出去。

她把手缩回来，放在膝盖上，指甲无声地掐进掌心。这个动作她做得很熟练了，力度刚好能让疼痛盖过别的什么——比如眼眶后面那股发酸的热意。

"哎，沈听晚的呢？"程以墨的声音从斜后方传来，带着不加掩饰的不满。

教室里安静了一秒。方芷萱转过身，表情是那种恰到好处的惊讶——像是真的不知道发生了什么似的："哎呀，是不是不够了？我印的时候好像少算了一份。"

她连"沈听晚"三个字都没有叫，就好像那个名字不值得从她嘴里说出来。

"我这有多余的，给你。"程以墨从自己桌上抽出一张递过来，沈听晚低声说了句谢谢，接过来，目光始终没有离开桌面。

这不是最糟的。最糟的在第四节课结束后。

沈听晚利用午休前的十分钟去办公室补交上周的数学作业——她病了三天，落了两次作业，昨晚补到凌晨一点才写完。她把作业本放在老师桌上正准备离开，方芷萱从门口进来了。

班长方芷萱来办公室是名正言顺的事。她拿了一沓需要批改的周测卷，回身时"不小心"碰到了沈听晚放在桌角的作业本。本子掉在地上，翻开到她写了满满三页的那部分。

"呀，这是谁的？"方芷萱弯腰捡起来，翻了翻，看到了名字。她的嘴角弯了弯，那种笑意不是嘲弄，比嘲弄更让人难受——是一种轻描淡写的不在意。"补交的啊？这都过了快一周了吧，老师估计都不记得要检查这份了。"

她把本子递还给沈听晚，手指松开的瞬间，本子掉落在地上。方芷萱说了句"哎手滑"，转身走了。

沈听晚蹲下来捡本子。封面上蹭了一道灰色的鞋印。

她把本子重新放回老师桌上，走出办公室。走廊里的阳光很好，打在水磨石地面上泛着白光，暖洋洋的，和她身体里那种冰凉的感觉完全是两个世界。

她没有回教室。

脚步像有自己的意志一样拐进了楼梯间，向上走。三楼，四楼，五楼。五楼东侧楼梯的尽头是一小块被墙壁和废弃课桌围出来的角落，头顶的监控摄像头正对着楼梯口，但这个角落恰好在它的视线之外——监控死角。

学校里没有人来五楼的这一头。这层的教室在去年暑假就清空了，说是要改建成自习室，但工程一直没开始，走廊里堆着灰扑扑的桌椅，空气中有一股久不通风的闷霉味。

沈听晚把自己塞进角落里，背靠墙壁，缓缓滑坐下去。

废弃的课桌堆在她右手边，桌腿上缠着蛛网。左手边是一扇锈死的铁窗，玻璃上蒙着厚厚的灰。光线从灰蒙蒙的窗户里透进来，把她整个人笼在一片浑浊的暗影中。

她把脸埋进膝盖，开始发抖。

不是冷。是那种积压了太久的东西终于找到一个没有人看见的地方可以释放出来。她不哭，只是抖，像一台过载的机器在嗡嗡地运转。

她开始数数。

这是她的习惯——每次躲到这里来的时候，她就在心里数秒，数到自己不再抖了为止。一、二、三……三十七、三十八……她的呼吸渐渐平缓下来，但身体还在轻微地颤动。……一百二十、一百二十一……

她想，今天也差不多是这个数字，一百多秒就能平静下来，然后回去上下午的课，假装什么都没发生。

……一百五十六、一百五十七……一百七十八、一百七十九——

一百八十。

她没有注意到这个数字意味着什么。但空气注意到了。

温度骤降。

那种冷不是从外面来的，而是从皮肤下面渗出来的，像血液突然被换成了冰水。沈听晚猛地抬起头，看到自己伸出的双手正在——

消失。

从指尖开始，像水彩被雨淋湿了一样，颜色一层一层地褪去。先是皮肤的暖色调变成灰白，然后灰白变成半透明，最后——她能透过自己的手指看到背后墙壁上的裂缝。

她尖叫了。

那声尖叫卡在喉咙里变成了一个嘶哑的气音——她连正常的声音都发不出了。她猛地站起来，膝盖撞翻了旁边的废弃课桌。桌腿扫过她的小腿——不，穿过了她的小腿。木质桌腿穿过她的腿，就像穿过一团空气，然后砸在地上发出巨大的声响。

她没有感觉到任何痛。

沈听晚低头看着自己的身体。校服还在，但颜色变得极淡，像褪色的老照片。她能看到自己的轮廓，但轮廓内部是半透明的——光从她身体里穿过去，在背后的墙壁上投不下一点影子。

恐惧让她的大脑嗡嗡作响。她转身冲下楼梯，脚步声消失了——鞋底踩在台阶上发不出声响。她跑过四楼、三楼的走廊，有学生端着饭盒从她面前经过，没有一个人侧目。她冲进教室，站在自己的座位旁边，胸口剧烈起伏，嘴巴一张一合地喊着什么。

没有人抬头。

方芷萱从她身边走过去拿水杯，肩膀直接穿过了沈听晚的手臂——像穿过一团雾。方芷萱打了个轻微的寒颤，皱了皱眉，继续走了。

沈听晚站在教室正中央，三十八个人，没有一个人知道她在这里。

她低头看着自己透明的手，看着阳光穿过她的身体在地板上画出一个没有她的光斑。一种比恐惧更复杂的东西慢慢升上来——不是痛苦，是某种苦涩的、荒诞的熟悉感。

被所有人忽视——这件事她不是第一次经历了。

只不过以前是比喻，现在是字面意思。

她缓缓松开掐得发白的手指，深吸了一口气。气息吸进去又呼出来，胸口没有正常呼吸的起伏感，像是空气直接穿过了她。

好吧。

她转身走出了教室。

既然没有人看得见她，那就没有人能伤害她。这个逻辑残忍得近乎温柔。

她在走廊里漫无目的地走。一切都变了——整个世界像蒙上了一层灰蓝色的滤镜，光线变得冷而干净，阳光打在地面上不再是暖色而是近乎银白。空气安静得不正常，那些嘈杂的人声、笑声、拖椅子的声音都隔了一层膜似的，变得遥远而模糊。

但她注意到了一些正常状态下不存在的东西。

走廊墙壁上，在灰蓝色的光线中浮现出了密密麻麻的字。

那些字不是用笔写的，像是有人用指甲一笔一笔刻在墙面上的，刻痕很浅，只有在透明状态下的特殊光线里才能看见。她凑近了看——

"第一天。我叫温既白。我不知道发生了什么。"

"第七天。没有人能看见我。我确认了。"

"第三十一天。我开始在墙上写字，因为我怕有一天我连自己说过什么都忘了。"

沈听晚的心跳突然加速。她认出了这个名字——温既白。学校公告栏里贴了三年的寻人启事上的名字。那个全校第一名，那个突然在某个下午消失、至今生死不明的少年。

她继续往前走，墙上的文字越来越密，字迹也从最初的凌乱慢慢变得工整。她看到日期标注从"第一天"一直排到了"第一千零九十六天"——今天的日期。

一千零九十六天。

三年。

有人在这面墙上写了三年的日记。

沈听晚的脚步在四楼物理实验室门口停了下来。门缝底部有一线微弱的光——不是正常的日光灯的光，是一种银蓝色的、只存在于透明层的光。

她伸手推开了门。

# 第二章 透明层的住客

物理实验室的门推开后没有发出声音。

室内的光线和走廊不一样。透明层的阳光透过窗玻璃照进来，变成了一种奇异的银白色，像融化的月光铺满了实验台的台面。灰尘在光束里悬浮着不动，像时间在这间教室里被按下了暂停键。

窗边的实验台上堆着一摞课本，封面磨得起毛，最上面那本的书脊开裂了，露出里面散落的书页。几支铅笔横七竖八地躺在课本旁边，有一支被削得只剩一小截，还插在一个用橡皮泥捏成的笔筒里。

实验台的边缘刻着密密麻麻的正字。

沈听晚还没来得及数那些正字有多少，就看到了那个人。

他坐在靠窗的实验台上，一条腿垂在台面边缘轻轻晃着，膝盖上摊着一本物理课本——封面上印着"高二物理·选修3-1"，纸张已经翻到卷边。他穿着一件洗得发白的校服，那种款式是三年前的旧版，领口的校徽颜色褪得快看不清了。

他在自言自语。

声音很轻，沙哑得像砂纸在摩擦——"……所以电磁感应定律的核心在于磁通量的变化率，不是磁通量本身……"他一边说一边用手指在空中比划，像在对着一群看不见的学生讲课。

沈听晚站在门口，呼吸停了一拍。

她认出了这张脸。

学校公告栏的寻人启事上贴着他的学生证照片：方正的下颌线，眉骨轮廓很深，眼睛在灰蓝色的光线中显得格外清亮。照片里的他十五岁，干净清朗。面前的他看起来更瘦了一些，颧骨的线条比照片里明显，下巴上有一层很浅的青色——像是很久没有好好照顾自己的痕迹。

温既白。

他显然感觉到了什么。手指在空中停住，头慢慢抬起来，目光越过课本的边缘，看向门口。

四目相对。

在一千零九十六天的沉默之后，温既白看到了另一个人的眼睛。

那本物理课本从他膝盖上滑落，砸在实验台上又弹到地面，书页散开，发出一连串清脆又空洞的声响。他没有去捡。他整个人僵在那里，像是被什么东西钉住了。

然后他的嘴唇动了。

"你……"

声音卡在喉咙里，沙哑到几乎破碎。他咽了一下，喉结上下滚动，再开口时那三个字像是用了全身的力气挤出来的：

"你能看见我？"

沈听晚还没从震惊中回过神来。她的嘴巴张着，大脑飞速运转——温既白。失踪三年。全校第一。这里是透明层。他在这里。三年。

"你、你是——"

"你能看见我。"温既白没有等她说完。他从实验台上站起来，太快了，踉跄了一下，膝盖撞在台角上，但他好像完全没有感觉到痛。他向她走了一步，又一步，然后强迫自己停了下来——在距离她两米远的地方。

他的眼眶红了。

不是一般意义上的红——那是一种从眼底深处涌上来的、压抑了太久的东西终于找到出口时的失控。他的手在发抖，两只手都在抖，十指攥成拳又松开、攥成拳又松开，指节绷得发白。嘴唇抖了很久才拼出第二句话。

"三年了。"他吸了一口气，那口气在他胸腔里卡了一下。"三年了，终于有人能看见我。"

沈听晚被他的反应吓到了，但更多的是被击中了。

她认得那种表情——在她每次忍着不在人前哭泣的时候，镜子里也是这种脸。那种明明已经碎了但还在拼命维持完整的、脆弱又倔强的神情。

"你……你真的是温既白？"她的声音也在抖。"三年前失踪的那个温既白？"

他点头。动作很小，但很用力，像是怕她不相信。

"你怎么——你怎么在这里？"

温既白张了张嘴，似乎有太多话想说但不知道从哪里开始。他最终选择了最简短的版本："和你一样。规则。"

"什么规则？"

"你在那个角落——五楼楼梯间的那个角落——待超过三分钟，就会变成这样。"他的声音慢慢稳定下来，每一个字都像是斟酌了很久才放出来的。"透明的，看不见的，碰不到的。二十四小时之后会恢复。"

"那你为什么——三年——"

"我的情况……不一样。"他的目光闪了一下，极快地移开又移回来。"我回不去了。"

沈听晚没有追问。她凭直觉感到那个"不一样"底下压着什么极重的东西，不是现在能撬开的。

她往前走了一步。

温既白下意识地往后退了半步，然后停住了。他的表情很复杂——想靠近又害怕靠近，像一只在寒冬里靠近火堆的动物，既渴望温暖又怕被灼伤。

"我不会消失。"沈听晚不知道为什么说了这句话。也许是因为他看起来太像一个随时会碎掉的人了。

"你会的。"温既白说。声音很轻，几乎像叹息。

"什么？"

他没有解释。他看了她一眼，目光停在她的脸上，像是在确认什么——确认她是真实的，确认这不是他在孤独中产生的又一个幻觉。

然后他说："你叫什么名字？"

"沈听晚。"

温既白重复了一遍这个名字，声音很低，像在把这三个字一个一个放进一个很重要的地方保管起来。

"沈听晚。"他点了点头。"我知道你。"

"你知道我？"

"你坐在五班最后一排靠窗的位置。你的数学不太好但语文很好，每次写周记都是全班最长的那个。你不怎么和人说话，只和程以墨走得近一点。方芷萱她们经常——"他停了一下，嘴角有一个不易察觉的紧绷，"经常欺负你。"

沈听晚愣住了。

"你怎么知道这些？"

"因为我虽然在透明层里，但我能看见外面的一切。"温既白偏了偏头，目光落在窗户上——窗外是正常世界的教学楼，有学生在走廊里打闹，午间广播正在放一首老歌。"三年了，我看了三年。我认识这个学校里每一个人，知道每个班谁和谁关系好、谁和谁有矛盾、哪个老师中午会在办公室睡觉。"

他停了一秒。

"也包括你。"

沈听晚的心脏跳了一下，很重。

"我——"她想说什么，但温既白继续说了下去，声音变得更轻了，像怕吓到她一样：

"我试过很多次。在你被欺负的时候，我就站在你旁边——很近，不到一步的距离。我对着你的耳朵说话——说'别理她们'，说'你没有做错任何事'，说'没关系的'。"

他的声音在"没关系的"那三个字上轻微地裂开了。

"每一次。但你从来听不见。"

物理实验室里安静了很久。银白色的光在实验台面上缓慢地流动，灰尘悬浮不动。沈听晚站在两米外的位置，看着面前这个穿着三年前旧校服的男生，忽然觉得胸口有什么东西堵住了——很硬、很重、酸得她鼻腔发胀。

她不知道自己曾经有过这样一个旁观者。

在所有她以为自己孤身一人扛过去的时刻——在作业被撕碎的时候、在被全班排斥的时候、在午休躲到五楼角落里发抖的时候——有一个人就站在她旁边，不到一步的距离，在她的耳边一遍遍说着她听不见的安慰。

三年。

她咬住了嘴唇内侧——用力咬，铁锈味在舌尖弥散开来，但她没有哭。

"那你呢？"她问。声音压得很低很稳，和她咬到出血的嘴唇形成了一种残忍的反差。"三年了，你一个人在这里，有谁在你旁边说过'没关系'？"

温既白愣了一下。

然后他笑了。

那个笑容很浅，嘴角只弯了一个几乎看不出来的弧度，但在灰蓝色的光线里，他的眼睛亮了一下——像是水面下沉了很久的东西忽然浮上来。

"没有。"他说。"所以我才说——三年了，终于有人能看见我。"

沈听晚不知道该说什么了。她想走过去拥抱他——那是本能反应——但她记起了一件事。

"你刚才说……碰不到？"

温既白点头。

她伸出右手，向他的方向慢慢递过去。他没有躲——甚至微微偏了偏身体迎向她的手，像是他也在期待什么奇迹。

她的手指穿过了他的前臂。

没有触感。没有温度。什么都没有。就像她的手穿过了一团有形状的光。

温既白低头看着她穿过自己手臂的那只手，笑容没有变，但眼底的那点光黯了一下。

"透明层的规则。"他说。"看得见，听得见，碰不到。"

看得见。听得见。碰不到。

沈听晚把手收回来，指尖还残留着一点若有若无的凉意——像是手指从冰水上方划过时带起的那层冷气。

她没有说话。温既白也没有说话。两个人隔着两米的距离站在物理实验室的银白光线里，一个透明的女孩和一个被困了三年的男孩，在一个没有人能看见他们的世界里，互相看着对方。

——这可能是沈听晚这辈子第一次被人认真地、长久地、安静地看着。

也是温既白三年来第一次被另一个人的目光接住。

窗外的午间广播换了一首歌。声音隔着层与层之间的薄膜传进来，模糊得像隔了一个世纪。

温既白低头看了一眼自己的手腕——那里用指甲刻着一串细小的数字，是某种倒计时的标记。他看了两秒，神情变了一瞬：某种温柔的、带着预知的疼痛从他的眼底掠过去，快得像错觉。

他没有告诉她那串数字意味着什么。

他只是重新抬起头，对她露出了一个比刚才更完整一点的笑容——嘴角弯起来的幅度大了一些，虽然眼底还是有化不开的东西。

"走吧。"他说。"既然你来了，我带你看看这个地方。"

他弯腰捡起地上散落的物理课本，一页一页地抚平折角，动作很慢很仔细，像是在对待一个老朋友。然后他把书夹在腋下，走向门口——经过沈听晚身边的时候留出了足够的距离，不远不近，刚好不会让她不舒服，也刚好不会产生穿过彼此的尴尬。

沈听晚跟上了他。

而在她看不见的角度，温既白走在前面的背影有一瞬间弓了下去——像是什么极重的东西压在了他的脊椎上。

他知道的。

二十四小时。

然后她会忘记他。忘记今天发生的一切。忘记她走进这间实验室时他有多高兴。忘记他在她耳边说了三年的话。忘记他的名字。

他什么都知道。

但他还是说了"走吧"。

因为比起二十四小时后的遗忘，此刻被看见的这一秒，已经是他三年来拥有过的最好的东西了。

# 第三章 二十四小时后

温既白带她走遍了整栋教学楼。

在透明层里，这栋楼和正常世界的版本既相同又完全不同。物理结构一样——走廊、教室、楼梯间、天台，每一面墙、每一扇窗都在它应该在的位置。但所有颜色都浸在灰蓝的底色里，像一张老旧的蓝晒照片。正常世界的人们像投影一样在走廊里走过，发出隔了一层膜的模糊声响——笑声、脚步声、广播声——全都隔着一道看不见的屏障，近在咫尺又远在天涯。

而透明层自己的"内容"，都是温既白三年来一个人填上去的。

三楼走廊的整面东墙写满了日记。字迹从最初的歪歪扭扭变得越来越工整。沈听晚一边走一边读——

"第十五天。我把所有能找到的书都搬到了物理实验室，还好图书馆的门锁不住透明层的人。但我碰不到正常世界的东西，只能碰透明层里本来就有的。好消息是粉笔在两个层都存在，所以我还能在黑板上写字。坏消息是值日生第二天就把它擦了。"

"第一百零八天。我今天发现一件事——我在墙上用指甲刻的字不会被正常世界的人看到，但在透明层里很清晰。所以这面墙就是我的日记本了。"

"第二百九十天。她今天又被欺负了。方芷萱把她的水壶藏到了垃圾桶后面。我在她旁边待了整整一节课，对着她的耳朵说了二十六遍'别哭'。她当然听不到。但后来她真的没哭——不知道是不是巧合。"

沈听晚在这一段前面站了很久。

她记得那天。高二上学期，有一次她的水壶怎么都找不到，最后在走廊垃圾桶后面发现了。她确实没有哭——不是因为坚强，而是因为那天不知道为什么，她觉得有人在旁边陪着她。

她当时以为是错觉。

"第八百九十一天，"温既白的日记继续往后，"今天是她的生日。十一月十七日。没有人记得——程以墨感冒请假了，其他人根本不知道她的生日。我在五班的黑板右下角用粉笔写了'生日快乐沈听晚'，字写得很小，希望她早读的时候能看到。"

沈听晚的脚步停住了。

"你……写过？"

温既白走在她前面，听到她的声音后停下来，回过头。银白色的光照在他的侧脸上，轮廓线条干净而薄。

"写了。"他说。"但值日生来得比你早，第二天早上黑板就擦干净了。"

"我从来没看到过。"

"我知道。"他转回头继续走。"所以后来我改成了在你课桌的灰尘上画太阳——那种简笔画的、圆形的、带着几条弯弯曲曲的光线的太阳。你趴在桌上的时候，我就在旁边的桌面上画一个。当然了，你一起身用胳膊一蹭就没了。"

他说这些话的时候语气很平，平得像在描述一件与自己无关的事。但沈听晚走在他身后，看着他说"当然了"那三个字时肩膀几乎不可察觉地缩了一下。

他们走上了天台。

高三教学楼的天台在正常状态下是锁着的，但透明层的人可以穿过任何门。温既白推开天台的门（门对他而言不存在实体，他只是走了过去，门自动打开了），傍晚的风扑面而来——但没有温度。

天台的视野很好。能看到整个校园——操场上有学生在跑步，食堂的烟囱冒着白气，校门口的梧桐树叶子被风吹得沙沙响。所有这些在透明层中都笼罩着灰蓝色的薄纱，像一个褪色的、慢了半拍的平行世界。

太阳正在落下去。

在正常世界里那应该是金红色的夕阳，但在透明层中，落日变成了银白色的，像一枚正在融化的硬币沉入地平线。没有暖意，但美得不像真的。

温既白在天台的栏杆旁坐下来，腿从边缘垂下去，轻轻晃着。沈听晚在他旁边坐下——中间隔着十厘米左右的距离。这是一个默契的距离：近到可以并肩，远到不会发生"穿过对方"的尴尬。

"你害怕吗？"温既白问。

"刚才害怕。"沈听晚说。"现在不了。"

"适应得很快。"

"我本来就是隐形人。"她说这话的时候嘴角甚至弯了一下。"差别没有想象中那么大。"

温既白看了她一眼——很短的一眼，但目光里有什么东西很沉。他没有接这句话。

他们在天台上坐了很久。天黑了。月亮在透明层中也是银白色的，比正常世界的月亮更大更亮，像被人掏空了内部只留下了一个发光的壳。

温既白给她讲了费曼路径积分。他讲得很认真，在空气中比划着公式，声音不再那么沙哑了——大概是因为今天说了太多话，声带终于被"重新激活"了。沈听晚听不懂，但她没有打断他。她看着他讲题时眼睛里的光——那种纯粹的、因为热爱而发亮的光——想，三年了，他就是靠这个撑过来的吗。靠给空气讲物理题。靠在墙上写没有人能看见的日记。靠在一个听不见的女孩耳边说"没关系"。

"我听不懂。"她最终还是诚实地说了。

温既白的手停在半空中。

"但你讲的时候好像在发光。"她说。

温既白愣了一秒。然后他笑了——不是之前那种浅到几乎看不见的笑，是真的笑了出来，胸腔震动了一下，甚至发出了一声轻微的、气音般的笑声。

三年来第一次笑出声。

那个声音在夜晚的天台上消散得很快，但沈听晚觉得自己好像会记住这个声音很久很久。

她不知道"很久很久"大概只有不到六个小时了。

接近凌晨的时候，他们从天台回到走廊。温既白不需要睡觉——透明层里的身体不会困也不会饿，这是三年来唯一的好处。沈听晚的透明身体也一样，没有困意。但时间还是在走。

走廊尽头，温既白看了一眼手腕上的刻痕——那串倒计时数字。

还剩不到四个小时。

他的表情变了。不是变得悲伤——而是变得非常、非常温柔。那种温柔里有疼，但被他压得很深，只在眼角的细微纹路里泄露了一点点。

"沈听晚。"

"嗯？"

"你明天……"他停了一下，斟酌了很久，"你明天还会来这里吗？"

她不假思索地回答："我每天中午都会来五楼那个角落。"

"为什么？"

"因为那是全校唯一没有人会来的地方。"她低头看着自己的手——依然是透明的，灰蓝色的光从骨骼的形状中透出来。"也是唯一不会有人看见我哭的地方。"

温既白沉默了。

"那你明天来的时候，"他说，"如果你想……可以再来这里坐坐。"

他说的是"如果你想"，不是"来找我"。像是不敢奢求太多。

"当然。"沈听晚说。她站起来，拍了拍校服裙子上并不存在的灰——习惯性动作。"明天中午，物理实验室。我带本小说来给你看，你待了三年估计——"

她的话在嘴里碎成了一个气泡。

毫无预兆地，一阵剧烈的眩晕从头顶灌下来。她的膝盖一软，整个人往前栽——手撑在墙壁上，这次她的手没有穿过去，指尖碰到了冰冷的、坚硬的、属于正常世界的墙面。

触感回来了。

她低头——自己的手正在恢复颜色。从指尖开始，像有人拿着水彩笔把褪掉的颜色一层一层涂回来：灰白变成肉色，半透明变成不透明，她的手变回了一只正常的、有血色的、投得下影子的手。

同时，世界在变。

灰蓝色的滤镜像被一只巨手掀开了——颜色回来了。走廊的墙壁变回了奶白色，地面变回了灰色水磨石，日光灯管发出正常的暖黄色光。那些墙上的字迹——温既白三年来写的日记——像隐形墨水遇到热一样迅速隐没，在她的眼前一行一行地消失。

而温既白——

他就站在她面前两米远的地方。他的轮廓正在变得模糊，像一张正在被擦除的铅笔画。他的脸、他的旧校服、他手臂下面夹着的那本翻烂的物理课本，所有细节都在快速褪去。

他在说什么。

他的嘴在动，嘴型像是在说一句话——但沈听晚听不到了。声音隔着越来越厚的层与层之间的屏障传不过来，像有人在水底说话，只剩下气泡在水面无声地破裂。

她拼命地看着他的嘴型。

"没——关——系——"

后面还有。他还在说。但他的身影已经像晨雾一样淡得快看不到了。

"你——忘了——也——没——关——系——"

然后他消失了。

走廊里只剩沈听晚一个人。

日光灯在头顶嗡嗡地响。远处有学生早读的声音——她不知道什么时候天亮了，不知道什么时候早自习开始了，不知道自己为什么站在四楼走廊的中间，不知道自己为什么脸上有泪痕。

她抬手摸了一下自己的脸——湿的。眼泪的温度还在，但她完全不记得自己为什么哭过。

大脑里像是有一块被人用橡皮擦得干干净净——没有留下任何字迹，只有纸面上轻微的凹痕，证明那里曾经写过什么。但写的是什么，她不知道了。

一种奇怪的空洞感从胸口弥漫开来。不是疼——如果是疼就好了，疼至少意味着有伤口，有来处。这种空洞是没有来处的，就好像有人从她身体里取走了一样东西，但连那样东西的形状都一起取走了，所以她甚至不知道自己少了什么。

她低头——

手腕内侧有一行字。

圆珠笔的笔迹，字很小但很清秀，一笔一划都认真到有点过分的程度。

"你不是隐形人，有人一直在看着你。"

沈听晚看着这行字看了十几秒。字迹陌生。她不记得自己什么时候被人写了这么一句话，也不记得是谁写的。

也许是程以墨的恶作剧？但程以墨的字很潦草，不是这种笔迹。

她皱了皱眉，用左手的拇指揉了揉那行字。圆珠笔的油墨有一点点晕开，但没有完全消掉。

上课铃响了。

她收回目光，把袖子拉下来盖住手腕，快步走向教室。走到三楼楼梯口的时候她的脚步顿了一下——一种说不出来的冲动让她想回头，想再上一层楼，想去看看四楼的物理实验室——但她不知道为什么想去那里。

她摇了摇头，下了楼。

教室里一切照常。方芷萱在前排和几个女生嘀嘀咕咕。程以墨看到她进来，挥了挥手："昨晚不回消息，干嘛去了？"

"我……"沈听晚愣了一下。昨晚？她不记得昨晚做了什么。"可能早睡了。"

程以墨盯着她看了两秒，目光滑到她的手腕上——袖子底下的字迹露出了一点点边缘。

"你手腕上什么？"

"不知道。"沈听晚把袖子往下拉了拉。"可能是我自己乱写的。"

程以墨的表情有一瞬间变得微妙，像是想说什么，但最终只是"哦"了一声，转回去翻课本。

放学。校门口。

沈听晚背着书包走出来，在人流里低着头——这是她的习惯，低头走路，不看任何人的脸，也不让任何人看到自己的脸。

她差一点撞到一个人。

那是一个头发半白的中年女人，穿着洗得发白的碎花外套，手里举着一张塑封过的照片，站在校门口正中间的位置，见到学生走过来就迎上去。

"同学——同学你好，你有没有见过这个人？这是我儿子，三年前在这个学校念书的——"

沈听晚被她拦住了。她低头看向照片——

照片上是一个穿着校服的少年，十五六岁的样子，眉眼干净，笑容清朗。照片的塑封边缘已经磨毛了，显然被无数次地握在手里、无数次地举起来给陌生人看。

一种毫无道理的心悸猛地击中了她。

她认识这张脸。不是那种"在公告栏寻人启事上看过"的认识——是更深层的、来自身体内部的某种认识。她的心跳加速，手心出汗，鼻腔发酸——所有这些反应都没有理由，因为她的大脑里没有任何对应的记忆。

"他叫温既白。"女人的声音在发抖，眼眶是红的。"你见过他吗？"

温既白。

这个名字像一块石头砸进了一片平静的水面。水花溅起来，涟漪一圈圈地扩散——但水底什么都看不见。

"我……没有。"沈听晚说。"对不起。"

她绕过温母往前走了。走出十几步后她回头看了一眼——温母还站在校门口，举着那张照片，夕阳把她的影子拉得很长很细。

沈听晚回到家，洗手的时候看到手腕上的字已经被水冲得模糊了。

"你不是隐形人，有人一直在看着你。"

她盯着那行正在消失的字看了很久。

然后她擦干手，拿起手机，对准手腕，拍了一张照片。

她不知道自己为什么要这么做——只是觉得不应该让这行字完全消失。就像有个声音在她意识的最底层、在记忆被清除后残留的最薄的一层回响里告诉她：

不要让它消失。

你不知道为什么。但不要让它消失。

快门咔嗒一声。

手机屏幕上，那行字模模糊糊的，但还能辨认。她把照片保存好，关了灯，上了床。

夜很安静。

在她看不见的透明层中，四楼物理实验室的窗边，温既白坐在实验台上，膝盖上摊着那本翻烂的物理课本。月光把他的影子投在地面上——不对，透明层的人没有影子。他只是坐在那里，一个人，在银蓝色的月光里。

他面前的实验台灰尘上，用手指写着一行字：

"第1097天。她来了。她看见我了。她笑了。然后她忘了。"

他把最后一个字的笔画收完，手指停在灰尘上没有拿起来。停了很久。

然后他用掌心把这行字抹掉了，在原来的位置重新写了一个简笔画的太阳。

和他三年来在她课桌上画的每一个太阳一样。

圆形的。带着几条弯弯曲曲的光线。

看不到也没关系。

他画了三年了，不差这一个。

# 第四章 陌生的痕迹

第二天醒来的时候，闹钟显示六点四十。

沈听晚在床上赖了两分钟，眼睛盯着天花板。有什么东西不对——说不清楚是哪里不对，就像做了一个很重要的梦但醒来后只剩下情绪残留：胸口有一小团化不开的东西堵在那里，不痛，但沉。

她翻了个身去拿手机，拇指习惯性地划开屏幕。

备忘录的图标上有一个红色小圆点——有新内容。

她不记得自己往备忘录里写过什么。

点进去，最顶上是一条昨天下午创建的笔记，时间戳显示"15:32"。那个时间她应该在——她想了想——在上课？她不记得了。

笔记内容：

"沈听晚，如果你看到了这段话，说明你已经忘了。不要害怕。我叫温既白，你在五楼楼梯间的监控死角待了超过三分钟后进入了一个叫'透明层'的地方。你看到了我。我在这里已经三年了。你和我待了二十四个小时，我们聊了很多，但时间到了之后你会失去所有记忆。这是规则的代价。如果你愿意再来见我，去五楼楼梯间的角落等三分钟就可以。如果你不愿意，那也没关系。但你要知道一件事——你不是隐形人，有人一直在看着你。——温既白"

沈听晚把这段话读了三遍。

心跳声大得像有人在她胸腔里擂鼓。

她坐起来，用力甩了甩头——这不可能是真的吧？什么透明层？什么失去记忆？这听起来像一个拙劣的恶作剧，或者是某个心理测试APP自动生成的玩意儿。

但这个笔记是从她自己的手机上写的。

她检查了输入法记录——确实是在她的手机上直接输入的，不是复制粘贴。时间戳、字数、输入法指纹全都对。

是谁？如果她当时在上课，怎么可能有别人拿到了她的手机？

然后她想起了手腕。

她翻了翻相册——昨天晚上她拍过一张照片。照片上是自己的手腕内侧，有一行模糊但还能辨认的字："你不是隐形人，有人一直在看着你。"

和备忘录里最后一句一模一样。

沈听晚打开搜索引擎，输入"温既白"。

结果出来的第一条就是三年前本地晚报的新闻：

"XX市第一中学高二学生温既白于X月X日下午在校期间失踪，至今下落不明。警方调查未果，学校及家属仍在寻找。温既白系该校连续两年年级第一名学生，成绩优异，失踪前无异常表现……"

配图是一张学生证照片——少年的脸干净清朗，校服领口别着蓝色的学生会徽章。

沈听晚盯着那张脸看了很久。

她的身体在发生一种奇怪的反应：心跳加速、手心微微出汗、鼻腔后面有一股酸意上涌——这些反应完全不受大脑控制，它们从身体的某个深处冒出来，像是肌肉记忆。就好像她的身体认识这张脸，即使她的大脑坚持说"不认识"。

她攥着手机出了门。

一路上她都在犹豫。

那段备忘录说的是真的吗？如果是真的，意味着她曾经进入过一个叫"透明层"的地方，见过一个失踪三年的人，和他相处了二十四个小时——然后全部忘记了。这种事情太荒谬了，荒谬到不值得当真。

但如果是假的，谁能解释手腕上的字？谁能解释备忘录？谁能解释她此刻胸口这种说不清道不明的空洞感？

到了学校，第一节课开始前十分钟。

沈听晚坐在自己的座位上翻课本，但一个字都看不进去。方芷萱从前面经过时扫了她一眼——那种一扫而过的、把她当空气的目光。沈听晚低下头，把袖子往下拉了拉，遮住手腕上已经褪得快看不见的笔迹。

课间。

程以墨趴在她桌上看手机，突然凑过来："你最近怎么了？昨天一整个下午我都没看到你，微信不回，晚自习也没来。"

沈听晚张了张嘴："我昨天……我忘了。可能是——"

"你忘了？一整个下午加一整个晚上你都忘了？"程以墨坐直了，带着揶揄的表情，"沈听晚同学这是失忆了？"

"可能……就是走神了吧。"

程以墨还想追问，但上课铃响了。她转回去之前瞥了一眼沈听晚的手腕——袖口没遮严的地方，圆珠笔字迹的尾巴露出来了一点点。

"你手上那个，"程以墨压低声音问，"是谁写的？你昨天也有——"

"不知道。"

程以墨的眼神变了一下。不是怀疑——是一种更认真的、想要搞清楚什么的神色。她看了沈听晚两秒，"嗯"了一声，转了回去。

午休铃响了。

教室里的人陆陆续续出去吃饭。方芷萱路过沈听晚桌子的时候对旁边的女生说了一句什么，语调不高但刚好能让沈听晚听见："又到她消失的时间了，猜猜这回是躲到哪里去。"

沈听晚没动。

她把手机拿出来，再看了一遍那段备忘录。

"如果你愿意再来见我，去五楼楼梯间的角落等三分钟就可以。"

她的手在抖。

不是因为害怕，是因为某种她无法命名的东西——比好奇更重、比冲动更深——从她的身体深处涌上来，推着她的脚步往外走。

她站起来了。

书包没拿，手机揣在口袋里，她走出教室，上楼。三楼，四楼——经过四楼的时候她下意识地朝走廊尽头看了一眼——物理实验室的门关着，看不出任何异常。

五楼。

楼梯间尽头的角落。

一切和昨天一样——不对，她不记得昨天来过这里。但身体记得。她的脚步在这里变得很确定，像走在一条走过无数次的路上。她站在监控死角的中央，背后是废弃课桌，头顶是坏了不知道多少次的灯管。

手机屏幕亮着，备忘录的页面还开着。

她开始数。

一、二、三……

心跳越来越快。

……五十八、五十九、六十……

第一分钟过去了。什么都没有发生。

# 第五章 第N次初见

"你又来了。"

这四个字里没有惊讶——这是沈听晚最先注意到的。

第一次见到一个透明人走进房间，正常反应应该是震惊。但温既白的表情里没有任何震惊的痕迹。他只是站在那里，手里的课本缓慢地合上，目光落在她身上，像在看一幅他已经看了很多次的画。

"什么叫'又'？"沈听晚问。她的声音还带着跑完楼梯后的喘息。

温既白没有立刻回答。他弯下腰从实验台下面的抽屉里拿出了一个被对折过的笔记本——封面是普通的牛皮纸，但因为反复翻阅而起了毛边。他翻开第一页，递到距离她能看清的位置。

上面是他的笔迹——那种她在墙壁日记上已经认得的、清秀到有点过分的字。

"第一次来访。时间：X月X日。持续时间：约18小时。她的第一句话是'你是温既白？'。她很害怕，但后来好了。我们在天台看了日落。离开前她说明天要带小说来。我在她手腕上写了字。24小时后她失去了所有记忆。"

下面还有一行小字：

"她的笑容和我想象的不一样。比想象的好。"

沈听晚的手指悬在笔记本上方，没有碰到——她不确定自己碰不碰得到。

"这是第二次。"温既白说。

"第二次？"

"你第一次来的时候，我们待了十八个小时。我给你讲了物理，你说听不懂但觉得我在发光——"他的声音在这里轻轻卡了一下，像是有什么东西梗在喉咙里。"然后二十四小时到了，你什么都不记得了。"

沈听晚看着那个笔记本上的记录，然后又看向手机里的备忘录——两边的内容可以互相印证。

"所以你在我手机里写备忘录——"

"是我写的。第一次你来的时候，我趁你不注意拿了你的手机写的。你的手机密码是111117。"

"你怎么知道我的——"

"你设置密码的时候我就在旁边。"温既白说这话的时候表情很平淡——不是故意的偷窥，只是一个在透明层里生活了三年的人必然会知道的事情。"111117，你的生日。十一月十七。"

沈听晚张了张嘴，一时间不知道该先处理哪一种情绪——被知道密码的微妙尴尬，被记住生日的意外，还是得知自己曾经经历过一段完整的人生但完全不记得的恐惧。

恐惧赢了。

"我丢了一段记忆。"她说。声音不大，但每个字都很清晰。"一整天——不对，十八个小时。我和你在这里待了十八个小时，说了话，看了日落，你说我笑了——然后我什么都不记得了。"

温既白看着她。

"你为什么不在第一次的时候就告诉我？"她问。声音开始有了棱角。"为什么不告诉我二十四小时后会失忆？如果你告诉了我，也许我可以——我不知道——做点什么——"

"做什么？"温既白安静地问。

"我——"她卡住了。她不知道自己能做什么。规则是规则，就像重力一样不可商量。

温既白等了几秒，确认她说不下去了，才开口。

"因为我自私。"

他说这两个字的时候嘴角甚至弯了一下——不是笑，是一种不带任何遮掩的自嘲。

"三年了。三年，一千零九十六天，我没有和任何人说过一句话。"他的手插在校服口袋里，指节的形状隔着布料鼓出来——他在用力攥着什么。"我对着每一个从我面前走过的人喊过，对着监控喊过，对着空气喊过，嗓子喊到出血都没有人回头看我一眼。"

他停了一秒，吸了一口气。

"然后你来了。你走进这间教室，你看着我——你的眼睛直直地看着我的眼睛——三年了第一次有人看着我。"

他把手从口袋里抽出来。手指在抖，指甲的边缘是破损的——透明层里没有指甲刀，三年来他只能在墙角磨。

"你知道那一刻我在想什么吗？我在想——如果我告诉她只有二十四小时，她会怎么做？她会去找出口。她会把所有时间花在'怎么离开这里'上面。她不会坐在天台上听我讲费曼路径积分。她不会笑。"

他看着她，眼睛里有什么东西在碎。

"我太害怕了。我害怕你把二十四小时全花在找出口上，一秒都不看我。"

物理实验室里安静了很久。

银白色的阳光在他们之间的空气里缓缓流动。灰尘悬浮不动。某间教室里传来隔了一层膜的、模糊的上课铃声。

沈听晚的愤怒在他说完之后就消散了——不是被说服了，是被理解压垮了。

她太懂那种感觉了。

被所有人当作空气。对着世界大声喊话但没有人回应。在人群中间孤独到快要崩溃但连"我好孤独"这句话都没有人能听到。

她过的每一天都是这样。

而他过了三年。

"你讲费曼路径积分的时候，"她开口了。声音已经不硬了，甚至有一点点抖。"你的声音是什么样的？你说的话我都不记得了，但是——笔记本上写着说我觉得你在发光。那是什么感觉？"

温既白愣住了。

他大概以为她会继续追问、继续质问、继续要求一个解释或者道歉。但她没有。她问的是一个他没有预料到的问题。

"我不知道什么感觉。"他老实地说。"但你说那句话的时候——我笑了。三年来第一次笑出声。"

沈听晚深吸了一口气。

"那我们不浪费时间了。"她说。

她从口袋里掏出手机，解锁（111117），打开备忘录，翻到一个新页面。

"你教我怎么在失忆之后还能记住你。"

温既白看着她，好一会儿没有说话。

然后他从实验台抽屉里翻出一支粗头的记号笔——他在透明层里能碰到的东西有限，但记号笔这种"跨层"物品两边都能用。他走到她面前，把笔帽拧开。

"左手。"

沈听晚把左手伸出来。

他开始写。

左手腕内侧："X月X日 第二次 物理实验室4F 温既白"

右手腕内侧："你已经来过两次了 看手机备忘录"

沈听晚按下了快门。屏幕上果然——她自己是正常的影像（虽然半透明），而温既白站在她旁边的位置只有一团模糊的、灰蓝色的光晕。但那团光晕隐约有人形的轮廓。

她照做了。

# 第六章 拼图

程以墨等了三天。

三天里她一直在观察沈听晚。

不是那种随意的关注——是刻意的、系统性的、带着目的的观察。她在自己的手机备忘录里建了一个文件夹，标题叫"沈听晚异常记录"。

记录如下：

"周二。午休时消失，下午第一节课前回来。手腕内侧有陌生笔迹（圆珠笔），她说不知道谁写的。当天晚自习缺席，微信不回。第二天早上表现正常但明显走神，说'可能早睡了'。"

"周四。午休时再次消失。这次从第四节课就不在了。放学后回来时两只手腕都有字（防水记号笔），字迹和周二不同但风格一致。手机备忘录里多出大量文字，她在走廊里站了十五分钟不动。"

"周五。她在课堂上反复看手机，下课后对着手机里的某段内容发呆。右手腕的字写着'你已经来过两次了 看手机备忘录'——她来过两次？来过哪里？"

周六是休息日。周一早读前，程以墨做了她这辈子做过的最大胆的事——她去监控室调了录像。

学校的监控系统不算复杂，程以墨作为广播站编辑有进办公区域的门禁卡。她趁着早读前保安交接班的空档溜进了监控室，在电脑上调出了周二和周四午休时段五楼楼梯间的监控录像。

画面里，沈听晚走上了五楼楼梯。

她走进了楼梯间尽头的那个角落——摄像头能拍到她走进去的最后一步，然后——

人不见了。

不是走出了画面。是就地消失了。她走进那个角落后，画面里再也没有出现任何人。没有人走出来，没有人经过，那个角落在接下来的二十多个小时里空空荡荡——直到第二天，沈听晚忽然出现在四楼走廊的监控里，像是凭空冒出来的。

程以墨把这段录像翻来覆去看了五遍。

"她在监控里消失了。"她对着自己的备忘录自言自语。"她走进了一个监控死角然后——消失了。"

这太不正常了。

程以墨把监控截图保存到手机里，然后做了第二件大胆的事：她在保安室门口拦住了早班保安钟岁安。

周一早读。

程以墨把沈听晚从教室后门拉了出来。走廊里没什么人，她靠在窗户边上，掏出手机，翻到监控截图页面。

"你看。"

屏幕上是监控录像的截图：沈听晚走进五楼楼梯间，然后下一帧画面里人消失了。

"这是你。周二和周四午休。你走进了五楼那个楼梯间的角落，然后在监控里消失了。"程以墨说，声音压得很低，但每个字都很硬。"两次。"

沈听晚看着截图，心跳开始加速。

"然后这是你手机上的。"程以墨翻到另一张截图——是沈听晚手机备忘录里"温既白"署名的那段长文。"我上周趁你去洗手间的时候拍的。别怪我偷看——你最近太不对劲了，我要搞清楚怎么回事。"

沈听晚的嘴巴张了张。

"这段文字署名温既白。"程以墨一字一句地说。"温既白——三年前在这个学校失踪的那个。你手腕上的字和这段备忘录里的最后一句话一模一样。你在监控里消失了两次。你什么都不记得。"

她看着沈听晚的眼睛："你到底遇到了什么？"

沈听晚的记忆里没有任何对应的画面——备忘录上的内容她今天早上已经看过了，但这些终究是"别人告诉她的事"而不是"她记得的事"。面对程以墨铺开的证据链，她能做的只有沉默。

"我不知道。"她最终说了。是实话。"我真的不知道。但我的手机、手腕上的字——这些不是我自己写的。"

"那谁写的？"

"他写的。"沈听晚指了指备忘录里的署名。"温既白。"

程以墨的眼睛眯了一下。

"你信？"

"我不知道该不该信。但是——"沈听晚犹豫了一秒，把手放在自己心口的位置。"每次看到那个名字，这里就会——不是疼，是有什么东西在动。我的身体在替我记某些事情。"

程以墨看着她按在心口的手，表情慢慢变了。不是怀疑了——是一种更复杂的、夹杂着担忧和认真的神色。

"跟我走。"程以墨拉起她的手。"有个人你得见一下。"

保安室在教学楼一楼的角落。门上贴着"保安值班室"的牌子，油漆斑驳。程以墨推开门，逼仄的小房间里堆满了旧报纸和几个闪着绿光的监控屏幕，空气中有一股烟味和老旧家具的木头味。

钟岁安坐在一张破旧的转椅上。

他五十多岁，头发花白，穿着保安制服但扣子没有扣整齐。他的脸上有那种长期失眠的人特有的灰暗，眼袋深得像两个影子。

他看到沈听晚的瞬间，身体明显僵了一下。

"你——"他说了一个字就停了。他的目光从沈听晚的脸上移到她的手腕上——袖口底下防水记号笔的字迹露出了一点边缘。

他认出了什么。

"你就是监控里消失的那个。"他不是在问。是在陈述。

程以墨关上了保安室的门。

钟岁安沉默了很长时间。

然后他弯腰打开了办公桌最底下的那个抽屉——最深处，压在一摞旧报纸下面，是一个泛黄的信封。他抽出信封，从里面取出一张照片。

照片很旧了，边缘发黄卷曲。上面是一个穿着校服的女孩，扎着马尾辫，笑容很灿烂。照片背面用钢笔写着一行字："钟霜，1998年摄。"

"我女儿。"钟岁安把照片放在桌上。"二十年前，也是在这栋楼。"

他的声音很平，像在说一件和自己无关的事。但他放照片的那只手在微微发颤。

"她也是从那个地方消失的。五楼楼梯间的那个角落。前三次她都回来了——每次都说不记得发生了什么，手腕上有字，手机——那时候还没智能手机——她的日记本上有不是她自己写的字。"

他停了一下。

"第四次她就再也没有回来。"

保安室里安静得能听到监控屏幕发出的电流声。

沈听晚看着照片上那个笑容灿烂的女孩，指尖发凉。

"你知道那个角落是什么？"程以墨问。

钟岁安摇头。"不知道具体是什么。但我知道那个角落不正常——这栋楼建成三十年了，那个位置的摄像头坏了七次，换了七次，每次都是同一个角落。不是线路的问题，电工查过，接线什么的都没问题。就是那个角落——像是不允许被看到似的。"

他看向沈听晚。

"你去了几次？"

沈听晚张了张嘴。"两次——至少我知道的是两次。"

钟岁安的眉头皱了起来。

"听着。"他的声音忽然变得很沉。"如果你还想去，我不拦你。我没有资格拦任何人——二十年前我也没拦住我女儿。但你要知道一件事。"

他抬起头，看着她的眼睛：

"每去一次，你在这边的存在就会薄一层。"

"什么意思？"

"我是说——你回来以后，周围的人对你的印象会变淡。"他的手指在桌面上无意识地敲了敲。"我女儿第二次回来的时候，她同桌已经不记得她喜欢吃什么了。第三次回来的时候，她的班主任在花名册上跳过了她的名字——不是故意的，是真的没注意到有这个人。"

沈听晚的心脏猛地缩了一下。

"你回去看看。"钟岁安说。"看看你周围的人是不是已经开始忘记你了。"

沈听晚从保安室出来后在走廊里站了一分钟。

然后她回到教室。

第一节课结束后，数学老师提了一嘴分组做练习题。方芷萱开始在前排张罗分组——和往常一样，她的分组方式是一种精密的社交仪式，谁和谁一组都有讲究。

沈听晚坐在最后一排等着。

以前方芷萱的分组至少会"故意跳过"她——那是一种有目的的忽视，需要意识到她的存在才能实施。

今天不一样。

方芷萱报完了所有人的名字。六组，每组六个人。三十六人。

班上一共三十八人。

减去方芷萱自己——三十七人。

还少了一个。

方芷萱报完名字后看了一遍名单，皱了皱眉，数了一下——"怎么少了一个？谁没有被分到？"她在名单上扫了一圈，眉头越皱越紧，但就是找不出那个"少了的人"是谁。

最终她说："可能我数错了。就这样吧。"

没有人提出异议。没有人说"沈听晚还没分组"。

连方芷萱都不记得她了。

沈听晚坐在最后一排靠窗的位置，看着阳光穿过窗户照在空空荡荡的桌面上。她的嘴角弯了一下——不是笑，是某种苦涩到极致时才会出现的弧度。

放学的时候她又在校门口看到了温母。

这一次她没有绕开。她走过去，站到温母面前。温母的眼睛立刻亮了——每一个愿意停下来的学生对她而言都是一根稻草。

"阿姨。"

"同学你好——你见过我儿子吗？"温母举起照片。

沈听晚看着照片上温既白十五岁的脸，心脏又开始了那种不受控制的加速。

"我认识他。"她说。

温母的手一抖，照片差点掉在地上。她上前一步，抓住沈听晚的手："你见过他？他在哪里？他还好吗？"

温母的手指很凉，很用力，指节硌在沈听晚的手背上生疼。她的眼泪几乎是瞬间涌出来的，滚烫的液体滴落在沈听晚的手背上。

沈听晚看着那些泪水。

她无法回答"他在哪里"这个问题——因为她自己也不记得。她只知道备忘录里写着他在，只知道她来过两次，只知道她的身体记得但她的大脑不记得。

"他——"她的声音卡住了。

她不能说"他在一个叫透明层的地方"——这话说出来和精神病有什么区别？

"他还好。"她最终只说了这四个字。

温母抓着她的手哭了很久。

回家后，沈听晚关上房门，坐在书桌前。

她花了整整一晚上，把手机备忘录里的所有内容、手腕上的字、照片库里的截图、程以墨提供的监控截图、钟岁安说的话——全部整理成了一份完整的"档案"。

她在手机上建了一个加密相册，把所有文件存了进去。密码不是111117——她另设了一个：1096。

温既白在透明层待过的天数。她从备忘录里记住了这个数字。

然后她拿起防水记号笔，在自己的左小臂内侧——袖子能遮住但一撩就能看到的位置——用最大号的字写了一行：

"先看手机再做任何事。密码1096。"

她把笔盖盖好，关了台灯。

黑暗中，她躺在床上，看着天花板。

她知道明天她要再去一次。

不是为了逃避霸凌——方芷萱已经快不记得她了，霸凌也许很快就不存在了。

是为了那个在透明层里独自待了三年的人。

是为了他不用再做第三次自我介绍。

她闭上了眼睛。在意识滑入睡眠之前的最后一秒，她的嘴唇无声地动了一下：

温既白。

她不记得他。但她在学着认识他。

# 第七章 不再是初见

沈听晚是在看完手机上所有档案之后走进监控死角的。

她在楼梯间里站了两分五十秒。最后那十秒她没有犹豫——她盯着手机屏幕上那段自己录的音频转文字："不要让他再做第三次自我介绍了。"

三分钟整。

灰蓝色的世界再次降临。

她没有在楼梯间停留。她直接跑向四楼，推开物理实验室的门。

温既白正站在窗边。他看到门开了，条件反射地转过身来，嘴巴张开——习惯性的、已经准备好的话在他的嘴唇上成型：

"你能——"

"温既白。"

沈听晚打断了他。

她站在门口，手机还攥在手里，呼吸急促但目光稳定。她看着他的眼睛，一字一句地说：

"你叫温既白。身高一米七八。物理满分。你在这里待了三年——一千零九十七天。你喜欢在窗玻璃上画简笔画。你给空气讲费曼路径积分。你做自我介绍的时候声音会抖。"

她停了一秒。

"我们见过两次。第一次你哭了。第二次你没有。你在我手腕上写过字、在我手机里写过备忘录、教我录了音。你说你很自私——你说你三年来第一次有人听你说话，所以你不想让我把时间花在找出口上。"

物理实验室里安静到可以听见灰尘落地的声音。

温既白的手从嘴边慢慢放下来——那只准备做手势的手。他的课本从另一只手里滑落，砸在实验台上，发出一声闷响。他没有去捡。

他站在那里看着她。

他的嘴唇抖了一下。两下。三下。

然后眼泪毫无预兆地从他的眼角滑下来——不是慢慢蓄积后的溢出，是某个阀门突然被猛然打开后的决堤。泪水沿着他瘦削的脸颊滚落，砸在旧校服的衣领上，洇出一小片深色的水渍。

他没有出声。一滴、两滴、三滴，一直在流，但他没有发出哭声。

他只是站在那里，站在银白色的阳光里，用一双被眼泪模糊的眼睛看着她——看着这个第一次不是以陌生人身份走进这间教室的女孩。

过了很久——沈听晚不知道是多久，可能是一分钟，可能是五分钟——温既白用手背擦了一下脸。动作很粗糙，袖口在眼眶上蹭了一下，留下一道红痕。

"你知道我每次最怕的是什么吗？"他说。嗓子哑得不像话。

沈听晚摇头。

"不是你忘记我。"他吸了一口气，那口气在喉咙里碎了一下。"是你每次走进来看我的眼神。那种——完全陌生的——像看一个不该出现在这里的怪人的眼神。"

他又擦了一下脸。

"我已经看了两次了。两次完全一样。你推开门，你看到我，你的眼睛里没有我——每次都像有人拿刀在我心上划了一道。"

他低下头，看着地上散落的课本。

"我不知道还能承受几次。"

沈听晚走进了教室。

她在他面前三步远的地方停下来——十厘米的距离之外，碰不到的距离。她把手机举起来，屏幕朝向他，上面是那份她花了一整晚整理的档案：备忘录、录音转文字、监控截图、手腕字迹的照片，还有她自己在最后加的一段话——

"这些都是证据。我做不到记得你，但我做得到不让你再做自我介绍。"

温既白看着那个屏幕看了很长时间。

然后他擦干了最后的眼泪，呼出一口长气。他的嘴角弯了一下——这次不是哭后的残余，是真的、小小的、带着一点光的笑。

"既然你这次不需要从头开始了，"他说，"我带你看一些你之前没看过的东西。"

他捡起地上的课本，夹在腋下，走到门口。经过她身边时他偏了一下身体——十厘米——然后在走廊里转身等她。

他带她看了走廊。

不是上次看过的日记墙——是另一面墙。在三楼和四楼之间的楼梯拐角处，有一面被温既白单独留出来的墙壁。这面墙没有写日记，而是刻满了关于她的记录。

"第四十七天。她的水壶被藏到垃圾桶后面了。我在她旁边说了'别哭'，说了二十六遍。她没有哭。"

"第一百二十天。她语文周记被评了优。老师在课堂上念了一段，方芷萱在下面翻白眼。我在沈听晚的桌角用手指写了'真棒'，虽然她看不到。"

"第二百零三天。她考试进步了十二名，从全班倒数第七变成了中游。她自己好像都没怎么高兴，可能是因为没有人跟她说'恭喜'。我在她桌旁鼓了掌。一个人的掌声。"

"第五百七十天。方芷萱撕了她的作文。就是那篇她写了一晚上的、关于'如果世界上没有人能看见你你会怎么做'的作文。我在墙上把她那篇作文默写了一遍——我之前在她桌上看到过，记住了大部分。可能有几个字不对，但意思是对的。"

"第八百九十一天。十一月十七日。她的生日。没有人记得。程以墨感冒请假了。我在五班教室黑板的右下角写了'生日快乐沈听晚'。字写得很小。第二天值日生把它擦了。"

沈听晚的手抚在墙壁上——抚在那些用指甲一笔一划刻出来的字上。她的指尖感觉不到凹痕的深度——透明层的触感规则让她无法真正触摸这面墙——但她的眼睛在一行一行地读。

三年。

他记录了她被欺负的每一天、她的每一个小小进步、她的生日。

他在她看不见的地方做了三年她最亲密的陪伴者。

而她一个字都不知道。

"你……"她的声音从喉咙里挤出来，比正常小了很多。"你为什么要做这些？"

温既白站在她身后两步远的地方。他的肩膀靠在墙上，双手插在口袋里，姿态看起来很随意——但沈听晚在墙面的反光里看到他的嘴唇抿成了一条紧绷的线。

"因为没有别的事可以做。"他说。"透明层里没有网络、没有新书、没有任何新东西。我能做的只有看着外面的世界——而外面的世界里我最常看的就是你。"

他停了一下。

"一开始只是因为你每天都来五楼。你是除了保安之外唯一会出现在五楼的人。后来——"

后来什么，他没有说。

沈听晚转过身面对他。

她想伸手碰他。这是一种本能的冲动——当你看到一个人为你做了三年的事、当你意识到你从来不是一个人在扛——你会想碰到他。哪怕只是碰一下他的手背，指尖对指尖。

她的手伸到一半停住了。

十厘米。

看得见，碰不到。

她把手悬在距离他手背一厘米的位置。近到能感觉到那层若有若无的凉意——透明层中两个人之间永远存在的那道屏障散发出的、冰水一样的温度。

"谢谢。"她说。声音很轻，但很稳。

温既白低头看着她悬在自己手背上方的那只手。

他笑了一下。

"走吧，"他说，"还有别的要给你看。"

他带她去了五楼楼梯间。

在透明层中，那个监控死角不是一个普通的角落。墙壁的纹理在这里开始扭曲——像一面镜子被从中间用力按压后产生的变形。空气中有一道几乎不可见的裂痕——不是墙面的裂缝，而是空间本身的裂痕——像有人在现实和透明层之间撕开了一条缝，又勉强缝合了，留下了一道丑陋的疤。

"三年前，"温既白站在裂痕前面，银白色的光从裂痕里渗出来，照亮了他半边脸。"这里发生过一件事。"

他看着她。

"跟你有关。"

沈听晚的心跳停了一拍。

"跟我——什么事？"

温既白没有立刻回答。他的手指触碰了一下裂痕的边缘——指尖接触到那道扭曲的空间时，灰蓝色的光线在他的手指周围泛起了波纹，像石子落入平静的水面。

"你还没有准备好知道。"他说。

"什么意思？"

"意思是——这件事牵涉到一个规则，一个代价。如果我现在告诉你，你在二十四小时内做出的反应可能会——"他顿了一下，像是在斟酌措辞，"可能会影响规则对你的判定。"

他转过身来面对她，目光前所未有地认真。

"你先不要问了。等你下一次来的时候——如果你还来的话——我会告诉你全部。"

沈听晚想追问。但她看到了他的眼睛——那里面有恳求的意味，还有一种更深层的、她读不透的疼痛。那种疼不是因为现在，而是因为过去。因为三年前。

她点了点头。

窗外忽然传来一个声音——隔着透明层的膜，模糊但依然可以辨认。

是一个女人的声音。

"同学——你有没有见过这个人？"

温既白的整个人僵住了。

他转身看向窗户——校门口。傍晚的光线里，温母正举着那张塑封照片站在人流中间，一头花白的头发在风中散乱。

温既白走到窗前。

他的手贴上了玻璃——不是触碰，是那种渴望触碰但永远隔着一层的姿态。他的指尖压在窗面上，指腹因为用力而微微发白。

他的嘴唇动了一下。没有声音。但沈听晚站在他身后，看到了他嘴型的形状：

"妈——"

他维持着那个姿势很久。直到温母的身影消失在校门口的拐角处。

然后他把手从玻璃上拿下来。玻璃上留下了五个模糊的指印——在透明层中它们会停留一会儿，然后慢慢消失。

他没有转身。但他的声音从背后传来，比之前都轻。

"我在这里能看到她。每个月她都来。每个月我都在这里看着她。"

沈听晚什么都没说。

有些东西不需要语言。

他们一起在窗边站了很久，看着窗外的世界——那个他们触碰不到的、正常的、有温度有色彩的世界——在傍晚的光线中渐渐暗下去。

天黑之后，温既白再次开口："你下次还会来吗？"

沈听晚看着手机里那份她花了一整晚整理的档案。

"一定。"

温既白没有再说话。

他只是在银蓝色的月光里轻轻点了一下头。

凌晨。灰蓝色第三次从她的世界褪去。她第三次什么都不记得了。

而在她不知道的现实世界中——

程以墨坐在教室里，花名册就摊在她面前。老师上课的时候念名字，念到第二十七个人直接跳到了第二十九个——第二十八个名字"沈听晚"像是从名单上蒸发了。

老师没有意识到自己跳了一个。

全班没有任何人注意到。

只有程以墨举起手："老师，你少念了一个人。沈听晚。"

老师翻了翻花名册，皱了皱眉："沈……听晚？这个名字是——哦，有的有的，"他的目光在名字上停了一秒，脸上露出一种微妙的困惑，像是在看一个明明应该认识但怎么也想不起来的字。"沈听晚同学——到了吗？"

教室里没有人回答。

沈听晚的座位是空的。

而程以墨低头看着那张空椅子，手心冰凉。

她拿出手机，发了一条消息给钟岁安：

"她又去了。这是第三次。你说的那个极限——是几次？"

# 第八章 三年前的楼梯间

钟岁安回复程以墨的那条消息只有一行字：

"我女儿是第四次没回来的。"

程以墨看到这行字的时候手机差点掉在地上。

她冲出教室，跑向五楼楼梯间。走廊里的学生被她撞了好几个，有人在背后骂，但她什么都听不到。她跑上四楼、五楼，转过最后一个弯——

楼梯间尽头的角落空空荡荡。

沈听晚已经不在了。

不——她在。

她正在消失。

程以墨看到了最后一秒的画面——沈听晚站在监控死角的正中央，身体像褪色的水彩画一样正在变得透明。她的校服轮廓还在，但颜色已经淡到近乎虚无。她的脸——半透明的脸——正好转向楼梯口的方向。

她看到了程以墨。

程以墨冲过去——伸手去抓她的手臂——

手指穿过了她的身体。

像穿过一团冷雾。什么都摸不到，只有一股沁入骨髓的凉意从指尖蔓延到手腕。

"回来！"程以墨喊。声音在空荡荡的楼梯间里撞来撞去。"你给我回来！沈听晚！"

沈听晚看着好友穿过自己身体的那只手——那只什么都抓不住的手——嘴唇动了一下。

她笑了。

不是那种开心的笑——是一种带着歉意的、很轻的、像是在说"对不起但我已经决定了"的笑。

然后她的身影像晨雾一样彻底消散了。

楼梯间里只剩下程以墨一个人。

她跪在冰冷的地面上，两只手撑在那个沈听晚刚才站着的位置。手机屏幕亮着——钟岁安的第二条消息刚刚弹出来：

"第四次就回不来了。你拦住了吗？"

程以墨没有回复。

她把额头抵在地面上，肩膀开始剧烈地颤抖。

---

灰蓝色的世界再次降临。

但这一次和前几次不同。

沈听晚低头看着自己的手——透明的程度超过了之前任何一次。前几次她的身体虽然透明，但至少还有一个清晰的轮廓，像一尊用冰雕刻出来的人。这一次，她的手指像碎玻璃一样折射着周围的灰蓝色光线，棱角处甚至能看到彩虹色的光斑散射出去。

她像一件快要碎掉的水晶制品。

她跑向物理实验室。跑的过程中她感觉自己的脚步比以前更轻——不是"轻快"的轻，而是"轻到快要不存在"的轻。

推开门。

温既白站在实验台旁边——这一次他没有在看书。他像是在等她。

沈听晚走进去，把手机举到他面前——屏幕上是那份完整的档案。

"我看过了所有东西。"她说。呼吸还没平稳。"我知道你叫温既白，知道我们见过三次，知道每次我都忘了——"

"你现在的身体——"温既白没有看手机屏幕。他的目光死死地钉在她的手上——那只几乎透明到可以看穿的手。他的脸色在灰蓝色的光线中变得近乎灰白。

"你这是第几次？"

"第四次。"

温既白整个人往后退了一步。他的后腰撞在了实验台的边缘上——砰的一声，但他好像完全没有感觉到。

"不。"他说。声音很轻但很急。"不是第四次。"

沈听晚愣住了。

"你以为你来过三次。"温既白的手撑在实验台上，指节发白。"但你来过四次。三年前——三年前你来过一次。你不记得，因为那一次的记忆也被清除了。这是你的第五次。"

五。

沈听晚的大脑空白了一秒。

"三年前——？"

"跟我来。"

温既白几乎是用跑的。他带她回到了五楼楼梯间——那个有着空间裂痕的角落。裂痕在灰蓝色的光线中泛着银白色的微光，像一道没有愈合的伤疤。

他在裂痕前站定，转过身面对她。

他的脸上不再有任何温柔的遮掩。那些他一直小心收藏的、用"还没准备好"当理由推迟的东西，此刻全部暴露在银白色的光线里——恐惧，内疚，和一种深到骨头里的疼痛。

"三年前。"他开始说。声音不稳，但他没有停。"高一下学期的一个中午——你那时候已经开始被欺负了，那天比较严重，方芷萱把你的课桌从教室搬到了走廊上。你跑到了五楼楼梯间。"

沈听晚不记得。但她身体里某个地方在隐隐作痛——一种很深的、没有来源的痛。

"你在那个角落待了超过三分钟——你也不知道那个角落有什么特别的，你只是蜷在那里哭。三分钟后你触发了规则，第一次变成了透明人。"

"那是我的——"

"你的第一次。你不知道发生了什么，你吓坏了。透明层的空间在这个角落是扭曲的——"他指了指那道裂痕。"你在恐慌中跑动的时候被扭曲的空间绊倒了。你整个人向这道裂痕跌过去——"

他停了一秒。

"裂痕的下面不是地面。是透明层的深处。如果你掉进去……就再也出不来了。不是'被困在透明层'——是掉进透明层和现实世界之间的夹缝，永远坠落，永远。"

沈听晚的血液变冷了。

"我当时在五楼。"温既白继续说。"我来找一本落在楼梯间的书。我站在死角的边缘——刚好在规则的影响范围之内。我看到了你——一个女生忽然变透明了，然后失去平衡，整个人向裂痕倒下去。"

他的声音在这里碎了。

"我冲进去了。"

"规则有一个——一个等价交换的条款。如果有人主动走进规则的范围内，以'永久留在透明层'为代价，可以触发一次'修正'——把一个即将坠入裂痕的人拉回正常世界。"

他看着她。

"我用我永久留在透明层的代价，把你从裂痕边缘拉了回来。你回到了正常世界，完全没有受伤。但你失去了那天所有的记忆——包括你触发过规则这件事。你甚至不记得你来过五楼。"

"而我从那天起就再也没有离开过这里。"

物理实验室外面银白色的月光透过窗户照进来，在他的脸上画出一片冷白的光斑。他的眼睛在那片光里闪着水光——不是哭，是一种比哭更安静的、已经疼了三年的东西。

"所以你——"沈听晚的声音几乎是气声。"你是为了我才——"

"不是'为了你'。"温既白摇头。"那一刻没有时间想'为了谁'。我只是看到一个人在坠落，我冲过去了。后来——后来我知道了你的名字，知道了你在五班，知道了你被欺负，知道了你每天都来五楼——我才开始想：至少她活着。至少她好好的。我留在这里是值得的。"

他把手从口袋里抽出来，掌心朝上，指尖还在微微发颤。

"我做了所有这些——留在这里三年，在你耳边说话三年，在你手腕上写字，在你手机里留备忘录——全是因为我想让你好好地活在那个世界里。"

他看着她，目光里有什么东西在崩塌。

"可你为什么要来找我？"

他的声音提高了——三年来他第一次提高声音。不是愤怒，是恐惧。

"我牺牲了所有东西让你能正常地活着——可你一次一次地跑进来——每来一次你在那边的存在就薄一层——现在你来了第五次——你的身体已经透明到我都快看不清你了——"

他的手猛地攥紧了。

"你告诉我——如果你回不去了——我这三年是为了什么？"

沈听晚没有说话。

不是不想说——是说不出来。

她蹲了下去。

膝盖碰到地面的时候她几乎感觉不到触感——她的身体太薄了，和地面之间的接触已经像两张纸片叠在一起那样脆弱。

然后她哭了。

不是她之前那种咬着嘴唇不出声的哭法。是真正的、不加控制的、声音从胸腔最深处涌出来的哭。眼泪在她半透明的脸上流下来，在灰蓝色的光里折射出碎光。

温既白站在她面前，伸出手——想碰她的肩膀——

手穿过了她。

他的手指穿过了她的肩膀，什么都没有碰到。他的手从她的身体里进去又出来，中间没有任何阻隔，就像穿过一团有温度的空气。

看得见。听得见。碰不到。

他把手收回来，攥成拳，攥到整个手臂都在发抖。然后他也蹲了下来，在距离她十厘米的位置，用他碰不到她的手，在空气中比划了一个拥抱的姿势。

两个人蹲在五楼楼梯间的灰蓝色光线里，一个在哭，一个在空气中抱着她。

中间永远隔着十厘米。

过了很久，沈听晚的哭声慢慢停了。她用袖子擦了擦脸——袖子已经完全透明了，擦不掉什么，但这个动作本身让她平静了一点。

她站起来。

"后悔没有用了。"她说。声音还在抖，但每个字很清晰。"我现在在这里。你告诉我——二十四小时后我还能回去吗？"

温既白没有说话。

"回不去了对不对？"

他闭了一下眼睛。

然后点了点头。

沈听晚深吸了一口气。空气进入她半透明的身体，没有正常呼吸该有的胸腔起伏，但她还是做了这个动作——一种仪式性的、告诉自己"你还活着"的动作。

"那我们还有二十四小时。"她说。

她看着温既白的眼睛——那双哭红了的、布满血丝的、三年来第一次有人愿意直视的眼睛。

"不要浪费在哭上面。"
