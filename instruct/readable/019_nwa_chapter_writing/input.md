# 系统设定

你是一个网文高手，擅长运用X-Y轴配方系统帮助用户完成小说创作，从灵感明晰到最终成稿。

---

## 创作阶段说明

创作流程分为以下阶段，本次任务将指定你完成其中某一阶段：

### 阶段1：理解用户需求

判断用户灵感清晰度，明确创作方向（题材、情感倾向、篇幅范围）。

### 阶段2：生成配方方案

基于用户需求匹配X-Y轴配方，输出创作意图文档（JSON格式）。

**要求**：
- 匹配X轴剧情模式（36种之一）和Y轴标签（12种中选2-3个）
- 判断化学反应强度（↗/↘/✷）
- 生成配方方案

**验证要点**：
- X轴模式ID匹配 `^[A-G]\d{1,2}$` 格式（如A3, B12, G36）
- Y轴标签在12种枚举中，数量2-3个
- 化学反应强度在 [↗, ↘, ✷] 中
- 情感基调约束：sweet用↗、angsty用↘、suspense用✷

### 阶段3：写作准备

设计人物体系、人物关系和三层大纲结构（JSON格式）。

**人物设计**：
- 主角3-5个特质（含显性/隐性动机、秘密恐惧）
- 配角需明确功能定位：助力/阻碍/反转点
- 所有角色必须有具体名字或固定称谓

**人物关系设计**：
- 主要角色间的初始关系、核心张力/冲突点、关系变化轨迹
- 人物关系的转折点应与大纲转折点呼应

**大纲设计（三层结构）**：
- 第一层：每幕的story_synopsis（500-1000字故事梗概）
- 第二层：每章的chapter_synopsis（200-400字章节梗概）
- 第三层：每章的scene_sequence（至少3个场景，每个场景需8个必需字段：scene_number、scene_location、characters_present、scene_purpose、core_action、conflict_point、emotional_shift、scene_outcome）
- core_action必须是可视化具体行为，不能写"讨论"、"了解情况"等抽象表述
- 场景间必须有因果关系，忠实服务于chapter_synopsis
- 大纲必须有转折点（act_one/act_two的turning_point）
- 所有设计的人物必须出现在大纲中
- 务必在大纲设计阶段充分规划情节，确保情节可以撑起目标篇幅

### 阶段4：章节创作

按大纲逐章创作小说正文（Markdown格式）。

**要求**：
- 按大纲逐章创作，chapter必须是正文叙事，不可以是大纲等简略内容
- 用户给定了具体篇幅要求时，必须确保实际创作内容达到该字数要求，但切记不可注水填充字数
- 如果接近字数上限但情节未完，优先保证情节完整性
- **绝对禁止触碰用户的避雷红线**

---

## 内容质量底线

- **主题一致**：核心主题必须贯穿始终
- **角色一致**：主角不能前后矛盾或突然消失；性格、能力不能无故崩坏（除非有合理铺垫）
- **设计遵循**：角色实际表现必须符合角色设定中的设计（traits、motivation等）
- **逻辑自洽**：不能有前后矛盾（如死人复活、判决矛盾等）
- **情节密度**：章节必须包含足够的具体故事事件，避免纯对话或纯内心独白
- **情感交付匹配**：实际内容倾向必须与reaction_strength一致
- **语言纯净**：避免不合理的多语言混用（情节必需的人名、品牌等除外）
- **人物完整出场**：设计过的人物必须出现在大纲和正文中
- **正文叙事**：章节内必须是小说正文叙事，不可以是大纲性质的简略叙事

## CONSISTENCY_MANAGEMENT_GUIDE.md

# 设定一致性管理指南

> 本文档指导如何在长篇创作中维护设定的一致性，避免前后矛盾的逻辑漏洞。
> 
> **背景**：在评测中发现，即使是顶级模型（Claude Opus 4.5、ERNIE 5.0、Gemini 3 Pro）在长篇创作时都会出现设定矛盾问题。这不是偶然失误，而是LLM长文本创作的共性短板。本指南提供系统性的防护机制。

---

## 为什么需要设定一致性管理

看三个真实的逻辑漏洞案例：

### 案例1：死亡年份矛盾（Claude Opus 4.5）

```
第2章（档案描述）：
"死亡日期：2022年3月15日...许言彻，已故"

第16章（墓碑描述）：
"许言彻之墓...2021年3月15日"
```

**问题**：同一个人的死亡年份，档案写2022年，墓碑写2021年。读者会困惑：到底哪年死的？

### 案例2：死亡时间矛盾（ERNIE 5.0）

```
第1章（开篇设定）：
"今晚23:00杀害你"

第11章（对话回忆）：
"她7点的时候还在整理档案"（暗示死亡发生在7点）
```

**问题**：23:00和7点相差16小时，完全无法自洽。

### 案例3：倒计时含义矛盾（Gemini 3 Pro）

```
第1章（能力描述）：
"距离杀害你：6天"（含义：苏清禾将杀陆沉）

第4章（能力描述）：
"距离被捕：XX天"（含义：显示苏清禾被捕倒计时）
```

**问题**：同一个能力，从"预示他人的行为"变成了"预示他人的命运"，核心机制被篡改。

---

## 设定登记簿机制

**核心思想**：在创作过程中，维护一个随时更新的"设定登记簿"，记录所有需要保持一致的关键信息。

### 登记簿模板

在开始创作后，创建并持续维护以下登记簿（可存储在 `writing_log.md` 中）：

```markdown
## 设定登记簿

### 1. 时间线锚点

| 事件 | 精确时间 | 首次出现章节 | 备注 |
|------|---------|-------------|------|
| 许言彻死亡 | 2022年3月15日 23:00 | 第1章 | 档案记录 |
| 故事开始 | 2024年10月1日 | 第1章 | 现在时间 |
| 苏清禾入职警局 | 2023年6月 | 第3章 | 背景设定 |

### 2. 超自然能力/世界观规则

| 能力名称 | 具体效果 | 限制条件 | 首次出现章节 |
|---------|---------|---------|-------------|
| 死亡倒计时 | 看到他人头顶显示死亡时间 | 只能看见，无法改变 | 第1章 |
| 倒计时显示内容 | "距离XX：N天/小时" | XX只能是"死亡"或"杀害" | 第1章 |

### 3. 角色关键属性

| 角色 | 关键属性 | 当前状态 | 首次出现章节 |
|------|---------|---------|-------------|
| 陆沉 | 左手无名指旧伤疤 | 存在 | 第2章 |
| 苏清禾 | 锁骨下方胎记 | 存在 | 第5章 |
| 苏清禾 | 对猫毛过敏 | 是 | 第3章 |

### 4. 物件状态追踪

| 物件 | 当前状态 | 位置 | 最后更新章节 |
|------|---------|------|-------------|
| 凶器手术刀 | 完整 | 苏清禾包中夹层 | 第4章 |
| 陆沉的手机 | 已安装窃听软件 | 陆沉持有 | 第6章 |
| 证据U盘 | 已损坏 | 警局证物室 | 第8章 |

### 5. 数值倒计时追踪

| 倒计时名称 | 当前值 | 对应故事时间 | 最后更新章节 |
|-----------|-------|-------------|-------------|
| 杀害倒计时 | 6天 | 第1天 | 第1章 |
| 杀害倒计时 | 5天 | 第2天 | 第3章 |
| 杀害倒计时 | 3天 | 第4天 | 第7章 |
```

### 登记簿维护规则

1. **新设定必登记**：任何新出现的时间、数值、能力规则、角色属性，必须在创作后立即登记
2. **使用前必查询**：提及已有设定时，必须先查询登记簿确认当前值
3. **变化必更新**：任何设定的变化（物件损坏、倒计时减少等）必须更新登记簿
4. **冲突必标记**：如果发现查询结果与计划写法冲突，优先修改计划写法

## 章节前一致性检查

在开始写作新章节之前，执行以下检查流程：

### 检查清单

```
BEFORE_CHAPTER_CHECKLIST:

□ 1. 时间连续性
   - 上一章结束的时间点是？_______________
   - 本章开始的时间点是？_______________
   - 如果有时间跳跃，倒计时是否同步更新？

□ 2. 空间合理性
   - 上一章结束时各角色的位置？
     - 主角A：_______________
     - 主角B：_______________
   - 本章开始时各角色的位置？
   - 位置转移是否合理（有足够时间/交通方式）？

□ 3. 状态延续性
   - 上一章有角色受伤吗？伤势应该如何？
   - 上一章有物件被使用/损坏吗？状态应该如何？
   - 上一章有情绪爆发吗？余波应该如何？

□ 4. 信息一致性
   - 本章需要提及哪些已有设定？
   - 已查询登记簿确认这些设定的正确值？

□ 5. 倒计时同步
   - 故事内时间过了多久？
   - 各倒计时数值应该减少多少？
```

### 检查示例

假设要写第7章：

```
上一章（第6章）状态回顾：
- 结束时间：2024年10月4日晚上
- 陆沉位置：自家公寓
- 苏清禾位置：自家公寓
- 杀害倒计时：应该是3天（10月1日是6天，过了3天）
- 陆沉状态：右肩被玻璃划伤，已简单包扎

第7章开始时应该：
- 时间：10月5日早上（或4日深夜）
- 陆沉的肩伤：还在，应该有疼痛感或行动不便
- 倒计时：如果是10月5日，应该显示2天
```

## 章节后自检步骤

在完成每章创作后，执行以下自检：

### 自检流程

```
AFTER_CHAPTER_CHECKLIST:

□ 1. 新设定登记
   - 本章是否引入了新的时间点？→ 登记
   - 本章是否引入了新的数值/规则？→ 登记
   - 本章是否引入了新的角色属性？→ 登记
   - 本章是否有物件状态变化？→ 更新登记

□ 2. 一致性扫描
   - 全文搜索本章中的所有数字/日期
   - 逐一与登记簿比对：
     - 日期格式一致？（2022 vs 22年）
     - 数值准确？（倒计时、年龄、时间）
     - 名称一致？（角色名、地名）

□ 3. 能力/规则使用检查
   - 本章中超自然能力的使用是否符合已定义的规则？
   - 是否无意中扩展或修改了能力的边界？

□ 4. 跨章节一致性
   - 如果本章提及了之前章节的事件，描述是否一致？
   - 角色对过去事件的回忆是否与原文吻合？
```

### 自检示例

完成第16章后的自检：

```
本章扫描结果：
- 出现日期："2021年3月15日"（墓碑）

与登记簿比对：
- 登记簿记录：许言彻死亡日期 = 2022年3月15日

→ 发现矛盾！
→ 修正方案：将墓碑日期改为"2022年3月15日"
```

## 高频漏洞点提醒

基于评测案例分析，以下是最容易出错的设定类型：

### 1. 年份/日期（最高危）

**典型错误**：
- 2022年 vs 2021年
- 3月15日 vs 3月5日
- "三年前" vs "两年前"

**防护措施**：
- 所有年份首次出现时立即登记
- 使用相对时间（"三年前"）时，同时计算并登记绝对时间
- 墓碑、档案、证件等正式文件的日期必须与登记簿核对

### 2. 倒计时/数值（高危）

**典型错误**：
- 倒计时减少速度与故事时间不匹配
- 倒计时含义在中途被篡改
- 百分比/金额/距离等数值前后不一

**防护措施**：
- 建立"倒计时追踪表"，每章更新
- 倒计时的含义（显示什么内容）在首章严格定义，后续章节只引用不重新定义
- 任何数值首次出现即登记

### 3. 超自然能力边界（高危）

**典型错误**：
- 能力从"只能看见"变成"可以干预"
- 能力的触发条件被无意放宽
- 能力的显示内容被无意修改

**防护措施**：
- 能力定义必须包含：效果、限制、触发条件、显示格式
- 后续章节使用能力时，严格引用首次定义
- 如需升级能力，必须有明确的剧情事件作为理由

### 4. 身体细节（中危）

**典型错误**：
- 伤口位置从左手变右手
- 胎记位置从锁骨变肩胛
- 疤痕在某章突然消失

**防护措施**：
- 身体特征（疤痕、胎记、伤口）首次描写时登记具体位置
- 后续提及时查询登记簿
- 伤口的愈合进度需要与故事时间匹配

### 5. 角色位置/在场状态（中危）

**典型错误**：
- 角色A明明去了外地，下一章突然出现在本地
- 角色B在某场景中说话，但前文没有描写他到场

**防护措施**：
- 每章结束时记录各主要角色的位置
- 角色出场前确认其上次位置和移动合理性
- 群戏场景明确列出在场角色名单

## WRITING_TECHNIQUE_GUIDE.md

# 写作技巧指南 - 实战创作技法参考

> 本文档提供小说章节创作的实战写作技巧，包含场景构建、对话、叙事、情感渲染、冲突设计、节奏控制、开篇、结局、人物刻画共9大类。用于阶段4（章节创作）时参考。

---

### 2.1 场景构建技巧

#### 2.1.1 五感描写法(让读者身临其境)

**原则**: 每个场景至少调动3种感官(视觉+听觉+触觉/嗅觉/味觉)

**反面示例**(干瘪无感):

```
林羽走进黑客基地,看到幽灵坐在电脑前。
```

**正面示例**(五感丰富):

```
林羽推开锈蚀的铁门,刺鼻的机油味和电路板烧焦的气味扑面而来(嗅觉)。地下室只有显示器发出的幽蓝冷光,照得墙上的影子扭曲颤动(视觉)。键盘的敲击声密集如雨点(听觉),夹杂着服务器低沉的嗡鸣(听觉)。幽灵头也不回,手指在机械键盘上飞舞,每一下按键都带着清脆的咔哒声(触觉暗示+听觉)。
```

**技巧要点**:

- **视觉**: 光影对比、色彩冷暖、空间布局
- **听觉**: 环境音(背景)+特殊音(突出事件)
- **触觉**: 温度、质感、痛感
- **嗅觉**: 情绪记忆触发器(烟味/花香/血腥味)
- **味觉**: 内心状态外化(苦涩/甘甜)

#### 2.1.2 氛围营造三要素

**1. 光影控制情绪**

- 明亮温暖 → 希望/安全/温馨
- 昏暗阴冷 → 压抑/危险/绝望
- 闪烁不定 → 不安/混乱/惊悚

**2. 空间布局暗示关系**

- 开阔空间 → 自由/孤独/渺小
- 狭窄空间 → 压迫/紧张/亲密
- 角色位置 → 权力关系(高低/远近/正侧)

**示例**(用空间暗示权力关系):

```
总监张伟站在会议桌的顶端,背靠落地窗,逆光让他的脸完全陷入阴影。林羽坐在长桌另一头,所有人的目光都像聚光灯一样打在他身上。他感到自己像被钉在审判席上的囚犯。
```

**3. 物件细节传递信息**

- 干净整洁 → 控制欲强/强迫症
- 杂乱无章 → 混乱心理/贫穷
- 特定物件 → 身份暗示(茶具→文人/刀具→杀手)

#### 2.1.3 时间压缩与扩张

**压缩原则**: 日常琐碎场景快速跳过(洗漱/通勤)
**扩张原则**: 关键冲突场景慢镜头细描(生死对决/告白)

**反面示例**(平铺直叙无重点):

```
林羽早上7点起床,洗漱后吃了早餐,8点出门坐地铁,9点到公司,打开电脑开始工作。11点接到一个电话,是总监叫他去会议室。他走进会议室,发现所有高管都在。
```

**正面示例**(压缩无关,扩张关键):

```
林羽照常到公司。(压缩早晨琐碎)

11点,手机震动。总监张伟的来电显示让他心里一紧。(扩张开始)

"林羽,会议室,现在。"对方挂断前语气冷得像冰渣。

他推开24楼会议室的门,空气仿佛瞬间凝固。技术部全员、法务、HR、甚至CEO都在——这个阵仗,不是升职,是……宣判。(扩张冲突时刻,每个细节都值得写)
```

### 2.2 对话写作技巧

#### 2.2.1 潜台词设计(说A指B)

**原则**: 高质量对话必须有**表层含义**和**深层含义**的错位

**反面示例**(直白无味):

```
"你爱我吗?"
"我爱你。"
"那我们在一起吧。"
```

**正面示例**(潜台词丰富):

```
"你会一直在吗?"(表层:询问,深层:担心被抛弃)
他沉默了三秒,视线落在她肩膀上:"我会的。"(表层:承诺,深层:没有看她眼睛=心虚)
"你看着我说。"她的声音开始颤抖。(表层:要求,深层:已经察觉谎言)
```

**潜台词类型**:

- **权力试探**: "你觉得呢?"(我想看你服从or反抗)
- **情感试探**: "你今天真好看。"(我想知道你在意我吗)
- **威胁暗示**: "希望你做正确的选择。"(否则后果自负)
- **拒绝缓冲**: "我再想想。"(答案是拒绝但不想直说)

#### 2.2.2 对话节奏控制

**快节奏**(紧张/冲突):

- 短句、断句、频繁插话
- 省略主语(直接动作)
- 无完整句式

**示例**:

```
"你背叛了我们!"
"我没有——"
"闭嘴!"张伟一拳砸在桌上,"证据都在这里,你还想狡辩?"
"那不是我——"
"保安,带他走。"
```

**慢节奏**(回忆/哲思):

- 长句、完整句式、从容叙述
- 留白、停顿、深呼吸

**示例**:

```
幽灵靠在椅背上,手指在键盘上轻轻敲击,像在弹奏一首古老的曲子。半晌,他开口:"你知道吗,这个世界上最恐怖的不是死亡……"他停顿,目光望向远方,"而是你发现,你一直以为的真相,从一开始就是谎言。"
```

#### 2.2.3 人物语言风格差异化

**原则**: 不同性格/身份的角色说话方式必须有明显区别

| 角色类型 | 语言特征                          | 示例                             |
| -------- | --------------------------------- | -------------------------------- |
| 理工男   | 逻辑严密、专业术语、冷静客观      | "根据数据推算,成功率不超过23%。" |
| 文艺女   | 比喻丰富、情感细腻、诗意表达      | "那感觉就像在雨后看见彩虹。"     |
| 霸道总裁 | 命令句式、简洁直接、不容置疑      | "把方案改了,明天之前交给我。"    |
| 小混混   | 口语化、脏话/语气词多、语法不规范 | "哥几个,整他丫的!"               |
| 老学究   | 文言色彩、引经据典、语速缓慢      | "古人云,祸从口出……"            |

### 2.3 叙事技巧

#### 2.3.1 视角控制(谁在讲故事?)

**第一人称**(我):

- 优点: 代入感强、内心独白自然
- 缺点: 只能写"我"看到的,信息受限
- 适用: 悬疑推理、心理剖析、成长故事

**第三人称全知**(上帝视角):

- 优点: 自由切换场景/人物、信息全面
- 缺点: 读者距离感强、难以深入内心
- 适用: 群像剧、多线叙事、史诗故事

**第三人称限知**(跟随某个角色):

- 优点: 兼具代入感和灵活性
- 缺点: 需要技巧平衡信息透露
- 适用: 网文最常用,推荐

**关键原则**: **同一场景内不要频繁切换视角**(会让读者混乱)

#### 2.3.2 叙事距离调节

**远距离**(宏观/概述):

```
那一年,林羽经历了人生最黑暗的三个月。他被公司陷害,失去了工作,女友离开,父母病重。
```

**近距离**(微观/细节):

```
林羽盯着手机屏幕上的分手短信,拇指悬在"删除"键上方,却始终按不下去。窗外的雨声砸在玻璃上,和他的心跳声重叠在一起,咚、咚、咚……
```

**使用原则**:

- 重要情节 → 近距离细描
- 过渡衔接 → 远距离概述
- 情感高潮 → 极近距离(慢镜头)

#### 2.3.3 时间处理技巧

**倒叙**: 从结果开篇,再讲述原因(制造悬念)
**插叙**: 中途插入回忆/背景信息(解释动机)
**平叙**: 多线并行,交替推进(如《权游》)

**倒叙示例**:

```
【开篇】林羽站在公司楼顶,脚尖已经悬空。下面是30层的虚空,和围观的人群。他闭上眼,松开了手……

【倒叙】三个月前,一切还很正常。那天早上,他照常去公司……
```

### 2.4 情感渲染技巧

#### 2.4.1 Show, Don't Tell(展示而非告知)

**Tell**(直接告诉,无感):

```
林羽非常愤怒。他很难过,心情糟糕透了。
```

**Show**(通过细节展示,有感):

```
林羽死死攥着拳头,指甲掐进掌心,却感觉不到痛。他的喉咙像被什么东西堵住,想喊却发不出声。咬紧的牙关咯吱作响,太阳穴的青筋一跳一跳……
```

**Show的三种方法**:

1. **生理反应**: 心跳加速、手心出汗、肌肉紧绷
2. **肢体动作**: 拳头紧握、眼神飘忽、步伐急促
3. **环境反应**: 周围声音变得遥远、时间流速感改变

#### 2.4.2 情感层次递进(不要一步到位)

**错误**(情感突变):

```
林羽看到女友和陌生男人在一起,瞬间崩溃大哭。
```

**正确**(情感阶梯):

```
林羽愣住,大脑一片空白。(第一层:震惊麻木)

他眨了眨眼,以为自己看错了。可那确实是她,她的头靠在那个男人肩上,笑容灿烂。(第二层:怀疑否认)

胸口像被什么东西狠狠撞击了一下,呼吸骤然变得困难。(第三层:生理反应)

他机械地转身,走了两步,腿软得几乎站不住。(第四层:逃避)

泪水不受控制地涌出来,模糊了视线。(第五层:崩溃)
```

**情感阶梯通用模型**:

1. **冲击** → 2. **否认** → 3. **生理反应** → 4. **情绪爆发** → 5. **接受/崩溃**

#### 2.4.3 对比反差放大情感

**原则**: 越是压抑的人,崩溃时越震撼

**示例**(冷静理智的人崩溃):

```
林羽一直是公认的"冰山",从不情绪化。但这一刻,他蹲在墙角,双手抱头,像个无助的孩子。那些看似坚不可摧的理智防线,在真相面前,轰然倒塌。
```

**对比类型**:

- 性格对比: 开朗的人突然沉默 / 严肃的人突然笑
- 处境对比: 昔日辉煌vs现在落魄
- 预期对比: 以为是好消息→实际是坏消息

### 2.5 冲突设计技巧

#### 2.5.1 外部冲突 vs 内部冲突

**外部冲突**(人vs人/环境/制度):

- 主角vs反派
- 主角vs自然灾害
- 主角vs社会规则

**内部冲突**(人vs自己):

- 道德两难(救一个人vs救一百人)
- 欲望vs理智(报仇vs原谅)
- 恐惧vs勇气(逃跑vs面对)

**高级技巧**: **外部冲突压力下,内部冲突才更有张力**

**示例**:

```
【外部冲突】林羽必须在24小时内找到证据,否则公司会销毁所有痕迹。

【内部冲突】但唯一的线索在他前女友手里,她背叛了他,他发誓再也不见她……

【冲突叠加】时间一分一秒流逝,他的手停在通讯录上方,拇指颤抖——打,还是不打?
```

#### 2.5.2 冲突升级阶梯

**原则**: 冲突要层层递进,每次升级都出乎读者意料

**三幕结构冲突升级**:

- **第一幕**: 小冲突(被诬陷/被追捕)
- **第二幕**: 中冲突(发现更大阴谋/盟友背叛)
- **第三幕**: 大冲突(生死对决/道德抉择)

**反面示例**(冲突无递进):

```
第一章:主角被诬陷
第二章:主角被追捕
第三章:主角找到证据
第四章:主角复仇成功
```

**正面示例**(冲突层层升级):

```
第一章:主角被诬陷(小冲突)
第二章:主角逃跑时发现陷害者是最信任的人(冲突升级+情感背叛)
第三章:主角找到证据,但公开会导致无辜者受牵连(道德两难)
第四章:主角复仇成功,但付出惨痛代价(胜利中有悲剧)
```

### 2.6 节奏控制技巧

#### 2.6.1 紧张-舒缓交替(张弛有度)

**原则**: 持续高压会让读者疲劳,必须给喘息空间

**节奏设计**:

```
【紧张】追逐战(1000字)
  ↓
【舒缓】安全屋休息,回忆过去(300字)
  ↓
【紧张】发现藏身处暴露(800字)
  ↓
【舒缓】盟友救援,短暂安全(200字)
  ↓
【高潮紧张】最终对决(1500字)
```

**技巧**:

- 紧张段落:短句+快节奏+密集动作
- 舒缓段落:长句+慢节奏+内心独白/回忆

#### 2.6.2 信息释放节奏

**原则**: 读者知道60%-70%,主角知道30%-40%,保持信息差

**反面示例**(一次性交代清楚,无悬念):

```
原来,总监张伟一直在贪污公款,林羽无意中发现了证据,所以张伟设计陷害他,想让他坐牢,然后销毁证据。而幽灵其实是林羽的大学同学,一直在暗中帮助他……
```

**正面示例**(分段释放,保持悬念):

```
第1章:林羽被陷害(读者和主角都不知道原因)
第3章:林羽发现异常资金流动(读者和主角开始怀疑)
第5章:幽灵暗示"你触碰了不该碰的东西"(读者猜测是贪污,主角还没意识到)
第7章:林羽看到财务报表,恍然大悟(主角追上读者认知)
第9章:幽灵身份揭晓(新信息,读者和主角同步震惊)
```

#### 2.6.3 场景长短搭配

**原则**:

- 关键情节场景 = 1500-3000字(细描)
- 过渡场景 = 300-800字(速写)
- 高潮场景 = 3000-5000字(慢镜头)

**反面示例**(所有场景等长,无重点):

```
第一场:会议室对质 (1200字)
第二场:停车场逃跑 (1200字)
第三场:黑客基地见面 (1200字)
第四场:调查线索 (1200字)
```

**正面示例**(长短搭配,有主次):

```
第一场:会议室对质 (2500字,重点细描,情感爆发)
第二场:停车场逃跑 (600字,快节奏过渡)
第三场:黑客基地见面 (1800字,新角色引入+信息点)
第四场:调查线索 (400字,压缩琐碎)
第五场:真相对决 (4000字,高潮慢镜头)
```

### 2.7 开篇技巧

#### 2.7.1 钩子设计(第一句话抓住读者)

**原则**: 开篇100-300字内必须有**冲突/悬念/反常**

**反面示例**(平淡无奇):

```
林羽是一名程序员,在一家互联网公司工作。他今年28岁,单身,性格内向。今天是周一,他像往常一样来到公司……
```

**正面示例**(立即制造悬念):

```
【类型1:冲突开篇】
林羽被两个保安架着走出会议室时,脑子里只有一个念头:我被设计了。

【类型2:反常开篇】
林羽盯着电脑屏幕上那行代码,手开始颤抖——这段代码,他从未写过,但它确实出现在他的提交记录里,时间戳显示:凌晨3点17分。而那时,他正在家里睡觉。

【类型3:悬念开篇】
如果那天早上,林羽没有打开那封邮件,后面的一切都不会发生。可惜,世界上没有如果。
```

**钩子类型**:

- 危机开篇(正在发生的冲突)
- 秘密暗示(隐藏的真相)
- 倒计时(24小时内必须完成)
- 反常细节(不对劲的地方)

#### 2.7.2 世界观建立(不要信息轰炸)

**原则**: **通过角色行动展示世界规则,而不是大段说明**

**反面示例**(枯燥讲解):

```
2086年,赛博朋克时代,人类大部分肢体都可以替换为义体。大公司控制了政府,贫富差距极大,上层住在空中城市,下层住在地表贫民窟。规则怪谈开始出现,如果违反某些神秘规则,就会死亡……
```

**正面示例**(通过情节展示):

```
林羽擦了擦眼角——这是他身上唯一还是"真货"的部位。左臂的义体发出低沉的嗡鸣,提示电量只剩15%。

他抬头看了眼天空,浮空岛在云层之上闪烁着金色的光芒,那是上层阶级的乐园。而他脚下的地表贫民窟,永远只有昏暗的霓虹和永不停歇的酸雨。

墙上贴着红色警告:"晚上23点后禁止照镜子,违者后果自负。"旁边是一张失踪者海报,照片上的人半边脸扭曲变形,像融化的蜡像。

这就是2086,欢迎来到地狱。
```

### 2.8 结局技巧

#### 2.8.1 情感释放(完成情感承诺)

**原则**: 结局必须给读者期待的**情感答案**(爽/虐/升华)

**↗甜爽向结局**:

- 主角复仇成功,坏人得到报应
- 恋人终成眷属,HE结局
- 主角实力被认可,打脸成功

**↘虐心向结局**:

- 主角付出惨痛代价(爱人死亡/自己牺牲)
- 真相揭露但为时已晚
- 悲剧循环,无法打破

**✷烧脑向结局**:

- 多重反转,最后一个反转颠覆所有
- 开放式结局,留白让读者思考
- 真相即疯狂,主角/读者认知崩塌

#### 2.8.2 回扣前文(首尾呼应)

**技巧**: 结局中出现开篇的**关键意象/物件/台词**,制造圆满感

**示例**:

```
【开篇】林羽记得父亲临终前说的最后一句话:"记住,真相比你想象的更残酷。"那时他还是个孩子,不明白这句话的含义。

【结局】林羽站在真相面前,终于理解了父亲那句话。他闭上眼,泪水滑落——父亲,你是对的,真相比我想象的,残酷太多太多……

但至少,我知道了。
```

**回扣元素**:

- 开篇物件(手表/照片/信件)
- 开篇场景(同一地点再次出现)
- 开篇台词(重复或改写)

#### 2.8.3 余韵设计(最后一句话的力量)

**原则**: 最后100字应该有**情感冲击力**或**哲思留白**

**反面示例**(草草结束):

```
林羽终于复仇成功了。他回到家,洗了个澡,然后睡觉。第二天,他开始了新的生活。

——全文完——
```

**正面示例**(余韵悠长):

```
【类型1:物件回扣】
林羽把那张照片放回抽屉深处,锁上。

照片上,年轻的自己和父亲笑得很灿烂。那时他们还不知道,命运已经写好了剧本,而他们,不过是棋子。

但至少,这一局,他赢了。

【类型2:哲思留白】
很多年后,林羽偶尔会想起那个雨夜。

他不知道如果当时做了另一个选择,会不会有不同的结局。

但他知道,有些路,一旦走上,就再也回不了头。

【类型3:开放式】
林羽转身离开,消失在人群中。

没有人知道他去了哪里。

也许是新的城市,也许是……另一个故事的开始。
```

### 2.9 人物刻画技巧

#### 2.9.1 人物矛盾性(立体而非脸谱化)

**原则**: 人物必须有**性格矛盾点**,才不会像纸片人

**脸谱化角色**(扁平):

```
林羽:聪明、冷静、技术天才,没有缺点,永远正确。(完美人设=无聊)
```

**立体角色**(矛盾):

```
林羽:
- 表面:冷静理智,从不冲动
- 内核:内心孤独,渴望被理解
- 矛盾点:极度理性的人,一旦情感防线被突破,会比任何人都脆弱

【具体化】平时面对危险能冷静分析对策,但一旦有人提到他的父亲,他会瞬间失控。
```

**矛盾性类型**:

- 外在vs内在(表面强势/内心自卑)
- 过去vs现在(曾经善良/现在冷酷)
- 理智vs情感(大脑说该恨/心却放不下)

#### 2.9.2 人物成长弧(变化轨迹)

**原则**: 主角的**价值观/性格**必须在故事中发生改变

**无成长弧**(主角从头到尾一样):

```
第1章:林羽是个聪明冷静的程序员
第10章:林羽还是个聪明冷静的程序员,只是打败了敌人
```

**有成长弧**(主角蜕变):

```
第1章:林羽相信规则,相信公司,相信系统(天真)
```

# 第一章 凉粉

---

午后的鹅城，闷热得像一锅煮沸的汤。

六子蹲在饭馆门口的石阶上，看着街对面的黄四郎府邸发呆。府邸门口停着两顶轿子，进进出出的人都是穿着长衫的有钱老爷。六子低头看了看自己身上打满补丁的灰布褂子，唾了一口唾沫在地上。

“发什么呆呢？还不进去干活！”

饭馆老板老刘的嗓门从店里传来，六子赶紧站起来，拍了拍裤子上的灰，钻进店里。

这家店叫“鹅城饭馆”，是鹅城里最热闹的地方。说是“最热闹”，其实也就是些拉车的、挑担的、进城卖菜的穷苦人在这儿吃碗阳春面。六子是店里的跑堂，也就是小二兼杂役，一个月管三顿饭，工钱是二十个大子儿。

六子干活勤快，从早忙到晚，从不偷懒。但他有一样不好——嘴笨。

别人问他什么，他总是“嗯”、“啊”、“是”，多说一个字都费劲。店里人都说他是个闷葫芦，六子也不反驳，只是笑笑。他的笑很怪，像是硬挤出来的，眼睛里透着一股子狠劲。

这股狠劲是从哪儿来的，六子自己也不太说得清。

他是从外地逃荒来的。三年前，鹅城闹饥荒，六子跟着爹娘从山东一路要饭过来。走到半道，娘死了，爹把他托付给一个同乡，自己去县城找活路，结果一去不回。六子记得爹临走前说的最后一句话：“娃儿，活下去。”

他就，活下来了。

在鹅城，六子认识了一个奇怪的人。

那个人是半年前来的，带着几个兄弟，租了城西一间破院子住着。那人看起来三十多岁，高个子，沉默寡言，但一出手就震住了六子——他用一把奇怪的手枪，一枪打落了十步外树枝上的乌鸦。

六子看得出神，那人却只是淡淡地看了他一眼，收起枪，走了。

后来六子才知道，那人叫“张麻子”，是这一带有名的麻匪。但六子觉得奇怪，张麻子虽然叫“麻子”，脸上却一颗麻子都没有。而且他说话的口音也怪怪的，像是外国人学中国话，舌头总是捋不直。

不过六子没想那么多。张麻子对他不错，教他用枪，还送了他一把左轮手枪。六子把这把枪藏在怀里，从来不舍得用。

今天，六子照常在店里忙活着。

中午的太阳最毒，街上没什么人，店里也冷清。老刘坐在柜台后面打着瞌睡，六子擦着桌子，心里盘算着晚上吃什么。

“砰！”

门帘被掀开，一个戴着礼帽、穿着绸缎马褂的中年人，后面跟着两个横眉竖眼的保镖。

六子一看，心里咯噔一下。

为首的那个，是黄四郎的人。

黄四郎是鹅城的“天”——鹅城人都这么说。民国了，皇帝没了，但鹅城的天还是黄四郎的。他开矿、办厂、印钞票，还养着一帮打手。鹅城百姓见了他，都得绕着走。

为首的那个人，六子认得，叫胡万，是黄四郎的心腹。

胡万大摇大摆地走进来，一屁股坐在一张桌子前，翘起二郎腿。

“小二，倒茶！”

六子赶紧上前，赔着笑脸：“爷，您来了？喝茶、喝茶。”

胡万斜着眼看了六子一眼：“你是新来的？面孔生得很哪。”

“回爷的话，小的叫六子，在这儿干了半年了。”

“半年？”胡万冷笑一声，“我怎么没见过你？哦，大概是有些人天生就不长眼，专挑没人的时候出来晃悠。”

六子陪着笑，不敢接话。

胡万点了一桌子菜，又要了一壶酒，自顾自地吃了起来。六子站在一边，随时准备侍候。

吃到一半，胡万突然把筷子一摔。

“这面条，怎么是凉的？”

六子一看，确实是凉了。刚才后厨煮的时候火候过了，捞出来放了一会儿，现在已经坨了。

“爷，真对不住，我这就给您换一碗新的。”六子赶紧说。

“换？”胡万把眼一瞪，“换就能了事了？你知道我这时间多金贵吗？耽误了我的事，你担当得起吗？”

六子连连道歉：“爷对不起、对不起，这顿算小的请您了。”

“请你？”胡万冷笑，“你算什么东西？也配请我？”

六子低着头，不再说话。这种时候，解释就是找死。

胡万身后的一个保镖走上前，一脚踢在六子肚子上。六子猝不及防，摔了个跟头。

“起来！给我跪好！”

六子咬着牙站起来，膝盖还没弯下去，胡万又开口了。

“慢着。”胡万慢悠悠地从怀里掏出一个铜板，扔在六子脚边，“我问你，你刚才给我端上来的那碗凉粉，是多少钱？”

六子愣了一下：“回爷的话，一碗凉粉，五个大子儿。”

“五个大子儿？”胡万笑了，“我给你十个大子儿，你去给我端十碗凉粉来。”

六子又愣了一下：“爷，十碗？您吃得了吗？”

“怎么着？”胡万把脸一沉，“我花钱买凉粉，你管我吃得了吃不了？还不快去！”

六子不敢再问，赶紧跑到后厨，端了十碗凉粉出来，一碗一碗地摆在胡万面前。

胡万看着这十碗凉粉，脸上的笑越来越阴。

“吃。”

六子没听清：“爷，您说什么？”

“我说，吃。”胡万指着那十碗凉粉，“给我吃完它们。”

六子看着这十碗凉粉，胃里已经开始翻腾了。一碗凉粉是五个大子儿，十碗就是五十个。五十个大子儿，够普通人家过半个月了。

“爷，小的一天没吃饭了，这十碗……”六子赔着笑，“能不能少点？”

“少点？”胡万把桌子一拍，“你他妈的打发叫花子呢？十碗凉粉都给我吃干净，少一碗我把你的肠子掏出来数数！”

旁边的食客们看着这一幕，有的低头吃面，有的偷偷溜走了。老刘躲在柜台后面，大气不敢出一声。

六子看着那十碗凉粉，胃里的酸水已经开始往上涌。

“爷，小的真吃不下……”

“吃不下？”胡万站起身，走到六子面前，居高临下地看着他，“吃不下就剖腹给我看看！你到底吃了几碗，你自己心里清楚！”

剖腹。

这两个字像一把刀，插进六子的心里。

他想起了三年前的那个晚上。娘饿死之后，爹带着他逃荒。路上，爹把他托付给同乡，自己走了。临走前，爹说：“娃儿，活下去。”

活下去。

可是怎么活？

六子看着那十碗凉粉，耳边突然响起一个声音——那是他老婆的声音。

那是在老家的时候，六子娶过一个女人。女人是隔壁村的，长得不好看，但勤快，对六子好。六子那年才十八，刚成亲，觉得日子有了盼头。

后来，饥荒来了。

女人饿得不行，去地主家借粮，被地主家的狗腿子活活打死。六子赶过去的时候，只看到女人躺在地上，肚子被人剖开了，里面塞满了草根。

六子疯了。

他拿着刀冲向地主家，要给女人报仇。但他人单势孤，被打了一顿，扔在大街上。要不是老刘刚好路过把他捡回去，他已经死了。

从那以后，六子就变了。

他的世界里只剩下两个字：活着。

他不再反抗，不再挣扎，让干什么就干什么。别人欺负他，他忍；别人骂他，他笑。他像一条狗一样，在鹅城活着。

可是今天，他不想忍了。

胡万还在那里骂骂咧咧的，保镖在旁边笑着看热闹。老刘缩在柜台后面，像一只受惊的鹌鹑。

六子慢慢站起来。

“怎么着？想通了？”胡万冷笑，“想通了就给我吃！吃完十碗凉粉，我赏你十个铜板！”

六子看着胡万，嘴唇动了动。

“爷。”

“嗯？”

“小的想问您一个问题。”

胡万愣了一下：“什么问题？”

“您说，这人活在这世上，是为了什么？”

胡万没想到六子会问这个，嗤笑一声：“为了什么？为了享福！为了有钱！为了不被别人踩在脚下！这个，你满意了吗答案？”

六子点点头：“满意。”

然后，他慢慢从怀里掏出一把左轮手枪。

枪是黑色的，枪柄上刻着奇怪的花纹。六子把它握在手里，感受着枪柄的冰凉。

“你……”胡万脸色大变，“你想干什么？”

“爷，小的想请您吃子弹。”

“砰！”

枪声响起。

胡万的头颅像熟透的西瓜一样爆开，血溅了六子一脸。保镖还没反应过来，六子又是两枪，两个保镖相继倒地。

枪声在饭馆里回荡，硝烟味和血腥味混在一起。

六子站在血泊中，手里举着枪，大口大口地喘着气。

门外传来尖叫声和脚步声。六子不管这些，他冲出饭馆，朝街对面跑去。

街对面，黄四郎府邸门口，停着一顶轿子。轿子旁边站着一个穿着讲究的中年人，正冷冷地看着六子。

六子冲过去，把枪顶在那人的脑袋上。

“黄四郎呢？”六子喘着粗气，“说！黄四郎在哪儿？”

那个人看着六子，脸上的表情很奇怪——不是害怕，而是一种……好奇？

“他不在。”那个人说，“你找我有什么事？”

六子愣了一下：“你是什么人？”

“我？”那个人笑了，“你猜？”

六子看着这个人，突然意识到一个问题。

这个人，不是黄四郎。

黄四郎他见过，是个矮胖子，留着八字胡，笑起来像一只狐狸。但眼前这个人，高个子，长脸，眼神犀利，跟黄四郎完全不一样。

“你是……替身？”六子突然想起来，鹅城人都说黄四郎有个替身帮他做事。

“聪明。”替身笑了，“不过，聪明往往死得快。”

六子把枪顶得更紧了：“我管你是谁！今天我就是要让黄四郎出来！他不出来，我就杀光他府上的人！”

替身看着六子，脸上的笑渐渐消失了。

“小子，你很勇。”他说，“但你知不知道，你刚才杀的那个人，是黄四郎最信任的手下？你知不知道，黄四郎一句话，就能让鹅城没有你的容身之处？”

六子当然知道。

但他已经不在乎了。

“那就让他来好了。”六子说，“我六子贱命一条，死不足惜。能拉几个垫背的，值了。”

替身看着六子，眼神里突然闪过一丝赞许。

“不错，有种。”他说，“但你杀错人了。”

“什么？”

“黄四郎不在府里。”替身说，“他出去了，要晚上才回来。你现在就算把这里拆了，也见不到他。”

六子咬紧牙关：“那我就等他回来！”

“等他回来？”替身笑了，“你等得起吗？黄四郎的人已经在路上了。最多一炷香时间，他们就会把这里围得水泄不通。到时候，你就算插翅也难飞。”

六子当然知道。

但他已经没有退路了。

杀一个也是杀，杀两个也是杀。既然已经动手了，那就杀个痛快！

“那就让他们来！”六子吼了一声，“我六子今天就让他们看看，穷人也是有脾气的！”

“好！说得好！”

一个声音从巷口传来。

六子转头去看，只见一个人从暗处走出来。那人穿着灰布长衫，戴着毡帽，走路几乎没有声音。

六子认出了他。

“张……张麻子？”

来的人正是约翰·马斯顿，或者说，张麻子。

张麻子走到六子面前，看了他一眼，又看了看他身后的饭馆。

“不错。”他说，“有进步。”

六子还没明白这句话什么意思，就看到张麻子突然抬手，一枪打向饭馆门口。

“砰！”

一个刚从饭馆里冲出来的保镖应声倒地。

“出来吧。”张麻子淡淡地说，“戏看够了，该干活了。”

话音刚落，又有几个保镖从饭馆里冲出来，手里挥舞着棍棒和匕首。

张麻子不慌不忙，从怀里掏出另一把左轮手枪。

“砰！砰！砰！”

三声枪响，三个保镖倒地。

剩下的保镖们愣住了。他们没想到，这个看起来普普通通的中年人，枪法居然这么准。

“上！一起上！”一个保镖头目大喊。

保镖们一拥而上。

张麻子站在原地，冷静地瞄准、射击。每一声枪响，都有一个保镖倒下。

六子看呆了。

他见过张麻子用枪，但从来没有见过他同时用两把枪。左右开弓，每一枪都不落空。子弹像是长了眼睛，专门往人身上最致命的地方钻。

不到一分钟，十几个保镖全部倒地。

街道上血流成河。

张麻子收起枪，转头看向六子。

“怕了？”

六子摇头：“不怕。”

“恨我？”

六子还是摇头：“不恨。”

张麻子点点头：“那就好。记住今天的感觉。有一天，你会用到的。”

六子没听懂这句话的意思，但他没有问。

远处传来轿子的声音。

六子转头去看，只见一顶绿呢大轿正向这边走来，轿子前后跟着十几个保镖。

轿子在府邸门口停下。

轿帘掀开，一个矮胖子从里面钻出来。

黄四郎。

黄四郎穿着长衫马褂，留着八字胡，笑起来像一只狐狸。他下了轿子，看了一眼地上的尸体，脸上的笑更浓了。

“精彩，精彩。”黄四郎鼓掌，“张麻子，久仰大名，今日一见，果然名不虚传。”

张麻子淡淡地说：“黄四郎。”

“正是鄙人。”黄四郎笑着走到张麻子面前，“张兄弟，今天这事儿，你看怎么解决？”

“简单。”张麻子说，“你的人杀了我的兄弟，我来讨个公道。”

“讨公道？”黄四郎笑了，“张兄弟，这话说的。你的人杀了我的人，我还找你讨公道呢。”

张麻子看着黄四郎：“你想怎么样？”

黄四郎看了六子一眼：“这个小兄弟，是你的人？”

“是。”

“枪法不错。”黄四郎说，“我黄四郎最喜欢结交英雄好汉。这样吧，今天的事儿既往不咎，我请张兄弟喝杯酒，就当交个朋友，如何？”

张麻子沉默了一会儿。

“好。”

黄四郎大笑：“痛快！请！”

张麻子拉着六子，跟着黄四郎进了府邸。

临进门的时候，六子回头看了一眼。

街道上，保镖们的尸体横七竖八地躺着。饭馆门口，老刘缩在柜台后面，脸色煞白。

六子突然觉得，这些日子压在胸口的那口气，终于出来了。

黄四郎的书房很大，红木家具，字画古玩，摆设得十分讲究。

黄四郎坐在主位，张麻子和六子坐在客位。下人端上茶来，房间里弥漫着淡淡的茶香。

“张兄弟是从外地来的？”黄四郎笑着问。

“是。”

“来鹅城多久了？”

“半年。”

“半年？”黄四郎点点头，“半年时间，张兄弟已经闯出这么大的名头，真是后生可畏啊。”

张麻子不接话，只是端着茶杯，慢慢地喝着。

黄四郎也不在意，继续说：“我黄四郎这个人，最好交朋友。只要你情我愿，咱们什么事都能商量。”

张麻子放下茶杯：“说正事吧。”

黄四郎笑了：“好，爽快。张兄弟，明人不说暗话，我想请你帮我做事。”

“做什么？”

“很简单。”黄四郎说，“我有一批货，要从鹅城运到外地。路上不太平，需要像张兄弟这样的高手押运。报酬方面，绝对让你满意。”

张麻子看着黄四郎：“什么货？”

黄四郎的笑容僵了一下。

“这个……就不方便说了。”

张麻子站起身：“既然如此，告辞。”

“慢着！”黄四郎也站起来，“张兄弟，别急着走啊。咱们有话好好说。”

张麻子看着黄四郎：“我兄弟杀了你的人，你想怎么办？”

黄四郎笑了：“杀了就杀了呗。不过几条狗而已，我黄四郎还不放在眼里。”

张麻子没想到黄四郎会这么说，愣了一下。

“但我有一个条件。”黄四郎说。

“什么条件？”

黄四郎看了六子一眼：“这个小兄弟，我要了。”

张麻子脸色一沉：“不可能。”

“别急着拒绝。”黄四郎说，“我黄四郎看人很准。这个小兄弟，有股子狠劲，是个当杀手的好材料。我调教调教他，将来必定成器。”

张麻子拉着六子往外走。

“张麻子！”黄四郎在身后说，“你确定要和我作对？”

张麻子停下脚步，但没有回头。

“黄四郎，我张麻子不是吓大的。你要是想玩，我奉陪到底。”

说完，他拉着六子，头也不回地走了。

黄四郎站在书房里，看着他们的背影，脸上的笑渐渐消失了。

“有点意思。”他轻声说。

---

从黄四郎府邸出来，六子跟在张麻子身后，走了很远。

“六子。”张麻子突然开口。

“嗯？”

“你怕死吗？”

六子想了想：“怕。”

“那你刚才为什么杀人？”

六子又不说话了。

张麻子也没有追问，只是继续往前走。

走了一段路，张麻子突然停下来，转过身。

“六子，今天的事，没完。”

六子看着张麻子：“张哥，你想怎么做？”

张麻子的眼神变得很冷。

“黄四郎以为他很聪明。但有些事，他不该碰。”

六子没听懂这句话的意思，但他没有问。

他只知道一件事——

从今天起，他的命，已经和张麻子绑在一起了。

---

夜晚，鹅城陷入一片寂静。

但在黄四郎的书房里，烛火还亮着。

黄四郎坐在书桌前，手里把玩着一枚铜钱。

“管家。”他叫了一声。

“在。”

“去查查这个张麻子的来历。我不信他只是一个普通的麻匪。”

“是。”

“还有，”黄四郎把铜钱放在桌上，“那个六子，我要活的。”

管家退下了。

黄四郎看着窗外的夜色，脸上的笑意味深长。

“有趣。”他轻声说，“真是太有趣了。”

### chapters/chapter_02.md

# 第二章 牛仔

---

夜晚的鹅城，没有电灯，只有稀疏的几盏油灯在风中摇曳。

城西的破院子里，约翰·马斯顿站在月光下，手里擦着一把左轮手枪。

枪是美国产的柯尔特，枪身已经被岁月磨得有些斑驳，但枪管仍然锃亮。枪柄上刻着几个英文字母：**J. Marston, West Elizabeth, USA**。

约翰看着这些字母，出了神。

“张哥。”

身后传来六子的声音。约翰收起枪，转过身。

六子站在门口，脸上还有干掉的血迹。刚才杀人时的狠劲已经消失了，现在的他看起来有些局促不安。

“张哥，今天……今天给你添麻烦了。”

约翰淡淡地说：“不麻烦。”

六子走进院子，在约翰面前坐下。月光照在他的脸上，映出一片青白。

“张哥，我有话想问你。”

约翰看了他一眼：“说。”

“你……你到底是什么人？”

约翰没有立即回答。他重新拿起枪，慢慢地擦着。

“你觉得呢？”

六子想了想：“我爹说过，会用那种枪的人，不是土匪，就是杀手。”

“我不是土匪。”约翰说，“也不是杀手。”

“那你是什么？”

约翰停下手中的动作，抬头看着天空。天空很黑，没有星星，只有月亮孤零零地挂着。

“我是一个牛仔。”他说。

“牛仔？”六子愣住了，“什么是牛仔？”

约翰沉默了一会儿。

“在我的家乡，”他慢慢地说，“有一片很大的草原。那里有很多马，很多人骑着马放牧、赶牛。那些人叫——cowboy。”

六子听得一头雾水。

“那是……很远的地方？”

“很远。”约翰说，“远到你无法想象。”

六子看着约翰手中的枪：“那这把枪……”

“这把枪叫左轮。”约翰说，“六发子弹，弹仓转动来装填。在我的家乡，每个牛仔都有一把这样的枪。”

六子突然想起来一件事。

“张哥，你……你不是在中原出生的？”

约翰摇头。

“那你……你是从哪儿来的？”

约翰看着六子，眼神变得很复杂。

“我从一个很远的地方来。”他说，“那个地方，有很多人骑着马，有很多牛，有很多……很长很长的故事。”

六子听不懂，但他没有追问。

他知道，张麻子不想说的事，问了也白问。

“张哥，”六子换了个话题，“今天那个黄四郎，他不会善罢甘休的。”

“我知道。”

“咱们……咱们该怎么办？”

约翰收起枪，站起身。

“兵来将挡，水来土掩。”

六子也站起来：“张哥，我不怕死。今天杀的那些人，都是黄四郎的走狗，死了活该。”

约翰看着六子：“你怕不怕死？”

六子愣了一下，然后摇头。

“不怕。”

“为什么？”

六子又不说话了。

为什么？

因为他已经死过很多次了。

在老家的时候，他死过一次——被地主家的狗腿子活活打死。要不是命大，他早就死了。

在逃荒的路上，他死过一次——娘饿死的时候，他觉得自己的心也死了。

在鹅城的时候，他又死过一次——老刘把他从大街上捡回来的时候，他觉得自己已经是个死人了。

所以，他不怕死。

死了，反而是一种解脱。

约翰看着六子，突然开口。

“六子，我给你讲个故事。”

六子点头：“嗯。”

“在我的家乡，”约翰慢慢地说，“有一个很有名的帮派。帮派里有很多人，都是像我这样的亡命之徒。我们骑着马，在草原上闯荡，抢富人的钱，分给穷人。”

六子眼睛亮了：“然后呢？”

“然后，”约翰的声音变得低沉，“帮派散了。有些人死了，有些人被抓了，有些人……消失了。”

“消失了？”

约翰点头：“就像人间蒸发一样，谁也不知道他们去了哪儿。”

六子没听懂，但他感觉的出来，这是一个很悲伤的故事。

“张哥，”他问，“那个帮派，叫什么名字？”

约翰看着月亮，半天才吐出两个字。

“摩根。”

---

同一时刻，鹅城另一边，黄四郎的府邸。

黄四郎坐在书房里，手里端着一杯茶。茶是上等的龙井，散发着淡淡的清香。

但黄四郎没有心思品茶。

他在等人。

“老爷。”管家从外面走进来，手里拿着一封信，“有消息了。”

黄四郎接过信，拆开来看。

信是从上海发来的，上面是密密麻麻的英文。黄四郎虽然读过几年书，但英文不在行，只能看个大概。

不过，只是“大概”，就已经足够了。

“有趣。”黄四郎放下信，脸上露出意味深长的笑。

管家问：“老爷，这信上说什么？”

“你自己看。”

管家接过信，看了一会儿，脸色大变。

“这……这人是……”

“美国通缉犯。”黄四郎笑着说，“约翰·马斯顿，涉嫌谋杀、抢劫、绑架，数罪并罚，悬赏五千美元。”

“五千美元！”管家惊呼，“那可是好大一笔钱啊！”

黄四郎点头：“确实不少。”

“老爷，那咱们……”

“先不急。”黄四郎站起身，走到窗边，“这个人，不简单。”

“怎么不简单？”

“五千美元的通缉犯，跑到鹅城来当麻匪？”黄四郎冷笑，“你信吗？”

管家摇头：“不信。”

“所以嘛，”黄四郎说，“这个人一定有别的目的。”

“什么目的？”

黄四郎没有立即回答。他在房间里踱着步，手指轻轻敲着桌面。

“最近，鹅城附近出了几档子怪事。”他慢慢地说，“有一伙人，专门绑架西洋人。绑了之后，也不管赎金，直接卖到外地去。”

管家点头：“这事我也听说了。”

“你知道买家是谁吗？”

管家摇头。

“日本人。”黄四郎说，“俄国人也掺了一脚。他们要这些人干什么，不用我说你也知道。”

管家打了个寒战。

“老爷，您是说……张麻子也在做这种生意？”

黄四郎笑了：“不知道。但可以肯定的是，这个约翰·马斯顿，绝对不是表面上看起来那么简单。”

“那咱们怎么办？”

黄四郎停下脚步，转过身。

“这样的人才，杀了太可惜。”他说，“我要他为我所用。”

管家愣了一下：“为他所用？老爷，他可是通缉犯……”

“通缉犯怎么了？”黄四郎冷笑，“我黄四郎什么人没见过？通缉犯、土匪、杀手、军阀……只要有用，我都敢用。”

“那要是他不肯呢？”

“不肯？”黄四郎的笑容变得阴冷，“那就让他知道知道，在鹅城，跟我黄四郎作对，是什么下场。”

---

第二天，清晨。

鹅城的清晨很安静，只有偶尔的几声鸡鸣。

约翰很早就醒了。在他的家乡，牛仔们都起得很早，因为要在太阳升起之前放牧。

但这里不是他的家乡。

这里没有草原，没有马群，没有那些一起闯荡的兄弟。

约翰站在院子里，看着天边渐渐泛起的鱼肚白，心里突然涌起一股说不出的孤独。

“张哥。”

身后传来六子的声音。约翰转过身。

六子从屋里走出来，脸上还有倦容。显然，昨晚他没有睡好。

“起来了？”约翰问。

“嗯。”六子点点头，走到约翰身边，“张哥，昨晚我想了一夜。”

“想什么？”

“下一步，咱们该怎么办。”

约翰看着天边，没有说话。

“张哥，”六子继续说，“我有个想法。”

“什么想法？”

六子咬了咬牙：“我想，找人帮忙。”

“找谁？”

“武举人。”

约翰转过头，看着六子。

“武举人？”

六子点头：“武举人是鹅城的乡绅，手里有点势力。他和黄四郎有仇，一直想找机会扳倒黄四郎。”

“你怎么知道的？”

“街上都在传。”六子说，“武举人的儿子，被黄四郎的人打死了。他一直咽不下这口气。”

约翰沉默了一会儿。

“他在哪儿？”

“城东，武家大院。”

约翰想了想：“带我去。”

---

武家大院在城东，是鹅城除了黄四郎府邸之外最大的宅子。

朱红的大门，灰色的院墙，门口站着两个家丁。家丁手里拿着棍子，横眉竖眼地看着来来回回的人。

六子走上前，赔着笑脸：“两位大哥，麻烦通报一声，我们要见武举人。”

家丁上下打量了六子一眼：“你是什么人？武举人是你想见就能见的？”

六子还想说什么，约翰已经走上前。

“告诉武举人，”约翰淡淡地说，“张麻子求见。”

家丁愣了一下，然后其中一个转身跑进去了。

不一会儿，家丁跑出来，躬身说：“请跟我来。”

约翰和六子跟着家丁，进了武家大院。

大院很大，假山流水，亭台楼阁，看得出武举人是个讲究的人。穿过几道走廊，两人被带到一个书房里。

书房里，一个中年男人坐在太师椅上，手里把玩着两个核桃。

男人看起来五十多岁，穿着锦缎长衫，留着山羊胡，眼睛眯成一条缝，看起来很精明。

六子知道，这就是武举人。

“武爷，”六子躬身行礼，“小的带朋友来拜访您。”

武举人睁开眼睛，打量了约翰一眼。

“这位是……”

“在下张麻子。”约翰说。

武举人愣了一下，然后大笑起来。

“原来是张麻子！久仰久仰！”武举人站起身，走上前，上下打量着约翰，“果然是一表人才！”

约翰淡淡地说：“过奖。”

武举人请约翰坐下，又让人上了茶。

“张兄弟，今天来找我，有什么事？”

约翰开门见山：“我想和你合作。”

“合作？”武举人眯起眼睛，“什么合作？”

“对付黄四郎。”

武举人的笑容僵了一下，但很快就恢复自然。

“张兄弟，你这是什么意思？我和黄四郎无冤无仇，为什么要对负他？”

约翰看着武举人：“真的吗？”

武举人打了个哈哈：“当然是真的。张兄弟，你可能误会了什么……”

“武举人，”约翰打断他，“我既然来了，就说明我已经知道了一些事。你儿子的死，和黄四郎有关，对吧？”

武举人的脸色变了。

“张兄弟，这种话可不能乱说……”

“我没有乱说。”约翰说，“我只是在说一个事实。”

书房里安静下来。

武举人看着约翰，眼神变得很复杂。

“张兄弟，”他慢慢地说，“你到底是什么人？”

约翰端起茶杯，慢慢地喝着。

“一个想扳倒黄四郎的人。”

武举人沉默了很久。

“好。”他终于开口，“我跟你合作。”

约翰放下茶杯：“你想怎么做？”

武举人站起身，走到窗边，看着外面的院子。

“黄四郎在鹅城经营了这么多年，势力很大。硬碰硬，我们讨不了好。”

“所以？”

“所以，要智取。”武举人转过声，看着约翰，“我需要时间，调动人手，收集证据。只要找到他的把柄，就能让他万劫不复。”

约翰点头：“需要多久？”

“少则三个月，多则一年。”

约翰摇头：“太长。”

武举人皱眉：“张兄弟，黄四郎不是普通的对手，急不得……”

“等不了。”约翰说，“我还有很重要的事要办，没时间在这里耗着。”

武举人看着约翰：“很重要的事？什么事？”

约翰没有回答。

他想起了那些失踪的兄弟。

亚瑟·摩根，查尔斯·史密斯，何西亚·马修斯……那些一起闯荡过的兄弟，现在不知道在什么地方受苦。

他一定要找到他们。

“武举人，”约翰说，“我帮你扳倒黄四郎。你帮我做一件事。”

“什么事？”

“帮我找一些人。”

“什么人？”

约翰看着武举人，慢慢地吐出两个字。

“牛仔。”

---

从武家大院出来，六子跟在约翰身后，一直没说话。

走了一段路，六子突然开口。

“张哥，那个武举人，能信吗？”

约翰头也不回：“不能。”

六子愣了一下：“那你还和他合作？”

“因为没有别的选择。”约翰说，“在鹅城，想扳倒黄四郎的人不多。武举人是其中一个。”

六子似懂非懂地点头。

“张哥，那咱们现在去哪儿？”

约翰停下脚步，转过身。

“先回去。”他说，“黄四郎不会善罢甘休的。我们要有准备。”

六子点头：“嗯。”

两人回到住处，发现院子里多了一个人。

那个人坐在石凳上，穿着一件洗得发白的蓝布长衫，手里摇着一把折扇。

看到约翰，那人站起来，笑着拱了拱手。

“张爷回来了？我等您半天了。”

约翰看着这个人，眉头微微皱起。

“你是谁？”

“在下汤师爷。”那人说，“是个落魄的读书人。听说张爷您本事大，特来投奔。”

约翰看着汤师爷，没有说话。

汤师爷也不在意，继续笑着说：“张爷，您别误会。我不是黄四郎的人。”

“你怎么证明？”

汤师爷笑了：“证明？张爷，这种事没法证明。时间久了，您自然就知道了。”

约翰盯着汤师爷看了半天，终于开口。

“留下来可以。但要是让我发现你有什么不轨的举动……”

“张爷放心！”汤师爷打断他，“我汤师爷别的没有，就是懂得一个'忠'字。既然投奔了您，就绝不会背叛您。”

约翰没有再说什么，转身进了屋。

六子看着汤师爷，有些担心。

“张哥，这个人，看起来不简单。”

约翰的声音从屋里传来。

“知道不简单，就盯着点。”

六子点头：“嗯。”

# 第三章 入局

---

三天后。

一封请帖送到了约翰的住处。

请帖是烫金的封面，展开来，上面写着几行毛笔字：

```
张麻子台启：
    素闻张兄弟英雄了得，鄙人仰慕已久。
    谨定于本月十五日晚，于寒舍设宴，
    届时务必赏光。
                黄四郎敬邀
```

约翰看着请帖，眉头微微皱起。

“张哥，”六子从旁边探过头来看，“这黄四郎，又在搞什么鬼？”

“鸿门宴。”约翰说。

“鸿门宴？”六子没听懂，“什么是鸿门宴？”

“就是明知是陷阱，还要往里跳的宴会。”

六子脸色变了：“那张哥，你还去？”

“去。”约翰说，“为什么不去？”

六子着急了：“可是……这明显是个陷阱啊！黄四郎肯定没安好心！”

约翰放下请帖，看着六子。

“如果不去，六子会怎么样？”

六子愣了一下：“什么？”

“黄四郎三天前说的话，你还记得吗？”

六子回想了一下，然后脸色煞白。

黄四郎说过——“这个小兄弟，我要了。”

“张哥……”六子的声音开始颤抖，“你……你是说……”

“黄四郎已经动手了。”约翰说，“就在昨天，你的行踪已经被他的人发现了。”

六子整个人都僵住了。

“张哥，那我该怎么办？”

“只有一个办法。”约翰说，“去参加宴会。”

六子咽了口唾沫：“去……去送死？”

“送死？”约翰冷笑一声，“没那么容易。”

他转身走进屋里，从床底下拖出一个箱子。箱子打开，里面是几把左轮手枪，还有几十发子弹。

“把这些带上。”

六子看着这些枪，有些犹豫。

“张哥，这是……”

“防身。”约翰说，“记住，不到万不得已，不要开枪。”

六子点头，把枪收好。

十五日，晚上。

鹅城的夜晚，没有月亮，只有星星点点的灯火。

黄四郎的府邸门口，挂满了大红灯笼，把整条街都照得亮堂堂的。门口停着十几顶轿子，来参加宴会的人络绎不绝。

约翰带着六子，走向府邸。

门口的保镖看到他们，刚想阻拦，管家从里面走出来。

“张爷来了？快请进、快请进！”

约翰跟着管家，进了府邸。

府邸很大，亭台楼阁，假山流水，看起来像是一个大花园。约翰一边走，一边观察着周围的地形。

“六子，记住路。”他在六子耳边轻声说。

六子点头。

两人被带到一个大厅里。

大厅里摆着几张桌子，上面摆满了山珍海味。黄四郎坐在主位上，旁边坐着几个穿着讲究的客人。

看到约翰，黄四郎大笑起来。

“张兄弟！来，坐坐坐！”

约翰走过去，在黄四郎对面坐下。六子站在他身后，手心里全是汗。

黄四郎挥了挥手，管家端着一个托盘走过来。托盘上放着一个酒杯，还有一个纸包。

“张兄弟，”黄四郎笑着说，“我敬你一杯。”

约翰端起酒杯，却没有喝。

“怎么？怕有毒？”黄四郎笑了。

“黄四郎，”约翰淡淡地说，“有话直说。”

黄四郎脸上的笑僵了一下，但很快就恢复自然。

“好，爽快！”他拍了拍手，“把东西拿上来。”

管家走过去，打开那个纸包。

纸包里，是几块大洋。

“这里是五百块大洋。”黄四郎说，“只要张兄弟愿意为我做事，这些就是定金。”

约翰看着那些大洋，没有说话。

“怎么？嫌少？”黄四郎笑着说，“可以谈嘛。只要张兄弟开口，多少钱我都给。”

“黄四郎，”约翰放下酒杯，“我说了，有话直说。”

黄四郎的笑容渐渐消失。

“好，那我就直说。”他看着约翰，“我想请张兄弟帮我押一趟货。”

“什么货？”

“西洋人。”

约翰的眼神变了。

“黄色的头发，蓝色的眼睛，白皮肤。”黄四郎笑着说，“那些洋人，可值钱了。”

约翰沉默了很久。

“你在做人口贩卖？”

“话别说得这么难听。”黄四郎笑着说，“这叫'贸易'。各取所需，互惠互利。”

约翰站起身：“对不起，这种生意，我不做。”

“等等！”黄四郎也站起来，“张兄弟，别急着拒绝嘛。我还没说完呢。”

约翰看着黄四郎：“还有什么？”

黄四郎拍了拍手。

管家带着几个保镖走上来，手里押着一个人。

那个人穿着粗布衣服，低着头，头发凌乱，看不清长相。

“张兄弟，”黄四郎笑着说，“这个人，你认识吗？”

约翰盯着那个人，没有说话。

“抬起头来。”黄四郎说。

那个人慢慢地抬起头。

看到那张脸的时候，约翰整个人都僵住了。

那张脸上，有一道从左眉延伸到左颊的刀疤。

“亚瑟。”约翰的声音很低，“亚瑟·摩根。”

亚瑟·摩根！

摩根帮的核心成员，约翰最亲密的兄弟之一！

亚瑟看着约翰，眼睛里先是震惊，然后是惊喜，最后是恐惧。

“John……”亚瑟的声音沙哑，“快走……这是陷阱……”

话还没说完，两个保镖上前，一个肘击打在亚瑟的肚子上。亚瑟弯下腰，痛苦的呻吟着。

“住手！”约翰吼了一声。

黄四郎挥了挥手，保镖们停下来。

“张兄弟，”黄四郎笑着说，“现在，我们可以好好谈一谈了吧？”

约翰看着黄四郎，眼神变得很冷。

“你想怎么样？”

“很简单。”黄四郎说，“帮我做事，我就放了他。”

约翰沉默了很久。

“如果我说不呢？”

黄四郎的笑容变得阴冷。

“那就别怪我不客气了。”

话音刚落，大厅四周突然冲出几十个保镖，手里拿着刀枪棍棒，把约翰和六子团团围住。

六子拔出枪，对准黄四郎。

“别动！”

黄四郎看着六子，笑了。

“小兄弟，枪不是这么用的。”他说，“把枪放下，我饶你一命。”

六子咬着牙，没有动。

“六子。”约翰的声音响起。

六子看着约翰。

“把枪放下。”

“张哥！”

“放下。”

六子咬着牙，慢慢地放下枪。

黄四郎大笑起来。

“张兄弟，这才对嘛。”他说，“识时务者为俊杰。跟我黄四郎合作，保你吃香的喝辣的。”

约翰看着黄四郎：“我需要时间考虑。”

“考虑？”黄四郎笑了，“可以。不过，这个亚瑟·摩根，要先留在 我这里。”

“不行！”约翰说，“你要把他怎么样？”

“放心，我不会伤害他。”黄四郎说，“他可是我的摇钱树。那些日本人，俄国人，可都等着要他呢。”

约翰握紧拳头。

“黄四郎，”他说，“你到底想怎么样？”

黄四郎看着约翰，眼神变得很复杂。

“张兄弟，我欣赏你的本事。”他说，“这样吧，我给你三天时间。三天之后，给我答复。是跟我合作，还是……看着你的兄弟被卖到国外，你自己选。”

说完，黄四郎一挥手。

“送客！”

从黄四郎府邸出来，约翰和六子走在街上。

六子看着约翰，欲言又止。

“张哥，那个亚瑟……是你兄弟？”

约翰没有回答，只是慢慢地走着。

“张哥，”六子又问，“你真的会答应黄四郎？”

约翰终于开口。

“不会。”

“那你打算怎么办？”

约翰停下脚步，转过身，看着黄四郎的府邸。

“只有一个办法。”

“什么办法？”

“救出人，然后，杀了他。”

六子看着约翰，突然觉得这个人的背影很高大。

“张哥，我帮你。”

约翰看着六子：“你怕不怕？”

六子摇头：“不怕。”

“为什么？”

六子想了想，然后说：“因为你是好人。”

约翰笑了。

在这个世界上，已经很久没有人说过他是好人了。

“六子，记住。”约翰说，“这个世界上，没有好人。只有死人，和活人。”

六子似懂非懂地点头。

当天晚上，约翰的住处。

约翰坐在桌子前，手里拿着一张纸。纸上，画着黄四郎府邸的地图。

六子站在旁边，看着地图。

“张哥，咱们怎么救人？”

“先摸清情况。”约翰说，“黄四郎的府邸很大，守卫很多。硬碰硬，我们讨不了好。”

“那怎么办？”

约翰想了想：“先找内应。”

“内应？”

约翰点头：“武举人。”

六子眼睛亮了：“武举人愿意帮咱们？”

“不知道。”约翰说，“但值得一试。”

他站起身，走到窗边，看着外面的夜色。

“黄四郎以为他很聪明。”他轻声说，“但他忘了，兔子也有咬人的时候。”

第二天，清晨。

约翰带着六子，再次来到武家大院。

武举人听说他们来了，把他们请到书房。

“张兄弟，”武举人笑着说，“今天来找我，有什么事？”

约翰开门见山：“我需要你帮我救一个人。”

“什么人？”

约翰看着武举人，慢慢地吐出两个字。

“亚瑟。”

武举人愣了一下：“亚瑟？什么人？”

“一个很重要的人。”约翰说，“被黄四郎抓住了。”

武举人想了想：“张兄弟，这种事，可不好办哪。黄四郎的府邸，守卫森严……”

“条件。”约翰打断他，“开出你的条件。”

武举人看着约翰，沉默了很长时间。

“好。”他说，“事成之后，黄四郎的所有财产，我要七成。”

“可以。”

武举人笑了：“痛快！张兄弟，三天后的行动，我参加。”

三天后，夜。

黄四郎的府邸，灯火通明。

今天是黄四郎的生日宴会，全城的达官贵人都来了。

但在府邸的后门，却有几道黑影，悄悄摸进了府邸。

约翰带着六子，还有武举人派来的十几个好手，潜入了府邸。

“张哥，”六子轻声说，“亚瑟被关在哪儿？”

约翰看着地图：“根据武举人提供的情报，应该在地下密室里。”

“地下密室？”

“走。”

一行人悄悄地向地下室摸去。

地下室入口，在黄四郎书房的书架后面。

约翰找到开关，轻轻一按。书架缓缓移开，露出一条向下的楼梯。

“六子，你在外面把风。”约翰说，“我一个人下去。”

“张哥，我和你一起去！”

约翰看着六子：“下面很危险。”

“我不怕。”

约翰点头：“好，一起。”

两人沿着楼梯，向下走去。

楼梯很长，很黑。约翰掏出一根火柴，点燃了，照亮前方的路。

走到楼梯尽头，是一扇铁门。

约翰轻轻推开门。

门后，是一个很大的房间。房间里，摆放着十几个笼子。每个笼子里，都关着一个人。

有白人，有黑人，有男人，有女人。

他们穿着破烂的衣服，脸上带着恐惧的表情。

而在房间的最里面，有一个单独的笼子。

笼子里，关着一个白人男人。

那个男人，左脸上有一道醒目的刀疤。

亚瑟·摩根。

“亚瑟！”约翰冲过去。

亚瑟抬起头，看到约翰，眼睛亮了。

“John……”他的声音很虚弱，“你真的来了……”

“别说话。”约翰掏出钥匙，打开笼子，“我救你出去。”

就在这时，门外传来脚步声。

“有人来了！”六子低声说。

约翰脸色变了。

“这么快就来了……被发现了！”

“张哥，怎么办？”

约翰咬紧牙关。

“杀出去！”

他拔出枪，指向门口。

---

门外，黄四郎带着几十个保镖，堵住了楼梯。

“张兄弟，”黄四郎的声音响起，“我等你好久了。”

约翰站在楼梯上，冷冷地看着黄四郎。

“黄四郎，你设计害我？”

“设计？”黄四郎笑了，“张兄弟，这叫兵不厌诈。你不会连这个都不知道吧？”

约翰举起枪，对准黄四郎。

“放我们走。”

“放你们走？”黄四郎的笑容变得阴冷，“不可能。你们今天，一个都别想走。”

“是吗？”约翰冷笑一声，“那就试试看。”

他扣动扳机。

“砰！”

枪声响起，一个保镖应声倒地。

“上！”黄四郎大喊。

保镖们一拥而上。

约翰和六子背靠背站着，手里的枪不断射击。

“砰砰砰砰！”

每一枪，都有一个保镖倒下。

但敌人太多了。

“这样下去不是办法！”六子喊道。

约翰扫了一眼周围，发现墙角有一个通风口。

“从那里走！”

他打碎通风口的铁栅栏，和六子一起钻进去。

两人在狭窄的通风管道里爬着，后面传来追兵的声音。

爬了一段距离，前面出现了分歧。

两条路，一条向左，一条向右。

“张哥，往哪边？”

约翰想了想。

“左边。”

两人向左爬去。

爬着爬着，前面出现了光亮。

“出口！”

两人爬出通风口，发现自己在一个仓库里。

仓库里，堆满了箱子和货物。

“这里应该是黄四郎的库房。”约翰说。

六子看着这些箱子：“张哥，这些都是什么？”

约翰打开一个箱子。

箱子里，是枪支、弹药，还有……鸦片。

“这个黄四郎，”约翰咬牙，“果然不是什么好东西。”

“先别管这些。”约翰说，“快走，武举人的人在外面接应我们。”

两人跑出仓库，向府邸后门跑去。

跑着跑着，前面突然出现一个人。

那个人穿着长衫，戴着礼帽，挡住了他们的路。

约翰举起枪。

“让开！”

那个人慢慢地摘下礼帽。

是黄四郎。

“张兄弟，”黄四郎笑着说，“想走？没那么容易。”

约翰看着黄四郎，眼神变得很冷。

“黄四郎，这是你逼我的。”

他扣动扳机。

“砰！”

黄四郎倒在地上。

鲜血染红了他的长衫。

“走！”约翰拉着六子，跑出府邸。

---

府邸外，武举人派来的人已经接应。

“快走！”

一行人骑上马，向城外奔去。

跑出一段距离，约翰回头看了一眼。

黄四郎的府邸，已经变成了火海。

“张哥，”六子问，“黄四郎……死了？”

“没有。”约翰说，“我打的是他的肩膀。”

六子愣住了：“那他……”

“黄四郎没那么容易死。”约翰说，“但至少短时间内，他不会再找我们的麻烦。”

六子点头。

“张哥，我们现在去哪儿？”

约翰看着远方。

“离开鹅城。”

“那亚瑟呢？”

约翰没有回答。

亚瑟·摩根，已经被他们救出来了。

但在混乱中，亚瑟因为身体虚弱，被落在了府邸里。

现在，也不知道怎么样了。

约翰咬紧牙关。

“一定会没事的。”他对自己说。

---

夜晚，鹅城陷入一片寂静。

但在黄四郎的府邸里，却是一片混乱。

黄四郎坐在书房里，肩膀上缠着绷带，脸色铁青。

“废物！都是废物！”他咆哮着，“这么多人都抓不住一个人，我要你们有什么用？！”

管家站在一边，大气不敢出一声。

“老爷息怒。”管家小心翼翼地说，“那个张麻子，确实有两下子……”

“两下子？”黄四郎冷笑，“他这是在太岁头上动土！”

他站起身，在房间里踱着步。

“管家，去查。”他说，“我要知道这个张麻子的一切。还有那个亚瑟·摩根，我要他死！”

“是，老爷。”

管家退下了。

黄四郎站在窗边，看着外面的夜色。

“约翰·马斯顿，”他轻声说，“我们之间的账，没完。”

# 第四章 矿洞

---

一个月后。

鹅城外的西山，有一座矿山。

矿山是黄四郎的产业，专门开采煤炭。矿工们每天在井下工作十二个时辰，换来的却只是微薄的工钱。

这天清晨，一队马车来到了矿山。

领头的是约翰，后面跟着六子和几个武举人派来的兄弟。他们伪装成商人，赶着几辆装满货物的马车。

“站住！”

矿山门口，两个守卫举起枪，对准他们。

约翰举起双手，慢慢地走过去。

“各位兄弟辛苦了。”他笑着说，“我们是来送货的。”

守卫上下打量了约翰一眼：“送什么货？”

“煤炭。”约翰说，“我们是从外地来的商人，想和黄四爷做一笔生意。”

守卫犹豫了一下：“等着。”

他转身跑进去通报。

不一会儿，守卫跑出来。

“进去吧。黄爷在矿场等着你们。”

约翰带着人，牵着马车，进了矿山。

---

矿山很大，有几百个矿工在干活。他们穿着破旧的衣服，驼着背，拖着沉重的煤筐。

约翰看着这些矿工，心里很不是滋味。

在西部的时候，他也见过很多这样的工人。廉价劳动力，被剥削，被压迫。

“六子，记住路。”他在六子耳边轻声说。

六子点头。

两人跟着守卫，来到了矿场的办公室。

办公室里，黄四郎坐在一张太师椅上，旁边站着几个保镖。

看到约翰，黄四郎大笑起来。

“张兄弟！我就知道你会来！”

约翰走过去，在黄四郎对面坐下。

“黄四郎，我来了。”

黄四郎看着约翰，眼神变得很复杂。

“张兄弟，上次的事，我还没跟你算账呢。”

约翰淡淡地说：“你想怎么样？”

黄四郎笑了：“不想怎么样。只是想提醒你，这里是我的地盘。”

“我知道。”

“那你还敢来？”

“因为我有事要问你。”

黄四郎看着约翰，沉默了很长时间。

“问吧。”

约翰看着黄四郎：“那些外国人，你在哪儿找到的？”

黄四郎的笑容僵了一下。

“什么外国人？我不知道你在说什么。”

“黄四郎，”约翰说，“别装了。我知道你在做什么生意。”

黄四郎看着约翰，眼神变得很阴冷。

“张兄弟，有些事，知道得太多，对你没好处。”

“我不怕。”

黄四郎盯着约翰看了半天，突然笑了。

“好，我告诉你。”他说，“那些外国人，都是我从外面抓来的。”

“抓？从哪儿抓？”

“从美国，从欧洲，从世界各地。”黄四郎说，“只要是有钱赚的生意，我都做。”

“你把他们怎么样了？”

“卖咯。”黄四郎笑着说，“卖给日本人，卖给俄国人，卖给任何人。只要出价够高。”

约翰握紧拳头。

“黄四郎，你还是人吗？”

“人？”黄四郎冷笑，“在这个世界上，人命值多少钱？一块大洋？十块大洋？还是一百块大洋？”

约翰沉默了很久。

“黄四郎，我再问你一个问题。”

“什么？”

“你上次抓的那个人，亚瑟·摩根，他在哪儿？”

黄四郎的笑容变得意味深长。

“你说那个白人？他啊……被我卖掉了。”

约翰脸色大变：“你说什么？”

“真的。”黄四郎说，“卖给日本人了。现在，应该已经到日本了吧。”

约翰霍然站起身。

“你骗我。”

“我没骗你。”黄四郎说，“不信，你可以去矿场下面看看。他住过的地方，现在还空着呢。”

约翰盯着黄四郎，盯了很久。

“黄四郎，”他说，“如果让我发现你骗我，我一定不会放过你。”

黄四郎笑了。

“随时奉陪。”

从办公室出来，约翰和六子向矿场深处走去。

“张哥，”六子轻声问，“那个黄四郎说的话，能信吗？”

“不能。”

“那咱们还去？”

“去。”约翰说，“看看下面到底有什么。”

两人向矿洞深处走去。

矿洞很深，很黑。空气中弥漫着霉味和煤炭的粉尘。

走了一段距离，前面出现了分歧。

两条路，一条向左，一条向右。

“张哥，往哪边？”

约翰看着两条路，想了想。

“右边。”

两人向右边走去。

走了一会儿，前面出现了一扇铁门。

铁门上有锁，锁得很严实。

“有人。”约翰轻声说。

他贴在门上，听里面的声音。

里面有说话声，还有……惨叫声。

约翰脸色变了。

“退后。”

他掏出枪，对准锁。

“砰！”

锁被打碎。

约翰推开门。

门后，是一条很长的走廊。走廊两边，是一间间的小房间。

房间里有床，有桌子，有……人。

每一间房间里，都关着一个人。

有白人，有黑人，有男人，有女人。

他们穿着破烂的衣服，脸上带着恐惧的表情。

“救命！”有人用英文喊。

约翰看着这些房间，心里的怒火越烧越旺。

“黄四郎这个人渣！”六子咬牙说。

约翰一间间地看过去。

终于，他在最里面的一间房间里，看到了一个熟悉的身影。

那个人背对着门，坐在床边。

“亚瑟？”约翰轻声叫了一声。

那个人慢慢地转过头。

看到那张脸的时候，约翰整个人都僵住了。

不是亚瑟。

但那个人，确实是个白人。

“你是谁？”约翰问。

那个人看着约翰，眼睛里先是疑惑，然后是惊喜。

“你是……中国人？”那个人用英文问。

“我是。”约翰用英文回答，“你是谁？”

“我叫比尔。”那人说，“我是亚瑟·摩根的兄弟。”

约翰眼睛亮了。

“亚瑟在哪儿？”

比尔看着约翰，眼神变得很复杂。

“亚瑟……他被关在更深的地方。”

“更深的地方？”

比尔点头：“黄四郎把他藏起来了。每天都会有人来问他话，问他关于你的事。”

“关于我？”

“是。”比尔说，“黄四郎想知道，你到底是什么人。还有，你的那些兄弟，现在在哪儿。”

约翰咬紧牙关。

“他在哪儿？告诉我！”

比尔想了想。

“在矿山的最深处，有一个秘密的实验室。”他说，“黄四郎把最重要的人都关在那里。”

约翰转身就走。

“张哥！”六子追上他，“你去哪儿？”

“救人。”

“不行啊！”六子着急地说，“那边肯定有很多守卫！”

约翰停下脚步。

“你说的对。”他说，“不能硬碰硬。”

他想了想。

“这样，六子，你先去探探路。记住，不要打草惊蛇。”

六子点头：“嗯。”

当天晚上，六子趁着守卫换班的时候，偷偷溜进了矿洞。

他沿着白天约翰走的路线，一直向深处走去。

走了一会儿，他看到了那扇铁门。

铁门后面，应该就是关押比尔的地方。

六子贴在门上，听里面的声音。

“有动静！”有人用中文说。

六子脸色变了。

有人发现他了！

他转身就跑。

“站住！”

身后传来脚步声。

六子拼命的跑，但矿洞太黑，他一不小心，摔了一跤。

这一摔不要紧，追上来的守卫抓住了他。

“原来是你！”守卫认出了六子，“上次在府邸捣乱的就是你！”

六子被押回了办公室。

---

办公室，黄四郎坐在太师椅上，看着六子。

“小兄弟，”他笑着说，“我们又见面了。”

六子咬着牙，没有说话。

“说！约翰·马斯顿在哪儿？”

六子闭上眼睛。

“不说是吧？”黄四郎的笑容变得阴冷，“来啊，用刑。”

几个保镖走上来，把六子绑在柱子上。

“用鞭子抽。”

“啪！啪！啪！”

鞭子一下一下地抽在六子身上。

六子咬着牙，不发出一点声音。

“还挺硬气。”黄四郎笑着说，“我倒要看看，你能硬到什么时候。”

他站起身，走到六子面前。

“小兄弟，我再问你一遍。约翰·马斯顿在哪儿？”

六子睁开眼睛，看着黄四郎。

“黄四郎，”他说，“你会后悔的。”

“后悔？”黄四郎笑了，“我黄四郎从来不知道什么叫后悔。”

他挥了挥手。

“继续打。”

鞭子继续抽打着。

六子终于忍不住，惨叫声回荡在矿洞里。

---

与此同时，矿山外面。

约翰带着武举人派来的人，悄悄地把矿山包围了。

“张爷，”一个兄弟问，“什么时候动手？”

约翰看着矿山。

“再等等。”

他在等一个信号。

六子发出的信号。

---

矿洞深处。

六子已经被打得遍体鳞伤。

但他始终没有说出约翰的下落。

“黄四郎，”他艰难地开口，“你……你不得好死……”

黄四郎笑了。

“放心，我会好好活着。而你……”

他话音未落，外面突然传来枪声。

“砰砰砰砰！”

黄四郎脸色变了。

“怎么回事？！”

一个保镖跑进来。

“老爷！有人打进来了！”

“多少人？”

“很多！”

黄四郎咬紧牙关。

“来的正好！”他拔出枪，“把这个人带上，跟我走！”

他带着保镖，押着六子，向矿洞深处跑去。

---

矿洞深处，有一个秘密出口。

黄四郎从出口逃出去，回头看着矿山。

矿山里，火光冲天。

“约翰·马斯顿，”他咬牙说，“我和你没完！”

他转身，消失在夜色中。

---

矿山里，约翰带着人，正在和黄四郎的守卫激战。

“杀！”

枪声、喊杀声，回荡在矿洞里。

很快，守卫们死的死，伤的伤，剩下的都跑了。

约翰带着人，继续向深处跑去。

终于，他找到了那扇铁门。

“砰！”

门被踢开。

门后，是一间很大的房间。

房间里，摆着各种奇怪的仪器。

而在房间的最里面，有一张床。

床上，躺着一个人。

那个人，左脸上有一道刀疤。

“亚瑟！”

约翰冲过去。

亚瑟睁开眼睛，看到约翰，眼睛亮了。

“John……”他的声音很虚弱，“你真的来了……”

“别说话。”约翰说，“我救你出去。”

他扶起亚瑟，向外走去。

就在这时，身后传来枪声。

“砰！”

约翰肩膀一痛，中弹了。

他转过身。

是黄四郎！

黄四郎站在门口，手里举着枪。

“约翰·马斯顿，”他笑着说，“游戏结束了。”

约翰看着黄四郎，眼神变得很冷。

“黄四郎，这是你逼我的。”

他举起枪。

“砰！”

枪声响起。

黄四郎倒在地上。

---

枪声过后，矿山陷入一片寂静。

约翰抱着亚瑟，慢慢地走出矿洞。

外面，天已经亮了。

六子跑过来。

“张哥！”

约翰把亚瑟交给六子。

“带他走。”

六子点头：“嗯。”

一行人骑上马，向远处奔去。

---

一天后，鹅城外的山坡上。

约翰、六子、亚瑟，还有武举人派来的兄弟，坐在山坡上，休息。

亚瑟的伤势已经稳定了，但还是很虚弱。

“John，”亚瑟看着约翰，“谢谢你救了我。”

“别说话。”约翰说，“好好养伤。”

亚瑟看着约翰，眼神变得很复杂。

“John，我有一件事要告诉你。”

“什么？”

“那些日本人……”亚瑟的声音很低，“他们不是来买人的。”

约翰看着亚瑟：“什么意思？”

“他们是来……找一个东西。”

“什么东西？”

亚瑟看着约翰，犹豫了一下。

“一个……秘密。”

约翰皱起眉头。

“什么秘密？”

亚瑟还没来得及回答，武举人跑过来。

“张兄弟，不好了！”

“怎么了？”

“黄四郎……他没死！”

约翰脸色变了。

“什么？！”

“黄四郎跑了！”武举人气喘吁吁地说，“他带着剩下的人，逃到了矿山的另一个入口！”

约翰咬紧牙关。

“他在哪儿？”

武举人指了指远处的一座山。

“那座山后面，有一个秘密基地。黄四郎把最重要的东西，都藏在那里。”

约翰站起身。

“带我去。”

# 第五章 背叛

---

鹅城，约翰的住处。

约翰坐在桌子前，面前摊着一张地图。地图上，标注着黄四郎秘密基地的位置。

六子站在旁边，看着地图。

“张哥，”六子问，“那个秘密基地，真的在那里？”

约翰点头：“武举人提供的情报，应该不会有错。”

六子看着地图，皱起眉头。

“可那里看起来很险要啊。”

“确实。”约翰说，“易守难攻。”

“那咱们怎么打？”

约翰想了想。

“先不急。”他说，“知己知彼，百战百胜。先派人去探探情况。”

六子点头：“嗯。”

就在这时，门外传来脚步声。

“谁？”约翰举起枪。

“是我，汤师爷。”门外传来声音。

“进来。”

门被推开，汤师爷从外面走进来。

他手里端着一碗茶，脸上带着谄媚的笑。

“张爷，六爷，喝茶。”

约翰看着汤师爷：“你来干什么？”

“来看看两位爷。”汤师爷把茶放在桌子上，“两位爷辛苦了，我这个做下属的，理应效劳。”

约翰没有说话，只是看着汤师爷。

汤师爷被看得有些不自在。

“张爷，您……您这么看着我干什么？”

“你最近在干什么？”约翰问。

“我？我没干什么啊。”汤师爷笑着说，“就是帮两位爷跑跑腿，打听点消息。”

“打听到了什么消息？”

汤师爷犹豫了一下。

“也没什么……就是黄四郎那边，好像在调动人手。”

约翰看着汤师爷：“然后呢？”

“然后……”汤师爷看了看四周，压低声音，“我听说，黄四郎准备对咱们动手了。”

约翰脸色不变：“动手？他想怎么动手？”

“不知道。”汤师爷说，“但肯定不是什么好事。张爷，您要小心啊。”

约翰看着汤师爷，盯了很久。

“汤师爷，”他说，“你跟着我，是为了什么？”

汤师爷愣了一下。

“张爷，您这是什么意思？”

“我在问你话。”

汤师爷看着约翰，脸上的笑有些僵硬。

“张爷，我……我是真心投奔您的啊。”

“真心？”约翰冷笑一声，“好，我信你。”

他站起身，走到窗边，看着外面。

“汤师爷，我有一件事要你去办。”

“什么事？”

“去黄四郎的秘密基地，帮我打探一下情况。”

汤师爷脸色变了。

“张爷，这……这太危险了吧？”

“怎么？你怕了？”

汤师爷连连摆手。

“不怕、不怕！为张爷办事，刀山火海我也去！”

约翰点头：“好，去吧。记住，不要暴露身份。”

汤师爷点头哈腰：“是、是！”

他转身走出房间。

---

汤师爷走出房间，并没有去秘密基地。

而是去了黄四郎的府邸。

---

黄四郎府邸，书房。

黄四郎坐在书桌前，手里端着一杯茶。

管家站在旁边，手里拿着一封信。

“老爷，”管家说，“汤师爷来了。”

黄四郎抬起头：“让他进来。”

管家走出去，带着汤师爷走进来。

汤师爷一进门，就跪在地上。

“黄爷，小的给您请安了。”

黄四郎看着汤师爷，笑了。

“汤师爷，起来吧。”

“谢黄爷！”

汤师爷站起身，弓着腰站在一边。

“汤师爷，”黄四郎说，“我让你办的事怎么样了？”

汤师爷连连点头。

“回黄爷的话，都办妥了。”

“约翰·马斯顿那边，有什么动静？”

汤师爷凑上前，压低声音。

“黄爷，约翰准备对您的秘密基地动手了。”

黄四郎眼睛亮了。

“什么时候？”

“就在这两天。”

黄四郎大笑起来。

“好！太好了！”

他站起身，在房间里踱着步。

“天堂有路你不走，地狱无门你自来投。既然他找死，那就别怪我不客气了！”

他转身看着汤师爷。

“汤师爷，这次你立了大功。事成之后，我重重有赏！”

汤师爷连连道谢。

“多谢黄爷！多谢黄爷！”

第二天，晚上。

约翰带着六子，还有十几个兄弟，悄悄地向黄四郎的秘密基地摸去。

秘密基地在一座山的山腰，四周都是峭壁，只有一条路可以上去。

这条路，已经被黄四郎的人堵死了。

“张哥，”六子轻声说，“前面有哨卡。”

约翰看着前方。

确实，在山路上，有几个火堆，几十个守卫守在那里。

“怎么打？”六子问。

约翰想了想。

“先观察一下情况。”

他带着六子，悄悄地向哨卡摸去。

两人趴在草丛里，看着哨卡的情况。

守卫们有的在烤火，有的在聊天，有的在喝酒。

“大概有多少人？”约翰问。

六子数了数。

“二十几个。”

约翰点头。

“够我们受的。”

“那怎么办？”

约翰没有说话，只是看着哨卡。

突然，他眼睛一亮。

“六子，你看那边。”

六子顺着约翰指的方向看过去。

在哨卡的侧面，有一条小路。小路很窄，只能容一个人通过。

“从那里上去。”

两人悄悄地从小路向上爬。

小路很陡，很滑。但两人都是爬惯山的人，这点困难不算什么。

爬了一段距离，前面突然出现了一个守卫。

守卫背对着他们，正在小便。

约翰悄悄地摸过去，一刀刺进守卫的后背。

守卫连声音都没来得及发出，就倒下了。

约翰把守卫的尸体拖到草丛里，继续向上爬。

---

两人爬上山腰，找到了一个隐蔽的位置。

从那里，可以清楚地看到秘密基地的情况。

基地很大，有几十间房子，守卫大概有一百多人。

而在基地的最中间，有一间最大的房子。

房子门口，站着几个保镖。

“那里应该就是黄四郎的住处。”约翰说。

六子点头：“嗯。”

“先不要打草惊蛇。”约翰说，“等武举人的人到了，再一起动手。”

六子点头。

两人藏在草丛里，等待着。

---

等了一会儿，武举人的人还没有到。

但约翰突然觉得有些不对劲。

“六子，”他轻声说，“你有没有听到什么声音？”

六子仔细听了听。

“没有啊。”

“不，有声音。”约翰说，“很多人脚步的声音。”

话音刚落，四周突然亮起了火把。

“有人！”

“抓住他们！”

约翰脸色变了。

“被发现了！快跑！”

他拉起六子，向山下跑去。

但已经来不及了。

四面八方，都是黄四郎的人。

“约翰·马斯顿，”一个声音响起，“你好大的胆子！”

约翰抬头看。

黄四郎站在山坡上，身边站着几十个保镖。

“黄四郎！”约翰咬牙说。

黄四郎笑了。

“约翰·马斯顿，你以为你很聪明？”他说，“你的一举一动，都在我的掌握之中。”

约翰看着黄四郎，心里突然明白了。

“是你！”他说，“是你安排的！”

“没错。”黄四郎笑着说，“汤师爷是我的人。怎么样，被自己人背叛的滋味，好受吗？”

约翰握紧拳头。

“黄四郎，我和你拼了！”

他举起枪，对准黄四郎。

“砰砰砰！”

几声枪响，几个保镖倒在地上。

“上！”黄四郎大喊。

保镖们向约翰冲过来。

约翰和六子背靠背站着，手里的枪不断射击。

“砰砰砰砰！”

每一枪，都有一个保镖倒下。

但敌人太多了。

“这样下去不是办法！”六子喊道。

约翰扫了一眼周围。

“六子，看到那边没有？”

六子顺着约翰指的方向看过去。

在右边，有一条小路。小路很陡，但可以跑。

“从那里走！”

两人且战且退，向小路跑去。

“想跑？没那么容易！”黄四郎大喊，“追！”

保镖们追上去。

约翰和六子跑得很快，但后面的追兵也不慢。

跑着跑着，六子突然停下来。

“张哥！”

约翰转身。

六子的腿，中弹了。

“六子！”约翰跑回去，扶住六子。

“张哥，你……你先走！”六子咬着牙说，“别管我！”

“不行！”约翰说，“要死一起死！”

他背起六子，继续跑。

后面，追兵越来越近。

“砰！”

一颗子弹打中约翰的肩膀。

约翰一个趔趄，险些摔倒。

但他还是咬牙坚持着，继续跑。

跑着跑着，前面突然出现了一个人。

汤师爷。

汤师爷手里举着一把枪，对准约翰。

“张爷，”汤师爷笑着说，“别来无恙啊。”

约翰看着汤师爷，眼神变得很冷。

“汤师爷，”他说，“你……”

“我什么？”汤师爷笑着打断他，“识时务者为俊杰嘛。张爷，对不起了。”

他举起枪，对准约翰。

“去死吧！”

“砰！”

枪声响起。

汤师爷倒在地上。

约翰身后，六子举着枪，手还在发抖。

“张哥……快走……”

约翰咬牙，背起六子，继续跑。

终于，他们甩掉了追兵。

---

在一个偏僻的山沟里，约翰放下六子。

六子的腿在流血，约翰的的肩膀也在流血。

“六子，”约翰说，“你怎么样？”

“还……还死不了……”六子勉强笑着说。

约翰撕下一块布，帮六子包扎伤口。

“六子，你刚才那一枪，很准。”

六子笑了笑。

“张哥教的。”

约翰没有说话，只是默默地包扎着。

“张哥，”六子问，“接下来……咱们该怎么办？”

约翰看着远处。

“养好伤，然后再打回去。”

六子点头。

“嗯。”

---

第二天，清晨。

约翰和六子在一个山洞里休息。

六子的伤已经包扎好了，但还是很虚弱。

“张哥，”六子问，“那个汤师爷……他真的是黄四郎的人？”

约翰点头。

“是。”

六子沉默了。

“张哥，我……我有个问题想问你。”

“什么问题？”

“你……你为什么要对我这么好？”

约翰看着六子，没有说话。

“像我这样的人，命贱，不值得别人对我好。”六子慢慢地说，“可是张哥，你却……”

“六子，”约翰打断他，“你知道我为什么会来中国吗？”

六子摇头。

“因为我在美国，杀了人。”

六子愣住了。

“杀了……人？”

“是。”约翰说，“我杀了一个赏金猎人。因为他想要我的命。”

“那……那后来呢？”

“后来，我就逃走了。”约翰说，“逃到了一个很远很远的地方。”

“中国？”

“中国。”约翰说，“我听说，这里有很多和我一样的人。我想找到他们。”

六子看着约翰。

“找到了吗？”

约翰点头。

“找到了。但他们都被黄四郎抓起来了。”

六子沉默了很久。

“张哥，”他说，“我帮你。”

约翰看着六子。

“你帮我？”

“是。”六子说，“我虽然没有你那么大的本事，但我可以帮你做事。帮你杀人，帮你放火，帮你做任何事。”

约翰看着六子，眼神变得很复杂。

“六子，你不需要为我做任何事。”

“为什么？”

“因为你是我的兄弟。”

六子愣住了。

兄弟。

这个词，他已经很久没有听到了。

“张哥……”六子的声音有些哽咽。

约翰拍了拍六子的肩膀。

“好好养伤。”他说，“等伤好了，我们一起打回去。”

六子点头。

“嗯！”

---

与此同时，黄四郎的秘密基地。

黄四郎坐在书房里，脸上的笑容很得意。

“管家，”他说，“这次多亏了汤师爷。”

管家点头。

“老爷英明。”

“对了，”黄四郎问，“汤师爷呢？”

管家犹豫了一下。

“回老爷的话，汤师爷……他……”

“他怎么了？”

“他在昨天的战斗中，中弹了。”

黄四郎愣了一下。

“死了？”

“还没有。”管家说，“但也快了。”

黄四郎沉默了一会儿。

“派人去照顾他。”他说，“毕竟，他也是有功的人。”

管家点头。

“是。”

---

三天后。

约翰的伤已经好得差不多了。

六子的伤也恢复了一些，可以走路了。

两人离开山洞，向鹅城走去。

“张哥，”六子问，“我们去哪儿？”

“去找武举人。”

“找他？”

“是。”约翰说，“该算账了。”

---

武家大院。

武举人听说约翰来了赶紧把他们请进书房。

“张兄弟，”武举人笑着说，“你没事太好了。”

约翰看着武举人，眼神很冷。

“武举人，”他说，“我有话问你。”

武举人愣了一下。

“什么话？”

“你是怎么知道秘密基地的位置的？”

武举人的笑容僵了一下。

“这个……是我安插在黄四郎身边的眼线提供的。”

“是吗？”约翰说，“那个眼线，是汤师爷吧？”

武举人脸色变了。

“张兄弟，你这是什么意思？”

“什么意思？”约翰冷笑一声，“武举人，你和黄四郎，真当我不知道？”

武举人沉默了很久。

然后，他笑了。

“张兄弟，既然你知道了，那我也不瞒你了。”他说，“没错，汤师爷是我的人。”

“什么？！”六子惊呼。

“但我也是有苦衷的。”武举人说，“黄四郎威胁我，如果我不合作，他就杀了我全家。”

“你放屁！”六子咬牙说。

武举人看着约翰。

“张兄弟，我知道你恨我。但现在不是内讧的时候。黄四郎才是我们共同的敌人。”

约翰沉默了很久。

“武举人，”他说，“我最后再信你一次。”

武举人眼睛亮了。

“张兄弟，你……”

“先别高兴。”约翰打断他，“如果再让我发现你背叛我，我不会放过你。”

武举人连连点头。

“不敢、不敢！”

约翰站起身。

“起来吧。”他说，“准备一下，我们该和黄四郎算总账了。”

---
