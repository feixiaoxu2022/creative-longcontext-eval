第1集 重生觉醒

场景1
地点：医院病房
时间：白天
氛围：迷茫、震惊

（方晓晓缓缓睁开眼睛，刺眼的白光让她眯起了眼。）
她下意识地摸向床头柜，想找手机看时间。
手指触碰到一个冰凉的东西——是她的手机。
方晓晓：（自言自语）几点了...
她拿起手机，屏幕亮起，显示的日期让她瞬间僵住。
（方晓晓的瞳孔骤然收缩，呼吸变得急促。）
方晓晓：（震惊）这...这不可能...
她颤抖着手指划开手机，屏幕上显示的日期是10年前。
（方晓晓感到一阵眩晕，脑海中闪过车祸的画面。）
方晓晓：（喃喃自语）我...我重生了？

场景2
地点：医院病房
时间：白天
氛围：震惊、困惑

方晓晓强忍着眩晕，开始检查手机里的内容。
她打开新闻APP，发现里面存着未来10年的重大新闻。
方晓晓：（难以置信）这...这些都是未来的新闻？
她继续翻看，发现还有股票信息、房价走势...
（方晓晓的手开始颤抖，她意识到自己拥有了一个不可思议的宝库。）
她注意到手机右上角的电量显示：37%。
方晓晓：（疑惑）37%？
她插上充电器，但电量纹丝不动。
方晓晓：（震惊）充不进去？
她拔掉充电器，电量依然显示37%。
（方晓晓感到一阵寒意，这个手机似乎有着神秘的力量。）

场景3
地点：大学宿舍
时间：傍晚
氛围：怀旧、复杂

方晓晓回到大学宿舍，看着熟悉的房间，心中五味杂陈。
（她看着镜中年轻的自己，眼眶微微泛红。）
方晓晓：（自言自语）33岁...我33岁就死了...
她深吸一口气，眼神逐渐变得坚定。
方晓晓：（坚定）这一世，我不会再犯同样的错。
她打开手机，开始规划如何利用这些未来信息改变命运。
（窗外的夕阳洒在她身上，映照出一个重获新生的灵魂。）
方晓晓站在窗前，看着城市的夜景，嘴角露出一丝微笑。
方晓晓：（内心独白）这一次，我要活出不一样的人生。

第2集 预知限制

场景1
地点：方晓晓的出租屋
时间：白天
氛围：兴奋、期待

方晓晓坐在电脑前，打开股票软件，准备用预知信息投资。
方晓晓：（兴奋）根据手机里的信息，这支股票明天会暴涨...
她毫不犹豫地投入了所有积蓄。
（方晓晓感到一阵激动，仿佛看到了财富在向她招手。）
第二天，她满怀期待地打开股票软件。
但屏幕上显示的却是：该股票已于昨日停牌。
方晓晓：（震惊）什么？停牌了？
她急忙打开手机，想查看那条新闻。
但那条关于股票暴涨的新闻已经消失了。
方晓晓：（惊恐）不见了？怎么会不见了？

场景2
地点：方晓晓的出租屋
时间：白天
氛围：困惑、恐惧

方晓晓疯狂地翻找手机里的其他信息。
她发现凡是她试图主动利用的信息，都开始消失。
方晓晓：（恐惧）这...这是什么意思？
她想起手机电量永远停在37%，心中涌起一股寒意。
方晓晓：（自言自语）手机里的未来，只在我不主动利用时才准确...
（方晓晓感到一阵恐惧，这个手机似乎有着自己的意志。）
她坐在床边，双手抱头，陷入深深的困惑。
方晓晓：（痛苦）那我重生还有什么意义？
她抬起头，眼神中闪过一丝不甘。
方晓晓：（坚定）不，一定有办法的...

场景3
地点：方晓晓的出租屋
时间：夜晚
氛围：沉思、决心

方晓晓坐在窗前，看着城市的夜景，陷入沉思。
（她回想起前世的悲惨遭遇，心中涌起一股强烈的复仇欲望。）
方晓晓：（内心独白）苏雅琴...陈浩然...你们害死了我，这一世，我要让你们付出代价。
她握紧拳头，眼神变得坚定。
方晓晓：（坚定）就算不能直接利用预知信息，我也要改变命运。
她拿起手机，开始研究如何在不触发限制的情况下利用信息。
（窗外的月光洒在她身上，映照出一个决心复仇的灵魂。）
方晓晓站起身，走向书桌，开始制定复仇计划。
方晓晓：（内心独白）这一世，我不会再犯同样的错。

第3集 前世记忆

场景1
地点：方晓晓的出租屋
时间：夜晚
氛围：震惊、痛苦

方晓晓在手机里发现了一个未完成的小说文档。
方晓晓：（疑惑）这是什么？
她点开文档，开始阅读。
（随着阅读的深入，方晓晓的脸色越来越苍白。）
方晓晓：（震惊）这...这是我的故事？
小说的内容竟然是她前世的真实经历，从出生到33岁死亡。
（方晓晓感到一阵眩晕，前世的记忆如潮水般涌来。）
她看到小说中描写自己被苏雅琴和陈浩然背叛的场景。
方晓晓：（痛苦）不...不要...
她看到小说中描写自己孤独死去的场景。
（方晓晓的眼泪夺眶而出，她捂住嘴，不让自己哭出声。）

场景2
地点：方晓晓的出租屋
时间：夜晚
氛围：痛苦、回忆

方晓晓瘫坐在地上，双手抱头，陷入痛苦的回忆。
（前世的记忆如电影般在脑海中回放。）
方晓晓：（内心独白）苏雅琴...你是我最好的朋友...
她想起前世被苏雅琴背叛的场景。
方晓晓：（内心独白）陈浩然...你说过会永远爱我...
她想起前世被陈浩然背叛的场景。
（方晓晓感到一阵心痛，前世的痛苦再次袭来。）
方晓晓：（痛苦）为什么...为什么要这样对我...
她抬起头，眼神中闪过一丝恨意。
方晓晓：（坚定）这一世，我要让你们付出代价。

场景3
地点：方晓晓的出租屋
时间：深夜
氛围：决心、复仇

方晓晓擦干眼泪，站起身来，眼神变得坚定。
方晓晓：（坚定）前世我太傻了，这一世，我不会再犯同样的错。
她拿起手机，开始研究如何利用预知信息复仇。
（方晓晓的眼神中闪过一丝冷意。）
方晓晓：（内心独白）苏雅琴，陈浩然，你们等着...
她打开手机，开始制定复仇计划。
（窗外的月光洒在她身上，映照出一个决心复仇的灵魂。）
方晓晓坐在书桌前，开始写下复仇计划的第一步。
方晓晓：（内心独白）这一世，我要活出不一样的人生。
