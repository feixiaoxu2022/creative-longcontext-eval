# 第一章：坠落的瞬间

## 场景1：体操馆的晨光

**场景编号**：1-1
**地点**：市体操队训练馆，清晨6:30，阳光透过高窗斜射在平衡木上
**在场角色**：林清羽、苏浅、陈教练、其他队员
**场景目的**：建立重生后的初始状态，引出核心冲突

**核心动作**：林清羽站在平衡木前，看着18岁的苏浅正在做赛前热身，突然意识到自己回到了十年前

**冲突爆发点**：苏浅在平衡木上做一个高难度的后空翻两周转体360度，落地时平衡木突然发出"咔嚓"一声

**情感转变**：从迷茫困惑 → 惊恐绝望 → 本能反应

**场景结果**：林清羽冲上去想接住苏浅，却被所有人误解为推人，苏浅重重摔在地上

**正文**：

清晨的体操馆弥漫着镁粉的味道，阳光像金色的纱幔一样从高窗垂落，照亮了平衡木上那个轻盈的身影。

林清羽站在场边，手指无意识地摩挲着护腕上的魔术贴。这是她熟悉的触感——十年前她还是运动员时最喜欢的护腕。但不对劲，太不对劲了。

她低头看自己的手，纤细、没有老茧，指甲修剪得整整齐齐。这不是32岁教练的手，这是18岁运动员的手。

"清羽，发什么呆呢？"陈教练的声音从身后传来，带着惯有的严厉，"苏浅马上要测那套动作了，你去给她做保护。"

林清羽猛地抬头。平衡木上，苏浅正在做赛前热身。那张脸——不是后来坐在轮椅上苍白消瘦的苏浅，而是18岁的苏浅，脸颊红润，眼神里燃烧着野心和自信。

记忆如潮水般涌来。2014年，全国青年体操锦标赛前夕，苏浅在训练中坠落，导致高位截瘫。而她，林清羽，被指控故意推人，从此背负着"凶手"的骂名，即使后来成为金牌教练也无法洗清这个污点。

"不..."林清羽的喉咙发紧，她想冲上去阻止，但身体像被钉在原地。

平衡木上，苏浅深吸一口气，开始了那套致命的动作——后空翻两周转体360度接直体后空翻。这是她的杀手锏，也是前世坠落的那个动作。

林清羽的瞳孔骤然收缩。她看到了——平衡木的一颗螺丝松动了，在苏浅落地的瞬间会发生微小的位移。

"停下！"林清羽终于喊出声，但已经晚了。

苏浅腾空而起，在空中完成了完美的两周转体。就在她即将落地的瞬间，平衡木发出一声清脆的"咔嚓"声，右侧支架突然下沉。

时间仿佛被拉长了。林清羽看到苏浅的身体在空中失去平衡，像一只折翼的鸟向下坠落。她的大脑还没来得及思考，身体已经做出了反应——

她冲向平衡木，双臂张开，试图接住下坠的苏浅。

"啊！"

一声惨叫划破了清晨的宁静。苏浅重重地摔在垫子上，但不是林清羽接住的位置——因为惯性，她从林清羽的手臂旁滑过，后脑勺狠狠撞在了平衡木的支架上。

"苏浅！"林清羽跪倒在地，双手颤抖着去扶苏浅的头，"别动，别动..."

"你干了什么！"陈教练的怒吼声炸响在耳边，"林清羽，你推她干什么？！"

林清羽猛地抬头，看到的是陈教练愤怒扭曲的脸，还有周围队员们惊恐、愤怒、鄙夷的目光。她这才意识到——在外人看来，她冲上去的动作不是救援，而是推人。

"不，我没有..."林清羽的声音颤抖着，眼泪已经夺眶而出，"平衡木坏了，螺丝松了..."

"闭嘴！"陈教练一把推开她，冲向苏浅，"快叫救护车！苏浅，苏浅你醒醒！"

苏浅躺在地上，脸色惨白，后脑勺的血迹在晨光中显得格外刺眼。她的眼睛微微睁开，目光涣散地看着林清羽，嘴唇动了动，似乎想说什么，但最终陷入了昏迷。

林清羽跪在原地，双手还保持着接人的姿势，但怀里空无一物。周围的指责声像潮水一样涌来：

"太狠毒了，嫉妒苏浅就直说..."
"这种人怎么配当运动员？"
"故意伤人，应该报警抓她！"

林清羽听不见这些声音。她只看见苏浅被抬上担架时，那双眼睛最后看向她的眼神——不是感激，不是疑惑，而是纯粹的、冰冷的恨意。

那是她前世噩梦的开始。

## 场景2：医院的走廊

**场景编号**：1-2
**地点**：市第一医院骨科病房外走廊，上午10:00，消毒水味弥漫
**在场角色**：林清羽、苏浅父母、陈教练、护士
**场景目的**：展现林清羽的愧疚和被误解的处境

**核心动作**：林清羽站在病房外，透过玻璃窗看着昏迷的苏浅，手里紧紧攥着从平衡木上拆下的松动螺丝

**冲突爆发点**：苏浅的母亲冲出来扇了林清羽一耳光，指责她是凶手

**情感转变**：从自责愧疚 → 委屈愤怒 → 坚定决心

**场景结果**：林清羽被禁止探视，但她发誓这一世一定要弥补过错

**正文**：

医院的走廊里弥漫着刺鼻的消毒水味，白色的墙面反射着冷光，让人眩晕。

林清羽站在骨科病房外，脸上还留着苏浅母亲那一耳光的红印。火辣辣的疼痛让她清醒——这不是梦，她真的重生了。

"滚！你给我滚！"苏母的哭喊声引来了护士和其他病人的侧目，"如果浅浅有什么三长两短，我要你偿命！"

"阿姨，我真的没有推她..."林清羽的声音沙哑，手里紧紧攥着那个从平衡木上拆下的螺丝，指节发白，"是平衡木的螺丝松了，我想接住她..."

"借口！全都是借口！"苏父红着眼眶冲过来，被陈教练拦住，"老苏，冷静点！现在最重要的是苏浅的治疗！"

林清羽透过病房的玻璃窗，看到苏浅躺在病床上，头上缠着厚厚的纱布，各种仪器的管线连在她身上。医生正在检查，眉头紧锁。

"情况不太乐观，"陈教练低声对苏家父母说，"颈椎受损，可能...可能会影响下肢功能。"

苏母发出一声撕心裂肺的哭喊，瘫软在地上。苏父抱着妻子，眼泪无声地流下来。

林清羽的心像被一只无形的手紧紧攥住。前世的记忆涌上来——苏浅瘫痪后的绝望，轮椅上的沉默，还有那句从未说出口的"对不起"。

"让我进去，"林清羽突然开口，声音不大但异常坚定，"我有话要对苏浅说。"

"你还想干什么？"苏母猛地抬头，眼神像刀子一样，"再伤害她一次吗？"

"不，"林清羽走近一步，目光直视苏母，"我要告诉她，我会负责。不管她变成什么样，我都会负责到底。"

"负责？你拿什么负责？"陈教练冷冷地说，"林清羽，队里已经决定暂停你的训练资格，等待调查结果。你现在最好回家反省。"

林清羽看着陈教练，又看看病房里的苏浅。她知道现在说什么都没用，所有人都认定她是凶手。

但她有一个秘密武器——重生的记忆。

她知道苏浅会瘫痪，但她也知道如何帮助苏浅康复。前世她作为教练积累的所有经验，现在都可以用来挽救苏浅。

"我不会走的，"林清羽把那个螺丝放进衣兜，转身走向走廊尽头的电梯，"我会证明我的清白，也会治好苏浅。"

身后传来苏母的咒骂声和陈教练的冷笑，但林清羽没有回头。她的眼神变了——不再是18岁少女的迷茫，而是经历过十年风雨的坚韧。

## 场景3：深夜的体操馆

**场景编号**：1-3
**地点**：体操队训练馆，深夜11:00，只有应急灯亮着
**在场角色**：林清羽
**场景目的**：展现林清羽的决心和行动力，为后续补偿做铺垫

**核心动作**：林清羽独自在平衡木前练习接人动作，一遍遍重复直到精疲力竭

**冲突爆发点**：林清羽在练习时摔倒，但她立刻爬起来继续

**情感转变**：从绝望无助 → 咬牙坚持 → 看到希望

**场景结果**：林清羽制定了详细的康复计划，决定从明天开始接近苏浅

**正文**：

深夜的体操馆像一座沉默的巨兽，只有应急灯发出微弱的黄光，在地上投下长长的影子。

林清羽站在平衡木前，这是事故发生的地方。现在这里空无一人，只有镁粉在空气中缓缓飘落。

她闭上眼睛，回忆前世苏浅康复训练的每一个细节。作为教练，她帮助过无数运动员康复，但苏浅是最特殊的一个——不仅因为她们的关系，更因为苏浅的倔强。

"如果我当时反应再快一点..."林清羽喃喃自语，手指抚摸着平衡木的支架，
"如果我能提前发现螺丝松动..."

但世上没有如果。她能做的只有向前看。

林清羽深吸一口气，踏上平衡木。她要练习那个动作——不是为了自己，而是为了将来能保护苏浅。

"一、二、三..."

她开始做后空翻，但在落地的瞬间故意让自己失去平衡，然后练习如何在空中调整姿势去接住一个下坠的人。

第一次，她重重摔在垫子上，肩膀撞得生疼。
第二次，她的脚踝扭伤了。
第三次，她的膝盖磕破了皮。

但她没有停下来。一次又一次，直到汗水浸透了训练服，直到肌肉开始抽搐。

"再来一次..."林清羽咬着牙，眼泪和汗水混在一起流下来，
"这一次，我一定要接住你。"

凌晨两点，林清羽终于瘫坐在垫子上。她的身体在抗议，但她的眼神却异常明亮。

她从包里拿出笔记本，借着微弱的灯光开始写计划：

*苏浅康复计划（第一阶段）：*
*1. 心理疏导：防止她陷入抑郁*
*2. 物理治疗：从被动运动开始*
*3. 核心训练：即使不能走路也要保持上半身力量*
*4. 最重要的：让她相信我不是凶手*

林清羽合上笔记本，望着黑暗中的平衡木。明天，她就要开始执行这个计划。不管苏浅怎么恨她，不管别人怎么看她，她都要弥补前世的遗憾。

"等着我，苏浅，"她轻声说，声音在空旷的体育馆里回荡，"这一次，换我来保护你。"

窗外，一轮弯月挂在夜空，像是在默默注视着这个重生者的誓言。

# 第二章：重生的恨意

## 场景1：苏醒的噩梦

**场景编号**：2-1
**地点**：市第一医院骨科病房，上午9:00，阳光刺眼
**在场角色**：苏浅、苏母、护士
**场景目的**：展现苏浅重生后的初始状态和对林清羽的仇恨

**核心动作**：苏浅从昏迷中醒来，摸到自己的腿，发现还有知觉，然后回忆起前世的一切

**冲突爆发点**：苏浅问母亲"林清羽呢"，得知林清羽被停训但还在队里

**情感转变**：从迷茫困惑 → 震惊狂喜 → 冰冷恨意

**场景结果**：苏浅决定这一世要让林清羽付出代价

**正文**：

头痛欲裂。

苏浅睁开眼睛，刺眼的阳光让她下意识地抬手遮挡。消毒水的味道充斥着鼻腔，这是医院特有的气息。

"醒了！浅浅醒了！"母亲的声音带着哭腔，紧接着是一阵慌乱的脚步声。

苏浅试图坐起来，但后颈的剧痛让她倒吸一口冷气。她的手无意识地摸向腿部——等等，腿部？

她的手指触碰到了被子下的双腿，有温度，有知觉。她动了动脚趾，清晰地感觉到了床单的触感。

"我的腿..."苏浅的声音沙哑，
"还能动吗？"

苏母的眼泪夺眶而出："能，医生说只是轻微脑震荡，腿部没有大碍，观察几天就能下地了。"

苏浅愣住了。前世的记忆如潮水般涌来——瘫痪、轮椅、绝望、还有林清羽那张"无辜"的脸。

她记得清清楚楚：平衡木上，林清羽冲上来"接"她，实际上是把她推了下去。那双手的力度，那个角度，绝不是救援。

"林清羽..."苏浅咬牙切齿地念出这个名字，指甲深深陷入掌心，
"她在哪里？"

病房里突然安静下来。苏母和护士交换了一个眼神，苏父从外面走进来，脸色阴沉。

"浅浅，你刚醒，别想这些..."苏父试图安抚。

"告诉我！"苏浅猛地转头，眼神凌厉得不像一个刚苏醒的病人，
"那个凶手在哪里？"

"她..."苏母犹豫了一下，"被停训了，在家反省。队里说要等调查结果。"

"停训？"苏浅冷笑一声，笑声里带着令人毛骨悚然的寒意，
"只是停训？她差点杀了我！"

"浅浅，"护士小心翼翼地插话，"虽然大家都说是意外，但平衡木的螺丝确实松了..."

"意外？"苏浅闭上眼睛，前世的画面在脑海中回放——林清羽站在平衡木旁，眼神复杂，然后冲上来...

不，那不是意外。那是精心策划的谋杀。林清羽嫉妒她的天赋，嫉妒教练的重视，所以选择在最重要的比赛前下手。

"我要见她，"苏浅睁开眼睛，目光如刀，
"现在就要见。"

## 场景2：病房的对峙

**场景编号**：2-2
**地点**：医院病房，上午10:30
**在场角色**：苏浅、林清羽、苏母
**场景目的**：第一次正面冲突，展现两人关系的对立

**核心动作**：林清羽带着水果和康复资料来探望，苏浅冷冷地让她滚出去

**冲突爆发点**：林清羽试图解释，苏浅直接把水果篮摔在地上

**情感转变**：从期待和解 → 震惊受伤 → 更加坚定

**场景结果**：林清羽被赶出病房，但留下了康复资料

**正文**：

敲门声响起的时候，苏浅正盯着天花板上的裂纹发呆。

"进来。"她冷冷地说。

门被推开，林清羽走了进来。她穿着便装，手里提着水果篮，另一只手抱着一叠文件。

看到林清羽的那一刻，苏浅感觉血液直冲头顶。就是这个人，前世毁了她的一生，现在却装作若无其事地站在这里。

"你来干什么？"苏浅的声音冷得像冰。

林清羽似乎被苏浅的态度吓了一跳，但很快调整过来，脸上带着温和的笑：
"我来看看你，听说你醒了..."

"看我死了没有？"苏浅打断她，眼神锐利如刀，
"可惜让你失望了，我还活着。"

"苏浅，我..."林清羽向前走了一步，"我知道你恨我，但我真的不是故意的..."

"不是故意的？"苏浅突然提高声音，引得外面的护士探头张望，
"平衡木的螺丝松了，你冲上来'接'我，结果把我推下去了。这叫不是故意的？"

"那是因为..."林清羽急切地想解释，
"平衡木真的有问题，我是想救你..."

"够了！"苏浅抓起枕头边的水杯，狠狠砸向地面，
"滚出去！我不想听你的谎言！"

水杯在地上摔得粉碎，水花溅到了林清羽的裤脚上。苏母赶紧上前拉住苏浅：
"浅浅，别激动，医生说你需要休息..."

"让她滚！"苏浅指着门口，胸口剧烈起伏，
"告诉她，如果再出现在我面前，我就报警！"

林清羽站在原地，脸上的血色褪去。她看着苏浅，眼神里不是愤怒，而是...愧疚？还有一丝悲伤？

"好，我走，"林清羽轻声说，把水果篮放在桌上，
"但这些资料你留着，是关于康复训练的，对你有帮助..."

"拿走你的假好心！"苏浅抓起水果篮，用力摔向门口，
苹果和香蕉滚了一地。

林清羽没有生气，也没有辩解。她只是默默地捡起地上的文件，放在床头柜上，然后转身离开。

在门口，她停下脚步，回头看了苏浅一眼：
"不管你信不信，我会证明给你看的。"

门关上了。病房里恢复了死寂。

苏浅喘着粗气，盯着那叠文件。康复训练计划？林清羽以为这样就能赎罪吗？

# 第六章：神秘的警告
